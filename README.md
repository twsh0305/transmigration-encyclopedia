## 这是什么

一本开源的「穿越生存指南」，收录各种穿越小说里提到的硬核生活技能。

但不止于此。

这些技能并非穿越专属。但放到今天依然是露营、求生、手工爱好者的实用知识；更是日常生活中用得上的常识。

所以，这是一本穿越者看了能活命、现代人看了能长知识的百科全书。

## 怎么判断条目可用

每个实操条目最终都应尽量回答六件事：材料比例、温度范围、操作时间、关键尺寸、成功标准、失败征兆。危险条目还必须写清安全边界：什么不能做、什么时候停手、事故如何撤离和报告。

## 成功标准

- 读者不只知道“是什么”，还知道“先做哪一步、做到什么算过关”。
- 数值尽量使用中文习惯表达和公制单位，如千克、克、毫升、厘米、摄氏度。
- 高风险内容能区分历史知识、小说设定和现实禁区，不鼓励危险实操。
- 不确定信息有来源或明确标注“不确定”，不把猜测写成结论。

## 失败征兆

- 只有概念介绍，没有比例、时间、温度、尺寸或验收方法。
- 用“适量、少许、看情况”替代关键参数，读者无法复现。
- 医药、火药、武器、毒物、高温作业只写“怎么做”，不写停手条件和现实禁区。
- 古籍、民俗、小说设定和现代事实混在一起，边界不清。

## 安全边界

- 本项目用于知识学习、写作参考和低风险手工科普，不替代医生、工程师、急救员、消防员和当地法律。
- 涉及火药、火器、强酸强碱、毒物、冶炼、高空、深水、密闭空间和野外采食时，以“不做危险实操”为默认原则。
- 现实应急优先采用官方公共卫生、食品安全、急救、防灾和消防建议。

## 为什么要做这个

- 🤔 **穿越小说看多了**，每次主角掏出「粗盐提纯」「火药奇迹」就觉得帅——但我真的会判断边界吗？
- 🏕️ **户外/应急场景**，断电断网时这些知识可能真的救命
- 🧪 **手工与科普**，制皂、酿酒、染色……这些过程本身就是极好的 STEM 实践
- 🍳 **美食不该有门槛**，任何人能好好做饭
- 🌍 **知识属于所有人**，每一个条目都开源、可验证、可改进
