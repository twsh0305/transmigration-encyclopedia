## 这是什么

一本开源的「穿越生存指南」，收录各种穿越小说里提到的硬核生活技能。

但不止于此。

这些技能并非穿越专属。但放到今天依然是露营、求生、手工爱好者的实用知识；更是日常生活中用得上的常识。

所以，这是一本穿越者看了能活命、现代人看了能长知识的百科全书。

## 为什么要做这个

- 🤔 **穿越小说看多了**，每次主角掏出「粗盐提纯」「黑火药配方」就觉得帅——但我真的会吗？
- 🏕️ **户外/应急场景**，断电断网时这些知识可能真的救命
- 🧪 **手工与科普**，制皂、酿酒、染色……这些过程本身就是极好的 STEM 实践
- 🍳 **美食不该有门槛**，任何人能好好做饭
- 🌍 **知识属于所有人**，每一个条目都开源、可验证、可改进

## 内容范围

### 第一卷：求生基石 🔥
水、火、盐——活下来才有一切

### 第二卷：食与饮 🍺
从采集到烹饪，从发酵到蒸馏

### 第三卷：器与物 🛠️
肥皂、纸张、玻璃、染料……造物者的手记

### 第四卷：医与药 💊
没有药房的时代，你还能靠什么

### 第五卷：武与工 ⚔️
火药、冶铁、弓弩——技术就是力量

### 第六卷：文与美 🎨
胭脂水粉、笔墨纸砚——文明不只是活着

### 第七卷：做饭指南 🍜
灶台，都是创造的地方

### 第八卷：走上巅峰 🏰
赚银子、掌权力——活下去只是开始

### 第九卷：修仙问道 🧙
功法、丹药、山海经——若世界有灵气，知识就是最好的功法
