# 3.26 纺线基础

> 最早的文明是一根线——人类第一次将纤维捻合，就开启了纺织的历史。

## 原理

纺线（纺纱）的本质是**把短纤维捻合成长线**：

1. **纤维**：棉、麻、丝、毛等短纤维
2. **加捻**：旋转使纤维缠绕抱合，摩擦力使它们不散开
3. **牵伸**：一边加捻一边拉伸，使粗细均匀

加捻方向：S 捻（右旋）和 Z 捻（左旋）。

## 所需材料

| 材料/工具 | 用途 | 说明 |
|-----------|------|------|
| 纤维原料 | 纺线材料 | 麻纤维、羊毛、棉、丝絮 |
| 纺锤（纺轮） | 加捻工具 | 最简单的纺线工具 |
| 纺车（进阶） | 高速加捻 | 比纺锤效率高 10 倍 |
| 梳理板/梳子 | 梳理纤维 | 使纤维平行排列 |
| 水碗 | 润湿 | 帮助捻合 |

## 操作步骤

### 纺锤纺线（最古老的方法）

1. **准备纤维**：将麻/毛/棉梳理整齐，平行排列成条状
2. **引线**：从纤维条一端捻出一小段线头，系在纺锤钩上
3. **加捻**：
   - 左手捏住纤维条，保持适当张力
   - 右手捻转纺锤（或用手指搓转）
   - 纺锤悬空旋转，带动线加捻
4. **牵伸**：
   - 旋转中右手向下拉纤维条
   - 保持均匀速度，控制线的粗细
5. **绕线**：捻好一段后绕在纺锤上，继续下一段

### 纺车（大大提高效率）

1. **纺车结构**：一个大轮（手摇/脚踏）+ 一个锭子
2. **操作**：
   - 手摇大轮带动锭子高速旋转
   - 另一只手牵伸纤维喂入
   - 线自动加捻并绕在锭子上

### 合股

将两根或多根单线反向加捻合在一起，制成更结实的合股线：
- 两股线 → 缝纫线
- 三股线 → 绳索
- 四股以上 → 细绳

## 不同纤维的纺线特性

| 纤维 | 纺线难度 | 成品特点 | 备注 |
|------|---------|---------|------|
| 麻 | ⭐⭐⭐ | 强力高、凉快 | 纤维短，不易纺细 |
| 羊毛 | ⭐⭐ | 弹性好、温暖 | 油脂润滑易纺 |
| 棉 | ⭐ | 柔软舒适 | 需要去除棉籽 |
| 丝 | ⭐ | 光滑亮丽 | 缫丝而非纺线 |

## 常见问题

- **Q**: 纺出来的线粗细不均怎么办？
- **A**: 牵伸手势不均匀。练习要点：①保持纤维条含水量一致 ②牵伸速度均衡 ③加捻速度稳定。

- **Q**: 线总是断？
- **A**: ①纤维太短（没梳理好）②加捻不够③张力太大。尝试减小牵伸力。

## 安全提醒

- 纺锤注意不要打到周围人
- 长期纺线注意手腕和肩颈健康
- 羊毛纺线前注意清洗去除油脂和杂物

## 穿越小说梗

纺线是穿越后最容易被低估的技能。小说里"主角穿越后一个月就开纺织厂"——但事实上，从种麻/养羊/种棉到纺出第一根合格的线，光是**纺线工艺**就需要数周练习。纺锤虽然简单，但要让线又细又均匀不亚于学一门手艺。先做纺锤，再做纺车，这是穿越者正确的纺织科技树。
