# 3.13 手工冷制皂（油脂 + 草木灰碱液）

> 当猪油遇上草木灰，人类第一次掌握了清洁的秘密。

## 原理

冷制皂法是不加热（或仅微热）的皂化工艺，核心是**控制温度让油脂和碱液在温和条件下反应**：

- 油脂 + 碱液 → 皂 + 甘油
- 低温皂化保留了甘油（保湿成分），皂体更滋润
- 冷制皂需要较长的晾皂期（4-6 周）

冷制皂 vs 热制皂：
- **冷制皂**：温度 40-50摄氏度，反应慢（需数周成熟），保留甘油
- **热制皂**：加热煮沸，反应快（数小时即可），甘油流失多

## 所需材料

| 材料 | 用量（按 1 千克油脂计） | 备注 |
|------|----------------|------|
| 动物油（猪油/牛油） | 1000克 | 硬脂酸含量高，成皂硬 |
| 植物油（橄榄油/椰子油） | 每份 500克 | 可调节皂的性质 |
| 草木灰碱液 | 约 350-400毫升（视浓度） | 浓度需达 20% 以上 |
| 水 | 适量 | 调节碱液浓度 |
| 香料（可选） | 适量 | 花草/精油 |

## 草木灰碱液制备

### 淋灰法（最实用）

1. 收集干燥的木柴灰（硬木灰最佳，含碱量高）
2. 将灰装入底部有小孔的容器（陶罐或木桶）
3. 压紧压实
4. 从上方缓缓倒入开水
5. 下方收集滤出的碱液
6. 将收集到的碱液重新倒在灰上再淋一次
7. 重复 3-4 次，浓度逐次提高
8. 最后一次的碱液浓度最高

**判断浓度**：放一个鸡蛋进去——如果蛋浮出水面约硬币大小，浓度够；蛋沉底，太稀。

### 烧灰技巧

- **硬木灰**（橡木、栗木、槭木）：含碱量最高
- **秸秆灰**（稻草、麦秆）：含碱量中等
- **软木灰**（松木、杉木）：含碱量较低
- 灰要烧透、呈灰白色、无黑色炭粒

## 操作步骤

### 第一步：准备碱液

1. 将草木灰碱液过滤（细布过滤，去除杂质）
2. 测试浓度（鸡蛋法或比重法）
3. 如浓度不够，小火蒸发浓缩
4. 冷却至 40-50摄氏度备用

### 第二步：准备油脂

1. 猪油/牛油加热融化，过滤去渣
2. 冷却至 40-50摄氏度（与碱液同温）

### 第三步：混合皂化

1. 碱液缓慢倒入油脂中，边倒边搅拌
2. 持续搅拌（手工 30-60 分钟，电动搅拌棒 10-15 分钟）
3. 直到皂液呈 Trace（画线不消失）

### 第四步：入模晾皂

1. 倒入模具
2. 保温 24-48 小时
3. 脱模、切块
4. 通风处晾 4-6 周（穿越版建议 6-8 周——草木灰碱液纯度不如化学纯 NaOH）

## 不同油脂配方的效果

| 油脂组合 | 硬度 | 起泡性 | 滋润度 |
|---------|------|--------|--------|
| 纯猪油 | 硬 | 中 | 好 |
| 猪油 70% + 椰子油 30% | 硬 | 好 | 中 |
| 纯橄榄油（马赛皂） | 软 | 差 | 极好 |
| 牛油 60% + 猪油 40% | 极硬 | 中 | 好 |
| 各种油脂混合 | 适中 | 适中 | 好 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 碱液 | 过滤清亮，鸡蛋法显示浓度足够，冷却至 40-50摄氏度 |
| 油脂 | 融化过滤无肉渣水珠，温度同样在 40-50摄氏度 |
| 搅拌 | 30-60 分钟达到 Trace，画线数秒不消失 |
| 成熟 | 保温 24-48 小时后脱模，通风晾 6-8 周 |
| 试用 | pH 试纸约 8-10，小样洗手无刺痛和明显油腻 |

## 失败征兆

- 皂液分层、表面冒油，说明乳化或皂化失败。
- 皂体白霜很厚且洗手刺痛，可能游离碱过多。
- 晾皂期不足就出售或使用，刺激风险高。
- 碱液溅到皮肤只用布擦，不冲水，会继续腐蚀。
- 工具混回厨房使用，造成碱和油脂污染。

## 常见问题

- **Q**: 草木灰制皂颜色发黑？
- **A**: 碱液过滤不够彻底。用细布多过滤几次，或静置沉淀后取上清液。

- **Q**: 皂体表面有白色粉末？
- **A**: 这是皂粉（未反应的碱与空气 CO₂ 反应生成碳酸盐）。无害但不美观，可刮掉或蒸汽修复。

- **Q**: 冷制皂能用草木灰碱做成功吗？
- **A**: 能，但难度比用纯 NaOH 大。原因：草木灰碱液浓度不稳定，且含其他杂质。第一次做建议用小批量试试水。

## 安全提醒

- 强碱操作必须戴手套（橡胶/丁腈）
- 碱液溅到皮肤上立即用大量水冲洗
- 制皂容器不可再用于烹饪
- 晾皂期间防儿童和宠物接触
