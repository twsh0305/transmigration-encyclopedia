# 3.10 木工工具

> 木材是最容易加工的结构材料——只要你有合适的工具。

## 原理

木工的核心工具原理都是**力的集中**：

- **锯**：齿状刃口交替左右偏斜（"锯路"），使锯缝宽于锯片厚度，防止夹锯
- **刨**：倾斜刃口切割木纤维，切出光滑表面
- **凿**：楔形刃口在锤击下沿木纹劈裂
- **钻**：螺旋刃口连续切削，排出木屑

## 所需材料

| 工具 | 关键部件 | 可用代替材料 |
|------|---------|-------------|
| 锯 | 齿状金属片 | 燧石片镶木柄、大骨片磨齿 |
| 刨 | 金属刃+木刨体 | 铁片/钢片磨刃 |
| 凿 | 楔形金属条 | 铁条磨扁、硬骨磨刃 |
| 钻 | 尖头金属条 | 燧石尖头、骨尖 |
| 锤 | 石块+木柄 | 天然卵石绑木柄 |
| 斧（进阶） | 铁刃+木柄 | 初期可用大石斧 |

## 操作步骤

### 一、简易手锯制作

1. **锯条**：用薄铁片或钢片（约 1mm 厚），打磨成略梯形
2. **开齿**：用三角锉或硬石片在一边开出锯齿
3. **锯路**：每隔一个齿向左、向右轻轻扳斜（交替）
4. **装柄**：一端打孔装木柄
5. **成品**：锯片长 30-50cm，齿距 2-3mm（横切）或 4-5mm（纵切）

### 二、简易刨子制作

1. **刨体**：硬木块（约 20×5×4cm），底面磨平
2. **刨刀槽**：在刨体中部开一个斜槽（45° 角）
3. **刨刀**：薄铁片磨成 25° 斜面刃口
4. **楔木**：用一块小木楔固定刨刀
5. **调节**：轻敲刨刀露出底面 → 调整切削深度

### 三、必备木工技术

| 技能 | 工具 | 要点 |
|------|------|------|
| 横切（断木） | 手锯 | 沿横纹锯断 |
| 纵切（开板） | 长锯+楔子 | 沿顺纹锯开，楔子防止夹锯 |
| 刨平 | 刨子 | 沿顺纹方向刨，吃刀要浅 |
| 凿孔 | 凿+锤 | 先画线，凿一半翻面再凿 |
| 钻孔 | 手钻（弓钻） | 参考"摩擦取火"弓钻原理 |
| 榫卯 | 凿+刨 | 先做榫头，再依榫头做卯眼 |

### 四、木工最基础项目：工作台

1. 选四根粗木（直径 10-15cm）× 1.2m 长（桌腿）
2. 选厚木板（厚度 3-5cm）作桌面（60×120cm）
3. 榫卯或铁钉固定桌腿与横梁
4. 桌面上加一个**木工钳**（两块木条 + 螺杆夹紧）
5. 完成——有了工作台，后续木工效率翻倍

## 常见问题

- **Q**: 没有铁怎么做木工工具？
- **A**: 初期可以纯木结构+石器。木槌、石凿（黑曜石镶柄）、骨锯，但效率极低。木工技术的飞跃依赖于**金属工具的获取**。

- **Q**: 怎么把原木变成木板？
- **A**: 传统方法：①用楔子顺纹劈裂原木 ②用大锯沿纵向锯开。穿越初期主要靠劈（锯很难批量造出木板），劈开的板再用刨子修平。

## 安全提醒

- 木工工具极其锋利，手指永远不要放在切割线上
- 刨刀和凿子应保持锋利（钝工具更容易滑脱伤手）
- 锯木时注意木屑飞溅，戴护目镜

## 穿越小说梗

小说里"主角用现代木工技术惊掉古人下巴"——中国古代木工技术（尤其榫卯）已经发展到登峰造极，穿越者在手工木工方面几乎没有优势。但穿越者的优势是**金属工具的制作知识**——你能做出更好的刨刀、更锋利的锯，这套工具系统才是核心竞争力。另外，**标准化的尺寸系统**（统一尺规）是穿越者可以引入的重要概念。
