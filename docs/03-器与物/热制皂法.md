# 3.16 热制皂法

> 不想等一个月？热制皂当天就能用。

## 原理

热制皂（Hot Process）是在高温下加速皂化反应的方法：

- 冷制皂：40-50°C，反应慢，需 4-6 周成熟
- 热制皂：80-100°C，反应快，数小时完成
- 加热加速了碱和油脂的反应速率，完成后无需长期晾皂

**代价**：热制皂的高温破坏了部分甘油，滋润度不如冷制皂。但好处是——今天做，明天用。

## 所需材料

| 材料 | 用途 | 与冷制皂的区别 |
|------|------|----------------|
| 油脂 | 主料 | 相同 |
| 碱液 | 皂化 | 相同 |
| 锅 | 加热 | 必须是不锈钢或搪瓷锅 |
| 温度计 | 控温 | 更关键——要控制不烧焦 |
| 模具 | 定型 | 相同 |

## 操作步骤

### 热制皂法（HP 法）

1. **油脂加热**：将油脂放入锅中，小火加热至 60-70°C
2. **加碱液**：缓慢加入碱液，边加边搅拌
3. **持续加热搅拌**：小火保持温度在 80-90°C，持续搅拌
4. **判断反应终点**（"皂糊"状态）：
   - 皂液变得像土豆泥一样浓稠半透明
   - 用刮刀取一点尝——没有"电击感"（游离碱刺激感）即可
   - 或者用 pH 试纸测——降至 8-9 即安全
5. **添加物**：可在此时加入香料、花草、磨碎的燕麦等
6. **入模**：趁热倒入模具
7. **冷却硬化**：24 小时后即可脱模使用

**总时间**：2-4 小时

### 热制皂 vs 冷制皂

| 对比 | 冷制皂 | 热制皂 |
|------|--------|--------|
| 制作时间 | 30-60 分钟搅拌 | 2-4 小时熬煮 |
| 成熟期 | 4-6 周 | 1-2 天 |
| 滋润度 | 高（甘油保留多） | 中（部分甘油流失） |
| 外观 | 光滑细腻 | 质地粗糙（"农家皂"质感） |
| 难度 | 需精确控温 | 需持续看火 |
| 适合场景 | 品质优先 | 急用或批量生产 |

### 再熔制法（Rebatching，适合回收旧皂）

1. 将旧皂或失败的皂刨成薄片/丝
2. 加入少量水或牛奶
3. 隔水加热，不停搅拌
4. 皂片融化成黏稠浆糊状
5. 可添加香料、花草等
6. 倒入模具冷却

**适合**：回收失败的肥皂、添加新成分、制作创意造型。

## 常见问题

- **Q**: 热制皂糊了/烧焦了？
- **A**: 火太大或水太少。热制皂要隔水加热或在锅底加少量水防焦，小火慢煮。

- **Q**: 热制皂表面有颗粒？
- **A**: 皂化反应不均匀。原因是搅拌不够充分或温度分布不均。出锅前充分搅拌可解决。

- **Q**: 热制皂用完皮肤还是痒？
- **A**: 游离碱残留。可能是皂化不彻底（温度不够或时间不足），或者碱量超标。

## 安全提醒

- 热制皂熬煮过程中温度高，注意防烫
- 皂液飞溅到皮肤上立即用冷水冲洗+醋中和
- 热制皂入模后很烫，需冷却至安全温度再脱模

## 穿越小说梗

如果想在古代批量生产肥皂卖钱，热制皂法更实用——今天投料，明天就能上架。虽然外观丑一点，但穿越世界的顾客哪有那么多讲究？关键是：热制皂不需要精确控温设备，一个铁锅一把火就能干，比冷制皂更适合古代的生产条件。
