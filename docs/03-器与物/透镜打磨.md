# 3.29 透镜打磨

> 两块磨好的玻璃，就能让人类看见月球的山脉——或者让细菌无处可藏。

## 原理

透镜利用**光的折射**使光线汇聚或发散：

- **凸透镜**（正透镜）：光线向中心汇聚 → 放大镜、显微镜物镜、望远镜目镜
- **凹透镜**（负透镜）：光线向外发散 → 近视眼镜、伽利略望远镜物镜

透镜的关键参数是**曲率半径**和**焦距**。焦距越短，放大倍数越大。

## 所需材料

| 材料/工具 | 用途 | 说明 |
|-----------|------|------|
| 玻璃片 | 透镜毛坯 | 透明无气泡的玻璃 |
| 铁碗/金属碗 | 研磨模具 | 凸面和凹面各一个 |
| 金刚砂（刚玉砂） | 粗磨料 | 不同粒度（粗→细） |
| 氧化铁/氧化铈 | 精磨/抛光 | 最后一道抛光 |
| 沥青/松香 | 抛光模 | 制作与透镜弧度匹配的模子 |
| 游标卡尺/弧规 | 测量曲率 | 确保精度 |

## 操作步骤

### 粗磨（成形）

1. **切圆**：将玻璃切割成圆形毛坯（用烧热的铁环接触玻璃，激冷开裂）
2. **粗磨曲率**：
   - 凸透镜：用凹形金属碗做模具，金刚砂+水研磨
   - 凹透镜：用凸形金属碗做模具
3. **逐级细磨**：依次用越来越细的砂（80目 → 120目 → 240目 → 400目 → 600目 → 1200目），每级磨到上一级的划痕消失

### 抛光

1. **制抛光模**：将沥青加热软化，浇在模具上成型
2. **加抛光粉**：用氧化铁（红粉）或氧化铈悬浊液
3. **抛光**：以 8 字形轨迹反复研磨，直至玻璃透明无雾

### 焦距检测

1. 将透镜对准太阳
2. 测量聚焦光点到透镜的距离→即为焦距
3. 做标记，记录焦距便于后续使用

## 简易显微镜/望远镜方案

| 类型 | 物镜 | 目镜 | 放大倍数 |
|------|------|------|---------|
| 简易放大镜 | 单个凸透镜 | — | 5-20× |
| 望远镜（伽利略式） | 凸透镜 | 凹透镜 | 3-10× |
| 望远镜（开普勒式） | 凸透镜 | 凸透镜 | 10-50× |
| 简易显微镜 | 短焦凸透镜 | 长焦凸透镜 | 50-200× |

## 常见问题

- **Q**: 没有金刚砂用什么？
- **A**: 石英砂过筛分级可用，但磨速很慢。更好的替代：碎玛瑙粉、刚玉砂（自然界存在）。

- **Q**: 镜片有气泡怎么办？
- **A**: 不影响使用，气泡只会轻微降低透光率。古代透镜几乎没有无气泡的。

- **Q**: 怎么把玻璃磨成凸面？
- **A**: 用凹形工具做模具。研磨时工具移动比镜片移动更容易控制曲率。

## 安全提醒

- 研磨玻璃粉吸入有害肺，操作时湿磨并戴口罩
- 切割玻璃时注意锋利的边缘
- 聚焦太阳光测试焦距时，不要让焦点对着易燃物或自己的手

## 穿越小说梗

"主角磨出了望远镜，在战场上料敌先机"——可行，但难度被严重低估。磨一副好的透镜（没有现代设备）需要数周甚至数月的耐心。穿越者最合理的路线：先做单透镜放大镜（检查伤口、取火），再做伽利略望远镜（物镜+目镜各一块），显微列和天文望远镜是更高级的追求。
