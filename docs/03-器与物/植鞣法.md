# 3.12 植鞣法

> 用树皮中的单宁，让兽皮变成永不腐烂的皮革——这是人类最古老的化学工业之一。

## 原理

植鞣利用植物中天然存在的**单宁酸（鞣酸）**：

1. 单宁分子渗入胶原纤维之间
2. 与胶原蛋白的氨基、羟基形成交联结合
3. 使胶原纤维不再腐烂、不再吸水膨胀、干燥后保持柔软

### 含单宁的植物

| 植物 | 部位 | 单宁含量 | 给皮革的颜色 |
|------|------|---------|-------------|
| 橡树 | 树皮、橡碗 | 高 | 浅棕 |
| 栗树 | 树皮 | 很高 | 深棕 |
| 栲树 | 树皮 | 高 | 黄棕 |
| 五倍子 | 虫瘿 | 极高（70%+） | 浅黄 |
| 石榴 | 果皮 | 中 | 金黄 |
| 柳树 | 树皮 | 中低 | 浅色 |
| 茶叶 | 叶片 | 中 | 红棕 |

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 裸皮 | 鞣制对象 | 已经过去肉脱毛的干净皮 |
| 树皮/植物材料 | 单宁来源 | 橡树皮最佳 |
| 陶缸/木桶 | 浸泡容器 | 非金属 |
| 温水 | 提取单宁 | 不要沸水 |
| 牛油/动物油脂 | 后期加脂 | 使皮柔软 |

## 操作步骤

### 一、鞣剂提取

1. 将树皮切碎或碾碎（增大接触面积）
2. 放入温水中浸泡 24-48 小时
3. 过滤得单宁溶液（鞣液）
4. 浓度越高效果越好——颜色深褐带涩味即为好

### 二、鞣制过程（需要数周到数月）

**初期（1-2 周）**：
1. 裸皮浸入稀鞣液（单宁浓度 5-10%）
2. 每天翻动一次
3. 逐渐加深鞣液浓度

**中期（2-6 周）**：
1. 转入浓鞣液（单宁浓度 20-30%）
2. 每周翻动 2-3 次
3. 检查鞣制进度——切口颜色是否均匀渗透

**后期**：
1. 切口从外到内颜色一致→鞣制完成
2. 皮呈均匀的浅褐色
3. 取出后用清水漂洗表面多余单宁

### 三、加脂（使皮革柔软）

1. 动物油脂（牛脂、猪油）加热熔化
2. 涂抹在微湿的皮革上
3. 揉搓使油渗入纤维
4. 挂在阴凉处干燥

### 四、干燥与拉软

1. 自然阴干（不要暴晒）
2. 半干时反复拉伸、揉搓
3. 完全干燥后再次拉伸 → 皮革变得柔软

## 常见问题

- **Q**: 植鞣要多长时间？
- **A**: 小型皮（兔皮、羊皮）2-4 周；牛皮 2-6 个月。古代厚牛皮有时要鞣一年。

- **Q**: 怎么判断鞣好了？
- **A**: 切一小块皮边观察：如果内外颜色一致（都是浅褐色），就是好了。如果中心还是白色，需要继续浸泡。

- **Q**: 鞣出来的皮革很硬怎么办？
- **A**: ①加脂不够（再补油）②揉搓不够（半干时反复搓揉）③鞣过头了（难救，只能下次缩短时间）。

## 安全提醒

- 单宁溶液对金属有腐蚀，用陶缸或木桶
- 某些树皮有刺激性（如栲树），处理时注意皮肤
- 植鞣的皮革耐水性好但怕反复浸泡，遇水后需阴干

## 穿越小说梗

小说里"主角用树皮水泡了泡皮就做成皮衣了"——现实中植鞣是**化学工程**，单宁渗透到胶原纤维深处需要时间，厚皮数月、薄皮数周是常态。但穿越者比原始人多知道一件事：**单宁的本质**，知道选哪种树皮效果好，知道不同浓度梯度的优势——这可以把鞣制周期从"瞎试几代人"缩短到"按科学流程走几个月"。
