# 3.27 织布基础

> 经线是骨架，纬线是血肉。一纵一横之间，布就诞生了。

## 原理

织布的本质是**经纬交织**：

- **经线**：纵向排列的线，固定在织机上下两端，提供基础结构
- **纬线**：横向穿过的线，由梭子承载，穿过经线间隙

### 织造四步循环

1. **开口**：提起部分经线，形成"梭口"
2. **引纬**：梭子穿过梭口，引入纬线
3. **打纬**：用筘将纬线推向织口
4. **翻综**：交换提起的经线组，准备下一梭

## 所需材料

| 材料/工具 | 用途 | 说明 |
|-----------|------|------|
| 经线 | 纵向线 | 需较结实（通常合股） |
| 纬线 | 横向线 | 可略细 |
| 腰机（最简织机） | 织造 | 绑在腰上操作 |
| 梭子 | 引纬 | 木/竹片制成 |
| 筘（打纬梳） | 打紧纬线 | 薄竹片 |
| 综杆 | 开口 | 挑经线的棍子 |
| 筒管 | 绕纬线 | 竹管 |

## 操作步骤

### 腰机（最简单的织机）

1. **制织机**：
   - 一根横杆绑在腰上（经线固定端）
   - 另一根横杆固定在对面的木桩或树上
   - 经线在两横杆之间绷紧排列

2. **穿经**：将经线均匀排布在两横杆之间，每根一根

3. **织造**：
   - 用综杆挑起奇数或偶数的经线 → 形成梭口
   - 插入梭子，纬线穿过
   - 用筘将纬线推紧
   - 交换综杆挑的经线 → 开口
   - 梭子穿回 → 打纬 → 重复

### 进阶：框架织机

1. 做一个矩形木框架（宽 ≈ 需要的布宽）
2. 上下横杆刻槽或钉钉子固定经线
3. 装 2-4 页综框（用脚踏板控制开口）
4. 效率提高 3-5 倍，解放双手

## 成功标准

- 经线张力均匀，梭子通过时不大面积挂线。
- 织 30厘米后布面密度基本一致，边缘不明显收缩。
- 纬线打紧但不把经线挤弯，布面平整、不起波浪。
- 取下后轻拉四角，布不散边、不连续断线。

## 失败征兆

- 布边越来越窄：纬线拉得太紧。
- 布面稀疏透光过多：经线间距大或打纬不够。
- 经线频繁断：线强度不够、张力过大或综杆有毛刺。
- 同一处花纹错乱：开口顺序或综杆控制错了，应拆回错误点。

## 常见问题

- **Q**: 布边松散不整齐？
- **A**: 两边的经线要拉紧一些，或者在边缘用更粗的线。保持经纬线张力一致。

- **Q**: 纬线一拉就断？
- **A**: 纬线捻度不够或纤维太差。合股后做纬线更结实。

- **Q**: 织出来的布太稀疏？
- **A**: ①经线间距太大 ②打纬不够紧 ③增加经线密度。

## 安全提醒

- 腰机长时间操作对腰部有负担，注意休息
- 梭子尖端可能伤手，打磨光滑
- 经线张力过大容易划伤手指

## 穿越小说梗

"主角用新技术改良织布机"——这是穿越小说经典桥段。腰机到框架织机的跨越，可以让织布效率提升数倍。但织布真正的瓶颈不是织机本身，而是**纺线**——没有足够的线，再快的织机也是空转。穿越者应先确保纺线产能（多台纺车同时工作），再升级织机。
