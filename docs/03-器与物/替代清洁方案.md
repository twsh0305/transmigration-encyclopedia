# 3.9 替代清洁方案（皂角、无患子、茶籽饼）

> 在古代，肥皂不是唯一的答案。皂角和无患子才是真正的原生清洁剂。

## 原理

某些植物含有**天然表面活性剂**——皂苷（Saponin）：

- 皂苷分子同时具有亲水端和亲油端
- 在水中搓揉时产生泡沫
- 泡沫能包裹油脂和污垢，使其被水冲走
- 与肥皂的化学原理相同（都是表面活性剂），但温和得多

## 常见天然清洁植物

### 皂角（Gleditsia sinensis）

中国传统清洁剂，历史悠久。

| 特性 | 说明 |
|------|------|
| 使用部位 | 豆荚（干透的皂角） |
| 使用方法 | 砸碎→泡水→搓揉出泡沫 |
| 适用 | 洗衣、洗澡、洗头 |
| 起泡性 | ⭐⭐⭐⭐ |
| 温和度 | ⭐⭐⭐⭐⭐ |
| 保存 | 干透可存数年 |

**使用方法**：
1. 取 2-3 片干皂角
2. 砸碎（用锤子或石头）
3. 放入锅中加水煮沸 10-15 分钟
4. 过滤取液
5. 冷却后即可用于洗涤
6. 或直接将皂角碎片放入布袋中，与水一起搓洗

**穿越评分**：⭐⭐⭐⭐⭐（分布广、效果好、制作零门槛）

### 无患子（Sapindus mukorossi）

俗称"洗手果"，含皂苷量极高。

| 特性 | 说明 |
|------|------|
| 使用部位 | 果实外皮（果肉部分） |
| 使用方法 | 取 3-5 颗去核→泡水/煮水→搓泡 |
| 适用 | 洗手、洗衣、洗碗 |
| 起泡性 | ⭐⭐⭐⭐⭐ |
| 温和度 | ⭐⭐⭐⭐⭐ |
| 保存 | 晒干可存 1-2 年 |

**使用方法**：
1. 取 3-5 颗无患子
2. 去核（去核防染色）
3. 放入布袋中
4. 在温水中揉搓
5. 产生丰富细腻的泡沫
6. 直接用泡沫洗涤

**穿越评分**：⭐⭐⭐⭐⭐（南方地区极常见）

### 茶籽饼（油茶籽榨油后的渣饼）

榨茶油剩下的渣饼，含茶皂素。

| 特性 | 说明 |
|------|------|
| 使用部位 | 榨油后的渣饼 |
| 使用方法 | 敲碎→泡水→沉淀取液 |
| 适用 | 洗碗、洗头、洗衣 |
| 起泡性 | ⭐⭐⭐ |
| 温和度 | ⭐⭐⭐⭐ |
| 特殊 | 有杀螺/杀虫作用（农业用途） |

**使用方法**：
1. 茶籽饼敲碎成小块
2. 热水浸泡 4-8 小时或煮沸 30 分钟
3. 过滤取液
4. 可直接用于洗碗洗头

**注意**：茶皂素对鱼类有毒，洗过碗的水不要倒入鱼塘。

## 三种比较

| 对比 | 皂角 | 无患子 | 茶籽饼 |
|------|------|--------|--------|
| 分布 | 全国温带 | 南方地区 | 产茶区 |
| 使用难度 | 需煮 | 简单揉搓 | 需浸泡 |
| 泡沫丰富度 | 中等 | 极丰富 | 中等 |
| 适合洗头 | 是（传统护发） | 是 | 是 |
| 适合洗衣 | 是 | 是 | 是 |
| 适合洗碗 | 可 | 可 | 可 |
| 取材季节 | 秋季成熟 | 秋季成熟 | 全年（压饼后） |

## 与现代肥皂的对比

| 对比 | 天然皂苷 | 肥皂 |
|------|---------|------|
| 去污力 | 中等 | 强 |
| 对皮肤 | 极温和 | 较刺激 |
| 泡沫 | 细腻但量少 | 丰富 |
| 制作 | 零门槛（采/捡即可） | 需要碱和油脂 |
| 保存 | 干果存数年 | 皂可存数月 |
| 在硬水中的效果 | 好（不易形成皂垢） | 差（硬水产生皂垢） |

## 常见问题

- **Q**: 皂角和无患子洗头后头发会不会涩？
- **A**: 会！因为天然皂苷去除了油脂。建议用醋或柠檬汁兑水冲洗一遍（酸碱中和），头发会恢复柔顺。

- **Q**: 这些植物古代就用来洗衣服吗？
- **A**: 是的。皂角在《本草纲目》中就有记载用于洗涤。无患子在南方民间使用千年以上。

- **Q**: 没有这些植物怎么办？
- **A**: 除了这三种，还有其他含皂苷的植物：木槿叶、假烟叶、丝瓜叶——揉搓出泡沫即可。但效果不如皂角和无患子。

## 安全提醒

- 天然皂苷无毒，但泡沫误入眼睛会刺激
- 茶籽饼水对水生生物有毒，不要排入鱼塘
- 无患子的核（种子）有毒，不可食用

## 穿越小说梗

穿越后最聪明的做法不是做肥皂——是直接用皂角/无患子。零成本、零门槛、零技术含量，捡一把就能用。如果真的要做肥皂，那也得等站稳脚跟、有大量富余油脂和草木灰之后。在此之前，皂角才是穿越第一天就能用的清洁方案。
