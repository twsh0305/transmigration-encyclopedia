# 3.6 天然染料提取

> 在没有化学染料的时代，一件衣服的颜色，决定了你的社会阶层。

## 原理

天然染料的本质是**植物/动物/矿物中含有色素的物质**。提取染料的原理：

1. **破碎细胞**：将植物组织打碎，释放内含的色素
2. **溶解/煎煮**：用热水或酒精将色素溶解出来
3. **浓缩**：蒸发水分得到浓缩染液
4. **媒染**：部分色素需要媒染剂才能牢固附着在纤维上

### 三大植物色素

| 色素类型 | 来源植物 | 颜色 | 媒染要求 |
|---------|---------|------|---------|
| 靛蓝（Indigo） | 蓼蓝、菘蓝、马蓝、木蓝 | 蓝 | 需要还原（不发烫水） |
| 茜素（Alizarin） | 茜草根 | 红橙 | 明矾媒染 |
| 类胡萝卜素 | 栀子、郁金 | 黄 | 弱或无媒染 |
| 单宁酸 | 五倍子、栎树皮 | 黑 | 铁媒染 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 染料植物 | 色素来源 |
| 大锅（铁锅/陶锅） | 煮染液 |
| 水 | 溶剂 |
| 明矾/铁锈/铜器 | 媒染剂 |
| 陶缸/木桶 | 浸泡发酵 |
| 纱布/滤布 | 过滤 |
| 待染布料/毛线 | 染色对象 |

## 操作步骤

### 茜草红（最简单的红色）

1. **采挖**：茜草根，秋季最佳
2. **洗净切碎**：干燥后碾成粗粉
3. **煎煮**：加水煮沸 30-45 分钟
4. **过滤**：取染液
5. **加媒染剂**：每升染液加 5-10克明矾
6. **染色**：布料浸入，微火煮 30-60 分钟
7. **洗净晾干**：不要暴晒

### 靛蓝（经典蓝色）

靛蓝需要**还原**——不溶于水，需在碱性还原环境中变为可溶：

1. **制靛泥**：蓝草浸泡 24 小时，加石灰搅拌，沉淀取靛泥
2. **配还原液**：靛泥 + 草木灰水 + 少量米酒（提供还原糖）
3. **还原**：保持 30-40摄氏度，12-24 小时，染液变成黄绿色即可
4. **染色**：布料浸入 10 分钟 → 取出氧化（变蓝） → 重复 3-5 次
5. **最后清洗**：醋水中和碱性

### 栀子黄（最简单）

1. 栀子果捣碎
2. 加水煮沸 20 分钟
3. 过滤即得金黄色染液
4. 不需媒染，直接浸泡染色

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 染液 | 过滤后无明显碎渣，颜色比目标布色深 2-3 个层级 |
| 媒染 | 明矾按每升染液 5-10克添加，布料煮染后颜色均匀 |
| 靛蓝 | 还原液呈黄绿色，布料出缸后 1-3 分钟逐渐转蓝 |
| 色牢度 | 阴干 24 小时后清水漂洗 2 次，水色明显变浅但布面不花 |
| 复染 | 颜色偏浅时重复浸染 2-5 次，而不是一次猛加媒染剂 |

## 常见问题

- **Q**: 为什么染上去的颜色一洗就掉？
- **A**: ①没有用媒染剂 ②染色时间不够 ③水温不对。茜素和单宁类染料必须加媒染剂。

- **Q**: 靛蓝还原失败怎么办？
- **A**: 染液必须是黄绿色才算成功。如果还是深蓝色，说明还原不充分：加更多米酒/糖，延长还原时间。

## 安全提醒

- 部分染料植物有毒（如某些漆树科植物），处理时戴手套
- 石灰水有强碱性，避免接触眼睛
- 铁锅煮单宁染料会使颜色暗沉，需注意

## 穿越小说梗

古代染料是"颜色即阶级"的经济基础——紫色来自骨螺极贵，红色来自茜草大量种植，黄色来自栀子很方便。穿越者如果懂得"不同媒染剂 + 同一种染料 = 不同颜色"的原理，就能用最少的原料调出最多的花色。靛蓝染的蓝印花布，直到近代都是中国农村的标配。
