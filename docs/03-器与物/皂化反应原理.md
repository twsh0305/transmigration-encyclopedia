# 3.20 皂化反应原理

> 油脂遇上碱，不再是油也不是碱——它是肥皂。

## 原理

皂化反应是**酯的碱性水解**：

```
油脂（甘油三酯） + 氢氧化钠（碱） → 肥皂（脂肪酸钠） + 甘油
```

油脂分子由一分子甘油和三分子脂肪酸组成，在强碱（NaOH/KOH）作用下，酯键断裂，生成脂肪酸盐（即肥皂）和甘油。

**皂化三要素**：
1. **油脂**：任何油脂都可皂化，不同油脂决定肥皂的硬度、起泡性和滋润度
2. **碱**：NaOH（氢氧化钠）→ 硬皂，KOH（氢氧化钾）→ 软皂/液体皂
3. **水**：溶解碱，参与反应

## 皂化值

每种油脂的皂化值不同——即完全皂化 1克油脂所需的 NaOH 量：
- 椰子油：0.190克氢氧化钠/克油脂（需碱较多）
- 橄榄油：0.134克氢氧化钠/克油脂（需碱较少）
- 猪油：0.138克氢氧化钠/克油脂
- 棕榈油：0.141克氢氧化钠/克油脂

**为什么重要**：碱量不足→肥皂太油（碱性不够，油脂未充分反应）；碱量过多→肥皂太刺激（游离碱伤害皮肤）。

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 油脂 | 主料 | 动物油或植物油 |
| NaOH（氢氧化钠） | 皂化剂 | 可用草木灰碱液替代 |
| 水 | 溶碱 | 纯净水或蒸馏水 |
| 温度计 | 控温 | 40-50摄氏度最佳 |
| 不锈钢锅 | 混合 | 铝锅会被碱腐蚀 |
| 模具 | 定型 | 硅胶模、纸盒、木盒 |

## 操作步骤

### 基础冷制皂流程

1. **溶解碱**：将碱缓慢倒入水中（**绝不可将水倒入碱中**，会剧烈喷溅），搅拌至完全透明
2. **加热油脂**：将油脂隔水加热至 40-50摄氏度
3. **混合**：碱水和油脂温度都达到 40-50摄氏度时，将碱水缓慢倒入油脂中
4. **搅拌**：持续搅拌 30-60 分钟，直到皂液呈"Trace"状态（能在表面画出明显纹路）
5. **入模**：倒入模具，表面抹平
6. **保温**：包裹毛巾，保温 24-48 小时
7. **脱模**：皂体硬化后脱模，切块
8. **晾皂**：通风处晾 4-6 周（成熟期），使 pH 降至安全范围

### 判断 Trace

用刮刀在皂液表面画一条线——线痕能清晰保持几秒钟不消失，即为 Trace（可入模状态）。

## 碱的来源

如果没有化学纯 NaOH，穿越后可用**草木灰碱液**替代：

1. 烧草木灰（优先用硬木灰、秸秆灰）
2. 将灰放入底部有孔的桶中
3. 淋入热水，收集过滤后的碱液
4. 重复淋灰几次，得到浓度足够的碱液
5. 用此碱液代替 NaOH 溶液

草木灰碱液的主要成分是碳酸钾（K₂CO₃），在水中呈碱性，可替代 NaOH 进行皂化。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 计量 | 按油脂皂化值计算碱量，不凭感觉加碱 |
| 温度 | 油脂和碱水混合时都在 40-50摄氏度 |
| Trace | 搅拌 30-60 分钟后表面线痕能保持几秒 |
| 成熟 | 脱模切块后通风晾 4-6 周，再接触皮肤 |
| 验收 | pH 试纸接近 8-10，小样洗手不刺痛、不滑腻 |

## 失败征兆

- 皂体表面冒油或分层，说明乳化或皂化失败。
- 皂体开裂、火山状突起，说明温度过高或反应过快。
- pH 过高、洗手刺痛，说明游离碱过多，不能用于皮肤。
- 皂体长期软黏不硬，可能水分过多或硬脂酸不足。
- 用舌头尝碱性是危险旧法，不作为验收方式。

## 常见问题

- **Q**: 皂化过程中温度太高了怎么办？
- **A**: 皂化反应放热，如果温度超过 70摄氏度可隔冰水降温。过热会导致皂体开裂或火山状突起。

- **Q**: 做出来的肥皂不硬？
- **A**: 硬脂酸含量不够。可以加入椰子油或棕榈油增加硬度。如果用的是纯橄榄油（马赛皂），需要更长的晾皂时间。

## 安全提醒

- **强碱腐蚀性极强**：必须戴橡胶手套、护目镜
- **碱水遇水放热**：溶碱时温度可达 80摄氏度以上，用耐热容器
- 碱液溅到皮肤上，立即用大量流动清水冲洗至少 15 分钟；不要在皮肤上用醋做酸碱中和实验
- 操作区域保持通风
- 制皂工具不可用于食品制作

## 穿越小说梗

穿越小说里主角"随手就做了块肥皂"——确实，皂化反应不难。难的是两件事：①你得有足够的碱——烧草木灰制碱其实挺费劲的，大概 10 斤灰才能做出够做几块肥皂的碱液；②晾皂要 4-6 周，不是当天就能用的。不过一旦量产成功，在古代这确实是个暴利生意——毕竟古人洗澡靠的是热水+刮泥。
