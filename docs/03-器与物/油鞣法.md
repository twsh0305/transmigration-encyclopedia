# 3.14 油鞣法

> 当没有树皮可用时，动物的脑子就是最好的鞣剂——这不是笑话，是千年工艺。

## 原理

油鞣利用**不饱和脂肪酸**渗入胶原纤维，与蛋白质结合，起到鞣制作用。与植鞣不同，油鞣的皮革更柔软、更耐水，但易吸收水分（油鞣革的标准用途：擦拭玻璃）。

### 脑浆鞣制（Native American 经典方法）

动物的大脑含有丰富的**卵磷脂和油脂**，其脂肪组成与皮革纤维天然亲和，是最有效的天然油鞣剂之一。

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 裸皮 | 鞣制对象 | 已去肉脱毛 |
| 动物脑浆/蛋黄 | 鞣剂 | 新鲜大脑，或蛋黄 |
| 动物油脂 | 润滑 | 鹿油、猪油、牛油 |
| 温火/烟熏 | 固化鞣剂 | 辅助渗透 |
| 木框架 | 绷皮 | 拉伸用 |
| 刮刀 | 挤油 | 见操作 |

## 操作步骤

### 一、准备鞣剂

1. 取 1-2 个动物的新鲜大脑（或 5-8 个蛋黄）
2. 捣碎成糊状
3. 加少量温水搅拌成乳液
4. 加入等量动物油脂

### 二、涂油（"油浸"）

1. 将裸皮浸泡在脑浆乳液中 2-4 小时
2. 取出后均匀涂抹乳液，确保全部覆盖
3. 重点涂抹皮最厚的部位（颈部、臀部）

### 三、揉搓与拉伸（最关键的一步）

1. 将皮挂在木架上
2. 用钝刀或刮刀反复**刮皮**，使乳液渗入纤维
3. 从中心向外刮，重复数十次
4. 皮在揉搓中逐渐变软、颜色变浅

### 四、烟熏（可选，但推荐）

1. 在小火堆上方架皮
2. 用腐木或绿草产生浓烟
3. 烟熏 1-3 小时，使鞣剂固化
4. 烟熏后的皮革不惧水洗（否则油鞣革遇水会变硬）

### 五、最后处理

1. 再次揉搓至柔软
2. 用温水漂洗多余油脂
3. 阴干后再次拉软

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 裸皮 | 去肉、脱毛干净，无腐臭和大片残脂 |
| 鞣剂 | 脑浆或蛋黄加温水打成乳液，能均匀涂开 |
| 浸润 | 浸泡 2-4 小时后厚处也变软，乳液能揉进纤维 |
| 拉软 | 干燥过程中持续拉伸，成品像麂皮一样柔软不板硬 |
| 烟熏 | 1-3 小时低温浓烟，颜色浅黄到烟褐，遇水后不立刻硬成板 |

## 失败征兆

- 皮上残肉残脂发臭，说明前处理失败，容易腐烂。
- 只涂油不揉搓，油停在表面，内部仍会干硬。
- 晾干时无人拉软，最终变成硬皮板。
- 烟熏火苗太大，把皮烤焦或烧洞。
- 成品潮湿后霉斑扩大，不适合做贴身衣物。

## 植鞣 vs 油鞣对比

| 特性 | 植鞣 | 油鞣 |
|------|------|------|
| 鞣剂 | 植物单宁 | 动物脑浆/油脂 |
| 时间 | 数周至数月 | 数天至两周 |
| 成品颜色 | 浅棕至深棕 | 米白至浅黄 |
| 柔软度 | 较硬 | 非常柔软 |
| 耐水性 | 好 | 怕水（除非烟熏） |
| 强度 | 高 | 较低 |
| 适用 | 鞋底、皮带、刀鞘 | 衣物、手套、软包 |

## 常见问题

- **Q**: 没有动物大脑怎么办？
- **A**: 蛋黄可以替代（富含卵磷脂），效果仅次于脑浆。也可用纯动物油脂+少量盐，但效果差些。

- **Q**: 油鞣的皮为什么湿了之后变硬？
- **A**: 这是油鞣的特性。解决方案：①烟熏处理②湿了之后浸水揉搓，干后重新拉软。

- **Q**: 怎么判断油鞣好了？
- **A**: 皮变得**柔软、米白色、像麂皮一样**，拉扯有弹性，就算好了。

## 安全提醒

- 脑浆鞣制的皮在烟熏前容易腐臭，需及时处理
- 烟熏时注意防火
- 揉搓过程体力消耗大，注意休息

## 穿越小说梗

"主角用野牛脑浆鞣出了比丝绸还软的皮"——脑浆鞣制是北美原住民几千年的智慧结晶，确实有效，而且比植鞣快得多。缺点是①很臭（脑浆腐味）②体力消耗极大（几个小时反复揉搓）③不烟熏就不防水。穿越者要是猎了头鹿，**脑浆鞣+烟熏**组合是短期内获得柔软皮革的最好路线。
