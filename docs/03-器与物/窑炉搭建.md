# 3.22 窑炉搭建

> 人类的文明史，某种意义上就是一部"烤东西"的历史。窑越好，文明越发达。

## 原理

窑炉的核心是**热工效率**：

1. **隔热**：阻止热量向四周散失
2. **烟气流动**：利用热空气上升的自然抽力
3. **热量利用**：让高温烟气先经过制品再排出

窑炉形制的演进: 地坑窑 → 升焰窑 → 倒焰窑 → 龙窑 → 馒头窑 → 现代隧道窑

## 所需材料

| 材料 | 用途 | 
|------|------|
| 土坯/砖 | 窑体 |
| 黏土砂浆 | 砌筑和密封 |
| 碎陶片/碎石 | 窑底垫层 |
| 木柴/木炭 | 燃料 |
| 陶管 | 烟囱/通风口 |
| 铁管/竹管 | 鼓风管 |

## 三种窑炉搭建

### 地坑窑（最简单）

适用：陶器、低温烧制。难度：低。

1. 挖一个直径 60-100cm、深 50-80cm 的圆坑
2. 坑底挖一个通风壕沟（从地面斜通到坑底）
3. 坑壁抹黏土砂浆，夯实
4. 顶部用泥盖封顶，留 1-2 个出烟孔
5. 在通风口处烧火，预热后放入陶坯

### 升焰窑（升级版）

适用：陶器、玻璃熔化、低温釉。难度：中。

1. 用土坯砌圆筒状窑体，高 80-120cm
2. 底部留火门（投柴口），宽 20cm、高 15cm
3. 内部距底部 20cm 处搭炉栅（用耐火泥条或铁条）
4. 窑顶逐渐收拢，留圆形出烟口
5. 侧面接一段陶管作烟囱

### 龙窑（最先进）

适用：瓷器、高温烧制。难度：高。

1. 选一个 15-30 度斜坡
2. 沿坡砌长条形窑室（长 5-15m、宽 1.5-2m）
3. 低端设火膛（燃烧室）
4. 高端设烟囱
5. 窑壁开投柴孔（侧壁）
6. 窑内壁涂耐火泥

**优势**：自然坡道产生的抽力极强，温度可达 1300°C+，适合瓷器烧制。

## 温度提升技巧

| 方法 | 可提升温度 | 说明 |
|------|-----------|------|
| 烟囱加高 | +50-100°C | 利用自然抽风 |
| 强制鼓风 | +100-200°C | 皮风箱/竹筒吹气 |
| 更换燃料 | +100-200°C | 木炭 > 干柴 > 湿柴 |
| 提高窑体密封 | +50°C | 减少热量损失 |

## 常见问题

- **Q**: 第一次烧窑要注意什么？
- **A**: ①充分干燥窑体（新建窑必须缓慢烘干一周以上）②第一次烧控制在低温 300-400°C 连续烧 12 小时③检查所有裂缝，用泥浆填补。

- **Q**: 窑烧不到预定温度？
- **A**: 按上述表格逐步排查：烟囱抽力→鼓风→燃料→密封性。

## 安全提醒

- 建窑选址远离房屋和易燃物
- 烧窑时必须有专人看管，防止火灾
- 新建窑第一次烧制可能有裂纹甚至坍塌，保持距离
- 一氧化碳中毒——窑炉排气不畅时致命

## 穿越小说梗

建窑是穿越者的"工程学入门考试"。现实中，新建一座窑从设计到能稳定出合格产品需要 2-4 周。如果你穿越后连个像样的窑都烧不出来，就别想玻璃、瓷器、钢铁这些高大上的东西了——**先把窑搞定**。龙窑是中国古人的伟大发明，穿越者如果能复现龙窑，就有了瓷器工业的基础。
