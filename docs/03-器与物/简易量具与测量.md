# 3.25 简易量具与测量

> 没有测量，就没有精确——没有精确，就没有工业。

## 原理

测量是将**未知量**与**已知标准**进行比较：

1. **长度**：以人体或自然物为基准（寸、尺、步）
2. **角度**：以直角三角形或圆为基准
3. **重量**：以常见物体为基准
4. **容量**：以容器为基准
5. **时间**：以天文现象或机械周期为基准

### 穿越者能获得的天然基准

| 基准 | ≈ 现代值 | 说明 |
|------|---------|------|
| 手掌宽（一握） | 约 8cm | 因人而异 |
| 拇指宽（一寸） | 约 2.5cm | — |
| 伸臂（一仞） | 约 1.8m | 两臂侧平举 |
| 一步 | 约 1.5m | 正常步幅 |
| 一撮 | 约 3g | 拇指食指捏起的盐量 |
| 一粒米 | 约 0.02g | — |

## 操作步骤

### 一、制作标准尺

1. **取临时基准**：用你自己的身体——量出从拇指尖到小指尖的距离（约 20cm = 古代"一拃"）
2. **刻基准尺**：在平整竹条上刻出等间距刻度
3. **校验**：用多个人体基准校验，取平均
4. **制作多把**：复制标准尺，确保统一
5. **细分**：将基本单位再细分为 10 等份

### 二、制作直角尺

1. 取两根直木条（长 30cm 和 40cm）
2. 按**勾股定理**：3:4:5 的比例
3. 30cm 和 40cm 两边成直角，斜边 50cm 验证
4. 固定成 L 形

### 三、制作简易秤

1. **秤杆**：取一根直木棍（约 50cm）
2. **支点**：在靠近一端打孔穿绳
3. **秤盘**：用竹/木/皮做盘
4. **砝码**：找一块石头，称出重量（例如用"一升水=1斤"）
5. **刻度**：挂已知重的物体，在秤杆上标记平衡位置
6. **可测范围**：从几克到几十斤

### 四、水平与垂直

1. **水平**：将水倒入浅槽，水面即水平面
2. **垂直**：用线+石头系成铅垂线
3. **45° 角**：在正方形对角线上

## 常见问题

- **Q**: 没有标准尺，古人怎么造建筑的？
- **A**: 古代也有自己的标准尺（官方颁发），但各地差异很大。穿越者的优势在于**统一标准**——你可以在领地内推行统一的度量衡。

- **Q**: 怎么在木头上画直线？
- **A**: **墨斗**——竹筒内装浸墨的丝绵，线穿过，拉紧弹出墨线。这是中国木匠两千年的传统工具。

## 安全提醒

- 切割量具时确保精度
- 铅垂线注意重物不要砸到脚
- 简易秤的支点要牢固，防止重物坠落

## 穿越小说梗

统一度量衡在穿越文里是"种田流"的必备操作。但说到具体的**每个单位怎么确定**，很多小说就含糊了。最简单的方法：**用你的身体先做一套基准，然后用水重做校准**——1升水=1公斤是最稳定可靠的标准。有了标准尺和标准砝码，你才能做出精密木工、标准零件、甚至可交换的金属部件——这是工业化的第一步。
