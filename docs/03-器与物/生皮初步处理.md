# 3.19 生皮初步处理

> 动物的皮如果不处理，三天就腐烂发臭；但如果处理得当，它可以陪伴你一辈子。

## 原理

生皮的主要成分是**胶原蛋白**。新鲜兽皮在常温下很快被细菌分解腐烂。处理的目的：

1. **防腐**：去除易腐烂的肉脂和水分
2. **清洁**：去除毛发和表皮
3. **准备**：为后续鞣制作好结构准备

### 生皮结构

| 层 | 说明 | 是否保留 |
|----|------|---------|
| 表皮层 | 最外层，含毛发 | 脱去 |
| 真皮层 | 胶原纤维，皮革的本质 | 保留并鞣制 |
| 皮下组织 | 脂肪和肉 | 去除 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 新鲜兽皮 | 原料 |
| 刮刀/竹片/骨片 | 去肉 |
| 盐 | 防腐（临时） |
| 清水 | 清洗 |
| 木灰/石灰 | 脱毛 |
| 木架 | 绷皮 |

## 操作步骤

### 一、清洗（立即进行）

1. 在兽皮还湿润时立即处理（拖延 = 腐烂）
2. 清水冲洗表面血污、泥土
3. 剪去残留的肉块和脂肪

### 二、去肉（最重要的一步）

1. 将皮平铺在倾斜的木板上（或绷在木架上）
2. 用刮刀从皮**尾部向头部**方向刮
3. 刮去残留的肉、脂肪和筋膜
4. **不要刮破皮**——破洞就是废品
5. 翻面，刮去皮内侧的薄层组织

### 三、脱毛

**石灰法（最常用）**：

1. 配制石灰水（熟石灰 10% 浓度）
2. 将皮浸入，完全浸泡
3. 室温下浸泡 3-7 天
4. 取出后用钝刀从**头部向尾部**方向刮，毛发应轻松脱落
5. 清水冲洗干净

**天然发酵法（无石灰）**：

1. 将皮浸入微温的稀草木灰水中
2. 每天换水，保持微碱性
3. 5-10 天后毛发松动即可刮除

### 四、脱灰与软化

石灰处理后，皮会膨胀变硬，需要中和：

1. 清水浸泡 2-3 天，每天换水
2. 加少量醋或发酵的面团水（乳酸）中和碱性
3. 皮恢复到柔软、适度膨胀的状态

### 五、准备鞣制

至此，生皮已变成**裸皮**——去除了毛发和非胶原成分，保留了胶原纤维网。接下来可以进行鞣制。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 时机 | 剥皮后当天处理；不能处理时先重盐防腐 |
| 去肉 | 肉脂、筋膜刮净，真皮不被刮穿 |
| 脱毛 | 石灰水 3-7 天后毛根松动，钝刀能轻推脱落 |
| 脱灰 | 清水 2-3 天反复换水后，皮不再滑腻刺手 |
| 裸皮 | 无腐臭、无绿斑、无蛆虫，厚薄基本完整 |

## 失败征兆

- 剥下后拖延到发热发臭，细菌已大量繁殖。
- 刮肉时出现大洞，后续鞣制只能做小件。
- 石灰水太浓或浸泡太久，皮纤维变脆。
- 脱灰不足，后续鞣剂难以均匀渗入。
- 处理病死动物皮，可能接触炭疽等严重病原。

## 常见问题

- **Q**: 皮发臭了还能用吗？
- **A**: 轻度异味还可以救（立即加盐或石灰水泡），但如果有绿斑或蛆虫就废了。**及时处理是关键**。

- **Q**: 怎么判断脱毛脱干净了？
- **A**: 用指甲轻刮皮面——如果毛发根部松动，轻轻一推就掉，说明石灰浸泡到位。

- **Q**: 小型动物皮（兔子、松鼠）能处理吗？
- **A**: 可以，但皮薄易破，去肉和脱毛都要更轻柔。

## 安全提醒

- 石灰水有强碱性，接触皮肤会腐蚀，戴手套操作
- 处理生皮时可能接触病菌（尤其野生动物），操作后彻底洗手
- 石灰水处理过的皮不要直接接触口鼻
- 盐水防腐的皮要远离金属容器（盐腐蚀金属）

## 穿越小说梗

小说里"主角猎了头鹿，三天后穿上了皮衣"——现实中从生皮到可用的皮革需要**数周甚至数月**。大部分穿越者忽略了这个时间跨度。三天能做的是**简单地晒干**（硬得像木板）或**烟熏**（粗糙但可用），真正的柔软皮革需要鞣制。
