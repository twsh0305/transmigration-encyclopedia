# 3.18 瓷器 vs 陶器区别与条件

> 陶器是泥土经过火的洗礼，瓷器是岩石经过火的升华。

## 原理

| 特性 | 陶器 | 瓷器 |
|------|------|------|
| 原料 | 普通黏土 | 高岭土（瓷土） |
| 烧成温度 | 800-1000摄氏度 | 1200-1400摄氏度 |
| 是否玻化 | 不完全（有微孔） | 完全玻化（致密） |
| 透光性 | 不透光 | 薄处透光 |
| 胎体颜色 | 红褐/灰色（铁含量高） | 白色（铁含量<1%） |
| 吸水率 | >5% | <0.5% |
| 声音 | 闷浊 | 清脆悠长（如金属） |

### 核心区别

| 维度 | 陶器 | 瓷器 |
|------|------|------|
| **原料不同** | 含铁量高的黏土 | 高岭土（纯净 Al₂O₃·2SiO₂·2H₂O） |
| **温度不同** | 1000摄氏度以下 | 1200摄氏度以上 |
| **玻化程度** | 颗粒间只是烧结 | 玻璃相填充颗粒间隙，完全致密 |

## 从陶到瓷的三个技术壁垒

### 壁垒一：高岭土（瓷土）

- 普通黏土含铁 5-8%，烧出来是红褐色
- 瓷土含铁 <1%，烧出来是白色
- **高岭土**是长石风化产物，一般产自花岗岩地区
- 缺乏高岭土，就别想做瓷器

### 壁垒二：高温窑炉（1200摄氏度以上）

- 800-1000摄氏度 → 地坑窑/升焰窑
- 1000-1200摄氏度 → 升焰窑 + 优质燃料 + 鼓风
- 1200-1300摄氏度 → **龙窑**（斜坡窑）或 **馒头窑**
- 1300摄氏度以上 → 龙窑 + 强鼓风 + 优质木炭

| 窑型 | 最高温度 | 适用 |
|------|---------|------|
| 地坑窑 | ~900摄氏度 | 陶器 |
| 升焰窑 | ~1050摄氏度 | 陶器、低温釉 |
| 馒头窑 | ~1250摄氏度 | 瓷器（北方） |
| 龙窑 | ~1350摄氏度 | 瓷器（南方） |

### 壁垒三：釉料配方

陶器上釉是"补救"（防止漏水），瓷器上釉是"增色"（已有玻璃质胎体）。

瓷器名釉：
- **青瓷**：铁还原呈青绿色（龙泉窑、越窑）
- **白瓷**：铁含量极低（邢窑、定窑）
- **青花**：钴料画花 + 透明釉（景德镇）

## 操作步骤：从陶到瓷的跨越路线

```
第一步：确认有高岭土矿（寻找白色黏土/花岗岩风化区）
↓
第二步：建龙窑（需要斜坡和大量人力）
↓
第三步：试烧（逐步提高温度，每次加 50摄氏度）
↓
第四步：配釉（石灰釉 → 石灰碱釉）
↓
第五步：大批量生产（需要完整的产业链）
```

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 陶器 | 800-1000摄氏度烧成后敲声不闷，吸水但不散 |
| 瓷土 | 白或灰白、杂质少，试片高温后不明显塌形 |
| 窑温 | 能稳定达到 1200摄氏度以上，并保持足够时间让胎体玻化 |
| 瓷器 | 薄处微透光，敲声清脆，吸水率低，不渗水 |
| 量产 | 同一窑位连续 3 次试片结果接近，才扩大烧制 |

## 失败征兆

- 只找到普通红黏土，却试图靠高温烧成白瓷。
- 窑温忽高忽低，试片一半生烧一半变形。
- 过早开窑，冷热冲击导致坯体炸裂。
- 釉和胎收缩不匹配，满身裂纹或剥釉。
- 没做试片就满窑烧成，失败会浪费整批燃料和坯体。

## 常见问题

- **Q**: 我怎么知道找到的是不是高岭土？
- **A**: ①白色或灰色②湿时有滑腻感③掺水后塑形好④烧到 1200摄氏度以上不变形。简易测试：取一小块烧到高温，白色不变就是好瓷土。

- **Q**: 没有温度计怎么知道窑内温度？
- **A**: 用**测温锥**或**观察孔**：
  - 800摄氏度：铁钉变暗红
  - 1000摄氏度：铁钉变橙红
  - 1200摄氏度：铁钉变亮黄白
  - 1300摄氏度：铁钉几乎不可见（太亮）

## 安全提醒

- 高温操作必须保持距离
- 瓷器烧成后冷却极慢（需 2-3 天），过早开窑会因冷空气进入导致坯体炸裂
- 瓷土矿选址注意塌方风险

## 穿越小说梗

瓷器是穿越小说里最经典的"发财路线"——这比玻璃靠谱，因为中国对瓷器有天然的审美和文化需求。但难点是：①你得正好穿越在瓷土产区 ②你得有建龙窑的工程能力 ③瓷器从试制到稳定量产需要数月。小说常忽略"试错成本"——每一次烧窑失败都是时间、燃料和人力的巨大消耗。穿越者的优势在于**知道方向是对的**，可以少走几百年的弯路。
