# 3.2 再生纸简易制作

> 古代没有废纸回收，但你可以教会他们。

## 原理

再生纸是**二次纤维利用**——将已有的纸张在水中分散，重新抄制成新纸：

1. 废纸浸泡在水中，纤维吸水膨胀
2. 机械搅拌使纤维重新分离（再制浆）
3. 去除油墨和杂质
4. 重新抄纸成型

每次回收，纤维会变短一点（机械损伤），品质会下降。一张纸最多可回收 5-7 次。

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 废纸 | 原料 |
| 水 | 浸泡和分散 |
| 盆/桶 | 泡纸容器 |
| 搅拌棒 | 搅碎纸浆 |
| 细网/纱窗 | 抄纸（简易版） |
| 压板 | 脱水 |

## 操作步骤

### 简易再生纸（无专业工具版）

1. **撕纸**：将废纸撕成小碎片（拇指大小）
2. **浸泡**：温水浸泡 2-4 小时，至纸片完全软透
3. **打浆**：用手搓揉、用木棒搅拌，或用石臼捣打
4. **调浆**：加水调成稀糊状（像稀粥）
5. **抄纸**：用细网/纱窗/密布代替纸帘，水平捞起纸浆
6. **脱水**：用布吸干或用木板挤压
7. **晾干**：贴在平滑表面晾干

**不需要纸帘，不需要大锅蒸煮**——有盆、有网、有废纸就能做。

### 添加填充物

再生纸中可以加入各种东西，制造特殊效果：
- **干花瓣** → 花草纸（装饰用）
- **碎草叶** → 粗糙的包装纸
- **丝线头** → 纹理纸
- **染色** → 用天然染料染纸浆（靛蓝、姜黄、茜草等）

### 去油墨（如需）

如果废纸上有印刷墨迹：

1. 废纸撕碎后加少量草木灰碱液
2. 浸泡 30-60 分钟
3. 充分搅拌搓揉，油墨会分散
4. 换清水漂洗 2-3 次
5. 油墨去除效果约 60-80%（不可能完全去除）

## 造纸失败的废料怎么办

造纸过程中产生的废纸浆和废纸边角料，别扔：

- 和泥混合，做纸泥墙（隔热隔音）
- 晒干后作为引火物（纸浆饼极易点燃）
- 掺入泥土做纸浆砖（轻质建材）
- 作为肥料回田（纤维素可降解）

## 常见问题

- **Q**: 再生纸颜色发灰？
- **A**: 废纸原有的油墨和杂质没有去除干净。可以尝试加漂白步骤。

- **Q**: 再生纸表面粗糙、厚薄不均？
- **A**: 纸浆不够细（再打一会儿），或者抄纸网太疏。换更细的网。

- **Q**: 再生纸容易破？
- **A**: 纤维在回收过程中受损变短。可以加一点新纸浆或捣烂的麻纤维增强。

## 安全提醒

- 再生纸不建议用于重要书写（耐久性差）
- 含化学油墨的废纸不宜用在食品包装
- 造纸废水含有机物，不要直接排入水源

## 穿越小说梗

穿越后如果急需用纸，再生纸是最快的方案——当天泡纸、当天出纸。不需要泡竹子几个月，不需要刮构树皮，不需要编纸帘（一块纱窗布搞定）。缺点是品质不高。应急可以，批量生产还得走正规造纸路线。
