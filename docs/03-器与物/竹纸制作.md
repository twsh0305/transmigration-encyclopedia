# 3.23 竹纸制作

> 竹子造的纸，是中国造纸技术的巅峰——也是穿越后最容易实现的方案之一。

## 原理

竹纸与麻纸/树皮纸的原理相同，但竹子有几个独特特点：
- 竹纤维较长（约 1.5-2毫米），纸张强度好
- 竹材生长快、分布广、产量大
- 竹子含大量木质素和胶质，处理难度大（需要长时间浸泡和多次蒸煮）

**竹纸痛点**：竹材处理周期长（传统工艺需要数月），但一旦掌握，竹纸产量和质量都远超其他原料。

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 嫩竹（一年生青竹） | 原料——老竹纤维太硬 |
| 石灰 | 蒸煮去胶 |
| 草木灰碱液 | 二次蒸煮漂白 |
| 大锅 | 蒸煮 |
| 石臼/水碓 | 打浆 |
| 纸帘 | 抄纸 |
| 压榨设备 | 脱水 |

## 操作步骤

### 第一步：浸竹（传统需数月，穿越速成版）

**传统法**：
1. 嫩竹砍下，截成 1-2 米段
2. 放入水池或溪流中，压上石头
3. 浸泡 2-3 个月（让微生物分解部分胶质和色素）
4. 竹材变软、颜色变浅

**穿越速成版**（等不了几个月）：
1. 嫩竹劈成细条（越细越好，筷子粗细）
2. 加石灰水煮沸 8-12 小时
3. 捞出，换清水再煮 4-6 小时去除石灰
4. 重复 2-3 次，直到竹条变得柔软

### 第二步：蒸煮（进一步去杂）

1. 浸泡后的竹料加草木灰碱液
2. 大火煮沸 6-8 小时
3. 竹料颜色变浅，纤维开始松散
4. 清水漂洗 2-3 次

### 第三步：打浆

1. 煮好的竹料放入臼中
2. 反复捶打至纤维分离
3. **最佳状态**：取少量放在水中搅拌——纤维呈均匀悬浮状，无硬块

**水碓升级**：如果有水力条件，用水碓打浆可以节省 90% 的人工。

### 第四步：漂白（可选）

如果需要白纸：

1. 纸浆中加入少量石灰水
2. 摊薄曝晒 3-5 天
3. 翻面再晒
4. 漂白后的纸浆颜色从灰黄变为白色
5. 清水漂洗去碱

### 第五步：抄纸

与通用造纸步骤相同——见"造纸术原理"篇。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 竹料 | 一年生嫩竹劈成筷子粗细，煮后能用手撕散 |
| 蒸煮 | 石灰水 8-12 小时、清水 4-6 小时循环后无硬芯 |
| 打浆 | 少量浆入水后均匀悬浮，无竹片硬块 |
| 漂洗 | 纸浆不再滑腻刺手，水色明显变清 |
| 成纸 | 厚薄基本均匀，干后能整张揭下，书写不大面积洇墨 |

## 失败征兆

- 老竹未充分蒸煮，纸面满是硬梗。
- 碱液残留，纸张发脆、发黄、刺激皮肤。
- 抄纸时纤维快速沉底，说明浆太浓、搅拌不足或缺纸药。
- 晾纸暴晒急干，纸张卷曲开裂。
- 浸竹池靠近住处和水源，发酵臭水会污染环境。

| 对比 | 竹纸 | 麻纸 | 构树皮纸 |

## 常见问题

- **Q**: 竹纸容易发黄？
- **A**: 竹子含木质素高，漂白不彻底的话会慢慢发黄。多漂洗和曝晒可以改善。

- **Q**: 没有一年生嫩竹怎么办？
- **A**: 老竹也能用，但需要更长的浸泡和蒸煮时间。老竹纤维硬化严重，打浆也更费力。

- **Q**: 用竹浆抄纸总是厚薄不均？
- **A**: 竹纤维在水中沉降速度快，需要快速搅拌保持悬浮。抄纸时动作要稳、快、均匀。

## 安全提醒

- 石灰水蒸煮时注意蒸汽烫伤
- 漂白后的纸浆碱性可能仍偏高，多洗几遍
- 浸泡竹子的水池会发臭（发酵过程中产生的气味），远离居住区

## 穿越小说梗

如果用竹子造纸，最大的问题不是技术——是时间。传统竹纸工艺从浸竹到出纸要 3-5 个月。穿越速成版（石灰+碱液反复煮）可以把时间压缩到 3-5 天，但需要大量的柴火和石灰。所以穿越后如果急着用纸，先用麻布或构树皮过渡，等站稳脚跟了再搞竹纸。
