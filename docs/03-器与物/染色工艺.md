# 3.11 染色工艺

> 从"不褪色"到"不掉色"，人类花了几千年——而穿越者可以一晚学会。

## 原理

染色是**染料分子与纤维结合**的过程：

1. **直接染色**：染料直接溶于水，被纤维吸附（如栀子黄）
2. **媒染染色**：染料需通过媒染剂（金属离子）作为"桥梁"连接纤维（如茜草红）
3. **还原染色**：不溶于水的染料被还原为可溶，染色后在空气中氧化为不溶态（如靛蓝）

### 媒染剂的作用

| 媒染剂 | 常见来源 | 与茜草产生的颜色 | 与五倍子产生的颜色 |
|--------|---------|-----------------|------------------|
| 明矾（铝盐） | 明矾石 | 红 | 浅灰 |
| 铁 | 铁锈+醋 | 紫褐 | 黑 |
| 铜 | 铜绿+醋 | 橙红 | 深灰 |
| 锡 | 锡器 | 亮红 | — |

## 所需材料

| 材料 | 用途 |
|------|------|
| 染液 | 已提取的天然染料 |
| 媒染剂 | 固色/调色 |
| 待染织物 | 须先脱浆煮练 |
| 大锅 | 染色 |
| 木棍/竹棍 | 翻搅 |
| 肥皂/皂角水 | 煮练去油 |
| 清水 | 漂洗 |

## 操作步骤

### 染前准备（煮练）

1. 将布料在碱水（草木灰水/石灰水）中煮沸 30 分钟
2. 清水漂洗干净——去除浆料和油脂，纤维才能吸收染料

### 同浴媒染法

1. 锅中加水和染液
2. 加入媒染剂（明矾等）
3. 放入湿布料，微火煮 30-60 分钟
4. 翻搅使着色均匀
5. 取出后自然氧化（晾挂）

### 后媒染法（更灵活）

1. 先用染液煮布
2. 取出后在媒染剂溶液中浸泡 15-30 分钟
3. 再次浸入染液加深颜色
4. 一布可以染出渐变效果

### 套染法（创造颜色）

| 第一步 | 第二步 | 结果 |
|--------|--------|------|
| 栀子黄 | 靛蓝 | 绿 |
| 茜草红 | 靛蓝 | 紫 |
| 栀子黄 | 茜草红 | 橙 |
| 靛蓝 | 五倍子黑 | 深蓝黑 |

## 成功标准

- 染前布料经煮练后能被水完全润湿，不挂油珠。
- 染色时布料能在染液中松散展开，30-60 分钟内颜色基本均匀。
- 清水漂洗到水色明显变淡，晾干后手搓不大片掉色。
- 记录染料重量、媒染剂用量、温度、时间和重复浸染次数。

## 失败征兆

- 布面花斑、折痕处发白：未展开、未翻搅或煮练不足。
- 一洗就掉色：缺媒染、温度不合适或染料浓度不足。
- 纤维发脆、发硬：铁媒染过量或碱处理太强。
- 染液发臭、长霉、表面漂油：原料腐败或容器不洁。

## 常见问题

- **Q**: 为什么染出来的颜色不均匀？
- **A**: 布料没有完全浸入、翻搅不够、或者染料没有充分溶解。染色时确保布料在染液中松散展开。

- **Q**: 怎么让颜色更深？
- **A**: 多次浸染（重复浸→氧化→再浸）比一次长时间煮耗更有效。每浸一次颜色加深一层。

## 安全提醒

- 铁媒染的布料会使纤维变脆（铁离子促进纤维氧化）
- 铜媒染时注意铜盐有微毒，染色后充分清洗

## 穿越小说梗

"穿越女在古代开染坊"——这是古代最赚钱的手工业之一，好的染匠在哪个朝代都吃香。现代穿越者最大的优势是**深知颜色搭配理论**（色环、互补色）和**科学的工艺流程**（媒染、套染），可以大幅缩短传统染坊"试错几代人"的研发周期。
