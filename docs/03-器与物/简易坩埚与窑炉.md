# 3.24 简易坩埚与窑炉

> 没有坩埚，熔化的玻璃就是一滩废料；没有窑炉，文明只能停留在石器时代。

## 原理

坩埚是承受高温熔炼的容器，窑炉是维持高温的设备。二者的本质都是**热学工程**：

- **窑炉**：利用隔热层减少热量散失，通过烟囱产生自然抽风，维持燃烧区的高温
- **坩埚**：用耐火材料制成的小型熔器，能承受 1000°C+ 而不熔化、不开裂

穿越初期，坩埚和窑炉往往需要**自己烧制自己**——用泥巴做第一个坩埚来烧正式的坩埚。

## 所需材料

### 简易坩埚

| 材料 | 作用 | 备注 |
|------|------|------|
| 耐火黏土 | 主体材料 | 含高岭土多的黏土最好 |
| 熟料（碎陶片粉） | 减少收缩开裂 | 旧陶器碾碎 |
| 石英砂 | 增加耐火度 | 河沙过筛 |
| 稻草/毛发 | 增强抗热震性 | 烧成后碳化留下微孔 |

### 简易窑炉

| 材料 | 作用 |
|------|------|
| 土坯/石头 | 窑体 |
| 黏土 | 粘合密封 |
| 烟囱管（陶管或砖砌） | 抽风 |
| 风箱/吹火筒 | 强制鼓风 |

## 操作步骤

### 坩埚制作

1. **配泥**：耐火黏土 70% + 熟料粉 30%，加水揉匀
2. **成型**：手捏或泥条盘筑成钵形，壁厚 1-2cm
3. **阴干**：避免暴晒，慢干 3-7 天（开裂则报废）
4. **素烧**：800°C 低温烧制，使坩埚具备基本强度
5. **上釉（可选）**：涂一层釉料减少熔体渗透

### 地坑窑（最简单）

1. 在地上挖一个直径 50cm、深 40cm 的坑
2. 坑壁抹一层黏土砂浆
3. 坑底留进风口（倾斜通道）
4. 用土坯在坑口砌一圈矮墙
5. 顶部用泥封住，留出烟孔

### 升焰窑（更实用）

1. 用土坯砌一个圆形窑体，高 60-80cm
2. 下部留火门（烧柴口）
3. 内部用泥条搭出炉栅（支撑燃料/坩埚）
4. 顶部留出烟口
5. 侧面装一个陶管作烟囱延伸

## 关键温度

| 工艺 | 所需温度 | 窑炉要求 |
|------|---------|---------|
| 陶器烧制 | 800-1000°C | 地坑窑即可 |
| 玻璃熔化 | 1000-1200°C | 升焰窑 + 鼓风 |
| 瓷器烧制 | 1200-1300°C | 龙窑/馒头窑 + 强鼓风 |
| 炼铁 | 1100-1250°C | 块炼炉 + 风箱 |
| 炼钢 | 1500°C+ | 高炉/坩埚钢 |

## 常见问题

- **Q**: 为什么窑烧不热？
- **A**: 检查三点：①燃料是否足够（木炭优于木柴）②烟囱抽风是否通畅③窑体是否漏气

- **Q**: 泥坩埚一加热就裂了？
- **A**: ①升温太快（要缓慢预热）②熟料比例不够③泥料没有充分揉匀

## 安全提醒

- 高温作业确保通风，防止一氧化碳中毒
- 烧窑时远离易燃物
- 用长柄工具操作，防止烫伤

## 穿越小说梗

小说里主角"三天就搭好了窑烧出玻璃"——现实中从挖泥到烧出合格的坩埚至少需要两周（光是阴干就急不得）。建窑是穿越后的**第一个工业基础设施**，直接决定了你能造什么。
