# 3.5 墨的制作（松烟墨、油烟墨）

> 纸是文明的载体，墨是文明的笔迹——没有墨，纸只是白纸。

## 原理

墨的主要成分是**碳**（炭黑）+ **胶**（动物胶/植物胶） + **添加剂**：

- 碳黑颗粒提供颜色（极细的黑色粉末，化学性质稳定）
- 胶使碳黑颗粒粘合成块，并使墨附着在纸上
- 添加剂（香料、防腐剂）改善气味和保存性

中国墨的两种主流：
- **松烟墨**：燃烧松木取烟，色黑但无光泽
- **油烟墨**：燃烧油脂（桐油/菜油）取烟，色黑且有光泽

## 所需材料

| 材料 | 松烟墨 | 油烟墨 |
|------|--------|--------|
| 烟料 | 松木燃烧的烟灰 | 桐油/菜油燃烧的烟灰 |
| 胶 | 骨胶/牛皮胶 | 骨胶/牛皮胶 |
| 水 | 溶胶用 | 溶胶用 |
| 香料（可选） | 麝香/冰片/丁香 | 同左 |
| 防腐剂（可选） | 白酒 | 白酒 |

## 操作步骤

### 第一步：取烟

**松烟法**：
1. 建烟室：密闭房间，顶部悬挂集烟罩（纸或布制）
2. 燃烧松木段（松脂含量高的老松木最佳）
3. 烟灰上升附着在集烟罩上
4. 每半小时收集一次烟灰（时间越长，碳粒越细）
5. 收集到的细烟粉 = 松烟

**油烟法**：
1. 灯碗中倒入桐油或菜油
2. 点燃灯芯（多盏灯同时点）
3. 灯上倒扣陶瓷碗或铁皮罩
4. 火焰燃烧产生的烟灰附着在碗底
5. 定期刮取碗底的烟灰
6. 油烟墨的炭粒比松烟更细，制成的墨更有光泽

**取烟效率**：松烟法一天可得数百克，油烟法一天只得几十克。

### 第二步：制胶

1. 骨胶/牛皮胶敲碎
2. 用冷水浸泡 2-4 小时至软化
3. 隔水加热融化（不要直接煮沸）
4. 胶水过滤去杂

### 第三步：和墨

1. 烟粉 + 胶水（比例约 1:1 到 1:1.5）
2. 加入少量酒（防腐、发墨）
3. 加入香料（研细的冰片/丁香粉）
4. 充分搅拌混合成墨泥

### 第四步：捶打

1. 将墨泥放在石板上
2. 用木槌反复捶打（炼墨）
3. 捶打过程使胶和烟充分融合
4. 捶打至墨团均匀细腻、不粘手
5. **捶打几百次是基本功**——捶得越多墨质越好

### 第五步：成型

1. 捶好的墨团放入模具（木质墨模）
2. 压紧压实
3. 脱模后修边
4. 放置在阴凉通风处自然阴干（**不能晒**）
5. 阴干过程持续 1-3 个月（干透的墨不开裂）

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 取烟 | 烟粉细黑，无明显灰渣、木屑和油滴 |
| 胶液 | 骨胶冷水泡 2-4 小时后隔水融化，不直接煮焦 |
| 比例 | 烟粉:胶水约 1:1 到 1:1.5，墨泥均匀不流淌 |
| 捶打 | 墨团细腻、不粘手、切面均匀，至少数百次捶炼 |
| 阴干 | 1-3 个月慢干，墨块不开裂、不发霉，研磨出墨顺畅 |

## 失败征兆

- 烟粉混入粗灰，写出来发灰、颗粒刮纸。
- 胶太多，墨块难磨且笔画发黏。
- 胶太少，墨块掉粉、字迹附着差。
- 阴干太快，墨条开裂弯曲。
- 集烟室通风差，头晕恶心是一氧化碳风险。

## 松烟墨 vs 油烟墨

| 对比 | 松烟墨 | 油烟墨 |
|------|--------|--------|
| 颜色 | 黑，无光泽 | 黑亮有光泽 |
| 炭粒粗细 | 较粗 | 极细 |
| 集烟速度 | 快（数百克/天） | 慢（几十克/天） |
| 适合用途 | 书法、大面积涂黑 | 绘画、工笔、精细书写 |
| 墨色层次 | 少 | 丰富 |
| 贵重程度 | 一般 | 昂贵 |

## 常见问题

- **Q**: 墨在砚台上磨不开？
- **A**: 胶放太多了。和墨时胶的比例过高。下次减少胶量。

- **Q**: 写出来的墨色淡？
- **A**: 烟粉不够细或胶太少。烟灰收集时取最细的那层（集烟罩最上层的烟最细）。

- **Q**: 墨块开裂？
- **A**: 阴干太快或捶打不够。阴干环境要保持一定的湿度，不要风直吹。

## 安全提醒

- 燃烧松木/油脂取烟，要保证通风（防止一氧化碳中毒）
- 集烟室内温度高，注意防火
- 干燥的墨块很脆，搬运时轻拿轻放

## 穿越小说梗

"主角用一块墨震惊文坛"——制墨其实不难，难的是时间。光阴干就要 1-3 个月。而且好墨讲究的是"捶打千杵"——一块好墨要反复捶打上千次才能让胶和烟充分融合，这是纯体力的苦活。穿越后想快速出墨，省掉捶打和阴干也不是不行——直接做墨汁：烟灰 + 胶水 + 水调匀，当天就能用。但墨汁不能长期保存，放几天就臭了。
