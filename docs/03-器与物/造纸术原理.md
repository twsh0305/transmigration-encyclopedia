# 3.30 造纸术原理

> 没有纸，文明只能刻在石头上——还好蔡伦想到了树皮。

## 原理

造纸的核心是**纤维的分散与交织**：

1. 植物纤维（纤维素）在水中分散成单根纤维
2. 用筛网捞出纤维，水从网孔流走
3. 纤维在筛网上相互交织，形成薄片
4. 压干水分、干燥后即得纸

**任何含纤维素的植物都可以造纸**，区别在于纤维的长度、纯度和加工难度。

## 纤维来源对比

| 原料 | 纤维特性 | 纸张品质 | 加工难度 |
|------|---------|---------|---------|
| 麻（大麻/苎麻） | 长而强韧 | 坚韧耐久 | 中等 |
| 树皮（构树/桑树） | 中长 | 柔软细腻 | 中等 |
| 竹 | 中等 | 光滑洁白 | 高（需长时间浸泡） |
| 稻草/麦秆 | 短而脆 | 粗糙 | 低 |
| 废纸/旧布 | 已处理 | 一般 | 最低 |

**最佳穿越材料**：构树皮（Broussonetia papyrifera）——纤维质量好、分布广，是中国古代造纸的主要原料。

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 植物纤维原料 | 纸浆来源 |
| 石灰水或草木灰碱液 | 蒸煮去杂，分离纤维 |
| 大锅 | 蒸煮原料 |
| 石臼/木槌 | 捣打纤维成浆 |
| 纸帘（竹帘+木框） | 抄纸用 |
| 压榨工具 | 压干水分 |
| 平滑的板/墙 | 晾纸 |

## 通用造纸流程

```
原料 → 浸泡 → 蒸煮 → 洗涤 → 捣打 → 调浆 → 抄纸 → 压榨 → 晾干 → 揭纸
```

### 第一步：备料

1. 原料切碎（树皮切 5-10cm 段，竹竿劈成细条）
2. 浸泡数日至数周（使纤维软化）

### 第二步：蒸煮

1. 原料放入大锅，加石灰水或草木灰碱液（pH 10-12）
2. 煮沸 4-8 小时（树皮/麻）或 1-2 天（竹子）
3. 煮至纤维松软、颜色变浅
4. 捞出用清水反复漂洗去除碱液和杂质

### 第三步：打浆

1. 煮好的纤维放入石臼
2. 反复捶打（或用水碓/磨盘）
3. 打至纤维分离成细浆状
4. 在水中搅拌时呈现均匀的悬浮液

**判断标准**：取少量纸浆放在手中搓——没有粗糙的硬块，只有细腻的纤维感。

### 第四步：抄纸

1. 纸浆倒入水槽，加水稀释（纸浆:水 ≈ 1:20）
2. 搅拌均匀，使纤维在水中悬浮
3. 纸帘插入水中，水平捞起
4. 轻微摇晃使纤维均匀铺展
5. 水流走，纤维留在帘上形成湿纸膜
6. 将湿纸膜翻扣在平板上

### 第五步：压榨干燥

1. 湿纸膜叠放（层间用布隔开）
2. 用重物或螺杆压榨，挤出水分
3. 将湿纸贴在平滑的板或墙上
4. 自然晾干或微火烘干

### 第六步：揭纸

1. 干透后从板/墙上揭下
2. 修边、整理

## 常见问题

- **Q**: 纸太薄容易破？
- **A**: 纸浆太稀或打浆不够（纤维没有充分分散，交织不密）。

- **Q**: 纸太厚且粗糙？
- **A**: 纸浆太浓或纤维打得不细。增加水量、继续打浆。

- **Q**: 纸的颜色发黄？
- **A**: 漂洗不彻底，或原料含杂质多。多蒸煮几次或加漂白步骤。

- **Q**: 纸从纸帘上揭不下来？
- **A**: 湿纸太薄，干燥后粘贴过牢。可以在纸帘上涂一层薄薄的菜油或蜂蜡作为脱模剂。

## 安全提醒

- 石灰水/碱液有腐蚀性，避免接触皮肤
- 蒸煮过程注意火源和烫伤
- 打浆时防止木槌砸到手

## 穿越小说梗

"造纸术是中国四大发明"——但你真的想穿越后造纸？得先找原料（构树皮或麻），蒸煮要一两天，打浆要人工捶打几百下，抄纸要手艺——没练过的人抄出来的纸厚薄不均。关键是纸帘的精细度决定了纸的品质，竹篾编得不够细，纸面就粗糙得像草纸。所以穿越后想靠造纸发家，先练好竹编手艺再做纸帘。
