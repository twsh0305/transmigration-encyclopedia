# 3.1 丝织基础

> 一根蚕丝可以长达一公里——这是自然界最完美的连续纤维。

## 原理

丝织与普通织布的核心区别在于**丝的来源和特性**：

1. 蚕吃桑叶 → 吐丝成茧（一根连续丝线可达 800-1500m）
2. 缫丝：将数根蚕丝合并成生丝（需要的粗细）
3. 织造：丝线极细且光滑，可织出极其细密的织物

### 丝织品种类

| 种类 | 特点 | 工艺 |
|------|------|------|
| 绢（生织） | 平纹，硬挺 | 生丝直接织，后脱胶 |
| 绸（熟织） | 柔软光泽 | 丝脱胶染后再织 |
| 缎 | 光滑亮丽 | 缎纹组织，长浮线 |
| 罗/纱 | 轻薄透明 | 加捻绞经结构 |

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 蚕茧 | 丝原料 |
| 热水锅（缫丝锅） | 溶解丝胶 |
| 缫丝架 | 集丝成线 |
| 经车/纺车 | 整理丝线 |
| 织机 | 织造（可用普通织机） |

## 操作步骤

### 一、养蚕（蚕的饲养）

1. **催青**：蚕卵在 20-25°C 孵化
2. **采桑**：每日喂新鲜桑叶 3-4 次
3. **除沙**：清理蚕粪，保持清洁
4. **上簇**：蚕成熟后（约 25-35 天），放上草簇吐丝结茧
5. **选茧**：好的白色茧用于缫丝，次品做丝绵

### 二、缫丝（提取丝线）

1. **煮茧**：蚕茧放入 70-80°C 热水中，煮 2-3 分钟溶解丝胶
2. **找丝头**：用毛刷在茧表面刷动，找到丝头
3. **集丝**：将 3-8 根蚕丝合并，穿过缫丝架的导孔
4. **卷绕**：将合并后的丝线绕在缫丝框上
5. **换茧**：一根丝线快尽时，接上新的茧丝

### 三、织造

1. **整经**：将丝线按需要的长度和密度整理成经线
2. **浆丝**：少量米汤浆丝（增加强度）
3. **上机织造**：参考"织布基础"的操作

## 丝与普通纤维的对比

| 特性 | 丝 | 棉 | 麻 |
|------|-----|-----|-----|
| 纤维长度 | 连续（~1000m） | 2-5cm | 10-30cm |
| 细度 | 极细（10-15μm） | 15-20μm | 20-30μm |
| 强力 | 高 | 中 | 很高 |
| 光泽 | 天然光泽 | 哑光 | 哑光 |
| 手感 | 润滑 | 柔软 | 粗硬 |

## 常见问题

- **Q**: 蚕不吃桑叶行不行？
- **A**: 蚕只吃桑叶（或柘叶）。没有桑树的地方无法养蚕产丝。

- **Q**: 缫丝时丝总断怎么办？
- **A**: 水温过高或过低。70-80°C 最佳——太热丝胶全溶，丝散开；太冷丝胶不溶，丝不易分离。

- **Q**: 一个茧能抽出多长的丝？
- **A**: 好的家蚕茧可抽出 800-1500m 的连续丝。6-8 个茧合成一根生丝。

## 安全提醒

- 缫丝热水注意烫伤
- 养蚕环境需保持卫生，防病菌
- 蚕有气味（桑叶+蚕沙），注意室内通风

## 穿越小说梗

丝织是古代中国最赚钱的产业，丝绸沿着丝绸之路从中国运到罗马，价格与黄金等价。穿越者想搞丝绸需要注意：①你需要桑树（北方没有）②养蚕到织出第一匹丝绸至少需要 2-3 个月③缫丝技术是关键——不会缫丝，满屋的茧子就是废品。但穿越者的优势在于：知道丝绸的**国际贸易价值**，可以更好地策划"丝绸之路"商业路线。
