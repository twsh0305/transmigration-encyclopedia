# 3.1 丝织基础

> 一根蚕丝可以长达一公里——这是自然界最完美的连续纤维。

## 原理

丝织与普通织布的核心区别在于**丝的来源和特性**：

1. 蚕吃桑叶 → 吐丝成茧（一根连续丝线可达 800-1500米）
2. 缫丝：将数根蚕丝合并成生丝（需要的粗细）
3. 织造：丝线极细且光滑，可织出极其细密的织物

### 丝织品种类

| 种类 | 特点 | 工艺 |
|------|------|------|
| 绢（生织） | 平纹，硬挺 | 生丝直接织，后脱胶 |
| 绸（熟织） | 柔软光泽 | 丝脱胶染后再织 |
| 缎 | 光滑亮丽 | 缎纹组织，长浮线 |
| 罗/纱 | 轻薄透明 | 加捻绞经结构 |

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 蚕茧 | 丝原料 |
| 热水锅（缫丝锅） | 溶解丝胶 |
| 缫丝架 | 集丝成线 |
| 经车/纺车 | 整理丝线 |
| 织机 | 织造（可用普通织机） |

## 操作步骤

### 一、养蚕（蚕的饲养）

1. **催青**：蚕卵在 20-25摄氏度孵化
2. **采桑**：每日喂新鲜桑叶 3-4 次
3. **除沙**：清理蚕粪，保持清洁
4. **上簇**：蚕成熟后（约 25-35 天），放上草簇吐丝结茧
5. **选茧**：好的白色茧用于缫丝，次品做丝绵

### 二、缫丝（提取丝线）

1. **煮茧**：蚕茧放入 70-80摄氏度热水中，煮 2-3 分钟溶解丝胶
2. **找丝头**：用毛刷在茧表面刷动，找到丝头
3. **集丝**：将 3-8 根蚕丝合并，穿过缫丝架的导孔
4. **卷绕**：将合并后的丝线绕在缫丝框上
5. **换茧**：一根丝线快尽时，接上新的茧丝

### 三、织造

1. **整经**：将丝线按需要的长度和密度整理成经线
2. **浆丝**：少量米汤浆丝（增加强度）
3. **上机织造**：参考"织布基础"的操作

## 温度与时间尺度

| 环节 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 催青 | 20 到 25 摄氏度 | 蚕卵整齐孵化 | 忽冷忽热，出蚕不齐 |
| 养蚕 | 每日喂桑叶 3 到 4 次，约 25 到 35 天上簇 | 多数蚕体透明发亮、开始结茧 | 蚕房霉味、大量死蚕 |
| 煮茧 | 70 到 80 摄氏度，2 到 3 分钟 | 丝头易找，丝线连续 | 水太热散成絮，水太冷找不到丝头 |
| 合丝 | 3 到 8 根茧丝合成一根生丝 | 粗细均匀，不忽粗忽细 | 频繁断丝或线粗不稳 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 养蚕 | 20-25摄氏度孵化，25-35 天后多数蚕体透明发亮、开始找地方结茧 |
| 选茧 | 白净、完整、手感紧实的茧用于缫丝，破茧和薄茧改作丝绵 |
| 缫丝水温 | 70-80摄氏度，丝头易找，丝线不断成团 |
| 合股 | 3-8 根茧丝合成一根生丝，粗细均匀，不忽粗忽细 |
| 织造 | 经线张力一致，布面无连续跳线、断经和明显油污 |

## 失败征兆

- 蚕房潮湿、有霉味或大量死蚕，说明通风和清洁失败。
- 水温过高，茧层散乱成絮；水温过低，丝胶不软、找不到丝头。
- 缫丝时频繁断丝，可能是茧质差、合股太少或卷绕过猛。
- 经线松紧不一，织出的绢面会起浪。
- 为追求产量把病蚕茧混入，会污染整批原料。

## 丝与普通纤维的对比

| 特性 | 丝 | 棉 | 麻 |
|------|-----|-----|-----|
| 纤维长度 | 连续（~1000米） | 2-5厘米 | 10-30厘米 |
| 细度 | 极细（10-15μm） | 15-20μm | 20-30μm |
| 强力 | 高 | 中 | 很高 |
| 光泽 | 天然光泽 | 哑光 | 哑光 |
| 手感 | 润滑 | 柔软 | 粗硬 |

## 常见问题

- **Q**: 蚕不吃桑叶行不行？
- **A**: 蚕只吃桑叶（或柘叶）。没有桑树的地方无法养蚕产丝。

- **Q**: 缫丝时丝总断怎么办？
- **A**: 水温过高或过低。70-80摄氏度最佳——太热丝胶全溶，丝散开；太冷丝胶不溶，丝不易分离。

- **Q**: 一个茧能抽出多长的丝？
- **A**: 好的家蚕茧可抽出 800-1500米的连续丝。6-8 个茧合成一根生丝。

## 安全提醒

- 缫丝热水注意烫伤
- 养蚕环境需保持卫生，防病菌
- 蚕有气味（桑叶+蚕沙），注意室内通风

## 穿越小说梗

丝织是古代中国最赚钱的产业，丝绸沿着丝绸之路从中国运到罗马，价格与黄金等价。穿越者想搞丝绸需要注意：①你需要桑树（北方没有）②养蚕到织出第一匹丝绸至少需要 2-3 个月③缫丝技术是关键——不会缫丝，满屋的茧子就是废品。但穿越者的优势在于：知道丝绸的**国际贸易价值**，可以更好地策划"丝绸之路"商业路线。
