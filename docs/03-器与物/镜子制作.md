# 3.32 镜子制作

> 在古代，一面清晰的镜子比黄金还珍贵——它是科技树的明显标志。

## 原理

镜子依赖**光的镜面反射**。在平整的透明玻璃背面镀上反光层，光线从玻璃穿过、在金属层上反射、再穿过玻璃回到人眼。

### 两种方法对比

| 方法 | 原理 | 难度 | 效果 |
|------|------|------|------|
| 锡汞法（传统） | 锡箔 + 汞形成锡汞齐镀层 | ⭐⭐⭐ | 明亮但有毒 |
| 银镜反应（化学） | 硝酸银还原为银镜 | ⭐⭐⭐⭐ | 非常明亮 |

## 所需材料

### 锡汞法

| 材料 | 用途 |
|------|------|
| 平板玻璃 | 基底（需打磨平整） |
| 锡箔 | 镀层金属 |
| 汞（水银） | 形成汞齐 |
| 重物（石板） | 加压 |

### 银镜反应

| 材料 | 用途 |
|------|------|
| 平板玻璃 | 基底 |
| 硝酸银 | 银源 |
| 氨水 | 络合剂 |
| 葡萄糖/甲醛 | 还原剂 |
| 氢氧化钠 | 碱性环境 |
| 蒸馏水 | 清洗 |

## 操作步骤

### 锡汞法（传统 16 世纪方法）

1. **玻璃准备**：将玻璃打磨平整，彻底清洁
2. **铺锡箔**：在平整石板上铺一层锡箔
3. **倒汞**：在锡箔上倒一层薄薄的汞
4. **覆玻璃**：小心将玻璃放在上面，让锡汞齐粘附
5. **压重**：放重物静置数小时
6. **倾斜**：将玻璃倾斜，让多余汞流出
7. **干燥**：通风处晾 1-2 周，汞齐完全固化

### 银镜反应（19 世纪工艺）

1. **清洗玻璃**：彻底去除油污（可用碱液+石灰粉打磨）
2. **配银氨溶液**：硝酸银 + 氨水，滴至沉淀刚好溶解
3. **加还原剂**：加入葡萄糖或甲醛
4. **镀银**：将混合液倒在玻璃上，摇晃均匀
5. **反应**：10-30 分钟，银层析出附着在玻璃上
6. **清洗**：蒸馏水冲洗
7. **保护**：涂一层清漆或油漆保护银层

## 常见问题

- **Q**: 古代没有硝酸银怎么做镜子？
- **A**: 锡汞法是古代（16-19 世纪）的标准工艺。银镜反应需要硝酸，建议先走锡汞路线。

- **Q**: 锡汞法的镜子能用多久？
- **A**: 维护得好可以用几十年。但汞齐会随时间老化变得模糊。

- **Q**: 穿越后没有汞怎么办？
- **A**: 汞在自然界以辰砂（朱砂，HgS）存在，可通过焙烧提取。但汞极毒，没有经验不建议尝试。

## 安全提醒

- **汞蒸气极度危险**，长期吸入会致神经损伤（水俣病）
- 锡汞法必须在通风极好的室外操作
- 银镜反应中的硝酸银有腐蚀性
- 古代的镜子作坊寿命都很短——工匠因汞中毒很少长寿

## 穿越小说梗

小说里"主角造出玻璃镜轰动京城"——历史事实是：威尼斯在 16 世纪就是靠镜子垄断发了大财，一面镜子值一座宫殿。但前提是你有**平板玻璃**（比吹制玻璃难得多）和**汞**（极毒且难以获取）。可行的穿越路线：先做小圆凸镜（首饰用），不做大穿衣镜。
