# 3.32 镜子制作

> 在古代，一面清晰的镜子比黄金还珍贵——它是科技树的明显标志。

## 原理

镜子依赖**光的镜面反射**。在平整的透明玻璃背面镀上反光层，光线从玻璃穿过、在金属层上反射、再穿过玻璃回到人眼。

### 两种方法对比

| 方法 | 原理 | 难度 | 效果 |
|------|------|------|------|
| 锡汞法（传统） | 锡箔 + 汞形成锡汞齐镀层 | ⭐⭐⭐ | 明亮但有毒 |
| 银镜反应（化学） | 硝酸银还原为银镜 | ⭐⭐⭐⭐ | 非常明亮 |

## 所需材料

### 锡汞法

| 材料 | 用途 |
|------|------|
| 平板玻璃 | 基底（需打磨平整） |
| 锡箔 | 镀层金属 |
| 汞（水银） | 形成汞齐 |
| 重物（石板） | 加压 |

### 银镜反应

| 材料 | 用途 |
|------|------|
| 平板玻璃 | 基底 |
| 硝酸银 | 银源 |
| 氨水 | 络合剂 |
| 葡萄糖/甲醛 | 还原剂 |
| 氢氧化钠 | 碱性环境 |
| 蒸馏水 | 清洗 |

## 现实安全边界

锡汞法和银镜反应都不适合作为无实验条件下的手工教程。汞蒸气会造成神经系统损伤；银氨溶液若残留干燥，可能生成危险爆炸性银化合物。穿越小说可以写“制镜作坊危险、利润高、工匠寿命短”，现实中不应照做。

## 低风险替代路线

| 路线 | 做法 | 成功标准 | 局限 |
|------|------|----------|------|
| 抛光铜镜 | 铜/青铜片反复打磨抛光 | 30厘米外能看清五官轮廓 | 色偏黄，需常抛光 |
| 黑曜石镜 | 黑曜石片磨平抛光 | 反光均匀，无深划痕 | 材料稀缺，易碎 |
| 水镜 | 黑碗盛清水 | 静水中能照见轮廓 | 只能临时使用 |
| 小玻璃背银 | 只作现代实验室演示 | 银层均匀附着 | 需要化学品和废液处理 |

## 成功标准

- 镜面平整，近距离照人不明显扭曲。
- 反光层均匀，无大片黑斑、流痕和脱落。
- 若走抛光金属路线，表面细划痕逐级减少，最终能看清眉眼。
- 制作过程没有汞、砷、强酸强碱和不明化学废液暴露。

## 失败征兆

- 镜面发雾、发黑、边缘剥落：镀层或保护层失败。
- 玻璃不平导致人脸拉伸变形：基底未磨平。
- 有金属汞珠、银氨残液、刺鼻气味或皮肤黑斑：立即停工隔离。
- 为追求亮度而在室内加热汞或强酸处理金属：越界危险操作。

## 常见问题

- **Q**: 古代没有硝酸银怎么做镜子？
- **A**: 更安全的路线是抛光铜镜或小型黑曜石镜。锡汞法历史上确实存在，但汞毒性极高，不建议复原。

- **Q**: 锡汞法的镜子能用多久？
- **A**: 维护得好可以用几十年。但汞齐会随时间老化变得模糊。

- **Q**: 穿越后没有汞怎么办？
- **A**: 不要为了镜子提取汞。抛光铜镜、银器抛光、水镜和小尺寸玻璃反光器更符合安全边界。

## 安全提醒

- **汞蒸气极度危险**，长期吸入会致神经损伤（水俣病）
- 锡汞法必须在通风极好的室外操作
- 银镜反应中的硝酸银有腐蚀性
- 古代的镜子作坊寿命都很短——工匠因汞中毒很少长寿

## 穿越小说梗

小说里"主角造出玻璃镜轰动京城"——历史事实是：威尼斯在 16 世纪就是靠镜子垄断发了大财，一面镜子值一座宫殿。但前提是你有**平板玻璃**（比吹制玻璃难得多）和**汞**（极毒且难以获取）。可行的穿越路线：先做小圆凸镜（首饰用），不做大穿衣镜。
