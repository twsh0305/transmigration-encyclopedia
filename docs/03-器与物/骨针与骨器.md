# 3.34 骨针与骨器

> 骨针是人类第一件精密工具。有了它，才有了衣服、帐篷和"文明"。

## 原理

骨器制作的本质是**硬组织加工**：

1. 动物骨骼由羟基磷灰石（无机）和胶原蛋白（有机）组成
2. 骨头有纵向纤维结构，沿纤维方向劈裂容易
3. 通过切割、打磨、钻孔等减材制造法成形

### 适合制器的骨骼

| 动物 | 最佳部位 | 用途 |
|------|---------|------|
| 鹿 | 胫骨、肩胛骨 | 针、刀、铲 |
| 牛/羊 | 跖骨（管状骨） | 针、锥、箭镞 |
| 猪 | 獠牙 | 刻刀、锥子 |
| 大型鸟 | 翅骨 | 骨笛、针管 |
| 鱼 | 大鱼刺 | 鱼钩、小针 |

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 动物骨骼 | 原料 |
| 硬石片（燧石/石英岩） | 切割和刮削 |
| 砂岩 | 打磨 |
| 尖石/尖锐骨头 | 钻孔 |
| 细砂+水 | 精磨 |
| 动物油 | 润滑和防腐 |

## 操作步骤

### 骨针制作

1. **取材**：取小动物（鹿/羊）的管状骨，敲碎取骨片
2. **劈料**：沿骨纤维方向用石片劈出细长条（约 2-3mm 粗）
3. **粗磨**：砂岩上将骨条磨成针形（一端尖、一端钝）
4. **钻孔**（最需耐心）：
   - 用尖石或骨尖在钝端钻凹坑
   - 快速来回旋转，加细砂和水（砂钻原理）
   - 钻透后用细骨棒扩孔
5. **精磨**：用细砂岩加水打磨至光滑
6. **抛光**：用皮革+细灰反复擦拭

### 骨刀/骨刮

1. **取材**：大型动物（牛/鹿）的肩胛骨或肋骨
2. **劈形**：沿骨纤维方向劈出刀形
3. **磨刃**：在砂岩上磨出刃口，角度 20-30°
4. **骨刀的局限**：不如燧石锋利，但比燧石耐用

### 骨锥（最实用的骨器）

1. **取材**：取管状骨中段
2. **一端磨尖**：如同削铅笔
3. **打磨光滑**：用于钻孔、缝皮、编篮
4. 这是骨器中制作最快速、用途最广的

## 骨器 vs 石器

| 比较 | 骨器 | 石器 |
|------|------|------|
| 锋利度 | 中等 | 极高（细石器） |
| 韧性 | 好（不易碎） | 脆（易碎） |
| 制作速度 | 快 | 慢 |
| 耐久性 | 一般（会老化） | 极好 |
| 易得性 | 易（狩猎就有） | 需要寻找合适石材 |

## 常见问题

- **Q**: 钻孔时骨头断了怎么办？
- **A**: 骨孔是骨针制作的最大难点。解决方案：①充分泡水软化②先用细石尖钻小凹坑，再逐步加深③工具保持湿润。

- **Q**: 新骨头能直接用吗？
- **A**: 新鲜骨头含油脂丰富，不易钻透且易臭。建议先**煮过**去除油脂，干燥后再加工。

- **Q**: 骨针能用多久？
- **A**: 合理使用可数月，但会逐渐磨损变钝。定期在细石上修磨可延长寿命。

## 安全提醒

- 碎骨飞溅可能伤眼，操作时注意
- 骨粉吸入有害，磨骨时保持通风
- 新鲜骨头可能携带病菌，建议煮沸消毒后再加工

## 穿越小说梗

"主角连骨针都做不出来就想造发电机"——骨针是技术树的起点。人类花了几万年才从打制石器进化到磨制骨针。穿越者也一样：做骨针是**精密加工能力的第一课**——如果你连骨针上的小孔都钻不透，就不要想精密齿轮了。骨针虽小，却是"制造工具的工具"的**元工具**。
