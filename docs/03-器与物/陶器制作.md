# 3.33 陶器制作

> 泥土经过火，就不怕水了——陶器是人类造出的第一种"新材料"。

## 原理

陶器制作的本质是**黏土烧结**：

1. 黏土矿物遇水具有可塑性，可成型
2. 干燥脱去物理水后保持形状
3. 高温烧制（800-1000°C）使黏土颗粒部分熔融，冷却后形成坚硬的陶瓷结合

黏土的主要成分是**高岭石**（Al₂O₃·2SiO₂·2H₂O），烧制过程中结晶水失去，形成莫来石等耐水矿物。

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 黏土 | 原料 | 河床/山坡挖掘，含砂量适中的最宜 |
| 水 | 塑形 | 适量 |
| 沙/熟料粉 | 调节收缩 | 减少干燥和烧制开裂 |
| 草灰/贝壳粉 | 掺和料 | 改善耐热性 |
| 窑/火堆 | 烧制 | 地坑窑最简单 |

## 操作步骤

### 手捏法（最简单的陶器）

1. **选土**：挖取黏土，去除石子、草根
2. **练泥**：加水揉至均匀，如耳垂般柔软
3. **成型**：直接用手捏成碗/罐形状
4. **修整**：用湿手或刮片修整表面
5. **阴干**：避风避光，慢干 3-7 天
6. **烧制**：800-1000°C 烧制数小时

### 泥条盘筑法（制作大件）

1. **搓泥条**：将泥搓成手指粗的均匀长条
2. **盘底**：先做一个平底泥饼
3. **盘壁**：将泥条沿底边盘绕而上，每圈用泥浆粘合
4. **内壁抹平**：用湿手或刮板将内壁抹平
5. **外壁修整**：刮平外部，加装饰纹
6. **阴干 → 烧制**（同手捏法）

### 露天烧制（无窑方案）

1. **干燥**：陶坯完全阴干（至少一周）
2. **堆柴**：在地上铺一层干柴
3. **放陶坯**：将陶坯放在柴堆上
4. **覆盖**：用干草+泥巴将陶坯和柴一起覆盖
5. **点火**：从底部点燃
6. **维持**：不断加柴，烧 3-6 小时
7. **冷却**：自然冷却（过夜），不要碰触

## 常见问题

- **Q**: 为什么烧出来的陶器一碰就碎？
- **A**: 温度不够（露天烧制很难超过 700°C）。挖个窑能把温度提高 200°C 以上。

- **Q**: 陶器漏水怎么办？
- **A**: 烧制前在表面涂抹细泥浆并磨光（"磨光陶"）。另一种方法：烧成后用稀米汤浸泡填补微孔。

- **Q**: 黏土去哪里找？
- **A**: 河岸断面、稻田下层、山坡冲积层。标准测试法：搓泥条绕手指一圈不断裂即可。

## 安全提醒

- 窑炉附近准备大量水/沙土灭火
- 烧窑时注意防止一氧化碳中毒（通风）
- 刚出窑的陶器表面极其脆弱，冷却前不要触碰

## 穿越小说梗

"主角在古代开了个陶器厂"——这是穿越者最早能掌握的手工业之一，古代已经极其成熟。除非你能突破**瓷器**（1200°C+ 的技术壁垒），否则普通陶器很难赚钱。但陶器**很有用**：做缸腌菜、做罐储粮、做壶酿酒，先解决温饱再谈致富。
