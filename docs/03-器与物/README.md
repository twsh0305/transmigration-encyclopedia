# 第三卷·器与物

> 没有工业，就用手造。陶器、玻璃、纸、皂、皮革、染料——生活需要的一切，从零开始。

## 使用顺序

1. 先做低温低风险器物：篮、绳、木工、石器、骨器和量具。
2. 再做陶器和造纸：能稳定成形、干燥、烧成或压干后，再扩大批量。
3. 接着做皂、染料、皮革：涉及碱液、腐败和染媒，必须分区记录。
4. 最后碰玻璃、窑炉和镜面：高温和碎裂风险高，只在场地合格后尝试。
