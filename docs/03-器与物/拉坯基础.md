# 3.8 拉坯基础

> 陶轮是人类最早使用的机床——旋转赋予了对称与效率。

## 原理

拉坯利用**旋转对称**原理：

1. 陶轮快速旋转，泥团在离心力和双手约束下成形
2. 手的形状决定了器物的内外轮廓
3. 湿润的黏土在旋转中被拉伸、收拢、开敞

相比手捏法和泥条盘筑法，拉坯的效率高 10 倍以上，且器壁均匀、形状规整。

## 所需材料

| 材料/工具 | 用途 | 说明 |
|-----------|------|------|
| 黏土 | 原料 | 练泥充分，无气泡 |
| 陶轮 | 旋转平台 | 见下文 | 
| 水盆 | 润滑 | 保持手和泥土湿润 |
| 刮片（木/竹/骨） | 修整 | 不同形状各备一个 |
| 线（铁丝/麻线） | 切底 | 从陶轮上切下成品 |
| 海绵/布 | 吸水 | 调整含水率 |

## 陶轮制作

### 手动慢轮（最简单的起步）

1. 找一个平的圆木板或石盘（直径 30-40cm）
2. 在底部中心装一个轴（木棍或铁棍）
3. 在地上挖一个浅坑，底部放一块凹石作轴承
4. 将轴插入轴承，手动旋转

### 快轮（脚踢式）

1. 做一个大直径木轮（直径 60-80cm），中心装轴
2. 轮盘顶部装一个小工作台面
3. 用脚踢动大轮，利用惯性旋转
4. 可加配重增加转动惯量

## 操作步骤

1. **练泥**：将黏土揉成均匀紧实的泥团（排除所有气泡）
2. **定中心（最关键）**：
   - 将泥团用力摔在陶轮中心
   - 双手沾水，抱住泥团
   - 旋转中双手向内挤压，直到泥团不再跳动
3. **开洞**：拇指从顶部中心垂直压下
4. **拉高**：
   - 双手内外夹住泥壁，由下向上提
   - 保持手稳定，感受泥的厚度
5. **塑形**：
   - 向外扩 → 碗/盘
   - 向内收 → 罐/瓶
   - 直上直下 → 杯/筒
6. **收口**：修饰口沿形状
7. **切底**：用线从底部将成品与陶轮分离
8. **阴干 → 修坯 → 烧制**

## 常见问题

- **Q**: 为什么泥团总是跑偏？
- **A**: 中心没定好。这是拉坯最难的一步，新手需数小时才能掌握。确保①陶轮平稳②泥团在正中心③手肘靠住身体固定。

- **Q**: 拉坯过程中坯体塌了？
- **A**: ①泥太软②底部太薄③转速太快。停下来重新开始。

- **Q**: 壁厚不均匀怎么办？
- **A**: 用竹片卡在内外壁之间作为厚度规。

## 安全提醒

- 陶轮旋转的泥浆飞溅，注意眼睛防护
- 练泥时注意排除气泡（烧制时气泡会炸裂）
- 长时间拉坯注意手腕保护

## 穿越小说梗

拉坯是古代陶器作坊的**核心技术**。穿越者如果能做出快轮，就比同时代的手工作坊领先一步。但拉坯需要大量练习——手艺人的"手感和肌肉记忆"至少需要数月训练。小说里"主角第一次拉坯就做出了完美花瓶"——那是开了挂。
