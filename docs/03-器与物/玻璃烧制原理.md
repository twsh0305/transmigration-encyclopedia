# 3.17 玻璃烧制原理

> 沙子烧到 1700°C 会变成透明的液体——这就是玻璃诞生的瞬间。

## 原理

玻璃不是晶体，它是一种**非晶态固体**（过冷液体）：

- 主要成分：**二氧化硅（SiO₂）**——沙子的主要成分
- 纯石英砂熔点约 1700°C，古代窑炉达不到这个温度
- 加入**助熔剂**（碱）可大幅降低熔点至 1000-1200°C
- **稳定剂**（石灰）防止玻璃溶于水

### 基本配方

| 成分 | 作用 | 比例 | 来源 |
|------|------|------|------|
| 石英砂 | 玻璃的主体（SiO₂） | 60-70% | 河沙/海沙 |
| 纯碱/草木灰 | 助熔剂（降低熔点） | 15-25% | 草木灰碱液 |
| 石灰石 | 稳定剂（防溶水） | 5-10% | 石灰石烧制 |
| 碎玻璃（可选） | 加速熔化 | 10-20% | 回收旧玻璃 |

**穿越者核心公式**：
```
沙子 + 草木灰 + 石灰石 + 高温（1000°C+）= 玻璃
```

## 所需材料

| 材料 | 用途 | 最佳来源 |
|------|------|---------|
| 石英砂 | 主料 | 河沙（白色/无色，不含铁杂质） |
| 纯碱 | 助熔剂 | 草木灰碱液提取+蒸发结晶 |
| 石灰石 | 稳定剂 | 烧石灰的原料 |
| 耐火粘土 | 坩埚 | 高岭土/耐火泥 |
| 窑炉 | 熔化 | 土窑改建，需达 1100°C+ |

### 选沙关键

不是所有沙子都能做玻璃——铁杂质会使玻璃发绿：
- ✅ 白色/无色河沙 → 透明玻璃
- ✅ 石英砂（SiO₂ > 95%）→ 最佳
- ❌ 黄色/红色沙子（含铁高）→ 绿色玻璃
- ❌ 含贝壳/石灰石块的沙 → 需先去除

**去铁法**：沙子用磁铁反复吸走含铁矿物，或酸洗。

## 坩埚制作

熔化玻璃需要耐 1200°C+ 的容器：

1. 取高岭土或耐火粘土（不含铁杂质的粘土）
2. 加少量石英砂和熟料（碎旧坩埚/碎砖）
3. 加水揉成团
4. 捏成厚壁坩埚（壁厚 2-3cm）
5. 阴干 3-5 天
6. 先用低温（500°C）素烧，再升温至 1000°C 烧结

## 窑炉要求

普通陶窑（800-900°C）不够烧玻璃，需要：
- **高温区**：至少 1000-1200°C
- **保温**：能维持高温 8-12 小时
- **鼓风**：用风箱/皮囊加大进氧量，提高温度

**穿越版方案**：
- 在陶窑基础上改造，加高烟囱增加抽力
- 使用风箱强制鼓风
- 燃料选用木炭（比木柴温度高）或焦炭

## 操作步骤

1. **配原料**：石英砂 60% + 纯碱 20% + 石灰石 10% + 碎玻璃 10%
2. **研磨**：所有原料研细成粉，混合均匀
3. **入坩埚**：装入耐火坩埚，不超过 2/3 满
4. **加热**：窑炉升温至 1100-1200°C，保温 6-12 小时
5. **观察**：原料熔化成黏稠透明的液体，无气泡时即可
6. **降温**：降至 800-900°C（适合吹制/成型的黏度）
7. **成型**：吹制、模铸或拉制
8. **退火**：成型后的玻璃放入 400-500°C 退火炉缓慢冷却（防止内应力导致开裂）

### 退火的重要性

玻璃冷却不均匀会产生内应力，轻轻一碰就裂。**退火** = 将玻璃放入保温炉中缓慢降温（数小时甚至数天）。没有退火炉，也可以埋在热灰中自然冷却。

## 有色玻璃配方

| 颜色 | 添加物 | 用量 |
|------|--------|------|
| 绿色 | 铁氧化物（Fe₂O₃） | 1-3% |
| 蓝色 | 氧化钴（CoO） | 0.1-0.5% |
| 琥珀色 | 硫+碳 | 微量 |
| 紫色 | 二氧化锰（MnO₂） | 2-4% |
| 红色 | 氧化铜（Cu₂O） | 1-3% |
| 乳白 | 骨灰/磷灰石 | 3-5% |

## 常见问题

- **Q**: 玻璃熔化后有大量气泡？
- **A**: 温度不够高或保温时间不够。气泡 = 原料分解产生的气体未排出。继续加热或提高温度。

- **Q**: 玻璃做出来是绿色的？
- **A**: 沙子中的铁杂质。换更纯的沙或添加少量二氧化锰（脱色剂）。

- **Q**: 玻璃一冷却就自己裂了？
- **A**: 退火不够。缓慢冷却是防止玻璃开裂的关键。

## 安全提醒

- 熔化玻璃温度极高（> 1000°C），严防烫伤
- 熔融玻璃沾到皮肤会粘连并深度烧伤
- 湿的工具接触熔融玻璃会导致剧烈飞溅（水分瞬间气化）
- 玻璃粉尘吸入有害，研磨时戴口罩

## 穿越小说梗

"穿越后烧玻璃暴富"——这是小说里最常见的套路。但真实情况是：①需要 1100°C+ 的高温，普通陶窑根本达不到 ②需要耐火坩埚，普通陶锅在 1000°C 就软了 ③退火是关键，没有退火炉玻璃一碰就碎 ④草木灰纯碱的纯度不如工业纯碱，玻璃品质会差一些。所以"穿越烧玻璃"不是不可能，但需要先升级窑炉——先学会改进陶窑、造风箱，再考虑烧玻璃的事。
