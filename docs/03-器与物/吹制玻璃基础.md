# 3.3 吹制玻璃基础

> 玻璃是液体——只是流动得极慢。吹玻璃就是用呼吸为这团液态混沌赋予形状。

## 原理

吹制玻璃利用**黏弹性流体成形**原理：

1. 玻璃在高温（约 1000°C）下呈黏稠可塑状态
2. 通过吹管向玻璃料泡内吹气，内部气压使玻璃膨胀
3. 借助重力和离心力控制形状
4. 冷却后保持形状

吹制是玻璃工艺中效率最高的成形方法，适合制造中空器皿（瓶、杯、壶）。

## 所需材料

| 材料/工具 | 用途 | 说明 |
|-----------|------|------|
| 吹管 | 吹气+挑料 | 铁管，长约 1.2m，一端有嘴 |
| 熔融玻璃料 | 原料 | 窑炉中 1000-1100°C 熔化的玻璃 |
| 石墨/木制模具 | 塑形 | 碗状、瓶状内模 |
| 剪刀（铁制） | 剪切玻璃料 | 长柄铁剪 |
| 沾水报纸/湿布 | 整形 | 接触处不黏连 |
| 退火炉 | 缓慢冷却 | 隔绝冷空气，防炸裂 |

## 操作步骤

1. **挑料**：将吹管前端伸入熔炉，旋转沾取适量玻璃料（约鸡蛋大小）
2. **滚圆**：在铁板或石板上滚动料泡，使其均匀
3. **吹小泡**：对着吹管轻轻吹气，形成一个小气泡
4. **塑形**：
   - 用湿报纸/石墨工具整形
   - 利用重力让玻璃自然下垂
   - 旋转吹管保持对称
5. **吹大**：继续吹气至所需大小
6. **开口（若做杯子/碗）**：用铁棒夹住底部，在吹管根部划水冷却使之断开
7. **退火**：将成品放入退火炉，12-24 小时缓慢冷却至室温

## 常见问题

- **Q**: 为什么玻璃一吹就破？
- **A**: 温度太低（玻璃不够黏稠）、或者玻璃中有气泡杂质。回炉重新加热。

- **Q**: 没有铁吹管怎么办？
- **A**: 可用陶管代替，但隔热性差，需用湿布包住手持端。不锈钢管也是好选择。

- **Q**: 退火炉必须要有吗？
- **A**: 不做退火，薄壁玻璃制品空气中冷却就会自行炸裂。退火炉的替代方案：埋在石灰粉或砂子中缓慢冷却。

## 安全提醒

- 熔融玻璃温度超过 1000°C，**绝对不能接触皮肤**
- 吹玻璃时注意水汽（吹管口有水会导致玻璃飞溅）
- 玻璃碎裂飞散很危险，操作时佩戴护目镜

## 穿越小说梗

"吹个玻璃瓶就发了财"——技术上可操作，但难点在：①需要持续保持 1000°C 的窑炉（燃料消耗惊人）②吹制技巧需要大量练习③古代已有成熟的陶瓷工业，玻璃并非必需品。最合理的路线是先做玻璃珠和小件（不需吹制），积累了资本再升级到吹制器皿。
