# 3.31 釉料配制原理

> 釉是玻璃化的泥土——它让陶器不再漏水，让瓷器光洁如玉。

## 原理

釉的本质就是**一层薄玻璃**：

1. 将矿物原料磨成细粉，加水调成浆
2. 涂在陶坯表面
3. 高温下熔融成液态玻璃
4. 冷却后形成光滑坚硬的玻璃质表层

### 釉的三大成分

| 成分 | 作用 | 常见原料 |
|------|------|---------|
| 网络形成剂 | 形成玻璃骨架 | 石英（SiO₂） |
| 助熔剂 | 降低熔化温度 | 草木灰、长石、石灰石、铅丹 |
| 稳定剂 | 增加化学稳定性 | 黏土、长石（Al₂O₃） |

## 所需材料

| 原料 | 作用 | 备注 |
|------|------|------|
| 石英/河沙（细粉） | 玻璃基质 | 主要原料，>50% |
| 长石 | 助熔+稳定 | 自然界广泛存在 |
| 石灰石/白云石 | 助熔 | 降低烧成温度 |
| 草木灰 | 助熔 | 不同植物灰成分各异（传统方法） |
| 黏土 | 悬浮剂+稳定 | 帮助釉浆附着 |
| 金属氧化物 | 着色 | 见下文 |

### 常见釉色着色剂

| 颜色 | 添加物 | 备注 |
|------|--------|------|
| 青/绿 | 铜（CuO/CuCO₃） | 还原焰呈青，氧化焰呈绿 |
| 铁红/褐 | 铁（Fe₂O₃） | 氧化焰呈褐/红/黄 |
| 青瓷色 | 铁（FeO） | 还原焰呈青绿 |
| 钴蓝 | 钴（CoO） | 极少量即可，很贵 |
| 锰紫/褐 | 锰（MnO₂） | 紫色到褐色 |
| 乳白 | 锡（SnO₂） | 使釉不透明 |

## 基础釉配方

### 草木灰釉（最简单的起步）

| 原料 | 比例 |
|------|------|
| 草木灰（过细筛） | 1 份 |
| 黏土 | 1 份 |
| 石英砂 | 2 份 |

### 石灰釉（中国古代主流）

| 原料 | 比例 |
|------|------|
| 石灰石（煅烧后） | 10-15% |
| 草木灰 | 10% |
| 石英 | 50% |
| 黏土 | 25-30% |

## 操作步骤

1. **原料粉碎**：将所有原料分别研磨成细粉（过 120 目筛）
2. **按比例混合**：干粉混合均匀
3. **调浆**：加水调成稀奶油状
4. **施釉**：浸釉（直接浸入浆中）、浇釉（浇在坯上）或刷釉
5. **干燥**：让釉层自然干燥
6. **入窑烧制**：按釉料要求温度烧成

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 粉碎 | 原料过 120 目筛，手捻无明显砂粒 |
| 记录 | 每次试釉写明比例、坯体、施釉厚度、窑位和烧成温度 |
| 釉浆 | 稀奶油状，搅拌后 30 秒内不明显分层 |
| 施釉 | 釉层均匀，厚薄约 0.5-1毫米，足圈擦净不上釉 |
| 试片 | 先烧小试片，确认不流釉、不针孔、不铅溶出风险再做器物 |

## 失败征兆

- 釉浆沉底结块，说明粉太粗或悬浮性差。
- 釉层太厚，烧后流釉粘窑板。
- 釉面针孔密集，可能升温太快、坯体排气不足或釉过熟。
- 龟裂贯穿且用于盛液，容易渗漏和藏污。
- 铅釉用于餐具，是不可接受的食品安全风险。

## 常见问题

- **Q**: 为什么釉面有针孔？
- **A**: 烧成温度太高或升温太快，釉中的气体来不及排出。尝试降低最高温度或延长保温时间。

- **Q**: 釉面龟裂怎么办？
- **A**: 釉的膨胀系数与坯体不匹配。增加石英含量可减少裂釉。

- **Q**: 釉不熔化/表面粗糙？
- **A**: 助熔剂不够或温度不足。增加草木灰或石灰石比例。

## 安全提醒

- **铅釉有毒**，不要做餐具！虽然铅丹（Pb₃O₄）是很好的助熔剂，但铅会溶出，绝对不要用在食器上
- 釉料粉末吸入有害，操作时戴口罩
- 配釉时做好记录——一次成功的配方就是宝贵资产

## 穿越小说梗

"主角配出了秘色釉"--釉料配方在历史上是家族最高的机密。穿越者最大的优势是知道**配方的科学原理**：助熔剂降低熔点、着色剂来自金属氧化物、石灰釉→石灰碱釉的历史演进路线。知道了方向，试验周期可以从几代人缩短到几个月。
