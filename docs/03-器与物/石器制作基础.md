# 3.21 石器制作基础

> 在金属出现之前，石头就是人类最强大的武器——也是最后的选择。

## 原理

石器的核心是**脆性材料的可控断裂**：

1. 特定岩石（燧石、黑曜石、石英岩）具有**贝壳状断口**
2. 用锤击或压制使岩石沿应力集中方向裂开
3. 控制打击点和角度，即可劈出锋利的刃口

### 适合制器的岩石

| 岩石 | 特点 | 适用 |
|------|------|------|
| 燧石/火石 | 硬度高、断口锋利 | 刀、箭头、刮削器 |
| 黑曜石（火山玻璃） | 刀刃可达原子级锋利 | 手术刀、箭头 |
| 石英岩 | 硬度极高但难加工 | 重砍砸器 |
| 砂岩 | 可做磨石 | 磨制工具 |
| 页岩 | 层理明显 | 磨制石斧 |

## 所需材料

| 材料/工具 | 用途 | 
|-----------|------|
| 石料（燧石/黑曜石等） | 原料 |
| 锤石（圆卵石） | 硬锤——大片剥片 |
| 鹿角/硬木 | 软锤——精细剥片 |
| 鹿角尖/骨尖 | 压剥——最终修整 |
| 皮垫/布垫 | 包裹石料防飞溅 |
| 护目镜 | 防石片飞溅 |

## 操作步骤

### 一、选材

1. 寻找河边或山麓的燧石/石英岩/黑曜石
2. **敲击测试**：两块石料相敲，声音清脆 = 好料；闷哑 = 有裂隙
3. 看断口：新鲜的贝壳状断口是好石料的标志

### 二、直接锤击法（快速出形）

1. 将石料放在腿上（垫皮革）
2. 用锤石（硬圆卵石）沿石料边缘 45° 角敲击
3. 控制敲击点——距边缘 1-2cm，角度精准
4. 每次敲下一片石片（石片→可用作刮刀）

### 三、压制法（精细修整）

1. 用鹿角尖或骨尖对准石料边缘（距刃口 2-3mm）
2. 用力向下前方压推
3. 压下一片小而薄的石片
4. 重复——沿刃口逐片修整，形成规整的刃缘

### 四、制作石刀

1. 选一块巴掌大的燧石
2. 用锤击法打出大形（一面较平、一面凸起）
3. 用压制法沿长边修出锋刃
4. 用鹿角轻敲修整刃口——最终刃口厚度应小于 1mm

### 五、装柄

1. 用骨锥在木柄上钻一个孔或开槽
2. 将石刀插入，用树胶/松脂固定
3. 再用细绳/皮条捆绑加固

## 黑曜石的特别说明

黑曜石刃口**可达纳米级**（比手术刀锋利），是穿越者能获得的最锋利刃器。但缺点是极脆，摔落即碎。适合做精细切割（手术、精细雕刻），不适合重砍。

## 常见问题

- **Q**: 为什么我敲的石头总是碎成渣？
- **A**: ①石料内部有裂隙（换一块）②打击角度太大或太小（45° 最佳）③锤石太软。多练。

- **Q**: 没有燧石用什么代替？
- **A**: 玻璃（瓶底、厚玻璃片）可代替黑曜石——破碎后形成的刃口同样锋利。但玻璃更脆。

- **Q**: 石器多久会钝？
- **A**: 用几分钟就会变钝。但重新修磨只需几秒钟——沿刃口再压剥一次即可。

## 安全提醒

- 石片飞溅极其锋利，必须戴护目镜
- 黑曜石刃比手术刀还锋利，绝对不要用手试刃
- 石粉吸入有害，操作时通风

## 穿越小说梗

小说里主角"用石刀轻松砍树"——不存在的。石刀虽然能锋利到刮胡子，但**做不了重活**。砍树需要石斧，劈肉需要石刀，剥皮需要刮削器——不同石器各有用途。穿越者最合理的策略：找到了优质燧石后，先做一批石片（一次敲击可得多个锋利石片），当做一次性手术刀/刮刀用，金属工具做出后再淘汰石器。
