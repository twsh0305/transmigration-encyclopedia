# 3.4 基础皮件制作

> 一块鞣好的皮革到一件成品，差的是裁切、缝合和耐心。

## 原理

皮革制品的核心在于**物理结构**：

1. 皮革的强度主要来自胶原纤维的编织方向（沿背脊线最强）
2. 缝合点往往是薄弱环节，需要合理设计间距和线材
3. 皮革有延展性，合理设计尺寸可贴合使用

### 皮料选择

| 用途 | 推荐皮料 | 厚度 |
|------|---------|------|
| 衣物/手套 | 羊皮、兔皮、鹿皮 | 0.8-1.5mm |
| 包/袋 | 小牛皮、山羊皮 | 1.5-2.5mm |
| 皮带/鞋底 | 牛皮、水牛皮 | 3-5mm |
| 刀鞘/工具套 | 植鞣牛皮 | 2-4mm |

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 鞣好的皮革 | 原料 |
| 锋利的刀（小刀/剃刀） | 裁切 |
| 锥子/尖刺（骨/铁） | 打孔 |
| 针（骨针/铁针） | 缝合 |
| 线（麻线/肠线/皮条） | 缝线 |
| 尺子/量规 | 测量 |
| 磨边棒/砂纸 | 修边 |

## 操作步骤

### 一、设计

1. 在纸或木片上画出形状
2. 考虑缝合余量（留 5-10mm 缝边）
3. 裁剪纸样 → 拼合试尺寸

### 二、裁切

1. 皮面朝上，用纸样压住
2. 在皮革**最结实的部分**（背脊附近）裁剪关键部件
3. 用锋利刀片一次划穿（不要来回锯）
4. 避免在皮面留下刀痕

### 三、打孔

1. 沿缝边标记孔位（间距 3-5mm）
2. 用锥子从皮面刺入（皮面美观）
3. 孔距均匀——这是手工作品精致度的关键

### 四、缝合

**马鞍针法（最结实的皮具缝法）**：

1. 单根线两头穿针
2. 第一针从左孔穿过，右针从右孔穿过交叉
3. 拉紧——两针在孔内交叉
4. 每针都拉紧，力度一致

**边距**：距切边 2-4mm，距边缘太近容易撕裂。

### 五、修整

1. 用刀片修去多余的毛边
2. 砂纸打磨边缘至圆滑
3. 用磨边棒蘸水打磨边缘（植鞣革可"烧边"深色）
4. 上油保养

## 制作示例：简易皮包

1. 裁切：两片矩形皮（30cm×40cm）
2. 打孔：三面边缘留 5mm 缝边，孔距 5mm
3. 缝合：马鞍针法缝三边
4. 加带：裁一条长皮条作提手
5. 修边：打磨所有边缘

## 常见问题

- **Q**: 皮边割不直怎么办？
- **A**: 用直尺和铁锥划出引导线。裁切时刀片刃角小一些（接近水平）。

- **Q**: 缝出来的线歪歪扭扭？
- **A**: ①孔位没有对齐 ②拉线力度不一致。用尺子画好打孔参考线，每针用力相同。

- **Q**: 皮革划破了能补吗？
- **A**: 小裂口从背面补一块皮补丁。大裂口只能废弃——所以裁切要谨慎。

## 安全提醒

- 裁切皮革的刀非常锋利，手指远离刀刃方向
- 锥子刺伤很深，注意控制力度
- 马鞍针法拉线时注意线缠住手指

## 穿越小说梗

小说里主角做了"精致的皮包皮鞋轰动全城"——现实中手工皮具极耗时间，一个简单的皮包需要熟练工 1-2 天。但穿越者有小优势：现代社会"手工皮具"的审美——简洁功能性设计 + 马鞍针法 + 扎实的裁切，在古代确实是有竞争力的产品。别一上来就做鞋子（需要楦头），先做袋子、皮带、刀鞘。
