# 1.23 简易编织（篮子、席子）

> 编织是人类最古老的手工技术之一——比陶器还早，比冶金更早。

## 原理

编织利用材料的柔韧性和交叉锁紧原理。经纬交叉后，每根材料都被相邻材料压住，形成稳定结构。不需要任何胶水或钉子。

本条目聚焦于不依赖竹篾/藤条的**草编/叶编**，材料更易获取。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 长草/稻草 | 干燥、长 30cm+ | 芦苇、蒲草 |
| 大片树叶 | 柔韧 | 棕榈叶、芭蕉叶 |
| 绳/线 | 辅助绑扎 | 撕成条的树皮 |

## 操作步骤

### 一、编篮子

#### 底部

1. 取 4-6 根草束作为经条，十字交叉排列
2. 用一根草束作为纬条，从中心开始绕圈编织
3. 每绕一圈，纬条从经条上方和下方交替穿过
4. 逐渐向外扩展，形成圆形底部

#### 立壁

1. 底部编到需要大小时，将经条向上弯折 90°
2. 继续用纬条绕圈编织
3. 每圈要紧贴上一圈
4. 逐渐向上编到需要的高度

#### 收边

1. 将经条末端向内弯折
2. 插入对面的编织缝中
3. 逐一处理

### 二、编席子

1. 将草/稻草排列整齐，根部对齐
2. 用两根绳子平行放在草束上，间隔 10-15cm
3. 将另一束草横向放在绳子上
4. 用绳子的两端交叉绑紧（打结）
5. 再放一束草，再绑一次
6. 重复直到所需长度
7. 最后修剪两端齐平

### 三、编草绳

1. 取 3-5 根长草
2. 一端打死结固定
3. 按三股编法编到底
4. 末端打死结

## 编织密度与用途

| 密度 | 用途 |
|------|------|
| 松编 | 透气篮筐、筛子 |
| 紧编 | 盛粮、盛水（内涂泥浆防水） |
| 超紧编 | 帽子、容器 |

## 常见问题

- **Q**: 草编的东西能装水吗？
- **A**: 草编本身漏水。但可以在内侧涂抹一层黏土，晾干后就能装水——这是古代常见的储水容器。

- **Q**: 干草太脆编不了怎么办？
- **A**: 稍微打湿再编。但不要泡透——湿草编完干了会收缩变松。

## 安全提醒

- 干草易燃，远离明火
- 草编容器不耐久，潮湿环境容易腐烂
- 存粮用草编容器要注意防虫防鼠

## 穿越小说梗

穿越后想装东西？没有塑料袋也没有纸箱——编个篮子是最实用的解决方案。材料遍地都是，方法只需要记住"上上下下"四个字。先编个粗糙的，能用就行。
