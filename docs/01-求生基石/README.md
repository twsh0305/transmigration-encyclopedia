# 第一卷·求生基石

> 穿越第一件事——活下去。水、火、庇护所、盐，缺一不可。

## 使用顺序

1. 先找水并净化：当天能喝安全水，比寻找食物更紧急。
2. 再取火：火用于煮水、保温、驱潮、照明和信号。
3. 接着搭庇护所：优先避风、避雨、离地、排水。
4. 然后补盐和基础工具：盐、绳、篮、刀具和木炭让生活开始稳定。

## 成功标准

- 水：清水滚沸 1 分钟再饮用；高海拔或低气压环境延长到 3 分钟。
- 火：火绒、引火物、细柴、粗柴四级材料齐备，雨后也能保留干火种。
- 庇护所：睡处离地或隔潮，屋顶 30 分钟小雨不滴到床位。
- 盐：能分清食盐、苦卤和可疑矿物，不把不明晶体直接入口。
- 绳具：成绳能承受日常捆扎、晾晒和拖拉，不用于吊人和高空承重。

## 失败征兆

- 只找到水源但没有净化，或只过滤泥沙就直接饮用。
- 夜间火堆靠近茅草、衣物和睡处，烟气倒灌进棚。
- 庇护所建在河滩、落石坡、枯树下或兽道上。
- 为了省时间跳过小样测试，直接做大批量盐、砖、绳或木炭。

## 安全边界

- 燃料、毒化学品、矿山污水和放射性污染水不能靠煮沸变安全。
- 封闭空间不生明火，不把炭火盆放进睡眠区。
- 野外求生先保守：不确定的植物、矿物、水源、洞穴和动物痕迹都按危险处理。
