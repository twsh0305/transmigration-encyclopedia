# 1.9 放大镜/透镜聚焦取火

> 阳光是免费的打火机，你只需要一片透镜。

## 原理

凸透镜（放大镜）将平行光线聚焦到一个点（焦点），焦点的光能密度极高，温度可达 300-500摄氏度以上，足以点燃大多数引火物。

焦点温度取决于：
- 透镜面积（越大聚光越多）
- 焦距（越短聚焦越强）
- 阳光强度

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 透镜 | 凸透镜，直径 5厘米以上 | 见下方替代方案 |
| 引火物 | 深色、干燥、细软 | 树皮纤维、干苔藓 |
| 阳光 | 晴天直射阳光 | — |

### 透镜的替代方案

| 替代物 | 说明 |
|--------|------|
| 眼镜 | 远视镜（凸透镜）可以；近视镜（凹透镜）不行 |
| 放大镜 | 最理想 |
| 望远镜/双筒镜片 | 拆下一个物镜即可 |
| 水球 | 透明气球/安全套装满水，形成球形透镜 |
| 冰透镜 | 将清澈冰块打磨成凸透镜形状（见下方） |
| 老花镜 | 可以，这就是凸透镜 |

### 冰透镜制作

1. 找一块清澈无气泡的冰
2. 用刀粗略削成圆盘形
3. 用体温（手掌）或布反复摩擦抛光两面
4. 使之一面凸一面平（平凸透镜）
5. 直径 8-10厘米，厚度 2-3厘米为宜
6. **注意**：手的热量会融化冰，动作要快

## 操作步骤

1. 准备好引火物（深色引火物更容易着火——黑色吸热最好）
2. 面朝太阳，将透镜对准引火物
3. 调整透镜与引火物的距离，使光斑最小最亮（这就是焦点）
4. 将焦点对准引火物中心
5. 保持不动，等待引火物冒烟
6. 轻轻吹气助燃
7. 引火物起火后移入柴堆

## 成功标准

- 焦点光斑小于黄豆大小，边缘清楚、亮度刺眼。
- 干燥火绒在晴天 10-30 秒内冒烟，1 分钟内能形成火星。
- 火星转入鸟巢状引火团后，轻吹 10-20 秒能起明火。
- 取火点周围 1米内没有干草堆、布料和油脂。

## 失败征兆

- 光斑大而散：焦距没找准或透镜不是凸透镜。
- 火绒只发热不冒烟：材料含水、颜色太浅或阳光不足。
- 一吹就灭：火星太小，先继续聚焦，不急着移入柴堆。
- 透镜放在太阳下无人看管：可能意外引燃。

## 常见问题

- **Q**: 多久能点着？
- **A**: 晴天用放大镜，3-10 秒即可。冰透镜较慢，可能需要 30 秒到 1 分钟。

- **Q**: 阴天能用吗？
- **A**: **不能**。必须有直射阳光。薄云偶尔可以，但极慢。

- **Q**: 近视眼镜能用吗？
- **A**: 单片不能。但可以把两片近视镜叠在一起凹面朝外，组成一个简易凸透镜——效果很差，但聊胜于无。

## 安全提醒

- 不用透镜时不要随意对准阳光——焦点可能引燃周围物品
- 甚至可能烧伤皮肤或眼睛
- 取火完成后将透镜收好或遮盖
