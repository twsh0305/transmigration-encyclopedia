# 1.17 火绒与引火物制备

> 有了火星只是第一步，没有火绒，再亮的火星也会熄灭在风中。

## 原理

引火物（tinder）是取火成功的关键中间环节。它的作用是将微小的热源（火星、余烬）放大为明火。好的引火物必须：

1. **着火点低**（300摄氏度以下）
2. **表面积大**（细纤维、蓬松）
3. **极度干燥**
4. **含有易燃挥发物**

## 引火物分级

取火是一个逐级放大的过程：

```
火星/余烬 → 一级引火物（火绒）→ 二级引火物（细枝）→ 三级引火物（拇指粗柴）→ 主柴
```

每一级都比上一级粗一点，火焰逐步增大。

## 一级引火物（火绒）——最关键

### 天然火绒

| 引火物 | 获取方式 | 着火难易 | 备注 |
|--------|---------|---------|------|
| 蒲绒 | 蒲公英成熟果穗 | ★★★★★ 极易 | 最佳天然火绒 |
| 干苔藓 | 树干、岩石上 | ★★★★ 易 | 需搓碎 |
| 桦树皮 | 白桦树干上剥取 | ★★★★★ 极易 | 含油脂，潮湿也能燃 |
| 干草/枯叶 | 草地 | ★★★ 一般 | 需揉碎 |
| 松针 | 松树下 | ★★★ 一般 | 干燥的 |
| 碎树皮纤维 | 搓揉树皮获得 | ★★★★ 易 | 内皮纤维 |
| 干菌类 | 树干上的马蹄菌 | ★★★★ 易 | 需捣碎成粉 |
| 棉絮 | 棉花 | ★★★★★ 极易 | 穿越后需有棉花 |
| 炭布 | 自制（见下方） | ★★★★★ 极易 | 最佳人造火绒 |

### 自制炭布（char cloth）

炭布是野外取火最可靠的引火物，强烈建议提前制作：

1. 找一个金属罐（如铁罐），盖上打一个小孔
2. 将纯棉布（旧衣服、纱布）撕成小块放入罐中
3. 不要装太满，留一半空间
4. 盖上盖子，放入火堆中
5. 加热后罐内棉布干馏（缺氧环境高温加热）
6. 小孔会冒出黄烟——那是可燃气体在排出
7. 当烟变少或停止时，取出铁罐
8. **不要打开盖子**！等完全冷却后再打开
9. 打开后布应变成黑色、柔软、极易点燃的炭布

**存放在防水容器中**，这是你取火最可靠的后备。

## 二级引火物

- 松枝、细小的干树枝（铅笔粗细）
- 竹片（劈成细条）
- 干草束
- 树皮碎片

## 三级引火物

- 拇指粗的干枝
- 劈开的木柴（劈开后内层更干燥）

## 引火物准备原则

1. **提前准备**：取火前先把所有级别的引火物准备好，按从细到粗排好
2. **比你想的多**：准备你认为需要量的 2-3 倍
3. **保持干燥**：贴身存放或放在防水容器中
4. **先点后取**：引火物都摆好了再开始取火，不要等火星出来了再找

## 时间与备量比例

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 备量 | 火绒、细枝、拇指粗柴都准备预计用量的 2 到 3 倍 | 起火后不断档 | 火苗起来才去找柴 |
| 炭布冷却 | 出火后等罐体完全冷却再开盖 | 炭布黑软、能阴燃 | 开盖复燃或变灰粉 |
| 吹燃 | 轻吹 10 到 30 秒扩大余烬 | 红点扩大成明火 | 猛吹把火星吹散 |
| 清场 | 取火点周围至少清出无干草圈 | 火只在目标火堆内 | 火星落到周围草叶 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 干燥 | 火绒手捏有脆响或蓬松感，无潮冷手感 |
| 分级 | 一级火绒、铅笔粗细枝、拇指粗柴、主柴全部提前备好 |
| 备量 | 准备量至少为预计用量的 2-3 倍 |
| 炭布 | 黑色、柔软、不断成灰粉，遇火星能稳定阴燃 |
| 点火 | 火绒团吹 10-30 秒能扩大成红亮余烬，再引燃细枝 |

## 失败征兆

- 火星落上去立刻熄灭，说明火绒潮湿或太粗。
- 只有火绒没有细枝，火苗会“亮一下就死”。
- 炭布出罐后变成灰白粉末，说明烧过头。
- 炭布未冷却就开罐，可能瞬间复燃。
- 在干草地直接试火，容易引发失控火灾。

## 常见问题

- **Q**: 野外哪里找最好的引火物？
- **A**: 桦树皮（全年可用，雨天都能点着）、松树下干燥的松针层、枯立木下方的碎木屑。

- **Q**: 雨天怎么办？
- **A**: 桦树皮含油可湿燃；枯立木内部通常是干的；松脂（松树上渗出的半透明硬块）是天然助燃剂。

## 安全提醒

- 引火物极度易燃，远离明火存放
- 炭布冷却前不要开罐，接触空气会立即自燃
- 取火前清理周围地面，防止引燃干草

## 穿越小说梗

主角"用嘴一吹，火苗就窜起来了"——实际上，吹早了把火星吹灭、吹猛了把余烬吹散，是新手最常见的翻车操作。正确做法是轻柔地、持续地吹气，让余烬慢慢扩大。
