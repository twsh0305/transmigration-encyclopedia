# 1.26 茅草屋顶铺设

> 没有瓦片？草也能盖出一个不漏水的屋顶。

## 原理

茅草屋顶的防水原理不是"不透水"——单根草茎是会透水的——而是利用**重力排水**和**毛细作用**：

1. **层层叠压**：草束从屋檐到屋脊层层压住，像鱼鳞/瓦片一样
2. **厚度足够**：30cm+ 厚的草层，雨水在渗透过程中就被下一层引导外流
3. **陡坡排水**：屋顶坡度至少 45°（越陡排水越快，越不容易漏水）

## 所需材料

| 材料 | 要求 | 最佳选择 |
|------|------|---------|
| 茅草/稻草 | 干燥、长杆、无霉 | 小麦秸秆、芦苇、芒草 |
| 绑扎材料 | 结实 | 绳索、藤条、树皮条 |
| 屋顶框架 | 木椽 | 圆木或方木 |
| 横杆 | 固定草束用 | 竹竿或细木棍 |

### 草料选择

| 草料 | 耐久性 | 防水性 | 获取难度 |
|------|--------|--------|---------|
| 芦苇 | ★★★★★ | ★★★★★ | 中 |
| 小麦秸秆 | ★★★ | ★★★★ | 易 |
| 芒草 | ★★★★ | ★★★★ | 易 |
| 稻草 | ★★ | ★★★ | 最易 |
| 竹叶 | ★★ | ★★ | 易 |

**芦苇是最佳选择**——中空结构自然防水，编好后可达 50-70 年寿命。

## 操作步骤

### 1. 准备草束

1. 将茅草梳理整齐，根部对齐
2. 扎成直径约 20cm 的草束
3. 每束用绳索在距根部 30cm 处扎紧
4. 草束长度至少 1 米

### 2. 铺设（从下到上）

1. **从屋檐开始**（最底层）
2. 将草束根部朝上、梢部朝下铺在屋顶框架上
3. 草束超出屋檐约 20cm（滴水檐）
4. 用横杆和绑绳将草束固定在椽子上
5. **第二层**压住第一层的上半部分（搭接约 60-70%）
6. 逐层向上铺，每层都压住上层的下半部分
7. **屋脊处**最后铺设，用特殊编法封顶

### 3. 关键参数

- **屋顶坡度**：≥ 45°，推荐 50-55°
- **草层厚度**：至少 25-30cm
- **搭接长度**：上层压住下层的 60-70%
- **固定间距**：每 30cm 一道横杆

### 4. 屋脊处理

屋脊是最容易漏水的部位：

1. 两侧草束在屋脊处交叉对搭
2. 上面再覆盖一层折叠的草束
3. 用横杆和绑绳压实固定
4. 有条件可用木板或陶瓦覆盖屋脊

## 常见问题

- **Q**: 茅草屋顶能用多久？
- **A**: 稻草顶 3-5 年，麦秸顶 5-10 年，芦苇顶 30-70 年。定期维护（更换表层腐烂的草束）可延长寿命。

- **Q**: 茅草顶防火吗？
- **A**: **不防火**。这是茅草屋最大的弱点。灶台烟囱必须远离屋顶，室内明火要极度小心。

- **Q**: 多久换一次？
- **A**: 不需要全部换。每年检查，更换腐烂的表层草束即可（就像换瓦片一样）。

## 安全提醒

- 茅草顶**极易着火**，烟囱周围必须用泥浆/陶瓦隔离
- 不要在茅草屋顶附近使用明火
- 潮湿气候下茅草容易发霉腐烂，注意通风

## 穿越小说梗

穿越后"盖了一间茅草屋"——盖得好的茅草屋可以住几十年，盖得不好一下雨就漏。关键是坡度够陡、草层够厚、搭接够多。这不是随便铺铺草就行的，是一门手艺。
