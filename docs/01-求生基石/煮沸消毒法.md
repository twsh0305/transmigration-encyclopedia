# 1.18 煮沸消毒法

> 人类学会烧开水的那一天，才是真正告别饮毛茹血的时刻。

## 原理

大部分致病微生物（细菌、病毒、寄生虫卵）在高温下蛋白质变性失活。水沸腾（100摄氏度，海拔高时低于 100摄氏度）持续一定时间即可杀灭绝大多数病原体。

**关键认知**：煮沸杀的是生物污染，对化学污染（重金属、农药、毒素）无效。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 水 | 任何水源 | — |
| 容器 | 金属锅、陶罐 | 石坑（用烧石加热法） |
| 热源 | 火堆 | 烧红的石头 |
| 燃料 | 木柴 | 干燥的动物粪便 |

## 操作步骤

### 标准煮沸法

1. 将水装入耐热容器
2. 放到火上加热至沸腾（水面翻滚冒大气泡）
3. **保持沸腾**：
   - 海拔 2000米以下：沸腾后继续 **1 分钟**
   - 海拔 2000-3000米：沸腾后继续 **3 分钟**
   - 海拔 3000米以上：沸腾后继续 **5 分钟**
4. 取下冷却后饮用

> CDC 的应急饮水建议是：清水达到滚沸后保持 1 分钟；海拔约 2000米以上保持 3 分钟。本条目中更高海拔延长时间属于保守野外余量。

### 为什么海拔高要煮更久？

水的沸点随海拔升高而降低：

| 海拔 | 水的沸点 | 建议沸腾时间 |
|------|---------|------------|
| 0米 | 100摄氏度 | 1 分钟 |
| 1500米 | 95摄氏度 | 2 分钟 |
| 3000米 | 90摄氏度 | 5 分钟 |
| 5000米 | 83摄氏度 | 8 分钟 |

温度越低，杀灭病原体需要的时间越长。

### 无容器时的烧石法

没有耐热容器时，可以用"烧石煮水法"：

1. 挖一个小坑或用不漏水的树皮槽
2. 将石头放在火中烧红（至少烧 30 分钟）
3. 用木棍夹出烧红的石头，放入水中
4. 一块拳头大的烧红石头可加热约 1 升水
5. 反复加入烧石直到水沸腾
6. 沸腾后等 1 分钟再取出石头

**注意**：不要用河边的湿石头——石头内部水分受热膨胀可能导致炸裂。用干燥的岩石。

## 煮沸能杀灭什么

| 病原体 | 是否有效 | 备注 |
|--------|---------|------|
| 大肠杆菌 | ✅ | 沸腾即刻杀灭 |
| 霍乱弧菌 | ✅ | |
| 伤寒沙门氏菌 | ✅ | |
| 贾第鞭毛虫 | ✅ | 需要沸腾 1 分钟以上 |
| 隐孢子虫 | ✅ | 需要沸腾 1 分钟以上 |
| 肝炎病毒 | ✅ | 需要沸腾 1 分钟以上 |
| 重金属 | ❌ | 煮沸无法去除 |
| 农药/化学品 | ❌ | 煮沸无法去除，反而浓缩 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 清水 | 水面滚沸冒大泡后计时，低海拔至少 1 分钟 |
| 高海拔 | 海拔约 2000米以上至少 3 分钟，更高海拔可取 5 分钟保守余量 |
| 浑水 | 先沉淀和过滤，再煮沸；不要把泥水直接当成已净化 |
| 储存 | 冷却后倒入干净有盖容器，取水不用手伸入 |
| 边界 | 化学污染、重金属、农药水源不靠煮沸解决 |

## 失败征兆

- 刚冒小泡就关火，未达到滚沸。
- 煮沸后敞口放置，灰尘、手和虫子造成二次污染。
- 用含铅、彩釉不明或油漆容器煮水，可能引入毒物。
- 河边湿石直接投入火中加热，可能爆裂。
- 多人饮用同一批水后腹泻，应怀疑处理或储存污染。

## 常见问题

- **Q**: 水开了就行，还是得一直煮？
- **A**: 必须保持沸腾状态至少 1 分钟。刚冒泡就关火，有些顽固的病原体还没死透。

- **Q**: 煮久一点会不会更安全？
- **A**: 不会更安全，但会浪费燃料，且水蒸发后溶解物浓度升高，口感变差。按推荐时间即可。

- **Q**: 煮沸能去除异味吗？
- **A**: 部分挥发性异味可以（如硫化氢），但泥土味、金属味去除不了。加木炭吸附可以改善。

## 安全提醒

- 取放沸水时注意烫伤
- 不要用含铅容器煮水（铅中毒）
- 煮沸后的水如果不立即饮用，应加盖保存，避免二次污染
