# 1.18 煮沸消毒法

> 人类学会烧开水的那一天，才是真正告别饮毛茹血的时刻。

## 原理

大部分致病微生物（细菌、病毒、寄生虫卵）在高温下蛋白质变性失活。水沸腾（100°C，海拔高时低于 100°C）持续一定时间即可杀灭绝大多数病原体。

**关键认知**：煮沸杀的是生物污染，对化学污染（重金属、农药、毒素）无效。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 水 | 任何水源 | — |
| 容器 | 金属锅、陶罐 | 石坑（用烧石加热法） |
| 热源 | 火堆 | 烧红的石头 |
| 燃料 | 木柴 | 干燥的动物粪便 |

## 操作步骤

### 标准煮沸法

1. 将水装入耐热容器
2. 放到火上加热至沸腾（水面翻滚冒大气泡）
3. **保持沸腾**：
   - 海拔 2000m 以下：沸腾后继续 **1 分钟**
   - 海拔 2000-3000m：沸腾后继续 **3 分钟**
   - 海拔 3000m 以上：沸腾后继续 **5 分钟**
4. 取下冷却后饮用

### 为什么海拔高要煮更久？

水的沸点随海拔升高而降低：

| 海拔 | 水的沸点 | 建议沸腾时间 |
|------|---------|------------|
| 0m | 100°C | 1 分钟 |
| 1500m | 95°C | 2 分钟 |
| 3000m | 90°C | 5 分钟 |
| 5000m | 83°C | 8 分钟 |

温度越低，杀灭病原体需要的时间越长。

### 无容器时的烧石法

没有耐热容器时，可以用"烧石煮水法"：

1. 挖一个小坑或用不漏水的树皮槽
2. 将石头放在火中烧红（至少烧 30 分钟）
3. 用木棍夹出烧红的石头，放入水中
4. 一块拳头大的烧红石头可加热约 1 升水
5. 反复加入烧石直到水沸腾
6. 沸腾后等 1 分钟再取出石头

**注意**：不要用河边的湿石头——石头内部水分受热膨胀可能导致炸裂。用干燥的岩石。

## 煮沸能杀灭什么

| 病原体 | 是否有效 | 备注 |
|--------|---------|------|
| 大肠杆菌 | ✅ | 沸腾即刻杀灭 |
| 霍乱弧菌 | ✅ | |
| 伤寒沙门氏菌 | ✅ | |
| 贾第鞭毛虫 | ✅ | 需要沸腾 1 分钟以上 |
| 隐孢子虫 | ✅ | 需要沸腾 1 分钟以上 |
| 肝炎病毒 | ✅ | 需要沸腾 1 分钟以上 |
| 重金属 | ❌ | 煮沸无法去除 |
| 农药/化学品 | ❌ | 煮沸无法去除，反而浓缩 |

## 常见问题

- **Q**: 水开了就行，还是得一直煮？
- **A**: 必须保持沸腾状态至少 1 分钟。刚冒泡就关火，有些顽固的病原体还没死透。

- **Q**: 煮久一点会不会更安全？
- **A**: 不会更安全，但会浪费燃料，且水蒸发后溶解物浓度升高，口感变差。按推荐时间即可。

- **Q**: 煮沸能去除异味吗？
- **A**: 部分挥发性异味可以（如硫化氢），但泥土味、金属味去除不了。加木炭吸附可以改善。

## 安全提醒

- 取放沸水时注意烫伤
- 不要用含铅容器煮水（铅中毒）
- 煮沸后的水如果不立即饮用，应加盖保存，避免二次污染

## 穿越小说梗

穿越主角动辄"烧了壶开水"——但古代烧开水并不简单：没有金属锅的年代，你用什么容器？答案是烧石法。先学会生火，再学会烧水，穿越的第一天才算及格。
