# 1.19 燧石取火

> 两块石头一撞，火花飞溅——这是人类用火历史上最优雅的方式之一。

## 原理

当硬质岩石（燧石/火石）以适当角度撞击含铁金属（高碳钢）时，从金属表面剥落的微小铁屑因撞击热量在空气中瞬间达到高温（>1000°C），与空气中的氧气剧烈反应（燃烧），形成飞溅的火花。

**关键**：火花的来源不是石头，而是**钢**。没有含碳量足够高的钢，燧石打不出火花。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 燧石/火石 | 硬度 > 7，断口锋利 | 石英、玛瑙、黑曜石 |
| 钢铁 | 高碳钢（含碳量 > 0.5%） | 旧锉刀、碳钢刀背 |
| 引火物 | 极干燥的细纤维 | 蒲绒、干苔藓、炭布 |
| 引火物容器 | 干燥的树皮、叶子 | — |

### 如何识别燧石

- **颜色**：黑色、深灰、深褐色
- **断口**：贝壳状断口，边缘锋利
- **硬度**：能划伤玻璃
- **质地**：致密、不透明、敲击有清脆声
- **分布**：石灰岩地区、河滩砾石中常见

## 操作步骤

1. **准备引火物**：
   - 将蒲绒/干苔藓揉成鸟巢状
   - 放在干燥的树皮或叶子上，方便拿取
   - **炭布（char cloth）是最好的引火物**——将纯棉布在密封铁罐中加热碳化制成

2. **握持**：
   - 左手（非惯用手）握燧石，锋利边缘朝上
   - 将引火物紧贴在燧石上方，靠近锋利边缘
   - 右手握钢铁（刀背、锉刀等）

3. **打击**：
   - 用钢铁**猛烈向下划击**燧石边缘
   - 打击角度约 30-45 度
   - 速度要快，力度要大
   - 正确的打击会产生明亮的火花飞向引火物

4. **引燃**：
   - 火花落在引火物上形成暗红亮点
   - 将引火物轻轻对折包住亮点
   - 缓慢吹气，亮点逐渐扩大
   - 引火物冒浓烟后起火

5. **转移**：将着火的引火物放入细柴堆

## 打击技巧

- 打击方向：**向下刮削**而非正面碰撞
- 有效火花 = 小角度快速刮削，不是大力磕碰
- 一分钟有效打击 10-15 次
- 每次打击应有 2-3 颗明亮火花

## 常见问题

- **Q**: 为什么打不出火花？
- **A**: 三个原因最常见：①钢的含碳量不够（不锈钢打不出火花）②燧石不够硬或边缘不够锋利 ③打击角度不对

- **Q**: 不锈钢刀能打火花吗？
- **A**: **不能**。不锈钢含碳量太低。必须用高碳钢。旧式碳钢刀、锉刀、碳钢火石棒都可以。

- **Q**: 没有燧石怎么办？
- **A**: 石英、玛瑙、坚硬的深色河石都行。关键是硬度够，能从钢上刮出铁屑。

## 安全提醒

- 燧石断口极其锋利，小心割伤
- 打击时火花飞溅，远离易燃物
- 确保周围没有干草等意外引燃物

## 穿越小说梗

穿越古代后，你手里那把不锈钢水果刀是打不出火花的——得找高碳钢。古代的锻铁含碳量不均匀，有些可以打火有些不行。最好找一把老铁匠打的碳钢刀。
