# 1.16 火种保存（火折子制作）

> 取火难，保火易。与其每次重新取火，不如让火种长明。

## 原理

燃烧需要三个条件：可燃物、氧气、温度（着火点以上）。火折子的原理是**控制氧气供给**，使可燃物处于"阴燃"状态——不冒明火但持续燃烧，只需要极少氧气就能维持。

打开盖子增加供氧，阴燃立刻转为明火；盖上盖子减少供氧，明火退回阴燃。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 竹筒 | 竹子截段 | 纸卷筒、空心木棍 |
| 藤条/草纸 | 干燥的草纸、蒲绒 | 干树皮纤维、干苔藓 |
| 火折子芯 | 草纸卷紧 | 棉布条、麻绳 |
| 盖子 | 木塞、竹节 | 布包+绳子 |

## 操作步骤

### 传统火折子制作

1. **制作筒身**：
   - 截取一节竹筒（长 15-20厘米，直径 3-4厘米）
   - 一端封闭（保留竹节），一端开口
   - 在竹筒壁上打几个小通气孔（可选）

2. **制作火折子芯**：
   - 将草纸/藤条/干树皮纤维反复折叠卷紧
   - 卷成直径略小于竹筒内径的紧密圆棒
   - 芯的密度很关键——太松烧得太快，太紧氧气不够容易灭
   - 一端露出竹筒口约 2厘米

3. **引燃火折子**：
   - 将火折子芯的一端放在明火中点燃
   - 让它燃烧几秒，确保端部完全红透
   - 轻轻吹灭明火，使端部保持暗红余烬状态

4. **收入竹筒**：
   - 将燃烧端朝内插入竹筒
   - 盖上盖子/塞子（不要完全密封，留一丝缝隙）
   - 火折子在低氧环境中进入阴燃状态

5. **使用**：
   - 需要火时，打开盖子
   - 对着暗红端轻吹气，火苗立刻窜出
   - 用火折子点燃引火物
   - 用完后重新盖上，火回到阴燃状态

### 简易版（纸卷火折子）

没有竹筒时：

1. 将报纸/草纸卷成紧实的纸棒
2. 点燃一端，烧 5 秒后吹灭明火
3. 保持端部暗红余烬
4. 放在避风处阴燃
5. 需要时对着吹气即可复燃

**缺点**：没有外壳保护，容易意外熄灭或完全烧尽。仅供短期使用。

## 保存时间

| 火折子类型 | 保存时间 | 备注 |
|-----------|---------|------|
| 竹筒+草纸芯 | 6-12 小时 | 传统水平 |
| 竹筒+棉布芯 | 4-8 小时 | 棉布燃速较快 |
| 纸卷（无筒） | 1-3 小时 | 容易灭 |
| 粗香/线香 | 8-24 小时 | 古代常用 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 筒身 | 竹筒长 15-20厘米、直径 3-4厘米，盖子能控风但不完全闷死 |
| 芯材 | 卷紧后能塞入筒内，露口约 2厘米，阴燃不散开 |
| 点燃 | 端部红透后吹灭明火，只留暗红余烬 |
| 保存 | 竹筒外壁不烫手，烟小，6-12 小时内可吹燃引火物 |
| 熄灭 | 不再使用时浸水或闷灭，确认无余温再收纳 |

## 失败征兆

- 芯太松，几分钟内烧尽；芯太紧，盖上就灭。
- 竹筒冒大烟或烫手，说明氧气过多或内部明火未退。
- 放在草堆、衣物、粮袋旁保存，是火灾隐患。
- 睡前仍有阴燃火源无人看管。
- 盖子完全密封后产生焦油水和异味，复燃困难。

## 常见问题

- **Q**: 火折子灭了怎么办？
- **A**: 只能重新取火。火折子不是永久的——它只是减少取火次数。

- **Q**: 阴燃有没有烟气？
- **A**: 有少量。竹筒+草纸的组合烟很小。如果烟很大说明氧气过多，盖紧一点。

- **Q**: 古代真的用火折子吗？
- **A**: 是的。中国从宋代开始就有商用火折子，古代客栈、驿站常备。西方也有类似装置（tinder box）。

## 安全提醒

- 火折子是阴燃火源，不要放在易燃物旁边
- 睡觉前确认火折子放置安全
- 完全不要火时，将火折子浸水或闷灭

## 穿越小说梗

火折子是穿越小说中主角装 X 神器——"从袖中掏出火折子一吹即燃"。但现实中你得先有火才能制作火折子，这是先有鸡还是先有蛋的问题。建议穿越第一天就想办法取到火，然后制作火折子持续保火。
