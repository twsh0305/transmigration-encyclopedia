# 1.1 井盐/岩盐开采

> 不靠海的地方也有盐——它藏在地下。

## 原理

古代内陆地区的盐主要来自三类：
1. **井盐**：地下盐卤水（溶解了盐的地下水），钻井取出后煎煮成盐
2. **岩盐**：地下盐矿，直接开采固态盐
3. **池盐**：内陆盐湖，日晒结晶

四川自贡井盐、山西运城池盐是中国古代最著名的内陆盐产地。

## 所需材料

| 材料 | 获取方式 | 备注 |
|------|---------|------|
| 盐卤水 | 盐井、盐泉 | — |
| 钻井工具 | 铁锉、竹缆 | 自贡传统用竹木工具 |
| 煎盐锅 | 铁锅、大口径 | — |
| 燃料 | 天然气（自贡特产）、柴火 | — |

## 操作步骤

### 井盐：打井取卤

1. **选址**：
   - 盐卤水通常出现在低洼地带
   - 地表有白色盐霜或咸味土壤是线索
   - 古代通过"看风水"（实际是观察地质特征）选井位

2. **凿井**：
   - 中国古代（宋代以后）发展了"卓筒井"技术——用铁制钻头冲击钻进
   - 井径约 10-15厘米，深度可达数十到数百米
   - 用竹制套管加固井壁，防止坍塌
   - 自贡最深盐井超过 1000 米

3. **汲卤**：
   - 用竹筒（底部有单向阀）放入井中汲取卤水
   - 用竹缆和辘轳提升
   - 一口井日产卤水数百到数千升

4. **煎盐**：
   - 将卤水倒入大铁锅中煎煮
   - 先加少量豆浆（去除杂质——蛋白质凝固吸附杂质）
   - 煮沸蒸发水分，盐晶析出
   - 捞出盐晶晾干

### 岩盐：开矿取盐

1. **找矿**：盐矿通常与石膏、石灰岩伴生
2. **开矿**：竖井或平硐开采
3. **碎盐**：将矿石破碎
4. **溶解重结晶**：将碎盐溶于水，过滤后煎煮结晶
   - 直接用矿石含杂质，溶解重结晶可提高纯度

### 池盐：日晒结晶

1. 在盐湖边挖浅池引水
2. 日晒蒸发
3. 收集结晶盐
4. 山西运城盐湖已有 4000+ 年产盐史

## 成功标准

- 卤水蒸发后能析出白色或浅色晶体，苦涩味不明显。
- 重结晶后盐粒干爽、不夹泥沙，少量溶于水后无明显浑浊和异味。
- 盐井、盐泉或盐池位置能稳定产卤，不是一次性雨后盐霜。
- 生产记录写清卤水量、煎煮时间、得盐重量和燃料消耗。

## 失败征兆

- 盐味强烈发苦：镁盐、苦卤或杂质过多，需要重结晶或弃用。
- 卤水有油膜、硫臭、腐臭、金属味：可能混入污染物。
- 井壁掉土、井口冒可燃气、工人头晕：塌方、甲烷或缺氧风险。
- 煎盐锅底焦黑、盐粒夹灰：火力和过滤不合格。

## 常见问题

- **Q**: 怎么判断地下有盐？
- **A**: ①土壤表面有白色盐霜 ②地下水尝起来咸 ③特定地质构造（背斜、断层）附近 ④植物稀疏、土地白化

- **Q**: 井盐和海盐哪个好？
- **A**: 化学成分基本相同（都是 NaCl），但井盐卤水矿物质含量不同，部分井盐含有较多的钾、碘等微量元素。

## 安全提醒

- 盐矿井深，注意通风（甲烷积聚可致窒息或爆炸）
- 卤水腐蚀性强，接触皮肤后及时清洗
- 深井有塌方风险

## 穿越小说梗

穿越到四川的人想打盐井？自贡的盐井深度可达千米，用宋代发明的卓筒井技术至少需要几个月才能凿成。但如果你只是找盐泉浅层卤水，凿个十几米的井就够。关键是你得先知道哪里有盐——四川盆地是天然盐卤区。
