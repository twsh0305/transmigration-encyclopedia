# 1.22 简易棚屋搭建

> 不会搭房子，你连第一晚都熬不过去。

## 原理

棚屋的核心功能是**阻隔热对流**——挡住风和雨，减少人体向环境的散热。结构越封闭，保温效果越好。

## 所需材料

| 材料 | 用途 | 替代方案 |
|------|------|---------|
| 长木棍/树枝 | 主框架 | 竹子 |
| 绳索 | 绑扎 | 藤条、树皮纤维、根须 |
| 大量树叶/草 | 覆盖屋顶 | 苔藓、树皮、土 |
| 石头 | 固定底部 | 土堆 |

## 操作步骤

### A 字棚（最经典、最推荐）

抗风性强，结构稳定，适合 1-2 人。

1. **选位**：平坦、干燥、背风的地面
2. **搭建脊梁**：
   - 找一根长约 2.5 米的横杆（脊梁）
   - 两端各架在一个分叉木桩上，或绑在两棵间距合适的树干上
   - 高度约 1 米（坐得进去即可）
3. **搭建斜面**：
   - 从脊梁两侧斜靠密集排列长树枝
   - 两侧各排列 10-15 根，形成 A 字形
   - 底部嵌入地面或用石头压住
4. **覆盖**：
   - 从底部开始向上铺草/树叶/苔藓（像瓦片一样层层压住）
   - **从下往上铺**，上层压住下层，雨水才能顺流而下
   - 覆盖厚度至少 20厘米（约手臂厚）才能有效防雨
5. **填充缝隙**：用细枝和碎叶塞满所有缝隙
6. **地面处理**：
   - 床铺至少 15厘米厚的干草/树叶
   - 隔绝地面寒气比盖好屋顶更重要

### 单坡棚（更简单）

适合一面有天然屏障（如大石、土坡）的情况。

1. 找一面天然屏障（石壁、土坡或大树）
2. 横搭一根横杆在屏障和两根立柱之间
3. 从横杆斜靠树枝到地面
4. 覆盖树叶/草
5. 开口朝向背风方向

### 搭建时间参考

| 棚屋类型 | 熟练者 | 新手 |
|---------|--------|------|
| A 字棚 | 1-2 小时 | 3-4 小时 |
| 单坡棚 | 30-60 分钟 | 1-2 小时 |

## 保温技巧

- 不要在封闭棚屋内生火；若必须短时使用蜡烛或炉具，入口和顶部通风口必须打开，并有人清醒看守
- 入口用一堆灌木或背包遮挡，减少冷风灌入
- 地面铺干草越多越暖，**隔绝地面寒气是保暖的第一要务**

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 位置 | 离河沟、崖壁、枯树至少 10米，地面不积水 |
| 尺寸 | 单人内宽约 1米、长约 2米、高约 1米；越贴身越保温 |
| 覆盖 | 草叶层厚 20厘米以上，从下往上搭接，泼水测试能顺坡流下 |
| 地铺 | 干草或树叶压实后仍有 15厘米厚，躺下不直接接触湿冷地面 |
| 稳定 | 手推脊梁不明显晃动，夜间风吹不会散架 |

## 失败征兆

- 屋顶能看到连续亮缝，夜间会漏风漏雨。
- 覆盖层从上往下铺，雨水会顺缝倒灌。
- 地铺一压就见泥，睡到半夜容易失温。
- 棚下有蚁穴、兽道、落石痕迹或枯枝悬挂，应立即换址。
- 生火后眼痛、头晕、犯困，是烟气或一氧化碳危险信号，马上离开通风。

## 常见问题

- **Q**: 下雨时棚屋漏水怎么办？
- **A**: ①增加覆盖层厚度 ②确保从下往上铺（像瓦片） ③在棚外加一道排水浅沟

- **Q**: 多大的棚合适？
- **A**: 越小越暖。1 人用 2米×1米足够。空间大了空气多，体热难以升温。

## 安全提醒

- 棚屋内若生火，必须留通风口——一氧化碳无色无味可致命
- 不要在枯树下搭棚——枯枝可能掉落
- 雷雨天不要在孤立大树下搭棚

## 穿越小说梗

主角"找了些树枝草草搭了个棚子就睡了"——现实是，覆盖不够厚=淋雨=失温=可能活不过第一晚。A 字棚的覆盖层至少要你小臂那么厚，这个量比你想象的大得多。
