# 1.25 粗盐提纯为细盐

> 粗盐→细盐，是穿越小说的入门级操作，原理简单到像是在做化学实验课后习题。

## 原理

粗盐中主要杂质：
- **不溶性杂质**：泥沙、有机物（通过过滤去除）
- **可溶性杂质**：氯化镁（苦味）、硫酸钙等（通过重结晶去除）

核心原理：**重结晶**。氯化钠的溶解度随温度变化很小（20°C 时 36g/100ml，100°C 时 39.8g/100ml），而杂质盐的溶解度随温度变化较大。利用这个差异，通过控制蒸发量让氯化钠优先结晶析出。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 粗盐 | 海盐、井盐 | 任何未精制的盐 |
| 水 | 淡水 | — |
| 容器 | 锅、陶罐 | — |
| 过滤材料 | 棉布、细沙 | — |
| 热源 | 火堆 | — |

## 操作步骤

### 第一步：溶解过滤（去除不溶性杂质）

1. 将粗盐放入容器中
2. 加入淡水，搅拌溶解
   - 不需要加热，常温即可溶解大部分盐
   - 盐与水的比例：约 1 份盐加 3 份水
3. 搅拌至不再溶解（达到饱和）
4. 用棉布或沙滤器过滤溶液
5. 不溶性泥沙和杂质被滤除
6. 得到澄清的盐水

### 第二步：蒸发结晶（重结晶）

1. 将过滤后的盐水倒入干净的锅中
2. 缓慢加热蒸发水分
3. **关键控制**：
   - 当溶液开始出现盐晶时，**调小火力**
   - 不要蒸干！在水量还剩约 1/3 时停止加热
   - 氯化钠会优先析出，镁盐等仍留在溶液中
4. 用布或竹漏勺捞出析出的盐晶
5. 剩下的母液（含大量镁盐）是苦卤，丢弃或另作他用

### 第三步：洗涤干燥

1. 将捞出的盐晶用少量冷饱和盐水快速冲洗
   - 不要用淡水洗！淡水会溶解盐晶
   - 用饱和盐水洗可去除表面附着的苦卤，但不会溶解盐晶本身
2. 将洗好的盐平摊在干净表面晾干
3. 完全干燥后即为较纯的细盐

### 可选第四步：加碘

如果穿越到了缺碘地区（内陆山区），碘缺乏症（甲状腺肿/大脖子病）是严重问题。

加碘方法：每公斤盐加入约 20-50mg 碘酸钾（KIO₃）。

**问题**：穿越后碘从哪来？
- 海藻灰中含碘（海带灰含碘约 0.1-0.5%）
- 可以将海藻烧成灰，用酒精或水提取碘
- 但实际操作难度极大，产量极低
- **更实际的方案**：多吃海产品（如果靠近海边）

## 提纯效果

| 步骤 | 氯化钠纯度 | 杂质含量 |
|------|-----------|---------|
| 粗盐（原始） | 80-90% | 高 |
| 溶解过滤后 | 90-95% | 中 |
| 一次重结晶 | 95-98% | 低 |
| 二次重结晶 | 98-99%+ | 极低 |

## 常见问题

- **Q**: 为什么盐有苦味？
- **A**: 氯化镁。重结晶时提前取出盐晶、用饱和盐水洗涤都可以去苦味。

- **Q**: 一次提纯够了吗？
- **A**: 日常食用，一次重结晶足够。如果要追求更高纯度，可以二次重结晶。

- **Q**: 加碘是必须的吗？
- **A**: 内陆缺碘地区非常重要。缺碘导致甲状腺肿、智力发育受损。海边居民通常不缺碘。

## 安全提醒

- 蒸发盐水时注意蒸汽烫伤
- 苦卤有腐蚀性，避免接触眼睛
- 加碘量极小，过量反而有害
