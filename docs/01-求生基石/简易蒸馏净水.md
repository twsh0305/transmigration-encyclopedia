# 1.24 简易蒸馏净水

> 蒸馏是唯一能同时去除悬浮颗粒、微生物和溶解盐的方法——把水变成蒸汽再变回水，一切杂质都留在了后面。

## 原理

水沸腾后变成蒸汽，蒸汽遇冷重新凝结为液态水。这个过程中：
- 不挥发的杂质（盐、重金属、泥沙）留在原容器中
- 挥发性比水低的杂质也留下
- 蒸汽凝结后的水接近纯水

**但注意**：沸点低于水的挥发性有机物（如酒精、某些溶剂）会跟着蒸汽跑过去，所以蒸馏不能去除所有化学污染。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 大容器（锅/罐） | 金属锅、陶罐 | — |
| 冷凝装置 | 金属管、竹管 | 中空植物茎 |
| 收集容器 | 碗、杯子 | — |
| 热源 | 火堆 | — |
| 盖子/密封材料 | 金属盖 | 布条+黏土密封 |

## 操作步骤

### 方法一：冷凝管蒸馏（经典法）

这是最接近实验室蒸馏的方法：

1. 在大容器中装入待净化的水（不超过 2/3）
2. 容器口安装一根倾斜向下的导管（金属管最佳，竹管也可）
3. 导管与容器口的接合处用湿黏土密封
4. 导管另一端通入收集容器
5. 对导管中段浇冷水或用湿布包裹（加速冷凝）
6. 加热容器至水沸腾
7. 蒸汽经导管流出，遇冷凝结成水滴，滴入收集容器

### 方法二：太阳能蒸馏（坑式法）

无需火源，利用阳光和温差：

1. 在湿润地面挖一个直径约 1 米、深约 0.5 米的圆坑
2. 在坑底中央放一个收集容器
3. 在坑上覆盖透明塑料薄膜（或大块透明材料）
4. 薄膜四周用土/石头压紧密封
5. 在薄膜正中央（对准收集容器）放一块小石头，使薄膜中间凹陷
6. 太阳加热坑内空气和土壤，水分蒸发
7. 蒸汽在薄膜下表面凝结，沿倾斜面流到最低点（凹陷处），滴入容器

**产量**：每天约 0.5-1 升/平方米，取决于阳光强度和土壤湿度。

### 方法三：两锅蒸馏（最简易）

1. 大锅中烧水
2. 在大锅上方架一个小锅（倒扣或用支架）
3. 蒸汽上升遇到小锅底部冷凝
4. 凝结的水滴沿小锅底部流入边缘的收集槽
5. 小锅中可放冷水或不断换水以保持低温

## 蒸馏效果

| 杂质类型 | 能否去除 |
|---------|---------|
| 泥沙/悬浮颗粒 | ✅ 完全去除 |
| 细菌/病毒/寄生虫 | ✅ 完全去除 |
| 溶解盐（海水淡化） | ✅ 完全去除 |
| 重金属 | ✅ 完全去除 |
| 挥发性有机物 | ⚠️ 部分去除 |
| 农药（非挥发性） | ✅ 去除 |

## 常见问题

- **Q**: 蒸馏水可以直接长期饮用吗？
- **A**: 紧急情况下可以。但长期只喝蒸馏水会缺乏矿物质。日常饮用应搭配其他水源或食物补充矿物质。

- **Q**: 能用蒸馏法淡化海水吗？
- **A**: 可以，蒸馏是海水淡化的经典方法。但产量低、耗能大，适合少量获取。

- **Q**: 导管密封不好怎么办？
- **A**: 用湿黏土涂抹接口处，干了之后就是很好的密封。也可以用面粉调浆糊密封。

## 安全提醒

- 蒸馏时注意蒸汽烫伤
- 收集容器不要离火源太近，避免蒸发损失
- 蒸馏不能去除所有有毒蒸汽——如果水源有化学污染，第一段蒸馏液（低沸点组分）应弃去

## 穿越小说梗

穿越小说里的蒸馏器通常画得太精致了。实际上用两口锅一根竹管就能搞出基础蒸馏装置——蒸馏酒和蒸馏水用的是同一套原理。
