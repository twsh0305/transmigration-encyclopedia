# 1.20 盐的保存与防潮

> 盐不会坏，但会变泥。

## 原理

氯化钠本身化学性质稳定，不会变质。但食盐极易**吸潮**（吸湿性），因为：
- 食盐中常含氯化镁（MgCl₂）杂质，氯化镁吸湿性极强
- 精盐颗粒细小，总表面积大，更容易吸潮
- 当空气湿度超过 75% 时，食盐开始吸水结块

吸潮的后果：结块→变硬→溶解成卤水→损失。

## 防潮方法

### 方法一：密封容器（最有效）

1. 将盐装入**完全干燥**的陶瓷罐或玻璃瓶中
2. 盖紧盖子，缝隙处用蜡或油纸密封
3. 每次取用后立即盖紧
4. 存放在阴凉干燥处

### 方法二：炒干法（定期维护）

1. 发现盐开始吸潮结块时，倒入干净的干锅中
2. 小火翻炒 5-10 分钟
3. 炒干水分后立即装入密封容器
4. 注意不要炒太久——盐不会焦，但温度过高会让碘流失

### 方法三：大米/豆子防潮

1. 在盐罐底部放几粒干大米或干豆子
2. 大米/豆子会优先吸潮，保护盐
3. 定期更换大米/豆子

### 方法四：炒热盐法（古人智慧）

1. 将一部分盐炒热（高温烘烤除湿）
2. 放入布袋中
3. 将布袋放入盐仓中——炒热的盐会吸收周围湿气

### 方法五：仓库级防潮

大量存盐时：

1. 盐仓建在高地、干燥处
2. 地面架空（铺木板或石板，下方通风）
3. 墙壁用石灰涂抹防潮
4. 盐堆用草席覆盖，减少与空气接触
5. 仓库保持通风但避免潮湿气流

## 不同盐的保存特性

| 盐类型 | 吸潮性 | 保存难度 |
|--------|--------|---------|
| 精盐（高纯度 NaCl） | 低 | 容易 |
| 粗盐（含 MgCl₂） | **高** | 较难 |
| 海盐 | 中 | 中等 |
| 岩盐 | 低 | 容易 |

**关键**：提纯度越高，吸潮性越低。这就是粗盐比细盐更容易受潮的原因——粗盐中镁盐杂质更多。

## 成功标准

- 盐粒倒出时干爽流动，不成泥、不挂壁。
- 梅雨或雨季连续 7 天后，罐底没有卤水。
- 大批量存盐时离地至少 20厘米，墙面和地面无返潮水印。
- 每月检查一次容器密封、结块、虫鼠和异味。

## 失败征兆

- 盐变湿、成团、罐底有液体：密封或仓库防潮失败。
- 盐有苦味、霉味、煤油味或化学品味：可能混入杂质或污染。
- 盐仓地面湿滑、墙根返潮：排水和架空不足。
- 用潮湿木勺反复取盐，导致整罐受潮。

## 常见问题

- **Q**: 盐结块了还能吃吗？
- **A**: 能。结块只是物理变化，不影响化学成分。捣碎或炒干即可。

- **Q**: 盐会过期吗？
- **A**: 纯氯化钠不会过期。但加碘盐的碘会逐渐流失，建议 2 年内用完。粗盐中的杂质可能因长期存放而影响风味。

- **Q**: 暴雨季节如何保存盐？
- **A**: 双重密封——先装进布袋（内层），再放入密封陶罐（外层），存放在室内最高的架子上。

## 安全提醒

- 盐仓严禁明火——大量粉尘有爆炸风险
- 潮湿的盐仓地面湿滑，注意安全
- 不要将盐和化学品同库存放

## 穿越小说梗

主角穿越后"存了几百斤盐当硬通货"——存盐不难，难的是防潮。古代盐商的仓库是专门设计的，地面架空、墙壁刷石灰、通风换气。你要是在地窖里随便堆了几百斤盐，梅雨季过后就变成一坑卤水了。
