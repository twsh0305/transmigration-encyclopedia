# 1.14 泥砖/土坯制作

> 万丈高楼平地起，而一切从一块泥砖开始。

## 原理

土坯砖是将黏土、沙子和纤维（稻草）按比例混合后晒干或阴干的建筑砌块。不需要烧制（区别于烧结砖），制作简单，成本低，是古代世界最常见的建材之一。

黏土干燥后颗粒间产生结合力，纤维起到类似钢筋的拉结作用，沙子调节收缩防止开裂。

## 所需材料

| 材料 | 比例 | 要求 |
|------|------|------|
| 黏土 | 约 60% | 黏性好，可搓成条 |
| 沙 | 约 30% | 细沙，减少收缩开裂 |
| 稻草/碎草 | 约 10% | 切成 5-10cm 段 |
| 水 | 适量 | 使混合物可塑 |

### 如何判断土壤是否适合做土坯

**搓条法**：取一把土加水搓成铅笔粗的泥条——
- 能搓成条不断裂 = 黏土够多，适合
- 一搓就散 = 黏土不够，需掺入黏土
- 搓成条后弯曲不断 = 黏土太多，需掺沙

## 操作步骤

### 1. 制模

- 用木板制作砖模（标准尺寸约 30×15×10cm）
- 内壁涂一层灰水或油（便于脱模）
- 没有木板可以用硬土挖出砖形凹槽代替

### 2. 备料

1. 将黏土粉碎，去除石块
2. 加入细沙混合均匀
3. 加入碎稻草
4. 加水和泥——像揉面一样反复踩踏翻拌
5. 和好的泥标准：用手抓一把捏成团，扔到地上不散不塌

### 3. 制坯

1. 将泥料用力摔入模具中（排出气泡）
2. 用木板刮平表面
3. 小心脱模
4. 将砖坯平放在平整干燥的地面上

### 4. 晾干

1. 前 1-2 天放在阴凉处阴干（避免表面速干开裂）
2. 第 3 天起可晒太阳
3. 一周后竖立晾干侧面
4. 完全干燥需要 **2-4 周**（取决于气候）
5. 干透的砖：颜色变浅、敲击声清脆、重量明显减轻

### 产量

- 一人一天可制作 100-200 块土坯
- 一间 4m×4m 的房屋约需 1000-1500 块

## 土坯建筑注意事项

- 地基用石块砌筑（防潮）
- 墙体用泥浆粘合
- 窗户上方的横梁要牢固
- 内外墙面抹石灰或泥浆（防雨水侵蚀）
- 屋顶出檐（保护墙面不被雨水冲刷）

## 常见问题

- **Q**: 土坯砖能用多久？
- **A**: 做好防水的情况下可用几十年甚至上百年。但雨水是最大敌人——墙面必须做好防水处理。

- **Q**: 为什么一定要加稻草？
- **A**: 稻草 = 古代钢筋。干燥收缩时纤维拉住泥土防止开裂。不加稻草的砖容易裂成两半。

## 安全提醒

- 土坯墙承重有限，不宜超过 2 层
- 地震区土坯建筑抗震性差
- 必须做好防水，否则雨季可能坍塌

## 穿越小说梗

主角"带着村民造砖建屋"——土坯砖不用烧、不用窑，本质就是高级泥巴。真正需要烧制的烧结砖，温度要达到 900°C 以上，那才是技术活。土坯房虽然简陋，但穿越初期足够遮风挡雨了。
