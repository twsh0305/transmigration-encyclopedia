# 1.21 石灰烧制与砂浆

> 没有砂浆，砖只是砖。有了砂浆，砖才能成为墙。

## 原理

石灰石（碳酸钙 CaCO₃）在 900-1000摄氏度高温下分解为生石灰（氧化钙 CaO）和二氧化碳：

```
CaCO₃ →(900摄氏度)→ CaO + CO₂↑
```

生石灰遇水发生剧烈放热反应，生成消石灰（氢氧化钙 Ca(OH)₂）：

```
CaO + H₂O → Ca(OH)₂ + 热量
```

消石灰与空气中的二氧化碳反应，重新生成碳酸钙并硬化——这就是石灰砂浆硬化的原理：

```
Ca(OH)₂ + CO₂ → CaCO₃ + H₂O
```

**石灰砂浆的硬化靠的是吸收空气中的 CO₂，这是一个缓慢的过程（数周到数月）。**

## 所需材料

| 材料 | 来源 | 要求 |
|------|------|------|
| 石灰石 | 山体、河滩 | 青灰色、滴盐酸冒泡 |
| 燃料 | 木柴、木炭 | 大量 |
| 窑 | 石砌或土砌 | 能达 1000摄氏度 |
| 沙 | 河沙 | 砌筑用 |

### 如何识别石灰石

- **外观**：青灰色或灰白色
- **硬度**：3 级（指甲刻不动，铁器可刻）
- **酸测试**：滴醋冒泡 = 含碳酸钙 = 石灰石
- **分布**：沉积岩地区、山体断面、河滩

## 操作步骤

### 第一步：烧制生石灰

1. 建窑：用石块砌成圆筒形窑，底部留进风口和出灰口
2. 将石灰石碎成拳头大小
3. 窑底铺木柴，逐层交替铺石灰石和燃料
4. 从底部点火
5. 持续加燃料维持高温 12-48 小时
6. 判断烧成：取出一块冷却，浇水后剧烈发热冒蒸汽 = 烧透了
7. 冷却后取出白色/灰白色块状物 = 生石灰

### 第二步：消解（熟化）

1. 将生石灰块放入坑或池中
2. **缓慢加水**——反应剧烈，会产生大量蒸汽和热量
3. 生石灰碎裂成粉末 = 消石灰（熟石灰）
4. 加水量约为生石灰重量的 70-100%
5. 消解后放置陈化 1-2 周（未完全消解的颗粒继续反应，避免砌墙后膨胀开裂）

⚠️ **安全警告**：生石灰加水时温度可达 150摄氏度以上，沸腾飞溅！必须缓慢加水、保持距离、穿长袖长裤。

### 第三步：配制砂浆

| 用途 | 消石灰 : 沙 | 水 |
|------|-----------|-----|
| 砌墙 | 1 : 3 | 适量 |
| 抹面 | 1 : 2 | 适量 |
| 勾缝 | 1 : 1 | 适量 |

1. 将消石灰和沙干拌均匀
2. 逐步加水搅拌至可塑状态
3. 以"能捏成团、落地散开"为宜

### 第四步：使用

1. 用泥刀将砂浆涂抹在砖/石上
2. 砌筑后刮去多余砂浆
3. 保持墙面湿润 3-7 天（洒水养护，防止开裂）
4. 砂浆完全硬化需要 2-4 周

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 石灰石 | 滴醋冒泡，敲碎后无大量泥质夹杂 |
| 烧成 | 浇少量水后剧烈发热、粉化，说明生石灰烧透 |
| 熟化 | 缓慢加水，陈化 1-2 周后无硬颗粒继续膨胀 |
| 配比 | 砌墙消石灰:沙约 1:3，抹面约 1:2，拌后可塑不泌水 |
| 养护 | 砌后 3-7 天保持微湿，2-4 周逐渐硬化 |

## 失败征兆

- 石灰烧不透，水化时有硬芯，砂浆后期开裂。
- 生石灰一次大量加水，飞溅和沸腾会严重灼伤。
- 砂浆太稀流淌，太干不粘砖。
- 未陈化的石灰颗粒上墙后继续膨胀，墙面鼓包。
- 石灰窑在低洼无风处运行，二氧化碳和一氧化碳风险上升。

## 常见问题

- **Q**: 石灰砂浆和水泥有什么区别？
- **A**: 石灰砂浆硬化靠吸收 CO₂（碳化），水泥靠水化反应。石灰砂浆硬化慢但韧性好（允许轻微变形不开裂），水泥硬化快但脆。

- **Q**: 可以直接用泥浆砌墙吗？
- **A**: 可以，但石灰砂浆强度远高于纯泥浆。穿越后有了石灰，建筑质量会上一个台阶。

## 安全提醒

- 生石灰遇水剧烈放热，操作时戴手套、护目镜
- 不要将生石灰存放在潮湿处（会自行吸水粉化）
- 烧石灰窑产生大量 CO₂，注意通风

## 穿越小说梗

"主角发现了石灰石就开始烧石灰"——找石灰石不难（滴醋冒泡就行），难的是烧到 1000摄氏度并持续十几小时。这需要大量的燃料和一个还过得去的窑。不过一旦有了石灰，建筑质量将从"泥巴糊的"直接升级为"千年不倒的"。
