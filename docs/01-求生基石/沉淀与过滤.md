# 1.13 沉淀与过滤

> 浑水变清水，不是魔法，是物理。悬浮颗粒比水重就沉下去，比水轻就拦下来。

## 原理

浑水中含有两类杂质：
- **悬浮颗粒**（泥沙等）：可通过静置沉淀或过滤去除
- **溶解物**（盐、矿物质等）：沉淀和过滤无法去除，需蒸馏或化学处理

过滤的本质是**机械拦截**——让水通过孔径小于杂质颗粒的介质，把杂质挡住。

## 所需材料

| 材料 | 获取方式 | 替代方案 |
|------|---------|---------|
| 容器（两个） | 陶罐、木桶、坑 | 任何能装水的器物 |
| 沙子 | 河滩 | 碎石 |
| 细砂 | 河滩、细土筛选 | 火山灰、灶灰 |
| 木炭 | 烧火后的炭 | 草木灰 |
| 砾石/碎石 | 河滩 | 陶片 |
| 布 | 衣物 | 草束、树皮纤维 |

## 操作步骤

### 方法一：静置沉淀

最简单，只需要时间和容器。

1. 将浑水装入容器，静置
2. 等待 1-6 小时（取决于颗粒大小）
3. 泥沙沉底后，**轻轻倾倒**上层清水到另一个容器，避免搅起沉淀
4. 如仍浑浊，重复 1-2 次

**加速技巧**：加入少量**明矾**（见明矾净水条目），可将沉淀时间从数小时缩短到 30 分钟内。

### 方法二：布滤

1. 将干净的布（棉布最佳）覆盖在容器口
2. 缓慢倒入浑水
3. 布会拦住大颗粒，滤过较清的水
4. 布脏了就清洗或更换

**注意**：布滤只能去除大颗粒，滤出的水仍需煮沸消毒。

### 方法三：沙石滤层（推荐）

这是野外最实用的过滤方法，可去除大部分悬浮颗粒。

1. 找一个底部有小孔的容器（竹筒底部穿孔、塑料瓶剪底倒置、或挖一个底部有排水口的土坑）
2. 从下到上依次填入：
   - **最底层**：砾石/碎石 5厘米（防止细砂流失）
   - **中层**：粗砂 5厘米
   - **上层**：细砂 10厘米
   - **最上层**：木炭碎块 5厘米（吸附异味和部分杂质）
3. 在最上面再铺一层砾石防止水流冲刷
4. 缓慢倒入浑水，从底部接取滤出液
5. **前几批滤出液要倒掉**（会把滤层中的细粉冲出来），直到出水变清

### 方法四：多层串联过滤

对极度浑浊的水，可以串联使用：
1. 先布滤 → 去除大颗粒
2. 再沙滤 → 去除细颗粒
3. 最后木炭吸附 → 去除异味和部分有机物

## 过滤层效果

| 滤层材料 | 去除对象 | 去除不了 |
|---------|---------|---------|
| 布滤 | 昆虫、树叶、大颗粒泥沙 | 细菌、溶解物 |
| 沙滤 | 悬浮颗粒、泥沙 | 细菌、溶解物 |
| 木炭 | 异味、部分有机物、部分重金属 | 细菌、溶解盐 |
| 沉淀 | 大颗粒 | 细小悬浮物、细菌 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 沉淀 | 静置 1-6 小时后底部有泥沙层，上层水明显变清 |
| 布滤 | 能拦住树叶、虫体和大颗粒泥沙，布面可清洗 |
| 沙滤 | 前几批浑水弃掉后，出水不再带明显砂粒和泥色 |
| 滤层 | 砾石 5厘米、粗砂 5厘米、细砂 10厘米、木炭 5厘米以上 |
| 饮用 | 过滤后仍煮沸至少 1 分钟，高海拔按煮沸条目延长 |

## 失败征兆

- 过滤后水仍浑黄，说明滤层太薄、流速太快或细砂被冲散。
- 滤速突然很慢，说明上层堵塞，需要刮去表层或更换滤料。
- 沙滤器有臭味、虫卵或黏滑膜，说明滤器本身污染。
- 把过滤水直接喝，仍可能感染细菌、病毒和寄生虫。
- 水源有油膜、农药味或矿山污染，沉淀过滤不能解决。

## 常见问题

- **Q**: 过滤后的水可以直接喝吗？
- **A**: **不行**。过滤只去除了颗粒，没有杀灭细菌和寄生虫。过滤后仍需煮沸至少 1 分钟。

- **Q**: 沙滤器能反复用吗？
- **A**: 可以，但需要定期更换木炭层和最上层细砂（容易堵）。如果滤速明显变慢，说明需要清理。

## 安全提醒

- 过滤≠消毒，过滤后的水**必须煮沸**才能饮用
- 不要用刚烧完还热的木炭，冷却后再用
- 沙滤器不用时要保持湿润或干燥存放，半干半湿容易滋生细菌
