# 7.4 食材替换表

> 没有某种食材时，先替换“功能”，再替换“名字”：咸、鲜、酸、甜、油脂、淀粉、膨发、增稠。

## 基础调料替换

| 现代调料 | 古代替代方案 | 起始比例 | 效果差异 |
|---------|-------------|----------|----------|
| 精制面粉 | 石磨全麦粉，粗筛去麸 | 水量增加 5%-10% | 更吸水，口感粗 |
| 酱油 | 发酵酱加盐水稀释 | 酱 1 份:水 2-4 份 | 颜色浅，豆香重 |
| 味精/鸡精 | 干贝、海带、香菇煮水 | 干货 10克:水 500毫升 | 鲜味柔和 |
| 酵母粉 | 老面 | 面粉重量 10%-30% | 发酵慢，风味酸香 |
| 泡打粉 | 草木灰碱水 + 酸味食材 | 极少量试加 | 碱味难控 |
| 料酒 | 米酒、黄酒 | 等量替换 | 去腥接近 |
| 玉米淀粉 | 红薯淀粉、葛根粉、藕粉 | 等量起步 | 透明度和黏度不同 |
| 白糖 | 饴糖、蜂蜜、浓缩果汁 | 甜度试加 | 含水高，需减水 |

## 主食与蛋白替换

| 现代食材 | 古代替代 | 操作要点 |
|---------|---------|----------|
| 大米饭 | 小米饭、麦饭、豆饭 | 杂粮先泡 2-8 小时 |
| 猪肉 | 野猪肉、家养猪 | 野味必须熟透，炖煮更久 |
| 牛肉 | 黄牛、水牛、羊肉 | 古代耕牛珍贵，剧情需考虑律令 |
| 鸡胸肉 | 土鸡、野鸡、兔肉 | 肉质紧，适合炖或撕丝 |
| 植物油 | 猪油、芝麻油、菜籽油 | 猪油香但冷后凝固 |
| 牛奶 | 羊奶、豆浆 | 羊奶膻，豆浆需煮沸 |
| 奶油 | 猪油、羊脂、芝麻酱 | 只能替代油润，不能完全乳化 |

## 替换决策步骤

1. 先判断原料功能：提供结构、香味、油脂、甜味、酸味还是鲜味。
2. 从少量开始替换：调料先按 1/2 量加入，试味后再补。
3. 涉及发酵、腌制、膨发时，不只看味道，还要看盐度、酸度和气泡。
4. 记录每次替换比例，能复现才算成功。

## 成功标准

- 味道方向接近原菜，不因替代品掩盖主料。
- 质地能成立：饭熟、面发、汤稠、肉不腥。
- 同一比例重复 3 次，结果差异不大。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 碱味、涩味重 | 草木灰水或野菜过量 | 加酸味食材，下一批减量 |
| 面团发不起来 | 老面少或温度低 | 增加老面，放温暖处延长时间 |
| 勾芡结块 | 淀粉直接入热锅 | 冷水调浆后再倒入 |
| 鲜汤腥苦 | 干货未泡洗或煮太久 | 先泡洗，缩短熬煮 |

## 安全边界

- 不用不能确认种类的野菜、蘑菇、矿物粉替代现代食材。
- 野味、禽肉、豆浆必须充分加热；“尝起来熟了”不是安全标准。
- 草木灰碱水只作为少量工艺材料，不直接当饮食调味品。

## 穿越小说梗

这个替换表本身就是金手指：古代厨师缺的不是聪明，而是“按功能替换、做小样、记录比例”的系统方法。
