# 7.4 食材替换表

> 没有X怎么办——古今食材对照手册，穿越厨房必备。

## 基础调料替换

| 现代调料 | 古代替代方案 | 获取难度 | 效果差异 |
|---------|-------------|---------|---------|
| 精制面粉 | 石磨全麦粉（粗筛即可） | 低 | 口感稍粗但营养更好 |
| 酱油 | 发酵酱加盐水稀释 | 高（需先做酱） | 颜色浅、味道更浓郁 |
| 味精/鸡精 | 干贝/海带/香菇煮水 | 低 | 天然鲜味，没有味精那种刺激感 |
| 酵母粉 | 老面（上次发酵留的面团） | 低 | 发酵慢一些，但风味更好 |
| 泡打粉 | 草木灰水（碱性，与酸反应膨发） | 低 | 需搭配酸性食材（如酸奶） |
| 料酒 | 米酒/黄酒 | 中 | 去腥效果一样好 |
| 小苏打 | 草木灰淋灰水浓缩结晶 | 中 | 纯度和碱性略有差异 |
| 玉米淀粉 | 红薯淀粉/葛根粉/藕粉 | 中 | 勾芡效果类似 |

## 食材替换

| 现代食材 | 古代替代 | 备注 |
|---------|---------|------|
| 猪肉 | 野猪肉/家养猪 | 圈养穿越后需半年起步 |
| 牛肉 | 黄牛肉/水牛肉 | 古代牛更珍贵，别乱宰 |
| 鸡胸肉 | 土鸡/野鸡 | 肉质更紧实，炖煮需要更长时间 |
| 大米饭 | 小米饭/麦饭/豆饭 | 口感不同但能吃饱 |
| 植物油 | 猪油/芝麻油/菜籽油 | 猪油最易获取 |
| 牛奶 | 羊奶 | 山羊奶更常见，风味不同 |
| 奶油 | 动物脂肪炼油 | 没有乳化效果但能替代油脂 |

## 穿越小说梗

这个替换表本身就是金手指——古代厨师不知道没有味精可以用干贝提鲜，不知道草木灰水可以替代泡打粉。
