# 7.5 食物保存古代方案

> 在没有冰箱的时代，保存食物是一门生死攸关的技术。

## 保存方案对照

| 方法 | 适用食材 | 保存时间 | 所需材料 | 难度 |
|------|---------|---------|---------|------|
| 盐渍 | 肉、鱼、蔬菜 | 数月至数年 | 大量盐 | 低 |
| 烟熏 | 肉、鱼、香肠 | 数周至数月 | 烟熏架+木屑 | 中 |
| 风干/晒干 | 肉干、鱼干、果干、蔬菜干 | 数周至半年 | 干燥通风环境 | 低 |
| 发酵 | 泡菜、腌菜、酱 | 数月 | 盐+容器 | 低 |
| 糖渍/蜜饯 | 水果 | 数月至年 | 大量糖/蜂蜜 | 中 |
| 窖藏 | 根茎类蔬菜 | 数周至半年 | 地窖 | 中 |
| 油封 | 肉、蔬菜 | 数月 | 油脂+密封容器 | 中 |
| 冰藏 | 易腐食材 | 数天至数周 | 冰窖/冰+保温 | 高（需有冰源） |

## 操作步骤

### 盐渍法（最基础）
1. 食材处理：肉切条、鱼去内脏、蔬菜洗净
2. 涂抹盐：食材表面均匀抹盐（肉约用食材重量10%~15%的盐）
3. 容器：陶罐或木桶，一层食材一层盐
4. 密封：顶部多铺一层盐，盖上盖子
5. 食用前：清水浸泡脱盐

### 烟熏法（风味最佳）
1. 先盐渍24小时脱水
2. 挂通风处晾干表面
3. 冷熏（30~40°C）：适合长期保存，需连续熏数天
4. 热熏（60~80°C）：适合短期保存，数小时即可

## 搭配策略

> 合理搭配保存方案：盐渍肉 + 窖藏菜 + 发酵酱，确保冬天不断粮。
