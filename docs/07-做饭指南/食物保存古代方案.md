# 7.5 食物保存古代方案

> 没有冰箱时，保存食物靠四件事：降低水分、提高盐糖酸、隔绝污染、控制温度。

## 保存方案对照

| 方法 | 适用食材 | 关键比例/温度 | 保存目标 | 主要风险 |
|------|----------|---------------|----------|----------|
| 盐渍 | 肉、鱼、蔬菜 | 肉鱼常用 10%-15% 盐作传统重盐 | 数周到数月 | 内部未腌透 |
| 乳酸发酵 | 菜、豆、酱 | 蔬菜盐度常见 2%-3% | 数周到数月 | 露出液面发霉 |
| 风干/晒干 | 肉干、鱼干、果干 | 表面干燥，内部无湿软 | 数周到半年 | 回潮霉变 |
| 烟熏 | 肉、鱼 | 先盐渍脱水，再熏 | 数周到数月 | 只熏不脱水 |
| 糖渍/蜜饯 | 水果 | 高糖、低水分 | 数月至一年 | 糖度不足发酵 |
| 窖藏 | 根茎、瓜果 | 阴凉、避光、通风 | 数周到半年 | 积水、鼠害 |
| 油封 | 熟肉、熟菜 | 食材全浸没，容器干净 | 短到中期 | 肉毒风险，不作长期常温万能法 |

## 盐渍肉鱼

1. 肉切 2-4厘米厚条，鱼去鳃去内脏，擦干表面水。
2. 按食材重量 10%-15% 取盐，内外均匀揉搓。
3. 陶罐中一层食材一层盐，顶部加重物压出水。
4. 腌 1-3 天后取出，挂在通风处风干或继续烟熏。
5. 食用前浸泡脱盐，再彻底加热。

## 发酵蔬菜

1. 蔬菜洗净沥干，切块或切丝。
2. 按蔬菜和水总重量的 2%-3% 加盐，揉出汁或配盐水。
3. 全部压在盐水下，液面高过食材 2厘米以上。
4. 18-24摄氏度放置 3-14 天，冒泡、变酸后转阴凉处。

## 成功标准

- 盐渍品：表面无黏滑臭味，切开内部也有咸味。
- 发酵菜：酸香、气泡细密，食材始终在液面下。
- 干制品：弯折有韧性，无湿心、无霉点。
- 窖藏：温度波动小，无积水，坏果能及时挑出。

## 失败征兆与处理

| 征兆 | 可能原因 | 处理 |
|------|----------|------|
| 腐臭、黏液 | 盐不足或温度高 | 丢弃，不尝味 |
| 黑绿毛霉 | 食材露出液面 | 大面积霉变丢弃 |
| 肉中心发灰臭 | 块太厚未腌透 | 丢弃，下一批切薄 |
| 罐内鼓胀喷液 | 异常发酵 | 隔离开启，勿食 |

## 安全边界

- 低盐、油封、真空、常温长期保存不能随意组合，肉毒风险高。
- 婴幼儿、老人、病人不要吃可疑发酵和腌制品。
- 任何保存食品只要出现腐臭、黏滑、霉变深入、异常胀气，直接丢弃。
