# 7.3 调味料从零获取

> 穿越后没有超市，所有调味料都得从原材料做起。

## 调味料制作时间线

不同调味料需要的时间差异很大，提前布局是关键。

| 调味料 | 制作周期 | 原料 | 难度 |
|--------|---------|------|------|
| 盐 | 当天~3天 | 海水/盐井/盐湖 | 低 |
| 蜂蜜 | 找到蜂巢即得 | 野生蜂巢 | 中（防蜂蜇） |
| 醋 | 1~3个月 | 酒+醋酸菌 | 中 |
| 米酒/黄酒 | 1~3个月 | 米+酒曲 | 中 |
| 酱油 | 3~6个月 | 大豆+小麦+盐+曲 | 高 |
| 饴糖（麦芽糖） | 1~2天 | 麦芽+糯米 | 低 |
| 味精替代 | 当天 | 干贝/海带/蘑菇 | 低 |

## 获取步骤

### 初期（穿越前3天）
1. 先搞定盐——海盐晒制或煮盐
2. 采集野生香料——花椒、八角（如果当地有）、紫苏、姜
3. 找蜂蜜——烟熏蜂巢取蜜

### 中期（第1~3周）
1. 酿第一批醋——用米酒做基底，加醋引子
2. 制饴糖——麦芽+糯米煮制
3. 准备做酱油的原料——大豆蒸熟制曲

### 长期（第1~6个月）
1. 酱油发酵——定期翻晒，至少3个月
2. 黄豆酱/豆瓣酱——发酵越久越香

## 穿越小说梗

穿越者知道发酵时间并提前布局——别人刚想起要酱油时，你已经有了发酵半年的成品。
