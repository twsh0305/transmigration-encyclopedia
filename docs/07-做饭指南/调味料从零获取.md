# 7.3 调味料从零获取

> 调味料不是一夜变出来的。盐当天能有，酱油要按月算，所以厨房规划要有时间线。

## 调味料时间线

| 调味料 | 制作周期 | 原料 | 起步标准 |
|--------|----------|------|----------|
| 盐 | 当天-3 天 | 海水、盐井、盐湖 | 能结晶、无泥砂 |
| 鲜味汤 | 当天 | 干贝、海带、蘑菇、骨头 | 小火煮 30-120 分钟 |
| 蜂蜜 | 找到蜂巢即得 | 野生蜂巢 | 过滤蜡渣和死蜂 |
| 饴糖 | 1-2 天 | 麦芽、熟米或糯米 | 甜、黏、无焦苦 |
| 米酒/黄酒 | 1-3 个月 | 米、曲、水 | 有酒香，无霉臭 |
| 醋 | 1-3 个月 | 酒、醋母、空气 | 酸香稳定 |
| 酱油/豆酱 | 3-6 个月以上 | 大豆、小麦、盐、曲 | 咸鲜、无败臭 |

## 初期 3 天

1. 先制盐或找盐：没有盐，保存、发酵和体力都受影响。
2. 收集姜、葱、蒜、紫苏、花椒等本地香辛物。不能确认种类的不入口。
3. 做鲜味汤：骨头或干货加水，小火煮到汤色变浓，作为当天提味。
4. 蜂蜜只做过滤，不高温久煮，避免焦化和香味损失。

## 第 1-3 周

1. 留老面，建立稳定面食发酵来源。
2. 做饴糖：麦芽提供糖化能力，熟米饭保温糖化后过滤熬浓。
3. 开第一坛醋：酒液放宽口容器，覆布防虫，保持通气。
4. 准备酱曲原料：大豆蒸熟，小麦炒香碾碎，保持器具清洁。

## 第 1-6 个月

1. 酱和酱油要定期翻晒，闻味、看色、查霉。
2. 盐水浓度要足，表面可有正常酱香膜，但黑毛、腐臭、蛆虫为失败。
3. 成品过滤后煮沸装罐，降低继续败坏风险。

## 成功标准

- 盐：白色或微灰，溶水后泥沙能沉降，咸味纯正。
- 醋：酸味明显，无腐臭和酒精刺鼻味。
- 酒：有酒香，不苦不霉。
- 酱：咸鲜厚重，表面无黑绿毛霉。

## 失败征兆与处理

| 征兆 | 可能原因 | 处理 |
|------|----------|------|
| 酱坛腐臭 | 杂菌污染或盐不足 | 丢弃，不混入新坛 |
| 酒液发霉 | 容器不洁或温度不稳 | 小样失败则弃 |
| 醋不酸 | 酒精不足或醋酸菌弱 | 加入成熟醋引子，延长通气 |
| 饴糖焦苦 | 火太大 | 加水重熬无效，下一批小火 |

## 安全边界

- 发酵失败品不靠加香料掩盖。
- 蜂巢采集要防蜂蜇；对蜂毒过敏者不参与。
- 盐、醋、酱都不能弥补腐败肉类的安全问题。

## 穿越小说梗

穿越者知道发酵时间并提前布局：别人刚想起要酱油时，你已经有了编号、日期、盐度和翻晒记录。
