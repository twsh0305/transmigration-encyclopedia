# 第七卷·做饭指南

> 现代厨房不能搬回古代，但做饭的原理是一样的。本卷教你用古代条件做出现代美食。

## 使用顺序

1. 先建厨房：排烟、取水、生熟分区和灭火点优先于菜谱。
2. 再定主食：米、麦、豆、薯哪个稳定，就以哪个为日常热量来源。
3. 接着做盐、油、酸、鲜味汤，解决“能吃”和“愿意吃”。
4. 最后布局长期发酵和保存，酱油、醋、酱、腊肉都要按周月规划。

## 成功标准

- 熟：主食无硬心，肉禽鱼最厚处无血水。
- 净：生食、熟食、污水、柴灰不交叉。
- 稳：同一菜式能按记录重复做出。
- 安：没有烟气中毒、烫伤、火灾和腐败食物入口。
- 温度：禽肉最厚处约 74摄氏度；肉馅约 71摄氏度；整块肉和鱼约 63摄氏度。

## 失败征兆

- 做饭时眼刺痛、头痛、犯困：优先怀疑排烟和一氧化碳。
- 食物酸臭、黏滑、胀气、长黑绿毛：不靠重煮挽救。
- 同一锅里先切生肉再盛熟食：生熟交叉，整餐风险升高。

## 安全边界

- 生食和熟食分刀板、分容器、分水盆；做不到就先简化菜谱。
- 室内炭火和柴火必须有稳定排烟，头痛、眩晕、恶心时先撤人再灭火。
- 腐败、霉变、胀气的食物不靠油炸、重煮、加酒或加辣掩盖。
