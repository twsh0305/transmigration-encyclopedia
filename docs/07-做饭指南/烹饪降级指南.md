# 7.2 烹饪降级指南

> 有电到无电，有锅到无锅——穿越者的烹饪降级兼容指南。

## 厨具降级对照

| 现代厨具 | 替代方案 | 替代难度 | 效果 |
|---------|---------|---------|------|
| 电饭煲 | 陶锅+慢火焖煮 | 低 | 米饭口感更好但容易糊底 |
| 冰箱 | 地窖+冰块/井水降温 | 中 | 温度不够低，保质期短 |
| 燃气灶 | 土灶+木柴 | 低 | 火候控制需要练习 |
| 烤箱 | 土窑/炭火烤 | 中 | 温度不均匀需翻面 |
| 微波炉 | 蒸笼隔水蒸/油锅煎 | 低 | 速度慢但效果不差 |
| 菜刀 | 石刀/铁匠打刀 | 低 | 不锋利但能用 |
| 破壁机/榨汁机 | 石臼/石磨 | 低 | 费体力但效果类似 |
| 不粘锅 | 铁锅/陶锅（养出油膜） | 低 | 养好油膜后也不粘 |

## 无器具烹饪法

连锅都没有的情况下，这些方法能让你吃上熟食：

### 石板烧
1. 找一块平整薄石板，架在火堆上烧热
2. 肉直接放石板上烤熟（类似铁板烧）
3. 石头可选砂岩或花岗岩，不要用河滩卵石（遇热开裂）

### 竹筒饭
1. 新鲜竹筒一节，一端留节一端开口
2. 米+水装入（米水比约1:1.5），略留空隙
3. 开口用草叶塞紧或湿泥封口
4. 扔进火堆侧边，烤20~30分钟
5. 劈开竹筒食用（竹膜不会粘饭）

### 叫花鸡
1. 鸡/鱼去内脏，抹盐和香料
2. 荷叶或大叶子包裹
3. 外层裹湿泥（厚约2~3cm）
4. 埋入炭火堆中，烤40~60分钟
5. 敲开泥壳，羽毛随泥脱落

## 穿越小说梗

降级指南体现系统化思维——不是对着遗迹哭，而是拿起竹筒做饭。
