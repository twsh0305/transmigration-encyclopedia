# 7.2 烹饪降级指南

> 降级烹饪不是将就，而是把现代厨具拆成基本功能：加热、保温、密封、搅碎、过滤、冷却。

## 厨具降级对照

| 现代厨具 | 替代方案 | 操作要点 | 失败征兆 |
|---------|----------|----------|----------|
| 电饭煲 | 陶锅慢火焖 | 米水约 1:1.2-1.5，沸后小火 | 糊底、夹生 |
| 冰箱 | 地窖、井水、蒸发冷却 | 阴凉、通风、离地 | 发黏、酸臭 |
| 燃气灶 | 土灶、炭炉 | 先烧旺再控风 | 黑烟倒灌 |
| 烤箱 | 土窑、炭火坑 | 预热后靠余温 | 外焦内生 |
| 微波炉 | 蒸、煎、回锅 | 少量加水防干 | 中心不热 |
| 菜刀 | 石刀、骨刀、粗铁刀 | 多用砸、剁、撕 | 滑刀伤手 |
| 破壁机 | 石臼、石磨 | 少量多次 | 粗细不匀 |
| 不粘锅 | 养好的铁锅、陶锅 | 热锅冷油 | 粘锅焦糊 |

## 石板烧

1. 选干燥、平整、无裂纹石板，厚 2-4厘米。
2. 先在火旁烘 10-20 分钟，再逐渐移到火上。
3. 滴水迅速蒸发即可放食材。
4. 肉片切薄到 3-5毫米，勤翻面。

成功标准：肉片无血水、表面焦香、石板不爆裂。河滩卵石、潮湿石头、夹层页岩不直接上火。

## 竹筒饭

1. 新鲜竹筒一节，一端留节，一端开口。
2. 米先浸 30-60 分钟，米水比约 1:1.3-1.5。
3. 装到竹筒 2/3 满，留膨胀空间。
4. 开口用叶片和湿泥封住，放火堆边烤 20-30 分钟。
5. 听到内部水声变弱、竹香明显时离火焖 5-10 分钟。

失败征兆：开筒后米心白硬是水少或时间短；底部焦黑是火太猛。

## 泥包烤

1. 鸡、鱼或薯类处理干净，抹盐。
2. 外包可食大叶或干净草叶，避免泥直接接触食物。
3. 湿泥厚 2-3厘米，完全包住。
4. 埋入炭灰中。小鱼 20-30 分钟，整鸡 60-90 分钟。
5. 取出后静置 10 分钟再敲开。

## 成功标准

- 主食熟透无硬心。
- 肉类中心无血水，汁液清亮。
- 烹饪器具没有裂、炸、倒、漏。

## 安全边界

- 不用潮湿密闭石头直接入火，爆裂会伤人。
- 任何肉、鱼、禽都要熟透；没有温度计时切开最厚处检查。
- 野外烹饪后火灰要完全熄灭，灰下暗火可持续数小时。

## 穿越小说梗

降级指南体现系统化思维：不是对着遗迹哭，而是知道每个现代厨具背后的功能，再找古代材料替代。
