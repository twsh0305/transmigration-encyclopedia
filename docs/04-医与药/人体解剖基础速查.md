# 4.1 人体解剖基础速查

> 你不需要成为外科医生，但你需要知道心脏在哪、血管怎么走、骨头怎么接。

## 原理

人体解剖基础是穿越者急救和基础医疗的**导航地图**。不用学全，但关键部位必须知道：主干血管位置、主要脏器投影、骨折常见部位。

## 骨骼系统速查

| 骨骼 | 数量 | 主要功能 | 常见骨折 |
|------|------|---------|---------|
| 颅骨 | 8片 | 保护大脑 | 颅顶骨折（严重！） |
| 锁骨 | 2根 | 连接肩与胸 | ⭐ 摔倒伸手撑地→易骨折 |
| 肱骨（上臂） | 2根 | — | ⭐ 中等 |
| 尺骨+桡骨（前臂） | 4根 | — | ⭐ 摔倒时撑地→最常骨折 |
| 肋骨 | 24根（12对） | 保护心肺 | 挤压伤/撞击 |
| 骨盆 | 1个 | 支撑+保护 | 坠落/撞击→大出血风险 |
| 股骨（大腿） | 2根 | 承重 | ⭐ 老年人最常见 |
| 胫骨+腓骨 | 4根 | 承重 | ⭐ 胫骨前缘无肌肉保护→易骨折 |

## 重要血管位置

**动脉出血最危险**——鲜红色、喷射状、出血量极大：

| 动脉 | 位置 | 按压止血点 |
|------|------|-----------|
| 颈总动脉 | 颈部气管旁 | 用力压向颈椎（慎用→脑缺氧） |
| 锁骨下动脉 | 锁骨内后方 | 按压锁骨上凹 |
| 肱动脉 | 上臂内侧 | 肱二头肌内侧沟 |
| 股动脉 | 大腿根部腹股沟 | 用力压向骨盆 |
| 桡动脉 | 手腕 | 中医把脉位置 |
| 腘动脉 | 膝弯 | 按压膝后方 |

**静脉出血**——暗红色、持续涌出——压迫止血效果好。

## 生命体征速查

| 指标 | 正常值 | 危险值 |
|------|--------|--------|
| 心率 | 60-100 次/分 | >120 或 <50 |
| 呼吸 | 12-20 次/分 | >30 或 <8 |
| 体温 | 36-37摄氏度 | 39摄氏度以上 发热 / 35摄氏度以下 低体温 |
| 瞳孔 | 双侧等大 2-4毫米 | 一大一小→脑损伤 |
| 意识 | 清醒 | 昏迷→危险 |

## 成功标准

| 场景 | 合格判断 |
|------|----------|
| 外伤初评 | 先看意识、呼吸、大出血，再看疼痛部位 |
| 生命体征 | 能在 1 分钟内数出心率和呼吸频率，并记录时间 |
| 大出血 | 识别鲜红喷射、暗红涌出和渗血的差异 |
| 腹部损伤 | 左上腹撞击后警惕脾破裂，右下腹持续痛警惕阑尾炎 |
| 头部损伤 | 记录“受伤时间、是否昏迷、是否呕吐、瞳孔变化” |

## 失败征兆

- 只盯着伤口大小，忽略意识、呼吸和循环。
- 伤者皮肤苍白、冷汗、口渴、脉快弱，却误判为“休息一下就好”。
- 头伤后短暂清醒又嗜睡，是颅内出血危险信号。
- 腹痛伴腹壁发硬、头晕或肩部牵涉痛，不应揉肚子或强行进食。
- 不懂解剖位置就盲目按压颈部、腹股沟或胸腹，可能加重损伤。

## 常见问题

- **Q**: 我需要学多少解剖知识才够用？
- **A**: 核心五条：①心脏在胸骨偏左（心肺复苏按压点）②锁骨下有大动脉（肩颈受伤大量出血时按压）③脾脏在左腹（左下肋骨撞击后警惕内出血）④阑尾在右下腹（右下腹痛考虑阑尾炎）⑤大腿根部有大动脉（下肢大出血时压这里）。

- **Q**: 内出血怎么判断？
- **A**: 撞击或跌落后出现：①面色苍白、头晕、口渴②脉搏快而弱③腹痛且腹壁硬→极可能是内出血，几乎无法自救。

## 安全提醒

- 不要随意切开身体探查——没有无菌条件切开腹腔=死亡
- 怀疑内出血：保温、平卧、禁食禁水、尽快就医
- 头部受伤后出现"清醒→嗜睡→昏迷"→颅内出血征象

## 穿越小说梗

小说里"主角切开病人的肚子，三下五除二切掉了坏掉的器官"——穿越环境中**开腹手术生存率极低**，主要原因不是技术，而是感染。在没有抗生素、没有无菌术、没有麻醉的环境下，切开腹膜几乎等于宣判死亡。穿越者更重要的是**诊断能力**（知道哪出了毛病）和**保守治疗能力**（协助身体自愈），而不是外科手术。
