# 4.18 饮水卫生与水源保护

> 穿越者最容易犯的错误不是打不过野兽，而是喝了一口没烧过的水。

## 原理

水中的致病体主要包括：

| 类型 | 典型病原 | 来源 | 导致的疾病 |
|------|---------|------|-----------|
| 细菌 | 霍乱弧菌、伤寒杆菌、大肠杆菌 | 粪便污染 | 霍乱、伤寒、腹泻 |
| 病毒 | 甲肝病毒、诺如病毒 | 粪便污染 | 肝炎、呕吐 |
| 原虫 | 隐孢子虫、贾第鞭毛虫 | 动物/人粪便 | 顽固腹泻 |
| 寄生虫 | 血吸虫、蛔虫卵 | 水中幼虫/卵 | 血吸虫病等 |

| 材料 | 用途 |
|------|------|

## 操作步骤

### 一、水源选择评估

| 水源类型 | 风险等级 | 处理难度 |
|---------|---------|---------|
| 流动溪水（上游） | ⭐ | 低 |
| 深井水 | ⭐⭐ | 低 |
| 流动溪水（下游有人烟） | ⭐⭐⭐⭐ | 中 |
| 池塘/湖泊 | ⭐⭐⭐ | 中 |
| 雨水收集 | ⭐ | 如果容器干净 |
| 雪融水 | ⭐ | 低 |
| 河水（下游城镇） | ⭐⭐⭐⭐⭐ | 高 |

### 二、净化方法（按优先级排序）

**方法一：煮沸（最可靠）**
1. 水烧至**剧烈沸腾**
2. 持续沸腾至少 1 分钟（海拔高时 3 分钟）
3. 冷却后饮用
4. 优点：杀灭一切致病微生物
5. 缺点：需要燃料

**方法二：沉淀 → 过滤 → 煮沸（完整流程）**
1. **沉淀**：浑浊水静置 2-4 小时
2. **沙滤**：用木桶做滤水器（从下到上：细沙→木炭→碎石→粗沙）
3. **煮沸**：过滤后水仍然需要煮沸

**方法三：明矾沉淀（只除浑浊，不杀菌）**
1. 每 10L 水加 1-2g 明矾
2. 搅拌 3 分钟后静置 1 小时
3. 上层清水倒出——**仍需煮沸**

**方法四：太阳能消毒（SODIS，无燃料）**
1. 透明瓶装水，放在阳光直射处 6 小时
2. 紫外线杀灭大部分病原体
3. 仅限清澈水，浊水无效

### 三、水源保护

1. **厕所远离水源**：厕所距离任何水源至少 30 米
2. **下游取水**：人用水取上游，洗涮/排放下游
3. **水井保护**：井口加高（30cm+）并加盖，防止地面水流入
4. **动物远离**：不让牲畜靠近饮用水源
5. **定期清淤**：水井/水池每年清理一次

## 常见问题

- **Q**: 看起来干净的水还需要烧吗？
- **A**: 必须烧。清澈不等于无菌——霍乱病人的粪便污染水源后，水看起来完全可以清澈透明。

- **Q**: 没有燃料天天烧水怎么办？
- **A**: 太阳能消毒（SODIS）是替代方案，或建造大型太阳能蒸馏器。但最好的策略是：一次烧大量水存储。

## 安全提醒

- 千万不要直接喝河水、溪水——即使看起来很干净
- 储水容器要加盖，防止动物污染
- 储水超过 2 天需要重新煮沸

## 穿越小说梗

"穿越者永远不会拉肚子"——这不太合理。穿越到一个没有现代卫生设施的时代，第一周基本都会水土不服。但你比古人多知道一件事：**腹泻多半来自水**。所以穿越后第一件事不是盖房子，而是**建一个无污染的饮用水系统**。上游取水、烧开了喝、厕所远离水源——这三个简单的操作能让穿越者的生存概率翻倍。
