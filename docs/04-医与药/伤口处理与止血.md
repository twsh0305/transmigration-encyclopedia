# 4.3 伤口处理与止血

> 在古代，一个小小的伤口就足以致命——不是因为伤太重，而是因为感染。

## 原理

伤口处理的核心目标：

1. **止血**：中断血液流出，防止失血过多
2. **清创**：去除污物和坏死组织
3. **防感染**：创造不利于细菌繁殖的环境
4. **促进愈合**：保持湿润、清洁、稳定的环境

## 所需材料

| 材料 | 用途 | 获取方式 |
|------|------|---------|
| 清洁布料/麻布 | 包扎 | 煮沸消毒 |
| 清水（煮沸后冷却） | 清洗 | 煮开 |
| 盐 | 生理盐水 | 精细盐 9克/升 水 |
| 蜂蜜 | 抗菌敷料 | 养蜂或采集 |
| 高度酒（>50%） | 擦拭伤口周围皮肤 | 不倒入深伤口 |
| 干净针线 | 专业缝合条件不足时不建议使用 | 深大伤口优先压迫、覆盖、固定和转运 |
| 绷带 | 加压止血 | 布条 |
| 夹板 | 骨折固定 | 木片/竹片 |

## 操作步骤

### 一、止血

**按压止血**：
1. 用清洁布块直接按压伤口
2. 持续按压 10-15 分钟（不要反复掀开查看）
3. 如果血渗透了布料，再加一层在上面按

**抬高受伤部位**：高于心脏水平，减少血流

**指压动脉止血**（大出血时）：
- 手臂出血→按压上臂内侧肱动脉
- 腿部出血→按压腹股沟股动脉
- 头部出血→按压耳前颞动脉

**止血带（仅限危及生命的四肢大出血）**：
1. 在伤口上方 5-10厘米处绑扎
2. 记录绑扎时间
3. 绑紧到出血停止、远端脉搏消失
4. 不要自行反复松开；交给后续救治者处理

### 二、清创

1. 用大量清洁水冲洗伤口（去除泥土、碎石）
2. 用稀盐水（9克盐/1升水）冲洗
3. 只移除表面松动的泥沙和小碎屑；深嵌异物不要硬拔，固定后求助
4. 伤周皮肤可用酒或高度蒸馏酒擦拭，酒精不要倒入深伤口

### 三、包扎

1. 小伤口：清洁敷料覆盖+包扎
2. 较大伤口：先止血、冲洗、覆盖、固定，尽快转运；没有专业条件不要强行缝合
3. 蜂蜜敷料：将蜂蜜涂在敷料上再覆盖伤口（古代有效抗菌方法）
4. 包扎力度：固定而不影响血供

### 四、深大伤口的边界

缝合是医疗操作，不是普通手工活。动物咬伤、粪土污染、深刺伤、挤压伤、超过 6-8 小时的脏伤口、能看到脂肪/肌腱/骨头的伤口，都不应强行缝死。低资源环境的目标是止血、冲洗、覆盖、固定、记录受伤时间和污染来源，并尽快交给有经验的医者处理。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 止血 | 直接按压 10-15 分钟不频繁掀开，血透布时只加新布继续压 |
| 大出血 | 四肢喷射性出血用止血带并记录时间，不反复松开 |
| 冲洗 | 可见泥沙、木刺、碎石被大量清洁水冲走 |
| 包扎 | 绷带固定但远端手指脚趾仍温暖、颜色正常、有感觉 |
| 观察 | 24-48 小时红肿不扩大、无脓液、无发热和红线向上蔓延 |

## 失败征兆

- 伤口仍喷血或浸透多层布，是危及生命的大出血。
- 包扎后远端发紫、冰冷、麻木，说明包得太紧。
- 泥土、粪便、动物咬伤被直接缝合，感染和破伤风风险极高。
- 化脓后自行深切探查，可能伤到血管神经。
- 发热、红肿扩大、恶臭、红线向心蔓延，是严重感染信号。

## 常见问题

- **Q**: 伤口化脓了怎么办？
- **A**: 浅表小脓点可热敷促排并保持清洁；深部脓肿、发热、红肿扩大时需要医疗处理。不要在不了解解剖的情况下深切探查。

- **Q**: 没有麻药能不能硬缝？
- **A**: 不建议。疼痛会导致挣扎、污染和二次损伤；深大伤口优先止血、覆盖、固定和转运。柳树皮等止痛材料剂量不稳，不能替代麻醉和清创条件。

- **Q**: 裂开的伤口不缝合行吗？
- **A**: 小伤口可以在清洁覆盖下自然愈合；超过 1 厘米、张力大、污染重、出血不止或位于脸手关节处的裂口，需要有经验者评估，不要只为“合上皮肤”而封住污染。

## 安全提醒

- 动物咬伤：不要缝合！开放清创，大量冲洗，暴露伤口
- 伤口沾了土壤/粪便：破伤风风险极高，务必彻底清创
- 自己处理时量力而行，大出血须求助他人
- 止血带用于保命，不用于普通小出血；一旦使用，清楚记录时间

"主角穿越后用现代急救知识救了个人，从此被奉为神医"——这是经典桥段。但注意：①给你的针线**消毒**②蜂蜜是穿越者能找到的最好的抗菌敷料③古今伤口感染率之差主要来自**清创的彻底程度**，不是你用什么"神药"。现代人最大的优势是知道**感染是细菌造成的**，所以会洗手、会消毒、会包扎干净。
