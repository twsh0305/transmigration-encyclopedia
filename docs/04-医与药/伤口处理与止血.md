# 4.3 伤口处理与止血

> 在古代，一个小小的伤口就足以致命——不是因为伤太重，而是因为感染。

## 原理

伤口处理的核心目标：

1. **止血**：中断血液流出，防止失血过多
2. **清创**：去除污物和坏死组织
3. **防感染**：创造不利于细菌繁殖的环境
4. **促进愈合**：保持湿润、清洁、稳定的环境

## 所需材料

| 材料 | 用途 | 获取方式 |
|------|------|---------|
| 清洁布料/麻布 | 包扎 | 煮沸消毒 |
| 清水（煮沸后冷却） | 清洗 | 煮开 |
| 盐 | 生理盐水 | 精细盐 9g/L 水 |
| 蜂蜜 | 抗菌敷料 | 养蜂或采集 |
| 高度酒（>50%） | 消毒 | 蒸馏酒 |
| 干净的针线 | 缝合 | 骨针+麻线消毒 |
| 绷带 | 加压止血 | 布条 |
| 夹板 | 骨折固定 | 木片/竹片 |

## 操作步骤

### 一、止血

**按压止血**：
1. 用清洁布块直接按压伤口
2. 持续按压 10-15 分钟（不要反复掀开查看）
3. 如果血渗透了布料，再加一层在上面按

**抬高受伤部位**：高于心脏水平，减少血流

**指压动脉止血**（大出血时）：
- 手臂出血→按压上臂内侧肱动脉
- 腿部出血→按压腹股沟股动脉
- 头部出血→按压耳前颞动脉

**止血带（仅限危及生命的四肢大出血）**：
1. 在伤口上方 5-10cm 处绑扎
2. 记录绑扎时间
3. 每 30 分钟松开一次恢复血供
4. 连续使用不超过 2 小时

### 二、清创

1. 用大量清洁水冲洗伤口（去除泥土、碎石）
2. 用稀盐水（9g 盐/1L 水）冲洗
3. 检查是否有异物嵌入，用消毒镊子取出
4. 伤周用酒或高度蒸馏酒擦拭消毒

### 三、包扎

1. 小伤口：清洁敷料覆盖+包扎
2. 较大伤口：缝合后再包扎
3. 蜂蜜敷料：将蜂蜜涂在敷料上再覆盖伤口（古代有效抗菌方法）
4. 包扎力度：固定而不影响血供

### 四、缝合（深大伤口）

1. 针线在沸水中煮 10 分钟消毒
2. 缝合时从皮缘外 3-5mm 进针
3. 间距约 5mm
4. 打结不要留在伤口表面
5. 7-14 天后拆线

## 常见问题

- **Q**: 伤口化脓了怎么办？
- **A**: 用煮沸过的布或刀片**切开排脓**，然后用盐水冲洗，蜂蜜敷料覆盖。

- **Q**: 没有麻药怎么缝合？
- **A**: ①速度要快②分散注意力③少量酒精（剧痛但短暂）④柳树皮煎水（含水杨酸，有轻微镇痛）

- **Q**: 裂开的伤口不缝合行吗？
- **A**: 小伤口可以自然愈合（肉芽填充）。超过 1cm 的裂开创口不缝合愈合极慢，且易感染。

## 安全提醒

- 动物咬伤：不要缝合！开放清创，大量冲洗，暴露伤口
- 伤口沾了土壤/粪便：破伤风风险极高，务必彻底清创
- 自己处理时量力而行，大出血须求助他人

"主角穿越后用现代急救知识救了个人，从此被奉为神医"——这是经典桥段。但注意：①给你的针线**消毒**②蜂蜜是穿越者能找到的最好的抗菌敷料③古今伤口感染率之差主要来自**清创的彻底程度**，不是你用什么"神药"。现代人最大的优势是知道**感染是细菌造成的**，所以会洗手、会消毒、会包扎干净。
