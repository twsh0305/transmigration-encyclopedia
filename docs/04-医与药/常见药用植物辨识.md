# 4.5 常见药用植物辨识

> 在古代，每一株草都可能是一种药——前提是你知道怎么用它。

## 原理

药用植物的有效成分是其**次生代谢产物**。植物为了抵抗病虫害和适应环境，合成了大量具有药理活性的化合物：

- **生物碱**：剧毒但低剂量有镇痛、麻醉作用（如乌头碱）
- **苷类**：多种药理活性（如强心苷、氰苷）
- **挥发油**：抗菌、驱虫（如薄荷油、桉叶油）
- **鞣质**：收敛止血（如五倍子）
- **黄酮类**：抗氧化、抗炎

### 穿越者最需要的药用植物按功效分类

| 功效 | 首选植物 | 用法 | 备注 |
|------|---------|------|------|
| **退热** | 柴胡、黄芩、青蒿 | 煎水服 | 病毒感染、疟疾 |
| **止痛** | 柳树皮、延胡索、川芎 | 煎水或外敷 | 含天然水杨酸 |
| **止血** | 三七、白及、地榆、艾叶 | 外敷或煎服 | — |
| **抗菌** | 黄连、金银花、大蒜、蒲公英 | 煎服或捣敷 | 伤口感染首选 |
| **驱虫** | 苦楝皮、使君子、南瓜子 | 煎服 | 肠道寄生虫 |
| **止泻** | 石榴皮、乌梅、山药 | 煎服 | 含鞣质收敛 |
| **通便** | 大黄、芒硝、火麻仁 | 煎服 | 少量用 |
| **安神** | 酸枣仁、远志、合欢皮 | 煎服 | 失眠焦虑 |
| **解毒** | 甘草、绿豆、金银花 | 煎服 | 食物中毒、解毒 |

## 操作步骤

### 植物辨识基本原则

1. **四看**：看叶形、看花序、看果实、看根茎
2. **三闻**：折断闻气味（薄荷→清凉、艾草→浓香）
3. **二问**：问当地人（最好的方法）
4. **一记**：记住有毒植物（不要混淆）

### 10种必识药用植物

**1. 蒲公英**
- 识别：黄花、蒲公英头状种子、锯齿叶
- 用途：抗菌、消炎、利尿，新鲜捣烂敷伤口
- 安全：全株可食，无毒

**2. 车前草**
- 识别：宽椭圆形叶基生，叶脉平行，花穗状
- 用途：利尿、止咳、消炎
- 安全：无毒

**3. 艾草**
- 识别：叶背面灰白色绒毛，浓烈香气
- 用途：艾灸、止血、驱虫
- 安全：外用安全

**4. 金银花**
- 识别：藤本，花初开白色后变黄色
- 用途：抗菌消炎，煎水治感冒发热
- 安全：无毒

**5. 马齿苋**
- 识别：肉质叶，茎匍匐紫色
- 用途：抗菌止泻，捣敷伤口
- 安全：可食

**6. 鱼腥草**
- 识别：心形叶，揉碎有浓烈鱼腥味
- 用途：抗菌消炎，煎水治肺热咳嗽
- 安全：可食，气味不好

**7. 紫花地丁**
- 识别：心形叶，紫色小花
- 用途：清热解毒，外敷治蛇虫咬伤
- 安全：无毒

**8. 益母草**
- 识别：茎方形，轮伞花序，叶深裂
- 用途：调经、产后血瘀
- 安全：孕妇禁用

**9. 甘草**
- 识别：羽状复叶，根甜
- 用途：解毒、调和药性
- 安全：大量用致高血压

**10. 黄连**
- 识别：基生叶，根茎黄色极苦
- 用途：极强抗菌，治痢疾、伤口感染
- 安全：味极苦，无毒

## 安全提醒

- **不认识绝对不要吃**——最常见的错误是用有毒植物当药用
- 部分"药食同源"植物大剂量也有毒（如苦杏仁含氰苷）
- 孕妇禁用大部分活血化瘀药
- 儿童用量为成人的 1/3-1/2

## 穿越小说梗

"主角随手在路边采了一把草就把病人治好了"——现实中辨识药用植物是最难的技能之一，许多植物长得极其相似，有无毒和剧毒之分（如芹菜 vs 毒芹）。穿越者的最佳策略不是"自学中药"，而是**找到懂草药的当地人**，然后用现代植物学知识来验证和优化他们的经验。千万不要随便采！一本《本草纲目》的现代注释版是穿越者最值得带的书之一。
