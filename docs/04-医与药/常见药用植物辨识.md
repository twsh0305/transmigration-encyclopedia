# 4.5 常见药用植物辨识

> 在古代，每一株草都可能是一种药——前提是你知道怎么用它。

## 原理

药用植物的有效成分是其**次生代谢产物**。植物为了抵抗病虫害和适应环境，合成了大量具有药理活性的化合物：

- **生物碱**：剧毒但低剂量有镇痛、麻醉作用（如乌头碱）
- **苷类**：多种药理活性（如强心苷、氰苷）
- **挥发油**：抗菌、驱虫（如薄荷油、桉叶油）
- **鞣质**：收敛止血（如五倍子）
- **黄酮类**：抗氧化、抗炎

### 穿越者最需要的药用植物按风险分类

| 用途线索 | 常见植物 | 更稳妥的用法边界 | 失败征兆 |
|------|---------|--------------|----------|
| **退热线索** | 柴胡、黄芩、青蒿 | 只作历史和辨识资料；持续高热先补水、降温、隔离和求医 | 给儿童、孕妇或昏迷者试草药 |
| **止痛线索** | 柳树皮、延胡索、川芎 | 柳树皮属弱效水杨酸来源，有禁忌；不叠加止痛药 | 胃痛、黑便、喘憋仍继续 |
| **止血线索** | 三七、白及、地榆、艾叶 | 外用前先压迫止血、冲洗和清洁覆盖 | 大出血时找草药而不压迫 |
| **抗菌线索** | 黄连、金银花、大蒜、蒲公英 | 不能替代清创、引流、换药和求医 | 红肿扩大、发热仍拖延 |
| **驱虫线索** | 苦楝皮、使君子、南瓜子 | 毒性和剂量差异大，不给儿童自行试药 | 腹痛、呕吐、神志异常 |
| **止泻线索** | 石榴皮、乌梅、山药 | 先补液、看尿量和精神状态 | 血便、脱水仍只喝草药 |
| **通便线索** | 大黄、芒硝、火麻仁 | 攻下药风险高，老人孕妇儿童不用 | 腹痛加重、脱水 |
| **安神线索** | 酸枣仁、远志、合欢皮 | 不替代心理危机干预、睡眠卫生和安全看护 | 自伤风险、意识异常 |
| **“解毒”线索** | 甘草、绿豆、金银花 | 不当通用解毒药；中毒先停止暴露、保存样本、求援 | 误食毒物后等待偏方 |

## 操作步骤

### 植物辨识基本原则

1. **四看**：看叶形、看花序、看果实、看根茎
2. **三闻**：折断闻气味（薄荷→清凉、艾草→浓香）
3. **二问**：问当地人（最好的方法）
4. **一记**：记住有毒植物（不要混淆）

### 10种必识药用植物

**1. 蒲公英**
- 识别：黄花、蒲公英头状种子、锯齿叶
- 边界：常作野菜和外用草药线索；伤口处理仍以冲洗、覆盖和观察感染为主
- 安全：一般食用风险低，但来源污染、误认和过敏仍需排除

**2. 车前草**
- 识别：宽椭圆形叶基生，叶脉平行，花穗状
- 边界：常作食药两用线索，不替代感染、咳喘和尿路症状的诊治
- 安全：一般风险低，污染地采集不可用

**3. 艾草**
- 识别：叶背面灰白色绒毛，浓烈香气
- 边界：艾灸和烟熏有烫伤、烟气和火灾风险；不用于室内大量烟熏
- 安全：外用也要避开破损皮肤和过敏反应

**4. 金银花**
- 识别：藤本，花初开白色后变黄色
- 边界：常作清热类草药线索；发热先看传染风险、补水和重症信号
- 安全：一般风险低，但不等于孕妇儿童可随意入口

**5. 马齿苋**
- 识别：肉质叶，茎匍匐紫色
- 边界：止泻不能替代补液和判断脱水；外用不能替代清创
- 安全：可食也要避开污染地和误认

**6. 鱼腥草**
- 识别：心形叶，揉碎有浓烈鱼腥味
- 边界：咳嗽、胸痛、呼吸困难先按呼吸道感染评估
- 安全：可食，气味明显；不用于替代抗感染治疗

**7. 紫花地丁**
- 识别：心形叶，紫色小花
- 边界：蛇咬、蜂蜇过敏和严重肿胀需固定、撤离和求医，不靠草药“解毒”
- 安全：一般风险低，仍需确认植物身份

**8. 益母草**
- 识别：茎方形，轮伞花序，叶深裂
- 边界：妇科出血、产后发热或腹痛是高风险信号，不自行用活血草药
- 安全：孕妇禁用

**9. 甘草**
- 识别：羽状复叶，根甜
- 边界：不当通用解毒药；中毒先停止暴露、保存样本、求援
- 安全：大量或长期使用可致高血压、低钾等风险

**10. 黄连**
- 识别：基生叶，根茎黄色极苦
- 边界：痢疾、血便、伤口感染先补液、隔离、清创和求医；不靠苦味判断药效
- 安全：入口剂量不稳，儿童、孕妇和重症者不用未知制剂

## 时间与核对比例

| 场景 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 辨识 | 同时确认叶、花、果、根茎、生境中的至少 3 项 | 不只靠颜色或气味 | 只说“长得像” |
| 来源核对 | 至少 2 个可靠来源或当地熟手交叉确认 | 名称、部位、季节一致 | 图谱和当地叫法对不上 |
| 试用边界 | 新植物最多做小范围皮肤接触观察 2 小时以上 | 无皮疹、喉紧、心悸 | 直接入口或给重病者 |
| 记录 | 当天写采集地点、季节、图样、反应 | 失败可追溯 | 药后不适却不知道是哪株 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 辨识 | 同时确认叶、花、果、根茎和生境，不只看单一特征 |
| 来源 | 至少由当地熟手或可靠图谱交叉确认 2 次 |
| 试用 | 新植物只做小范围外用观察，入口用途必须有可靠来源和禁忌核对 |
| 记录 | 记录采集地点、季节、图样、用量和反应 |
| 禁忌 | 孕妇、儿童、重病者不用未知草药试验 |

## 失败征兆

- 只凭“长得像”就入口或入药。
- 采自农药地、污水沟、矿区、路边尘土重处。
- 药后出现喉咙紧、皮疹、呕吐、心悸、麻木，应立即停止。
- 活血、攻下、毒性药材给孕妇或儿童使用。
- 把“天然”当成“安全”，忽略剂量和误认风险。

## 安全提醒

- **不认识绝对不要吃**——最常见的错误是用有毒植物当药用
- 部分"药食同源"植物大剂量也有毒（如苦杏仁含氰苷）
- 孕妇禁用大部分活血化瘀药
- 儿童、孕妇、老人和重病者不拿未知草药试验；不要凭“成人几分之一”的比例自行给药

## 穿越小说梗

"主角随手在路边采了一把草就把病人治好了"——现实中辨识药用植物是最难的技能之一，许多植物长得极其相似，有无毒和剧毒之分（如芹菜 vs 毒芹）。穿越者的最佳策略不是"自学中药"，而是**找到懂草药的当地人**，然后用现代植物学知识来验证和优化他们的经验。千万不要随便采！一本《本草纲目》的现代注释版是穿越者最值得带的书之一。
