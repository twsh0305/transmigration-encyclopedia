# 第四卷·医与药

> 在古代，一个伤口感染就可能要命。掌握基础医疗知识，留给你的容错空间远比现代小得多。

## 使用顺序

1. 先学卫生：洗手、煮沸饮水、生熟分离和粪便隔离，比任何药方都优先。
2. 再学急救：止血、降温、固定、搬运和观察意识呼吸。
3. 然后学隔离：发热、腹泻、咳嗽、皮疹和不明死亡都先按传染风险处理。
4. 最后看草药和制药：只做已辨识、低风险、外用或辅助用途，不把未知药物当灵丹。

## 成功标准

- 伤口：能先压迫止血、清洁覆盖、记录时间，再判断是否需要转运。
- 烧烫伤：能立刻用清洁凉水降温 5-20 分钟，不抹油、不撒粉、不挑破水疱。
- 饮水：清水能滚沸 1 分钟后再饮用；可疑化学污染水不靠煮沸解决。
- 隔离：病人、排泄物、餐具、衣物和照护者动线分开，不让全屋交叉暴露。
- 用药：每次只改变一个变量，记录剂量、时间、症状变化和副作用。

## 失败征兆

- 只背药名，不会清创、止血、补水和隔离。
- 发热腹泻时共用水桶、餐具、毛巾，导致多人连续发病。
- 把古籍“久服”“无毒”直接当安全结论。
- 伤者意识变差、呼吸困难、出血不止、胸腹剧痛，却仍在试偏方。

## 安全边界

- 医疗条目不能替代现实医生、急救培训和当地急救系统。
- 不自制内服抗生素、麻醉药、毒性草药丸散和矿物药。
- 不接触不明尸体、脓血、呕吐物和粪便；必须处理时先做隔离和防护。
- 任何入口药物都要保守，儿童、孕妇、老人和重病者风险更高。
