# 4.14 药粉与药丸制作

> 药粉和药丸是古代制剂形态；现实中要先确认药材身份、入口边界和剂量风险，不能把“能成丸”当成“能治病”。

## 原理

药粉和药丸的制作核心是**粉碎 + 成型**：

- **药粉**：干燥药材粉碎，可作为外用、香料、标本或历史制剂研究
- **药丸**：药粉 + 粘合剂→成型→干燥，便于保存，但入口剂量更难判断

### 药丸vs药粉vs煎剂

| 剂型 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| 煎剂 | 易观察煎煮过程 | 不能久放、剂量仍不稳 | 历史制剂 |
| 药粉 | 长期保存、方便 | 易吸入、易误服、易霉变 | 外用或研究 |
| 药丸 | 携带方便、缓释 | 制作复杂、入口剂量更难判断 | 历史制剂 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 干燥药材 | 原料 |
| 石臼/研钵 | 粉碎 |
| 细筛（纱布/竹筛） | 过筛 |
| 蜂蜜（首选粘合剂） | 制丸（蜜丸） |
| 米糊/面粉 | 制丸替代粘合 |
| 蜡/油纸 | 包装保存 |

## 操作步骤

### 一、制作药粉

1. 药材彻底干燥（阳光下暴晒或微火烘干）
2. 在石臼中反复捣碎
3. 过细筛（筛下的细粉→成品，粗粒→继续捣）
4. 粗纤维多的药材先切段再粉碎

**细度标准**：
- 外用粉：过 40-60 目筛，避免尖硬颗粒
- 历史内服粉：古籍常追求过 80-100 目筛，但现实不自制未知药粉入口

### 二、制作药丸

**蜜丸（最好吃的药丸）**：

1. 蜂蜜在微火上加热至起小泡（炼蜜，去除水分）
2. 药粉 1 份 + 炼蜜 0.8-1.2 份（视药材吸水性而定）
3. 趁热揉和成团
4. 搓条 → 切段 → 搓圆（每丸约 3-5克）
5. 用蜡纸/油纸包裹，密封保存

**水丸（不甜的药丸）**：

1. 药粉放入浅盆
2. 用喷壶洒少量水，摇晃盆使粉粒凝成小颗粒
3. 反复洒水、摇晃，颗粒逐渐长大至合适大小
4. 晒干或阴干

### 三、历史药丸示例（不作为入口建议）

| 药丸 | 配方 | 用途 |
|------|------|------|
| 蜂蜜丸 | 草药粉 + 炼蜜 | 展示蜜丸成型和保存思路 |
| 三七丸 | 三七粉 + 糯米糊 | 古代跌打叙事材料 |
| 艾绒丸 | 艾绒搓成小丸 | 艾灸材料，注意火灾和烫伤 |
| 绿豆甘草丸 | 甘草粉 + 绿豆粉 + 蜜 | 民间“解毒”叙事，不替代中毒救治 |

## 比例、尺寸与保存时间

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 药粉细度 | 外敷粉过 40 到 60 目；历史内服粉常追求 80 到 100 目 | 无尖硬颗粒 | 砂粒感明显、吸入呛咳 |
| 蜜丸比例 | 药粉 1 份配炼蜜 0.8 到 1.2 份 | 能成团不粘手 | 太湿发酵、太干开裂 |
| 丸重 | 常用蜜丸每丸 3 到 5 克，小丸约黄豆大小 | 每丸大小接近 | 一批丸重差异很大 |
| 保存 | 蜜丸阴凉密封半年到一年，水丸 3 到 6 个月 | 无霉点、酸味、胀气 | 发霉、虫蛀、异味 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 药材 | 完全干燥、无霉味、无虫蛀，来源和名称可确认 |
| 细度 | 外敷粉无尖硬颗粒；不把细粉当成入口安全证明 |
| 蜜丸 | 药粉 1 份配炼蜜 0.8-1.2 份，能成团不粘手 |
| 丸重 | 常用蜜丸每丸 3-5克，小丸按黄豆大小分装 |
| 保存 | 阴凉密封，蜜丸半年至一年内无霉点、酸味、胀气 |

## 失败征兆

- 药材身份不确定，不能靠气味和传闻入药。
- 药粉有霉味、结块、虫蛀，不可再制丸。
- 蜜未炼就混粉，水分高，药丸容易发酵酸败。
- 毒性药材与普通药材共用研钵未清洗，会交叉污染。
- 任何自制丸粉被误服后出现皮疹、喉紧、呕吐、心悸，应停止使用并求医。

## 常见问题

- **Q**: 没有石臼怎么粉碎？
- **A**: ①碗+硬石头替代 ②布包药材后用锤子砸 ③两片粗石头对磨。最原始的"研钵+研杵"可以用粗陶碗+石头替代。

- **Q**: 药丸能放多久？
- **A**: 蜜丸密封阴凉处可保存半年至一年。水丸保存期较短（3-6 个月）。如有发霉/异味则不可再用。

- **Q**: 药丸太大吞不下去怎么办？
- **A**: 不要为了吞服而改变大小或次数。自制丸药没有稳定剂量，入口前必须能确认药材、适应证、禁忌和可靠来源；否则只作外用或历史展示。

## 安全提醒

- 不研磨乌头、马钱子等毒性药材；毒粉残留会交叉污染
- 药粉吸入可能刺激呼吸道，戴口罩
- 丸药制作注意手部卫生

## 穿越小说梗

"主角随手搓了一颗药丸塞病人嘴里"是高风险爽点。更可信的写法是：主角能做出形态稳定、标识清楚、保存不霉的外用或备用制剂，但面对入口药物仍要核对药材、剂量、禁忌和病因，不能把成丸技术当作神医证明。
