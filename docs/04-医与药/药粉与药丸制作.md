# 4.14 药粉与药丸制作

> 煎药需要每天生火熬煮，但药丸做好一颗就可以随身携带好几个月。

## 原理

药粉和药丸的制作核心是**粉碎 + 成型**：

- **药粉**：干燥药材粉碎，直接服用或外用
- **药丸**：药粉 + 粘合剂→成型→干燥，持续释放有效成分

### 药丸vs药粉vs煎剂

| 剂型 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| 煎剂 | 吸收快、药力强 | 不能久放、每天要煎 | 急性病 |
| 药粉 | 长期保存、方便 | 部分药材难粉碎 | 慢性调理 |
| 药丸 | 携带方便、缓释 | 制作复杂、吸收慢 | 长期服药 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 干燥药材 | 原料 |
| 石臼/研钵 | 粉碎 |
| 细筛（纱布/竹筛） | 过筛 |
| 蜂蜜（首选粘合剂） | 制丸（蜜丸） |
| 米糊/面粉 | 制丸替代粘合 |
| 蜡/油纸 | 包装保存 |

## 操作步骤

### 一、制作药粉

1. 药材彻底干燥（阳光下暴晒或微火烘干）
2. 在石臼中反复捣碎
3. 过细筛（筛下的细粉→成品，粗粒→继续捣）
4. 粗纤维多的药材先切段再粉碎

**细度标准**：
- 内服粉：过 80-100 目筛（如面粉般细）
- 外敷粉：可稍粗（过 40-60 目）

### 二、制作药丸

**蜜丸（最好吃的药丸）**：

1. 蜂蜜在微火上加热至起小泡（炼蜜，去除水分）
2. 药粉 1 份 + 炼蜜 0.8-1.2 份（视药材吸水性而定）
3. 趁热揉和成团
4. 搓条 → 切段 → 搓圆（每丸约 3-5g）
5. 用蜡纸/油纸包裹，密封保存

**水丸（不甜的药丸）**：

1. 药粉放入浅盆
2. 用喷壶洒少量水，摇晃盆使粉粒凝成小颗粒
3. 反复洒水、摇晃，颗粒逐渐长大至合适大小
4. 晒干或阴干

### 三、特殊药丸示例

| 药丸 | 配方 | 用途 |
|------|------|------|
| 蜂蜜丸 | 黄连粉 + 炼蜜 | 止泻健胃 |
| 三七丸 | 三七粉 + 糯米糊 | 活血止血 |
| 艾绒丸 | 艾绒搓成小丸 | 艾灸用 |
| 解毒丸 | 甘草粉 + 绿豆粉 + 蜜 | 食物中毒 |

## 常见问题

- **Q**: 没有石臼怎么粉碎？
- **A**: ①碗+硬石头替代 ②布包药材后用锤子砸 ③两片粗石头对磨。最原始的"研钵+研杵"可以用粗陶碗+石头替代。

- **Q**: 药丸能放多久？
- **A**: 蜜丸密封阴凉处可保存半年至一年。水丸保存期较短（3-6 个月）。如有发霉/异味则不可再用。

- **Q**: 药丸太大吞不下去怎么办？
- **A**: 把丸子搓小（每次黄豆大小），一次多服几颗。

## 安全提醒

- 研磨毒性药材（乌头、马钱子等）后彻底清洗工具
- 药粉吸入可能刺激呼吸道，戴口罩
- 丸药制作注意手部卫生

## 穿越小说梗

"主角随手搓了一颗药丸塞病人嘴里"——古代药丸（尤其是蜜丸）的制造确实不复杂，但注意蜂蜜要先"炼"（加热除水），不然蜂蜜里的酵母菌会让药丸发酵酸败。穿越者的优势：知道蜜丸、水丸、蜡壳丸的不同保存特性。
