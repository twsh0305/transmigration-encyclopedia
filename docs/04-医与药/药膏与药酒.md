# 4.15 药膏与药酒

> 药膏涂抹伤口，药酒温暖筋骨——一个是外用的物理屏障，一个是内服的液体良药。

## 原理

### 药膏（外用）

药膏是**药物 + 基质**的混合物：
- **基质**：油脂（动物油、植物油）或蜂蜡，作为载体
- **药物**：粉碎的草药或提取液
- 药膏涂在皮肤上形成保护膜，缓慢释放药效

### 药酒（内服）

药酒是**乙醇提取**：
- 酒精溶解药材中的有效成分（尤其是一些水不溶性成分）
- 酒精本身防腐，药酒可长期保存
- 酒精促进血液循环，加速药效吸收

## 所需材料

### 药膏

| 材料 | 用途 |
|------|------|
| 干净的动物油/猪油 | 基质 |
| 蜂蜡 | 调节硬度（可选） |
| 干燥草药粉 | 药物成分 |
| 陶罐/竹筒 | 容器 |
| 干净的布 | 涂抹和包扎 |

### 药酒

| 材料 | 用途 |
|------|------|
| 高度蒸馏酒（50-60% 最佳） | 溶剂 |
| 干燥药材（切片） | 药物成分 |
| 玻璃/陶瓷罐 | 浸泡容器 |
| 纱布 | 过滤 |

## 操作步骤

### 药膏制作

1. **准备基质**：动物油加热熔化（猪油约 30g）
2. **加药**：加入草药粉（约 10g），微火搅拌 10-15 分钟
3. **调节硬度**：如需更硬→加少量蜂蜡；更软→加更多油
4. **过滤**：用纱布趁热过滤药渣
5. **冷却**：倒入陶罐，自然凝固

**常用药膏配方**：
- **创伤膏**：猪油 + 蜂蜜 + 黄连粉，外敷伤口
- **消炎膏**：猪油 + 蒲公英粉 + 金银花粉
- **烧伤膏**：猪油 + 大黄粉（冷却后涂抹）

### 药酒制作

1. **药材处理**：干燥药材切片/切段（增大接触面积）
2. **浸泡**：药材 + 酒 = 1:5 至 1:10
3. **密封**：容器密封
4. **存放**：阴凉处浸泡 7-30 天（每周摇晃一次）
5. **过滤**：取出药材，得药酒

**常用药酒配方**：
- **活血酒**：当归 + 川芎 + 红花 + 酒，浸泡 14 天
- **祛风湿酒**：威灵仙 + 独活 + 防风 + 酒
- **跌打酒**：三七 + 乳香 + 没药 + 酒

## 常见问题

- **Q**: 药膏凝固后太硬/太软怎么办？
- **A**: 挖出来重新熔化，太硬加更多油，太软加少量蜂蜡或更多的药粉。

- **Q**: 药酒多久能好？
- **A**: 一般 2-4 周。药材切得越碎，浸泡时间越短。如果急用，可将药材与酒一起隔水蒸 40 分钟（但效果不如浸泡）。

- **Q**: 没有高度酒能做药酒吗？
- **A**: 酒精浓度至少 >30% 才有防腐和提取效果。米酒（10-15%）提取效果差，且易变质。

## 安全提醒

- 外用药膏**不得口服**
- 药酒需标注外服还是内服（很多外用活血药有毒）
- 药膏发霉、药酒浑浊有味→不可使用
- 孕妇禁服活血化瘀类药酒

## 穿越小说梗

"主角拿出一瓶自制的跌打药酒救了摔伤的猎人"——药酒在古典小说中是最常见的"武林人士必备"。但对于穿越者来说，药酒真正的价值不只是配方，而是你知道**乙醇是比水更好的溶剂**，以及**酒精本身就是防腐剂**。一个好的改进：先做高浓度酒精，再做药酒，效果远超古代用黄酒做的药酒。
