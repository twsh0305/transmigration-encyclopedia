# 4.15 药膏与药酒

> 药膏和药酒常见于古代医药与武侠叙事；现实中最重要的是分清外用、完整皮肤、入口风险和污染边界。

## 原理

### 药膏（外用）

药膏是**药物 + 基质**的混合物：
- **基质**：油脂（动物油、植物油）或蜂蜡，作为载体
- **药物**：粉碎的草药或提取液
- 药膏涂在皮肤上形成保护膜，缓慢释放药效

### 药酒（多作外用或历史制剂）

药酒是**乙醇提取**：
- 酒精溶解药材中的有效成分（尤其是一些水不溶性成分）
- 酒精本身防腐，药酒可长期保存
- 入口药酒会同时带来酒精、药材误认、剂量不稳和肝肾负担风险

## 所需材料

### 药膏

| 材料 | 用途 |
|------|------|
| 干净的动物油/猪油 | 基质 |
| 蜂蜡 | 调节硬度（可选） |
| 干燥草药粉 | 药物成分 |
| 陶罐/竹筒 | 容器 |
| 干净的布 | 涂抹和包扎 |

### 药酒

| 材料 | 用途 |
|------|------|
| 高度蒸馏酒（50-60% 最佳） | 溶剂 |
| 干燥药材（切片） | 药物成分 |
| 玻璃/陶瓷罐 | 浸泡容器 |
| 纱布 | 过滤 |

## 操作步骤

### 药膏制作

1. **准备基质**：动物油加热熔化（猪油约 30克）
2. **加药**：加入草药粉（约 10克），微火搅拌 10-15 分钟
3. **调节硬度**：如需更硬→加少量蜂蜡；更软→加更多油
4. **过滤**：用纱布趁热过滤药渣
5. **冷却**：倒入陶罐，自然凝固

**常用药膏配方**：
- **创伤膏**：猪油 + 蜂蜜 + 黄连粉，外敷伤口
- **消炎膏**：猪油 + 蒲公英粉 + 金银花粉
- **皮肤护理膏**：猪油 + 蜂蜡 + 少量草药粉，只用于完整皮肤或愈合期干裂处

### 药酒浸泡边界

1. **药材处理**：干燥药材切片/切段（增大接触面积）
2. **浸泡**：药材 + 酒 = 1:5 至 1:10
3. **密封**：容器密封
4. **存放**：阴凉处浸泡 7-30 天（每周摇晃一次）
5. **过滤**：取出药材，得药酒

这些比例和时间只能说明历史上“酒精可萃取并保存药材”的思路，不等于入口剂量。现实中不自制内服药酒；外用跌打酒也必须清楚标识，不用于破损皮肤、儿童、孕妇、酒精过敏者和大面积皮肤。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 药材 | 干燥、可确认、无霉虫，毒性药材不入家庭药酒 |
| 药膏 | 油脂加热后无水珠爆响，药渣过滤干净，冷却后质地均匀 |
| 药酒 | 酒精度 50-60% 更利于提取和防腐，药材:酒约 1:5 到 1:10；只作外用或历史制剂边界 |
| 标识 | 每瓶写清“外用、药材、日期、禁忌、不得入口” |
| 观察 | 7-30 天浸泡期无霉膜、异味、胀气和浑浊腐败 |

## 失败征兆

- 外用跌打酒被当成内服药，可能中毒。
- 药膏用于新鲜深伤口或烧伤早期，容易封住污染和热量。
- 药酒出现酸败、霉膜、沉淀腐臭，不能继续使用。
- 孕妇、儿童、肝病患者服活血或高度酒药酒，风险高。
- 药材名称不确定仍泡酒，等于把未知毒物浓缩。

## 常见问题

- **Q**: 药膏凝固后太硬/太软怎么办？
- **A**: 挖出来重新熔化，太硬加更多油，太软加少量蜂蜡或更多的药粉。

- **Q**: 药酒多久能好？
- **A**: 历史浸泡常见 2 到 4 周；时间变短只会让萃取不稳定。不要为“急用”加热浓缩含酒精药液，明火、蒸汽和误服风险都会上升。

- **Q**: 没有高度酒能做药酒吗？
- **A**: 酒精浓度至少 >30% 才有防腐和提取效果。米酒（10-15%）提取效果差，且易变质。

## 安全提醒

- 外用药膏**不得口服**
- 药酒默认不得入口；很多外用活血药、矿物药和未知草药有毒
- 药膏发霉、药酒浑浊有味→不可使用
- 孕妇禁服活血化瘀类药酒
- 新鲜烧伤先冷却和覆盖，不用油膏当急救第一步

## 穿越小说梗

"主角拿出一瓶自制的跌打药酒救了摔伤的猎人"——药酒在古典小说中是最常见的"武林人士必备"。但对于穿越者来说，药酒真正的价值不只是配方，而是你知道**乙醇是比水更好的溶剂**，以及**酒精本身就是防腐剂**。一个好的改进：先做高浓度酒精，再做药酒，效果远超古代用黄酒做的药酒。
