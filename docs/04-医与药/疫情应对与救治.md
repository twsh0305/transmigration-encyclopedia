# 4.20 疫情应对与救治

> 在古代，一次瘟疫能灭掉一座城、一个朝代。穿越者最大的价值之一，就是知道瘟疫是怎么回事、该怎么防。

## 原理

古人眼中的"瘟疫"其实是多种传染病的统称。穿越者需要先辨别是哪种病，再对症施策。

### 古代常见瘟疫速辨

| 病症特征 | 可疑疾病 | 传播途径 | 死亡率（无现代医疗） |
|---------|---------|---------|-----------------|
| 发热+淋巴结肿大（尤其是腹股沟/腋下） | **鼠疫（黑死病）** | 跳蚤→人，人→人飞沫 | 50-90% |
| 严重水样腹泻+呕吐+迅速脱水 | **霍乱** | 污染的水/食物 | 30-60% |
| 发热+全身红色皮疹+水疱 | **天花** | 飞沫+接触 | 20-60% |
| 阵发性剧烈咳嗽+咳血 | **肺结核** | 飞沫（长期密切接触） | 50-80% |
| 发热+头痛+肌肉剧痛+皮疹 | **斑疹伤寒** | 虱子/跳蚤 | 10-40% |
| 发热+剧烈头痛+颈项强直+呕吐 | **流行性脑膜炎** | 飞沫 | 50-80% |
| 发热+咳嗽+呼吸困难 | **流感/肺炎** | 飞沫 | 5-30%（古代较高） |
| 高热+寒战+大汗周期发作 | **疟疾** | 蚊子 | 10-40% |
| 发热+黄疸+出血 | **黄热病/出血热** | 蚊子/接触 | 20-50% |

## 疫情应急预案

### 第一阶段：发现疑似病例（黄金24小时）

1. **立即隔离病人**：单独房间/帐篷，距其他人至少 10m
2. **报告**：通知所有接触过的人，统计接触者名单
3. **初步诊断**：根据症状判断最可能的疾病
4. **划定警戒区**：标记疫区边界，限制出入
5. **物资清点**：检查药品、食物、水、消毒物资存量

### 第二阶段：全面防控（第1-7天）

| 措施 | 具体操作 | 关键点 |
|------|---------|-------|
| **封锁** | 封闭疫区出入口，设卡检查 | 越早越好，犹豫一天就多一批感染者 |
| **口罩** | 所有人佩戴多层布口罩 | 飞沫传播疾病尤其重要 |
| **洗手** | 进出疫区必须用肥皂/草木灰洗手 | 最简单但最有效的措施 |
| **消毒** | 石灰粉撒布地面；醋/酒熏蒸室内 | 生石灰兑水成石灰乳，刷墙/地面 |
| **灭鼠灭蚤** | 烟熏、毒饵、火烧 | 鼠疫必须灭鼠+灭蚤（老鼠死了跳蚤会跳人身上） |
| **垃圾处理** | 焚烧或深埋（1.5m以上） | 垃圾堆是病媒生物的温床 |
| **水源保护** | 疫区水源专人看守，禁止污染 | 霍乱、伤寒通过水源传播 |

### 第三阶段：病人救治（持续）

#### 鼠疫
- **严格隔离**：单独房间，防跳蚤（床腿涂油、地面撒石灰）
- **对症**：退热、补液、淋巴结肿大处冷敷
- **护理**：高营养流食，保持体力
- **注意**：腺鼠疫有部分存活率（30-50%），肺鼠疫基本必死

#### 霍乱
- **补液是生死关键**：糖盐水口服（1升水+3g盐+20g糖）
- 严重脱水者需肛门灌注补液（古代无输液条件）
- 病人排泄物用生石灰覆盖 30 分钟后再处理
- 水源彻底消毒（煮沸或用漂白粉）

#### 天花
- **人痘接种可作为预防**（见"接种与免疫原理"）
- 病人隔离至所有痂皮脱落（约3-4周）
- 病人衣物、被褥全部焚烧
- 护理过天花的人具有免疫力，优先安排护理

#### 流感/肺炎
- 通风！通风！通风！（古人认为"风邪入体"，会关窗，反而加重）
- 保持呼吸道湿润（蒸汽吸入）
- 大蒜、生姜水辅助（虽不能抗病毒，但能改善症状）

### 第四阶段：恢复与复盘（疫情结束后）

1. **终末消毒**：疫区全面消毒，尸体焚烧/深埋区覆盖石灰
2. **观察期**：最后一例病人痊愈/死亡后再观察最长潜伏期（通常14-21天）
3. **总结**：记录疫情始末、有效和无效的措施
4. **改进**：完善卫生制度，建立常设隔离设施

## 古代防疫工具体系

### 消毒剂制作

| 消毒剂 | 制作方法 | 适用范围 | 备注 |
|-------|---------|---------|------|
| 石灰水 | 生石灰:水=1:3，搅拌溶解 | 地面、墙面、排泄物 | 便宜量大，首选 |
| 醋熏蒸 | 醋煮沸，蒸汽弥漫房间 | 空气消毒（效果有限） | 聊胜于无 |
| 艾草烟熏 | 干艾草点燃，烟熏房间 | 空气消毒（古代经验） | 有一定驱虫作用 |
| 酒精（高度酒） | 蒸馏至70%以上浓度 | 小物件、伤口 | 成本高，用于关键处 |
| 草木灰水 | 草木灰浸泡后过滤 | 洗手、清洗衣物 | 碱性，有一定消毒作用 |
| 沸水 | 100°C煮沸20分钟 | 餐具、布料 | 最可靠 |

### 防护装备制作

| 装备 | 制作 | 防护等级 |
|------|------|---------|
| 布口罩 | 4-6层棉布缝制，中间夹活性炭粉 | ⭐⭐⭐ 基础防护 |
| 防护服 | 油布/蜡布外套，罩住全身 | ⭐⭐⭐⭐ 防飞沫 |
| 手套 | 厚棉布或皮革缝制 | ⭐⭐⭐ 防接触 |
| 护目镜 | 透明玻璃/云母片 | ⭐⭐⭐⭐ 防喷溅 |
| 靴套 | 油布套鞋 | ⭐⭐⭐ |

## 常见问题

- **Q**: 没有现代检测手段，怎么确定是什么病？
- **A**: 按症状组合判断（参考上面的速辨表）。但即便不确定具体病名，**隔离+口罩+消毒+补液**这套基础方案能应对大多数传染病。

- **Q**: 村民不信有瘟疫怎么办？
- **A**: 古代人把瘟疫当"天谴"。穿越者策略：①利用当地信仰（"天神托梦说疫至"）②现身说法（先隔离自己人做示范）③利用权威（说服族长/县令下令）

- **Q**: 封城引起恐慌/暴动怎么办？
- **A**: ①封城不等于放弃——保证城内粮食供应 ②设专门通道送物资 ③公开透明告知隔离期限 ④对违抗者严惩不贷（古代疫区封锁靠的是军令而非自觉）

- **Q**: 没有现代药品能治好瘟疫吗？
- **A**: 补液能救霍乱（死亡率从50%降到5%），隔离能控制鼠疫传播。古代瘟疫致死的主因往往不是"没有特效药"，而是"不知道要隔离"和"不会补液"。

## 穿越小说梗

"主角在古代阻止了瘟疫"——这可能是穿越者**对整个世界最大的贡献**。一个懂得基础防疫知识的穿越者，其价值堪比几座城池。

但要注意几个现实问题：

1. **古代交通慢，疫情扩散也慢**——这给了穿越者反应时间。与现代社会航空时代不同，古代一个村庄爆发疫情后，可能需要数周甚至数月才会传到下一座城。

2. **"隔离"在古代是反人性的**——古人讲究"仁义"，把病人关起来被认为是残忍。穿越者需要足够的权威（族长/官员身份）才能推行强制隔离。

3. **瘟疫是最好的权力催化剂**——如果你能在一个瘟疫横行的时代保护一座城的人平安，你会瞬间获得无上的威望。很多穿越小说里，主角正是通过防疫积累第一桶政治资本。
