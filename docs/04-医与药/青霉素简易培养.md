# 4.17 青霉素发现与简易培养

> 这是穿越者最大的"终极挑战"——从发现到应用，每一步都是技术与运气的结合。

## 原理

青霉素是**青霉菌**（Penicillium）分泌的代谢产物，它能抑制细菌细胞壁合成，从而杀灭多种致病菌。

1. 青霉菌在营养物上生长→分泌青霉素到培养基中
2. 青霉素抑制细菌细胞壁合成→细菌因渗透压而破裂
3. 青霉素对**革兰氏阳性菌**（葡萄球菌、链球菌等）效果极好
4. 对人体细胞基本无害（人类细胞没有细胞壁）

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 发霉的食物 | 青霉菌来源 | 甜瓜/橘子皮上的蓝绿色霉斑**
| 培养容器 | 扩大培养 | 陶罐/玻璃瓶 |
| 营养液/培养基 | 菌种生长 | 见下文 |
| 干净布/纱布 | 过滤 | 初步粗滤 |
| 无菌水/蒸馏水 | 配制 | 煮沸后冷却 |
| 米汤/土豆水 | 简易培养基 | 含淀粉和糖 |

## 操作步骤

### 一、发现青霉菌

1. 寻找自然霉变的食物：橘子、甜瓜、面包、米饭
2. 目标霉菌特征：**蓝绿色**霉斑（不是黑色、红色、白色）
3. 用放大镜观察：蓝色菌丝和绿色孢子的典型结构

### 二、分离培养（纯化菌种）

1. **配培养基**：土豆削皮切碎煮水 + 少量糖，灭菌
2. **接种**：挑少量蓝绿色孢子放入培养基
3. **培养**：20-25°C，静置 5-7 天
4. **观察**：表面形成蓝绿色菌膜→成功

### 三、青霉素提取（最简化版）

1. 菌膜形成后，在培养基表面出现**黄色澄清液体**
2. 用干净的布过滤（去除菌丝和孢子）
3. 收集**黄色滤液**——这就是粗制青霉素液
4. 调节 pH 到 6-7（用醋或草木灰水调）
5. **立即使用**——粗提液极不稳定

### 四、抗菌测试（验证有效性）

1. 切一小块煮熟的肉或土豆
2. 滴几滴青霉素液
3. 另一块不滴（对照）
4. 24 小时后：滴了青霉素的没有腐败→有效

### 五、使用（粗制青霉素）

1. **仅限外用**——粗制青霉素含有大量杂质和毒素，不可注射
2. 用干净布蘸取青霉素液敷在伤口上
3. 每天换药 2-3 次
4. 观察感染是否消退

## 常见问题

- **Q**: 为什么我培养出来的没有效果？
- **A**: ①可能不是正确的青霉菌种（自然界只有少数青霉菌产青霉素）②培养基成分不对③温度不合适④被杂菌污染了

- **Q**: 粗制青霉素能保存多久？
- **A**: 常温下**不超过 24 小时**。低温（地窖）可保存 3-5 天。冻干可保存更久（但穿越环境难做到）。

- **Q**: 口服行不行？
- **A**: 粗制青霉素口服会在胃酸中分解，而且含有大量杂质可能中毒。**穿越者的青霉素只能外用**，不要口服或注射。

## 安全提醒

- **不要注射！** 粗制青霉素不是药厂的纯品，注射=自杀
- 初次使用前做皮肤测试：涂少量在手腕内侧，观察 30 分钟有无红肿
- 霉菌孢子可能引起过敏，操作时戴口罩
- 部分青霉菌产生**黄曲霉毒素**——确认霉菌颜色（蓝绿色为好，黑色/黄色不要用）

## 穿越小说梗

"主角在穿越后制出了青霉素救了很多人"——这是穿越小说里最"高阶"的医疗成就。但真实难度被严重低估了：①找到**正确**的青霉菌株需要运气②培养过程中极易被杂菌污染③粗提液极不稳定④不能注射只能外用。合理的方案：在建立了基本的**无菌操作台**（封闭空间+酒精+煮沸器具）之后再进行培养。外用治疗伤口的成功概率较高，但想做出能注射的青霉素——那是工业级的难度。最有用的青霉素是**外用治疗伤口感染**，仅此就足以降低穿越者的死亡率 50%。
