# 4.16 金鸡纳霜与奎宁

> 柳树皮是弱效退热药，金鸡纳树皮是真正的抗疟特效药——可惜它只在南美洲生长。

## 原理

金鸡纳树皮含有**奎宁**（Quinine）和**奎尼丁**（Quinidine）等 20 多种生物碱，是抗疟特效药：

1. 奎宁进入疟原虫体内，抑制其 DNA 复制和蛋白质合成
2. 对红细胞内的疟原虫有强力杀灭作用
3. 对间日疟、三日疟、恶性疟均有效

### 金鸡纳 vs 青蒿素

| 对比 | 金鸡纳（奎宁） | 青蒿（青蒿素） |
|------|--------------|---------------|
| 原产地 | 南美洲安第斯山脉 | 中国/亚欧大陆 |
| 有效性 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 副作用 | 耳鸣、恶心、心律不齐 | 极低 |
| 获取难度 | **必须进口** | 中国境内就可找到 |
| 历史 | 17 世纪传入欧洲 | 1972 年发现 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 金鸡纳树皮（Cinchona） | 奎宁来源 |
| 水 | 煎煮 |
| 酒 | 酊剂提取 |
| 炭粉 | 包裹服用（减轻刺激） |

## 操作步骤

### 金鸡纳树皮煎剂

1. 树皮碾碎成粗粉
2. 加水煮沸 15 分钟
3. 过滤后服用
4. 每次 3-5g 树皮量
5. 极苦——可加蜂蜜或包裹在米饭中吞服

### 奎宁酊剂（更有效）

1. 树皮粗粉 + 高度酒（50%+），浸泡 7 天
2. 每天摇晃
3. 过滤后酊剂深褐色
4. 每次 5-10ml，兑水服用

### 穿越后的残酷现实

金鸡纳树**只生长在南美洲安第斯山区**（秘鲁、厄瓜多尔、玻利维亚），在中国/亚洲无法自然种植。如果你穿越到古代中国：

1. **没有金鸡纳树皮**——除非你正好穿越到南美
2. **替代方案**：①青蒿（黄花蒿）——中国本土就有 ②常山（有一定效果但致吐）③鸦胆子（有毒）
3. **如果能到南美**：想办法把金鸡纳种子带回你所在的大陆——历史上这也是欧洲人做的事

## 如何引进金鸡纳（如果条件允许）

历史上荷兰殖民者在 19 世纪偷运金鸡纳种子到爪哇岛种植，垄断了全球奎宁市场近百年。穿越者如果做了同样的操作：

1. 在南美收集金鸡纳种子
2. 在适宜气候地区种植（海拔 1000-2000m、温暖湿润）
3. 树苗 3-4 年后可以采皮
4. 一棵树可持续采皮数十年

## 常见问题

- **Q**: 穿越到中国能用金鸡纳吗？
- **A**: 除非你带了种子并成功引种。否则老老实实找青蒿。

- **Q**: 金鸡纳树在中国能种吗？
- **A**: 云南南部、台湾海拔 1000-2000m 的山丘可以试种。但不是原产地，生长较慢。

- **Q**: 奎宁的副作用？
- **A**: 恶心、耳鸣、头痛是常见反应（"金鸡纳中毒"），大剂量可致失明和心律不齐。孕妇禁用（可能致流产）。

## 安全提醒

- 奎宁毒性：治疗剂量和中毒剂量接近，慎用
- 孕妇和心脏病患者不要使用
- 如出现视力模糊、严重耳鸣→立即停药

## 穿越小说梗

"穿越者用金鸡纳霜治好了疟疾"——这是经典的"南美发现"剧情。一个合理的桥段：穿越者恰好流落南美或在航行中经过南美，认识了金鸡纳树。如果不走南美路线，青蒿素才是中国穿越者的真正选择。金鸡纳的历史隐喻：谁控制了特效药，谁就控制了通往世界贸易的钥匙——荷兰人靠金鸡纳发了大财。
