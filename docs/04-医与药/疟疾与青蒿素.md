# 4.10 疟疾与青蒿素

> 疟疾在古代是比战争更可怕的杀手——直到青蒿素被发现。

## 原理

疟疾由**疟原虫**（Plasmodium）引起，通过**按蚊**叮咬传播：

1. 按蚊叮咬感染者的血液后，疟原虫在蚊体内发育
2. 带虫的蚊子叮咬健康人→疟原虫进入血液→到肝脏繁殖→侵入红细胞
3. 红细胞内疟原虫繁殖→红细胞破裂→周期性发冷发热

### 症状特征

| 阶段 | 表现 | 持续时间 |
|------|------|---------|
| 寒战期 | 剧烈发冷、发抖 | 1-2 小时 |
| 发热期 | 高烧 40-41摄氏度、头痛 | 4-6 小时 |
| 出汗期 | 大量出汗、体温骤降 | 2-4 小时 |

**周期性发作**——隔天（间日疟）或三天（三日疟）发作一次，是典型特征。

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 青蒿（黄花蒿） | 青蒿素来源 | 仅作植物识别和历史资料 |
| 冷水 | 历史绞汁概念 | 粗浸液不能保证剂量 |
| 纱布 | 过滤示意 | 不能保证无菌或药效 |
| 纱布帐篷 | 防蚊 | 第一道防线 |

## 低资源历史方法（不能替代正规抗疟药）

现代疟疾治疗依赖明确诊断和标准化抗疟药。世界卫生组织推荐以青蒿素为基础的联合疗法治疗多数非重症疟疾；CDC 对重症疟疾强调尽快使用静脉青蒿琥酯。野外冷浸青蒿只能作为理解历史的低资源方案，不能保证剂量、疗程和治愈率。

### 青蒿冷浸史料边界

《肘后备急方》的“青蒿绞汁”说明古人注意到青蒿热敏成分，但野外叶片的青蒿素含量、植物身份、污染和服用量都不可控。本项目不提供采集量、浸泡比例、疗程或儿童/孕妇用量；若能获得正规抗疟药，应优先使用正规药物。

### 古代对照方法（青蒿绞汁法）

《肘后备急方》（葛洪·东晋）记载：
"青蒿一握，以水二升渍，绞取汁，尽服之。"

注意：**不要加热**——这正是关键。葛洪比屠呦呦早了一千多年发现了这个秘密。

### 其他古代抗疟方案

| 方法 | 说明 | 效果 |
|------|------|------|
| 青蒿绞汁 | 冷浸取汁 | ⭐⭐⭐⭐⭐ |
| 常山（黄常山根） | 水煎 | ⭐⭐⭐ 但致吐 |
| 鸦胆子 | 民间曾用于抗疟 | ⭐⭐ 毒性大，不自制服用 |
| 针灸 | 刺激穴位 | 基本无效 |

## 预防

1. **蚊帐**：穿越后第一晚就应该挂蚊帐（最重要措施）
2. **纱窗**：门窗加装纱网
3. **排水**：清除积水（蚊虫繁殖地）
4. **熏蚊**：艾草/青蒿点燃熏蚊
5. **药物预防**：不要自行长期服青蒿水预防；现代预防用药需按地区耐药性和医生建议选择

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 识别 | 发冷、40摄氏度左右高热、出汗退热呈周期发作，结合蚊虫暴露史 |
| 防蚊 | 夜间使用蚊帐，住处周围 100米内清理积水 |
| 植物 | 能确认是黄花蒿，而不是普通艾草或其他蒿属植物 |
| 治疗边界 | 把青蒿冷浸视为低资源历史方案，优先寻找正规抗疟药 |
| 重症判断 | 意识差、抽搐、黄疸、呼吸困难、休克、尿少时按重症疟疾处理 |

## 失败征兆

- 把“退烧了”当成治愈，疟疾可反复发作或转重症。
- 用青蒿煮沸久煎，可能破坏有效成分。
- 只治疗不防蚊，病人和同住者会被反复叮咬传播。
- 儿童、孕妇、高热昏迷者靠草药拖延，是高死亡风险。
- 单用来源不明植物，混入毒草或剂量不可控。

## 常见问题

- **Q**: 黄花蒿和普通蒿草怎么区分？
- **A**: 黄花蒿（Artemisia annua）叶细碎、有特殊香气、茎绿色带红、一年生。常见蒿草多为多年生，气味不同。

- **Q**: 青蒿素煮沸后还有效吗？
- **A**: 青蒿素受热会分解，这解释了古籍“绞汁”思路。但知道热敏性不等于能自制可靠抗疟药，正规诊断和标准化抗疟药仍优先。

- **Q**: 穿越后没有抗疟药怎么办？
- **A**: 第一要务是防蚊、隔离发热者、记录周期性发作、补水降温并寻找正规抗疟来源。种植和识别黄花蒿可作为长期资源，但不能代替急症处理。

## 安全提醒

- 青蒿本身无毒，但野生植物可能与其他毒草混淆
- 疟疾可能与其他发热疾病并存，需要鉴别
- 防蚊比治疗重要一万倍——穿越后第一件事就是做蚊帐
- 疑似重症疟疾是急症，不要等待草药“慢慢见效”

## 穿越小说梗

"主角靠青蒿改变疟疾局面"可以写，但关键不是一碗草汁包治全村，而是先做蚊帐、排水、病例记录和重症转运，再把黄花蒿识别、种植和标准化提取作为长期工程。真正可信的冲突来自剂量不稳、植物误认、孕妇儿童风险和正规抗疟药缺失。
