# 4.11 病媒生物控制

> 蚊子、老鼠、跳蚤、苍蝇——这四样东西在古代夺走的生命比战争多百倍。

## 原理

病媒生物（Vector）是指**传播疾病的生物媒介**：

| 病媒 | 传播的疾病 | 致死率（无现代医疗） |
|------|-----------|------------------|
| 蚊子 | 疟疾、登革热、丝虫病 | ⭐⭐⭐⭐⭐ 极高 |
| 老鼠 | 鼠疫（通过跳蚤传播） | ⭐⭐⭐⭐⭐ 极高 |
| 跳蚤 | 鼠疫（鼠→人） | ⭐⭐⭐⭐⭐ 极高 |
| 苍蝇 | 霍乱、痢疾、伤寒、沙眼 | ⭐⭐⭐⭐ 高 |
| 虱子 | 斑疹伤寒、战壕热 | ⭐⭐⭐ 中 |
| 臭虫 | 皮肤瘙痒（不传播主要疫病） | ⭐ 低 |
| 蜱虫 | 多种人畜共患病 | ⭐⭐ 中 |
| 蟑螂 | 污染食物传播多种病菌 | ⭐⭐ 中 |

## 所需材料

| 材料 | 用途 | 针对病媒 |
|------|------|---------|
| 蚊帐（纱布/细麻布） | 隔蚊 | 蚊子 |
| 艾草/蒿草 | 烟熏驱蚊 | 蚊子 |
| 捕鼠夹/猫 | 灭鼠 | 老鼠 |
| 生石灰 | 消毒+覆盖 | 跳蚤、苍蝇 |
| 密封垃圾坑 | 控制繁殖 | 苍蝇 |
| 清洁环境 | 消除繁殖地 | 所有 |
| 排水沟 | 消除积水 | 蚊子 |
| 肥皂水 | 驱杀跳蚤 | 跳蚤 |

## 操作步骤

### 一、蚊子（第一号敌人）

**繁殖需求**：只要有**积水**，雌蚊就能产卵。

**控制措施**：
1. **清除积水**（最有效）：
   - 倒掉所有容器中的积水
   - 填平水坑和低洼
   - 疏通排水沟
   - 水缸加盖
2. **物理防护**：
   - **蚊帐**——穿越者第一晚前就应该准备好
   - 门窗装纱网（细竹编/薄纱）
   - 穿长袖长裤（尤其黄昏）
3. **驱蚊**：
   - 艾草/青蒿点燃烟熏
   - 薄荷、香茅等植物种植在房前
   - 夜间在室内燃烧干橘子皮（传统方法）

### 二、老鼠（第二号敌人）

**控制措施**：
1. **堵洞**：老鼠能钻过 1厘米的缝隙，用石块+泥灰堵住所有孔洞
2. **粮食密封**：所有粮食放缸中或悬挂
3. **养猫**：这是古代最好的灭鼠方案
4. **捕鼠器**：木板式/笼式捕鼠夹
5. **不投毒**：老鼠药可能误杀猫狗或人

### 三、苍蝇（污染源）

**控制措施**：
1. **管理垃圾**：
   - 厨余垃圾及时深埋
   - 粪坑加盖
   - 设置远离居住区的垃圾坑
2. **物理防护**：
   - 食物用纱罩盖住
   - 厨房安装纱门
   - 腌制的食物要密封
3. **捕蝇**：甜醋+水的陷阱瓶

### 四、跳蚤

**控制措施**：
1. **环境**：保持地面干净，喷洒草木灰水
2. **宠物**：给猫狗洗澡，睡垫勤换
3. **个人**：勤洗澡、勤换衣
4. **植物驱蚤**：艾草、薄荷铺在床铺下

## 时间与距离尺度

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 积水检查 | 每 7 天检查一次，雨后当天复查 | 住处周围 100 米内无长期积水 | 水缸、坑洼、破罐积水 |
| 鼠洞封堵 | 大于 1 厘米的缝隙都堵住 | 粮袋无咬痕、无新鼠粪 | 白天见鼠，说明鼠量已高 |
| 垃圾粪坑 | 厨余当天处理，粪坑加盖覆土 | 苍蝇不成群 | 粪坑和厨房都见大量苍蝇 |
| 死鼠处理 | 发现后立即隔离，不徒手触碰 | 用工具夹取、焚烧或深埋 | 死鼠后人群发热、淋巴结肿大 |
| 例行记录 | 每周记录虫咬、腹泻、发热和鼠迹 | 能发现趋势 | 只等大规模发病才行动 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 蚊子 | 住处周围 100米内无长期积水，水缸加盖，夜间有蚊帐 |
| 老鼠 | 粮食密封或悬挂，墙洞大于 1厘米的缝隙被堵住 |
| 苍蝇 | 粪坑、厨余、肉类残渣加盖或深埋，食物有纱罩 |
| 跳蚤 | 宠物睡垫每周暴晒或更换，死鼠不徒手接触 |
| 记录 | 每周检查鼠洞、积水、虫咬和腹泻发热病例 |

## 失败征兆

- 黄昏室内蚊声密集，说明防蚊和排水失败。
- 白天见鼠粪、咬痕和粮袋破口，说明鼠量已高。
- 粪坑和厨房周围苍蝇成群，肠道传染病风险上升。
- 发现死鼠后不久人群发热、淋巴结肿大，要警惕鼠疫样风险。
- 用毒饵灭鼠却不控制粮食和孔洞，会造成误食和反复回潮。

## 常见问题

- **Q**: 为什么说蚊子比老鼠更危险？
- **A**: 在现代中国，鼠疫已被控制，但疟疾在历史上南方地区每年夺走数百万人的生命。如果你穿越到南方湿热地区，**蚊子是你的头号生物敌人**。

- **Q**: 穿越后怎么快速判断当地有什么病媒威胁？
- **A**: ①水多不多→蚊子②老鼠多不多→鼠疫③苍蝇多不多→肠道传染病④观察当地人的健康状况

## 安全提醒

- 不要徒手触摸死老鼠（可能携带跳蚤和鼠疫菌）
- 死鼠用火钳夹取焚烧或深埋
- 处理垃圾后必须洗手
- 蚊香燃烧注意防火

## 穿越小说梗

"主角穿越后第一件事就是做蚊帐"——这是一个体现穿越者专业性的好细节！古代不是没有蚊帐，但质量参差不齐。穿越者知道驱蚊防疟的重要性，可以在定居点系统性地做环境改造：排水、垃圾处理、猫粮留猫。真正的高阶操作：**用青蒿做蚊香**（燃烧青蒿驱蚊+杀灭空气中的疟原虫），这是古代就有但在小说里很少写的细节。
