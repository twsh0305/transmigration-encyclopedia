# 4.2 传染病隔离原则

> 黑死病杀死了欧洲三分之一的人口——不是因为治不好，而是因为不知道要隔离。

## 原理

传染病传播三要素：**传染源 → 传播途径 → 易感人群**

隔离的核心理念：**切断传播途径**。

### 常见传染病的隔离期

| 疾病 | 传播方式 | 隔离期 | 穿越后风险评估 |
|------|---------|--------|--------------|
| 鼠疫（黑死病） | 跳蚤/飞沫 | 痊愈后 7 天 | ⭐⭐⭐⭐⭐ 极高 |
| 霍乱 | 粪口 | 症状消失后 7 天 | ⭐⭐⭐⭐ 高 |
| 麻疹 | 飞沫 | 出疹后 5 天 | ⭐⭐⭐ 高 |
| 肺结核 | 飞沫 | 治疗 2 周后 | ⭐⭐⭐⭐ 极高 |
| 麻风病 | 长期接触 | 数年 | ⭐⭐ 但毁容 |
| 天花 | 飞沫/接触 | 痂脱落后 | ⭐⭐⭐⭐⭐ 极烈 |
| 伤寒 | 粪口 | 症状消失后 2 周 | ⭐⭐⭐⭐ 高 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 隔离房间/帐篷 | 收治病人 |
| 干净布料（口罩） | 防护呼吸道 |
| 通风口/窗扇 | 稀释飞沫和污浊空气 |
| 生石灰 | 消毒排泄物 |
| 火 | 处理无法清洗的污染物，须远离房屋和草垛 |
| 肥皂/草木灰 | 洗手 |
| 单独餐具 | 病人专用 |

## 操作步骤

### 一、发现病例后的立即行动

1. **报告**：告知所有接触者
2. **隔离**：将病人移至单独房间或帐篷
3. **划定区域**：标记隔离区，禁止无关人员进入
4. **用具专用**：病人有单独的餐具、衣物、毛巾
5. **专人护理**：固定 1-2 人负责护理

### 二、隔离措施

**病人端**：
1. 病人戴口罩（多层布缝制）
2. 病人咳痰/排泄物用石灰覆盖
3. 病人餐具单独清洗，用沸水煮 20 分钟
4. 病人衣物用沸水浸泡后清洗

**护理人端**：
1. 护理人戴口罩、手套（多层布）
2. 每次接触病人后**必须洗手**
3. 不要直接接触病人的排泄物
4. 护理人的衣物每天更换消毒

**公共端**：
1. 隔离区入口放石灰水脚垫
2. 限制人员流动
3. 健康人与隔离区保持距离（3 米以上）
4. 优先开窗通风、减少同屋停留时间；熏醋和烟熏不能可靠消毒空气

### 三、死亡病例处理

1. **尸体尽快处理**——尸体是巨大的传染源
2. 深埋（1.5米以上）或火化
3. 接触尸体的人必须戴口罩+手套
4. 尸体所在房间彻底消毒

### 四、疫情控制（重点）

| 措施 | 效果 | 说明 |
|------|------|------|
| 封村/封城 | ⭐⭐⭐⭐⭐ | 切断传播最有效 |
| 隔离病人 | ⭐⭐⭐⭐ | 越早越好 |
| 追踪接触者 | ⭐⭐⭐ | 时间久了难追踪 |
| 消毒环境 | ⭐⭐⭐ | 减少环境传播 |
| 口罩 | ⭐⭐⭐⭐ | 飞沫传播有效 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 早隔离 | 发热、腹泻、皮疹、咳血等疑似病例当天分开居住 |
| 分用具 | 餐具、毛巾、衣物、便器专人专用并煮沸或石灰处理 |
| 护理人 | 固定 1-2 人护理，进出洗手，外层衣物单独清洗 |
| 距离 | 呼吸道症状至少保持 3米以上距离，房间通风 |
| 解除 | 症状消失后再按最长潜伏期或疾病规则观察 |

## 失败征兆

- 病人仍参加集市、祭祀、聚餐、同屋睡眠。
- 护理人轮流太多，接触者名单无法追踪。
- 排泄物直接倒入水沟或厕所不覆灰。
- 只熏香熏醋却不隔离、不洗手、不处理排泄物。
- 隔离区断水断食，病人逃离或家属强行探视。

## 常见问题

- **Q**: 没有现代检测手段怎么判断是什么病？
- **A**: 根据症状组合判断：
  - 发热+咳嗽+呼吸困难→呼吸系统传染病
  - 严重腹泻+脱水→霍乱/痢疾
  - 发热+皮疹→天花/麻疹/水痘
  - 发热+淋巴结肿大→鼠疫

- **Q**: 隔离多久才能解除？
- **A**: 一般症状消失后再隔离相当于**最长潜伏期**的时间。鼠疫：症状消失后 7 天。霍乱：症状消失后 7 天。天花：痂完全脱落（约 3-4 周）。

## 安全提醒

- 隔离不是"等死"——要给病人提供食物、水、基本护理
- 照顾者自己要做好防护，不要成为下一个病人
- 当地官府/村民们可能不理解隔离——穿越者需要用自己的声望或权威来推动
- 醋、艾草、香料和烟熏不能替代通风、洗手、煮沸、石灰处理排泄物和接触者追踪

## 穿越小说梗

"主角在古代成功阻止了瘟疫蔓延"——这可能是穿越者**最大的贡献之一**。在古代，人们对传染病的理解是"瘴气""天谴""鬼怪作祟"，隔离被视为不仁不义。如果穿越者知道"隔离+口罩+消毒"这套基础防疫手段，就能有效降低死亡率。但要注意政治因素：在古代封闭村庄/封锁道路会引起恐慌甚至暴力抗拒。穿越者需要用威望和信任来推行现代防疫措施。
