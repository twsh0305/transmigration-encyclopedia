# 4.19 骨折固定与搬运

> 断骨不可怕，可怕的是断骨在愈合时错位——或者搬运中二次损伤。

## 原理

骨折处理三原则：

1. **复位**：将断骨恢复到正常对位
2. **固定**：用夹板使骨折端不再活动
3. **制动**：受伤部位停止活动 4-8 周

### 骨折判断

| 症状 | 可能性 |
|------|--------|
| 受伤后剧痛，无法活动 | 极可能骨折 |
| 听到"咔嚓"声 | 骨折 |
| 肢体变形/异常角度 | 明显骨折 |
| 肿胀+压痛+淤血 | 可能骨折或骨裂 |
| 骨端露出皮肤 | **开放性骨折**（紧急！） |

## 所需材料

| 材料 | 用途 | 替代品 |
|------|------|--------|
| 夹板 | 固定 | 木板、竹片、硬纸板、树枝 |
| 绑带/布条 | 绑扎 | 撕成条的布料 |
| 软垫（衣服/布） | 衬垫 | 任何柔软物 |
| 三角巾/吊带 | 上肢固定 | 大块布料 |
| 木棍+布料 | 临时担架 | 两根长棍+床单 |

## 操作步骤

### 一、闭合性骨折固定

**上臂/前臂骨折**：
1. 用软布衬垫夹板内侧
2. 两块夹板夹住手臂两侧
3. 布条在**骨折上方和下方**各绑一道
4. 屈肘 90°，用三角巾吊在胸前

**小腿/大腿骨折**：
1. 将伤腿与健腿并拢
2. 健腿本身就是最好的"夹板"
3. 在膝、踝两处将双腿绑在一起
4. 或者用长夹板从大腿到脚踝

**手指骨折**：
1. 与相邻手指并拢
2. 用窄布条缠绕固定
3. 用冰棍/小木片固定更好

### 二、开放性骨折

1. **不要试图将露出骨端推回体内！**
2. 用干净敷料覆盖伤口
3. 用夹板固定骨折（**不要压迫骨端**）
4. 优先处理失血
5. 尽快就医——需要清创

### 三、脊柱/颈部骨折（极危险）

- **不要移动伤者！** 移动可能造成瘫痪
- 保持头颈和躯干在一条直线上
- 两侧用衣物/沙袋固定头部
- 用硬板从身下入位搬运

### 四、担架制作

1. 两根长木棍（约 2m）
2. 中间编结绳索/藤条作网
3. 或铺上大衣/毯子
4. **搬运时伤者脚朝前、头朝后**（便于观察）
5. 上下坡保持伤者水平

## 常见问题

- **Q**: 怎么知道骨头接好了？
- **A**: 对比健侧肢体：①长度一致②无异常角度③骨端无台阶感。如果摸到骨头有明显"台阶"，就是没对好。

- **Q**: 骨折需要多久愈合？
- **A**: 上肢 4-6 周，下肢 6-12 周，老年人加倍。固定期间不要拆除夹板。

## 安全提醒

- 如果伤者失去知觉、呼吸困难 → 优先处理气道和呼吸
- 固定不要太紧（检查远端手指/脚趾是否发紫发凉）
- 骨折后 24-48 小时肿胀达到高峰，绑带可能需要调整

## 穿越小说梗

"主角用木板给村民接好了断腿"——夹板固定骨折确实是古代就已掌握的技术，穿越者没有太多知识优势。但你比古人**多知道一点**：固定要包括上下两个关节（如小腿骨折要固定踝和膝）、夹板要垫软材料、不要太紧以免影响血供。这些细节决定了愈合质量。
