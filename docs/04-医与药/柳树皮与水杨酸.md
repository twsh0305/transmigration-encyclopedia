# 4.7 柳树皮与水杨酸

> 两千多年来，柳树皮是人类最重要的止痛药——直到拜耳把它变成了阿司匹林。

## 原理

柳树皮含有**水杨苷**（Salicin），在人体内代谢为**水杨酸**（Salicylic acid），具有退热、镇痛、抗炎作用。

化学转化：柳树皮 → 水杨苷 → 水解 → 水杨酸 → 乙酰化 → 阿司匹林（19 世纪拜耳公司）

## 所需材料

| 材料 | 用途 |
|------|------|
| 柳树皮（白柳/紫柳） | 药源 |
| 水 | 煎煮 |
| 锅 | 煎煮用 |
| 布 | 过滤 |

### 更好的柳树品种

| 品种 | 水杨苷含量 | 常见地 |
|------|-----------|--------|
| 白柳（Salix alba） | 高 | 河边、湿地 |
| 紫柳（Salix purpurea） | 很高 | 河边 |
| 毛柳（Salix viminalis） | 中 | 河边 |
| 旱柳（Salix matsudana） | 中低 | 各地 |
| **杨树皮**（Populus） | 有类似作用 | 各地 |

## 操作步骤

### 柳树皮煎剂

1. **采皮**：春季最佳，选手指粗的嫩枝（老树皮水杨苷含量低）
2. **剥皮**：取树皮的**内层皮**（韧皮部），绿色的部分最有效
3. **切碎**：越小越好，增大接触面积
4. **煎煮**：一把树皮 + 500ml 水，小火煮 15-20 分钟
5. **过滤**：得褐色苦味液
6. **服用**：成人每次 50-100ml 温服

### 剂量参考

| 症状 | 剂量 | 频率 |
|------|------|------|
| 普通头痛 | 50ml | 每 4-6 小时 |
| 发热 | 50-100ml | 每 4-6 小时 |
| 关节炎症 | 100ml | 每天 2-3 次 |

### 更纯的提取（穿越者进阶）

1. 将柳树皮煎液浓缩（小火蒸发至 1/4 体积）
2. 加入少量高度酒（酒精溶解水杨苷）
3. 过滤掉不溶物
4. 得浓缩酊剂——效果更强

## 常见问题

- **Q**: 柳树皮和现代阿司匹林哪个强？
- **A**: 现代阿司匹林（乙酰水杨酸）效果是柳树皮的 10 倍以上，副作用（胃出血）也更小。柳树皮温和但效果有限。

- **Q**: 柳树皮煎水什么味？
- **A**: 极苦，带涩。建议兑少量蜂蜜服用。

- **Q**: 所有柳树都行吗？
- **A**: 所有柳属植物都有水杨苷，但含量差异大。白柳和紫柳含量最高。垂柳也可以但含量较低。

## 安全提醒

- 水杨酸**儿童不能大量使用**（可能引起 Reye 综合征——罕见但致命）
- 水杨酸刺激胃黏膜，长期大量使用可能导致胃出血（和现代阿司匹林一样的副作用）
- 孕妇慎用
- 不要和类似的药重叠使用
- 柳树皮有抗凝血作用，外伤出血时慎用

## 参考资料

## 穿越小说梗

"主角发烧了，吩咐人去弄点柳树皮来"——讲道理、有知识、还能救命。柳树皮止痛的记载横跨数千年、遍及各大文明，穿越者用它完全符合历史逻辑。但也不要神化它——它只是弱效阿司匹林。真正的价值在于：①消炎退热确实有效 ②原料随处可见 ③增加了"穿越者确实懂医"的说服力。
