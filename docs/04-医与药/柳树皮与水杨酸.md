# 4.7 柳树皮与水杨酸

> 两千多年来，柳树皮是人类最重要的止痛药——直到拜耳把它变成了阿司匹林。

## 原理

柳树皮含有**水杨苷**（Salicin），在人体内代谢为**水杨酸**（Salicylic acid），具有退热、镇痛、抗炎作用。

化学转化：柳树皮 → 水杨苷 → 水解 → 水杨酸 → 乙酰化 → 阿司匹林（19 世纪拜耳公司）

| 材料 | 用途 |

| 柳树皮（白柳/紫柳） | 药源 |
| 水 | 煎煮 |
| 锅 | 煎煮用 |
| 布 | 过滤 |

| 品种 | 水杨苷含量 | 常见地 |
|------|-----------|--------|
| 白柳（Salix alba） | 高 | 河边、湿地 |

| **杨树皮**（Populus） | 有类似作用 | 各地 |

## 操作步骤

### 历史低风险用法

1. **识别**：只在能确认柳属植物、采集地无污染时记录。
2. **取材边界**：历史上常取嫩枝树皮内层，现实不为了试药环剥活树。
3. **煎煮概念**：古籍和民间常用小段树皮煎出褐色苦味液，本项目只保留历史尺度，不给入口剂量。
4. **试用边界**：只作为成人短期、轻症、无禁忌情况下的历史替代思路；不替代退烧药、止痛药和医生判断。

### 更纯的提取（穿越者进阶）

不建议在现实中自制浓缩酊剂。浓缩会放大剂量不确定性和胃肠出血风险；小说中可把它写成“药效更强但副作用更明显”的危险进阶工艺。

## 时间与禁忌边界

| 环节 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 识别 | 至少确认柳属特征、采集地和树皮内层 | 不把未知树皮当药 | 只凭“像柳树”就采 |
| 煎煮 | 作为历史描写可写小火 15 到 20 分钟 | 得到褐色苦味液 | 浓缩到膏状或混酒 |
| 观察 | 只用于成人短期轻症思路 | 症状轻、能进水、能休息 | 高烧、脱水、剧痛仍拖延 |
| 禁用 | 儿童、孕妇、哺乳者、阿司匹林过敏、胃溃疡、出血风险者不用 | 先排除禁忌再考虑 | 服后胃痛、黑便、喘憋、耳鸣 |

## 成功标准

- 能识别柳属植物，并知道它只是弱效、剂量不稳定的水杨酸来源。
- 用于写作时能标明禁忌：儿童、孕妇、哺乳者、阿司匹林过敏、胃溃疡、出血风险者不用。
- 现实轻症处理仍优先补水、休息、降温观察，重症及时求医。

## 失败征兆

- 把柳树皮当成万能退烧药，给儿童、病毒感染者或出血伤者使用。
- 服后胃痛、黑便、呕血、皮疹、喘憋、耳鸣或意识异常。
- 高烧不退、颈项强直、呼吸困难、脱水或持续剧痛，却继续拖延。
- 自行浓缩、混酒、叠加其他止痛退烧药，导致剂量不可控。

## 常见问题

- **Q**: 柳树皮和现代阿司匹林哪个强？
- **A**: 现代阿司匹林（乙酰水杨酸）效果是柳树皮的 10 倍以上，副作用（胃出血）也更小。柳树皮温和但效果有限。

- **Q**: 柳树皮煎水什么味？
- **A**: 极苦，带涩。味道只能帮助辨识史料描述，不能证明剂量合适或适合入口。

- **Q**: 所有柳树都行吗？
- **A**: 所有柳属植物都有水杨苷，但含量差异大。白柳和紫柳含量最高。垂柳也可以但含量较低。

## 安全提醒

- 儿童和青少年发热时不要使用柳树皮或阿司匹林类思路，可能出现罕见但致命的 Reye 综合征风险
- 水杨酸刺激胃黏膜，长期大量使用可能导致胃出血（和现代阿司匹林一样的副作用）
- 孕妇慎用
- 不要和类似的药重叠使用
- 柳树皮有抗凝血作用，外伤出血时慎用

## 穿越小说梗

"主角发烧了，吩咐人去找柳树皮"符合历史逻辑，但不能写成万能退烧神药。更可信的写法是：主角先补水、休息、降温观察和排查传染病，再把柳树皮当作有禁忌、剂量不稳的弱效水杨酸来源，而不是给儿童、孕妇或出血伤者随便使用。
