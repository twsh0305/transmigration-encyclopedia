# 4.21 自然灾害应对

> 地震、洪水、蝗灾、旱灾——古代没有天气预报和应急救援，每一次大灾都是一次"人口筛选"。穿越者知道提前准备，就能少死很多人。

## 使用优先级

1. 先保命：撤离危险区、点名、救重伤、切断火源。
2. 再保水食：净水、干粮、临时厕所和垃圾区优先建起来。
3. 接着防次生灾害：火灾、滑坡、洪水污染、传染病、一氧化碳和抢粮冲突。
4. 最后恢复生产：以工代赈、修路、修堤、补种和重建。

## 成功标准

- 每户知道集合点、撤离路线、携带物资和失散后的报到方式。
- 水源、厕所、尸体/动物尸体、厨房和伤员区分开，最少相距 30米。
- 灾后 24 小时内完成伤员分级、饮水净化、临时排泄区和夜间照明。
- 赈灾记录能写清粮食数量、发放对象、工时和剩余库存。

## 失败征兆

- 人员回危房、河滩、滑坡坡脚和断桥附近抢财物。
- 灾后水源混浊、尸体暴露、蚊蝇增多，仍未设置厕所和消毒区。
- 室内炭火取暖后多人头痛、眩晕、恶心、犯困，提示一氧化碳风险。
- 救灾只发粮不组织秩序，几天后出现抢夺、传染病和二次伤亡。

## 安全边界

- 不把动物异常当可靠地震预报；地震准备应建立在建筑加固、避难演练和物资储备上。
- 洪水中不涉水、不驾车冲水、不打捞漂浮财物；退水后仍按污染水处理。
- 封闭空间不烧炭、不用油灯取暖，头痛眩晕时先撤到新鲜空气处。

## 时间与距离尺度

| 场景 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 灾后首日 | 24 小时内完成点名、伤员分级、净水、厕所和照明 | 人、水、厕、食、伤员区分开 | 只抢财物、不建秩序 |
| 卫生分区 | 水源、厕所、尸体/动物尸体、厨房、伤员区至少相距 30 米 | 粪污和尸体不污染水源 | 灾后腹泻发热集中出现 |
| 避洪台 | 高出历史最高水位至少 2 米 | 洪峰来时能容纳人员和粮食 | 高台被水漫过 |
| 动物尸体 | 深埋 1.5 米以上并撒石灰 | 蚊蝇和异味减少 | 尸体暴露、犬鼠啃食 |
| 暴雪燃料 | 按正常消耗 3 倍储备 | 封路期间不断火 | 室内烧炭却无通风 |

## 地震

### 原理与预兆

古代地震预测常看**动物行为异常**和**地下水变化**，但现代地震学并不认为这些能可靠预测地震。它们只能作为“提高警惕”的异常线索，不能替代房屋加固、避难演练和应急物资。

| 预兆 | 说明 |
|------|------|
| 动物异常 | 牛羊不进圈、老鼠白天跑、蛇出洞冬眠期、狗集体狂吠、鸡上树 |
| 地下水异常 | 井水突然上升/下降/变浑/变味、泉水断流或涌新泉 |
| 地声地光 | 地下传来闷雷声/撕布声，天空出现蓝/白色光 |
| 小震频繁 | 地区内小震次数突然增加（但有些大地震前无小震） |

### 应对流程

#### 震前准备

1. **加固房屋**：木结构（榫卯）比砖石结构抗震；墙体和房梁用木柱支撑加固
2. **储备物资**：水、干粮、药品、火柴、毛毯集中存放于安全处
3. **规划避难区**：选空旷平地（远离山体、河岸、建筑物）
4. **固定家具**：大型柜子固定于墙面，重物放低处
5. **建立预警**：安排人夜间观察动物/水井异常

#### 震时

- **室内**：躲在坚固桌子/床下，或墙角、门框处，护住头部
- **室外**：远离建筑物、围墙、电线杆，向开阔地跑
- **绝对不要**：使用明火（燃气泄漏会爆炸）、乘电梯、跳楼

#### 震后救治

| 伤情 | 处理 |
|------|------|
| 挤压伤（被压肢体） | 先补液再解除压迫，突然松开会引发"挤压综合征"（肾衰竭死亡） |
| 骨折 | 夹板固定，搬运时保持脊柱直线 |
| 头部外伤 | 止血包扎，意识不清者侧卧位防窒息 |
| 呼吸道损伤 | 清理口鼻粉尘，人工呼吸 |
| 心理创伤 | 陪伴、安抚、分担重体力劳动（让人有事做，减少恐惧） |

### 次生灾害

| 次生灾害 | 危害 | 应对 |
|---------|------|------|
| 火灾 | 地震后燃气/油灯泄漏引发 | 震后立即灭火，切断火源 |
| 山体滑坡/滚石 | 山区地震后山石松动 | 震后立即撤离至开阔地 |
| 堰塞湖 | 山体滑坡堵塞河道 | 派人监测水位，必要时人工引流 |
| 瘟疫 | 尸体腐烂、水源污染 | 立即掩埋尸体，消毒水源 |

## 洪水

### 洪水预兆

| 预兆 | 说明 |
|------|------|
| 连续暴雨 | 上游持续降雨，下游数小时到数天后洪峰到达 |
| 河水变浑、水位上涨 | 最简单直接的信号 |
| 上游漂来杂物 | 树枝、家具、牲畜尸体顺流而下 |
| 动物上山 | 鼠、蛇、蚁往高处迁移 |

### 应对流程

#### 防洪准备

1. **加高河堤**：沙袋/土袋/石笼加固薄弱段
2. **疏通河道**：清除阻碍水流的杂物和沉积物
3. **建避洪台**：每个村庄建一个高台（高出历史最高水位至少2米）
4. **准备救生工具**：木筏、葫芦/皮囊（浮水）、绳索
5. **转移物资**：粮食、重要器物提前移至高处

#### 洪水期间

- **放弃财物保命**：人先走，不要舍不得家产
- **往高处走**：高地、山丘、坚固楼房上层
- **绝不涉水过河**：看似平静的水面下可能有暗流
- **小心蛇虫**：洪水把蛇、鼠、虫从洞穴中赶出，它们会附着在一切突出物上
- **饮用水净化**：洪水过后水源全部被污染，必须煮沸方可饮用

#### 水灾后防疫

| 措施 | 理由 |
|------|------|
| 全面消毒 | 洪水带来大量泥沙、粪便、尸体，地面用石灰覆盖 |
| 煮沸饮水 | 水源被严重污染，霍乱、痢疾、伤寒高发 |
| 清理淤泥 | 洪水退去后淤泥中藏有大量病菌 |
| 灭鼠 | 洪水后鼠类聚集，鼠疫风险上升 |
| 处理动物尸体 | 深埋（1.5米以上）+ 撒石灰 |

## 蝗灾

### 原理

蝗虫在干旱年景大量繁殖，成群迁飞，一天可吃掉上万人的口粮。古代中国蝗灾频率极高——"蝗灾之后，赤地千里"。

### 应对策略

#### 预防

- **翻耕灭卵**：秋季翻耕土壤，将蝗虫卵暴露于天敌和严寒
- **养殖天敌**：鸡、鸭、椋鸟都是蝗虫天敌。一群鸭子一天可吃掉数万只蝗虫（明清时期就有"鸭兵灭蝗"的记录）
- **早期发现**：在蝗虫若虫（未长翅）阶段消灭，效率最高

#### 灭蝗方法

| 方法 | 操作 | 效果 |
|------|------|------|
| 挖沟陷杀 | 挖深沟，驱赶若虫入沟后掩埋 | ⭐⭐⭐⭐ 高效 |
| 火烧 | 夜间用火把吸引成虫，集中焚烧 | ⭐⭐⭐ 有效 |
| 网捕 | 用大网/布匹捕捉成虫做饲料 | ⭐⭐⭐ 可做蛋白质来源 |
| 烟熏 | 用湿草+硫磺发烟驱赶 | ⭐⭐ 驱赶而非灭杀 |

#### 蝗灾后自救

蝗虫不吃**绿豆、豇豆、薯类、芋头**——如果预知蝗灾，提前种这些作物可以减少损失。
蝗虫本身**高蛋白可食用**：油炸蝗虫、烤蝗虫、蝗虫干——不仅是活命粮，有些地方至今还是名菜。

## 旱灾

### 征兆

雨水明显少于往年、河流水位下降、水井干涸、土地龟裂、次年的蝗灾风险上升。

### 应对

| 措施 | 说明 |
|------|------|
| 储水 | 雨季提前蓄满水库、水窖、水缸 |
| 深挖井 | 旱季地下水位下降，需挖更深的水井 |
| 节水灌溉 | 滴灌/沟灌（用陶罐装水埋土中做简易滴灌） |
| 选耐旱作物 | 高粱、粟（小米）、薯类比小麦水稻耐旱 |
| 分粮配给 | 按人头定量分配存粮，严禁囤积居奇 |
| 以工代赈 | 修水利、筑路来换取粮食，避免白白施舍导致依赖 |

### 旱灾后次生灾害

旱灾 → 蝗灾（干旱利于蝗虫繁殖） → 饥荒 → 流民 → 瘟疫

**穿越者必须把这条链切断在第一环**。旱灾年开始就要储备粮食、准备灭蝗、控制人口流动。

## 暴雪/极端低温

| 应对 | 说明 |
|------|------|
| 储备燃料 | 木柴/木炭需达到正常消耗的3倍 |
| 加固屋顶 | 清除积雪防止压塌 |
| 防寒衣物 | 皮毛→棉衣→干草填充多层穿着 |
| 食物储备 | 暴雪封锁期间无法外出获取食物 |
| 冻伤处理 | 温水复温（**不要用热水或火烤**），冻伤部位不要揉搓 |
| 防一氧化碳 | 室内取暖必须留通风口 |

| 盐 | 消毒、保存食物、补充电解质 | ⭐⭐⭐⭐⭐ |
| 糖 | 紧急能量来源、补液 | ⭐⭐⭐⭐⭐ |
| 生石灰 | 消毒、做砂浆防水 | ⭐⭐⭐⭐ |

| 干净布料 | 包扎、做口罩、做过滤器 | ⭐⭐⭐⭐ |

## 穿越小说梗

"穿越者利用现代知识在古代大灾中救下一城百姓"——这是穿越文的经典桥段，但几个细节容易被忽略：

1. **越早知道越好**：穿越者如果能记住历史上某年某地发生了什么灾害（比如"1346年，黄河流域大旱，次年蝗灾"），提前半年准备，效果是灾难发生后再应对的100倍。

2. **救灾需要组织力**：不是穿越者一个人知道怎么做就够了——需要动员成百上千人。穿越者的真正价值不是做所有事，而是**指挥别人做正确的事**。

3. **赈灾是双刃剑**：白白施粮会养成依赖、消耗有限资源。穿越者应该用"以工代赈"的方式（修路、修水利、筑城墙），既赈了灾，又有了建设成果。
