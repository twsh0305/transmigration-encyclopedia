# 4.6 接种与免疫原理

> 人类战胜天花——这个杀死上亿人的病毒——靠的不是药，而是"免疫记忆"。

## 原理

免疫系统有**记忆功能**：

1. 第一次感染某种病原体时，免疫系统产生抗体和记忆细胞
2. 第二次接触同种病原体时，记忆细胞迅速响应，在病原体造成症状前将其清除
3. **接种**就是利用这个原理：用低毒或无毒的病原体预先"训练"免疫系统

### 关键概念

| 概念 | 说明 |
|------|------|
| 抗原 | 能刺激免疫反应的外来物质（细菌、病毒等） |
| 抗体 | 免疫细胞产生的蛋白质，特异性结合抗原 |
| 记忆细胞 | 长期存活，记住抗原——再次接触时快速响应 |
| 主动免疫 | 身体自己产生抗体（接种/自然感染） |
| 被动免疫 | 直接输入抗体（母体传给婴儿） |

## 操作步骤

### 人痘接种术（古代中国发明）

这是天花疫苗出现之前，中国人在 16 世纪发明的免疫方法——**人类最早的接种术**。

**原理**：取天花患者身上减毒后的痘痂，接种到健康人身上，引起一次温和的天花感染，从而获得终身免疫力。

**方法一：鼻苗法（最安全）**

1. 追踪新近痊愈的天花患者（出痘结束 20-30 天）
2. 取其干燥的痘痂，研磨成粉
3. 将粉末吹入健康人的鼻腔
4. 观察 7-14 天：通常出现轻微发热和少量皮疹
5. 恢复后获得终身免疫

**方法二：衣法（更低毒）**

让健康儿童穿天花轻症患者的贴身衣物，接触其皮肤脱屑。

### 牛痘接种术（詹纳，1796——更安全）

**原理**：牛痘病毒（Cowpox）与天花病毒（Variola）抗原相似，但牛痘对人基本无害。接种牛痘后，免疫系统产生的抗体也能"认出"并抵抗天花。

**获取牛痘（如果穿越时有牛）**：

1. 找到患有牛痘的奶牛（乳房上有水泡和溃疡）
2. 用消毒刀片取牛痘水泡中的**浆液**
3. 在健康人的上臂划 2-3 道浅痕（深度以不出血为度）
4. 将牛痘浆液涂在划痕上
5. 7-10 天后局部出现痘疱和轻微发热
6. 恢复后获得免疫

### 穿越者可做的其他免疫

| 免疫对象 | 方法 | 可行性 |
|---------|------|--------|
| 天花 | 人痘鼻苗或牛痘接种 | ⭐⭐⭐ 可行 |
| 狂犬病 | 被咬后无法（需现代疫苗） | ⭐ |
| 破伤风 | 彻底清创（预防为主） | ⭐⭐ |
| **小规模群体免疫** | 隔离+消毒 | ⭐⭐⭐ 更务实 |

## 常见问题

- **Q**: 人痘接种安全吗？
- **A**: 大约 1-2% 的接种者会发展成重症天花并死亡（远低于自然感染 30% 的死亡率）。在古代没有更好的选择，人痘法已经足以拯救大量人口。

- **Q**: 怎么存储痘苗？
- **A**: 痘痂密封阴凉处可保存数月。牛痘浆液需当天使用或存储在密封玻璃管中（地窖可保存数日）。

- **Q**: 穿越后在古代推行接种会遇到什么问题？
- **A**: ①宗教/文化阻力——有人会认为"这不是在种毒吗"②人口流动——接种需要组织③痘苗供应——需要持续有患者提供痘痂。穿越者需要用治疗效果来建立信任。

## 安全提醒

- 人痘接种有 1-2% 致死率——这在古代是可以接受的，但现代人心理上可能难以承受
- 接种后的人也是传染源（有轻微传染性），需要隔离
- 牛痘接种要确认是牛痘不是真天花
- 刀片每次使用前必须消毒

## 穿越小说梗

"主角通过种人痘/牛痘拯救了疫情"——这是穿越小说最硬核的医疗成就之一。穿越者的知识优势：①知道"接种"的原理（比古人只知道"经验"高出维度）②知道人痘→牛痘的进化路线③知道「用减毒病原体做疫苗」。如果穿越到清朝康熙年间，天花是真正的"国祸"——康熙本人就得过天花。穿越者若能帮助推广人痘/牛痘，就是**拯救了无数生命+获得了皇帝的信任**两开花。
