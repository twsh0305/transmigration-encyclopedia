# 4.6 接种与免疫原理

> 人类战胜天花——这个杀死上亿人的病毒——靠的不是药，而是"免疫记忆"。

## 原理

免疫系统有**记忆功能**：

1. 第一次感染某种病原体时，免疫系统产生抗体和记忆细胞
2. 第二次接触同种病原体时，记忆细胞迅速响应，在病原体造成症状前将其清除
3. **接种**就是利用这个原理：用低毒或无毒的病原体预先"训练"免疫系统

### 关键概念

| 概念 | 说明 |
|------|------|
| 抗原 | 能刺激免疫反应的外来物质（细菌、病毒等） |
| 抗体 | 免疫细胞产生的蛋白质，特异性结合抗原 |
| 记忆细胞 | 长期存活，记住抗原——再次接触时快速响应 |
| 主动免疫 | 身体自己产生抗体（接种/自然感染） |
| 被动免疫 | 直接输入抗体（母体传给婴儿） |

## 历史接种术（不作为复现教程）

### 人痘接种术（古代中国发明）

这是天花疫苗出现之前，中国人在 16 世纪发明的免疫方法——**人类最早的接种术**。

**原理**：取天花患者身上减毒后的痘痂，接种到健康人身上，引起一次温和的天花感染，从而获得终身免疫力。

历史上的鼻苗法、衣法等做法都存在把轻症变成重症、引发局部疫情、传播其他感染的风险。它们适合写历史和理解免疫思想，不应在现实中复现。

### 牛痘接种术（詹纳，1796——更安全）

**原理**：牛痘病毒（Cowpox）与天花病毒（Variola）抗原相似，但牛痘对人基本无害。接种牛痘后，免疫系统产生的抗体也能"认出"并抵抗天花。

牛痘接种比人痘安全得多，是现代疫苗史的关键节点。但“从动物病灶取材料接种人”仍然是高风险生物操作：病原体身份、杂菌污染、接种剂量和传播隔离都不可凭经验保证。

### 更务实的公共卫生路线

| 风险对象 | 可写做法 | 现实边界 |
|---------|----------|----------|
| 天花 | 历史上可写人痘、牛痘推动免疫观念 | 不复现取痂、研磨、划刺、鼻吸或动物病灶接种 |
| 狂犬病 | 写成“暴露后必须现代疫苗”带来的无力感 | 不靠草药、烧灼、放血替代疫苗 |
| 破伤风 | 彻底清创、减少污染伤口、保持器械清洁 | 深刺伤和污染伤需专业处置 |
| 小规模群体免疫 | 隔离病例、追踪接触者、消毒污染物、减少聚集 | 比自制疫苗更务实、更可控 |

## 时间与隔离尺度

| 场景 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 疑似烈性传染病 | 立刻隔离病例，记录发病日期和接触者 | 接触链能追到人 | 病人继续赶集、聚会 |
| 历史接种描写 | 写出接种后 2 到 3 周隔离观察 | 体现风险、社会阻力和失败可能 | 写成无痛无风险神技 |
| 污染物处理 | 衣物、被褥、器具单独标记和消毒 | 不与健康人用品混放 | 同一刀片或布料多人共用 |
| 现代语境 | 疑似天花材料按公共卫生事件上报 | 不接触、不转移、不私藏 | 试图自行采样或保存 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 现实边界 | 不自行采集、研磨、接种痘痂或动物病灶材料 |
| 历史理解 | 能区分人痘、牛痘、现代疫苗和自然感染 |
| 疫情处置 | 优先隔离病例、追踪接触者、消毒污染物、减少聚集 |
| 小说可信 | 写出接种后隔离 2-3 周、社会阻力、痘苗供应和失败风险 |
| 资料核对 | 疫苗和传染病信息核对 CDC、WHO 等可靠来源 |

## 失败征兆

- 把人痘写成无风险神技，忽略死亡和传播风险。
- 把牛乳房水泡都当牛痘，可能混入其他病原。
- 接种后不隔离，反而可能制造新的传播链。
- 用同一刀片连续操作多人，传播血源感染。
- 在现代语境中试图复现人痘或自制疫苗，越过生物安全边界。

## 常见问题

- **Q**: 人痘接种安全吗？
- **A**: 不安全。它属于把活病原暴露给健康人的历史做法，可能导致重症、死亡和新的传播链；现代语境中不应复现。

- **Q**: 怎么存储痘苗？
- **A**: 本项目不提供痘苗采集、保存、转运或接种细节。现代疑似天花或相关材料应视为公共卫生和生物安全事件，交由专业机构处理。

- **Q**: 穿越后在古代推行接种会遇到什么问题？
- **A**: 宗教和文化阻力、人口流动、隔离执行、记录混乱、感染传播风险都会成为难点。更可信的写法是先建立隔离、告知、登记、观察和失败处置制度。

## 安全提醒

- 人痘接种是暴露活病原的历史做法，存在死亡、重症和制造新传播链的风险，现代语境中不应复现。
- 历史接种术只能作为制度和观念描写：隔离、登记、观察、失败处置比“神技成功”更可信。
- 不自行采集、保存、转运、研磨或接触痘痂、动物病灶、脓液和疑似疫苗材料。
- 现实中天花已被根除；任何疑似天花材料都属于公共卫生与生物安全事件，应交给专业机构。

## 穿越小说梗

"主角通过免疫观念改变防疫制度"比"现场自制疫苗"可信得多。穿越者的知识优势不是会动手取痂，而是能说清隔离观察、接触者登记、病例追踪、失败风险和公众沟通。若写到清代天花背景，更稳妥的写法是推动官方隔离、记录和专业接种机构，而不是让主角亲手复现高风险生物操作。
