# 4.4 基本卫生与感染预防

> 在微生物被发现之前，人类唯一对抗感染的手段就是洗手——但他们不知道为什么要洗。

## 原理

感染三要素：**传染源 → 传播途径 → 易感人群**

切断任何一个环节即可阻止感染。穿越者最大的医疗优势不是"治疗"，而是**预防**——知道感染是怎么传播的。

### 常见感染途径

| 途径 | 穿越环境中的高发情况 | 预防方法 |
|------|---------------------|---------|
| 接触传播 | 手→伤口、共用毛巾 | 洗手、隔离 |
| 水源传播 | 饮用生水 | 煮沸/消毒 |
| 食物传播 | 不洁厨具、苍蝇叮爬 | 盖好食物、煮熟 |
| 空气传播 | 咳嗽、打喷嚏 | 通风、掩口鼻 |
| 媒介传播 | 蚊虫、老鼠、跳蚤 | 纱帐、灭鼠、清洁 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 清水 | 清洁 |
| 草木灰/肥皂 | 去油脂和细菌 |
| 高度酒（蒸馏酒） | 临时擦拭皮肤周边或小物件，不能替代灭菌 |
| 大锅 | 煮沸消毒 |
| 干净的布料 | 敷料、口罩 |
| 生石灰 | 消毒排泄物 |

## 操作步骤

### 一、洗手（最重要的卫生措施）

**什么时候必须洗手**：
- 处理伤口前后
- 接触病人前后
- 做饭前
- 上完厕所后
- 接触动物后

**方法**：
1. 流动水冲洗
2. 用肥皂/草木灰搓洗至少 20 秒
3. 重点：手掌、手背、指缝、指甲
4. 干净布擦干或晾干

### 二、器械消毒

在没有高压蒸汽灭菌的条件下：

| 方法 | 适用 | 操作 |
|------|------|------|
| 煮沸消毒 | 金属/骨制器械 | 沸水中煮 10 分钟以上 |
| 火焰灼烧 | 金属工具表面应急去污 | 会留下烟灰和氧化物，不能保证无菌手术 |
| 酒精擦拭 | 小物件或皮肤周边 | 高度酒不是现代 70% 医用酒精，不能用于深伤口 |
| 日光消毒 | 敷料、布料 | 烈日下暴晒半天 |

### 三、伤口护理的卫生

1. 处理伤口前**必须洗手**
2. 用煮沸冷却后的水清洗
3. 敷料必须干净
4. 用过的敷料**烧掉**（不要重复使用）
5. 包扎后不要在伤口上摸来摸去

### 四、环境清洁

1. **垃圾**：远离居住区，定期清理
2. **厕所**：盖土处理粪便（远离水源 30米以上）
3. **排水**：清除积水（蚊虫繁殖地）
4. **通风**：保持居住空间空气流通

### 五、个人卫生

1. 剃发/短发（减少头虱）
2. 定期清洗床铺
3. 衣物在阳光下暴晒
4. 不与病人共用碗筷

## 时间、距离与消毒尺度

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 洗手 | 肥皂或草木灰搓洗至少 20 秒 | 手掌、手背、指缝、指甲都洗到 | 做饭、包扎、接生前不洗手 |
| 器械煮沸 | 沸水中煮 10 分钟以上 | 金属、骨针冷却后再用 | 用脏刀反复处理伤口 |
| 酒精擦拭 | 高度酒只作临时去污 | 作为小物件或皮肤周边处理 | 把饮酒当人体消毒，或倒入深伤口 |
| 厕所距离 | 距水源至少 30 米 | 防蝇、防渗、覆土 | 粪坑靠近井或厨房 |
| 病人用品 | 餐具、毛巾、寝具单独使用 | 护理后洗手换外层衣物 | 多人共用毛巾和敷料 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 洗手 | 关键时刻用水和肥皂或草木灰搓洗至少 20 秒 |
| 饮水 | 饮用水过滤后煮沸，储水容器加盖并用专用勺取水 |
| 伤口 | 接触伤口前后洗手，敷料煮沸或日晒，脏敷料烧毁 |
| 厕所 | 距水源至少 30米，如厕后覆土或灰，防蝇防渗 |
| 病人 | 单独餐具、毛巾、寝具，护理后洗手换外层衣物 |

## 失败征兆

- 做饭、接生、包扎前不洗手。
- 生水看着清就喝，或开水敞口存放被二次污染。
- 苍蝇能接触粪坑和食物，粪口传播链没有切断。
- 多人共用毛巾、碗筷和敷料。
- 同一住处连续多人腹泻、发热或伤口化脓，说明卫生系统失效。

## 常见问题

- **Q**: 没有肥皂怎么洗手？
- **A**: 草木灰水有弱碱性，去油脂效果好，是古代最重要的洗手材料。皂角、无患子也可用。

- **Q**: 敷料不够怎么办？
- **A**: 洗净的布料煮沸消毒后可重复使用。但用于化脓伤口的敷料建议烧毁。

- **Q**: 穿越后最容易得的病是什么？
- **A**: 腹泻（水源性）和伤口感染（外伤性）。只要做好这两件事就赢了 80%：**喝开水+彻底清创**。

## 安全提醒

- 生石灰遇水产热，不要直接接触皮肤
- 草木灰水有腐蚀性，洗手后清水冲净
- 隔离病人时注意自我保护

## 穿越小说梗

"主角穿越后很少生病"——这不是主角光环，而是现代人潜意识里的卫生习惯在起作用。穿越者最重要的医疗贡献不是发明什么神药，而是**教会当地人洗手**。历史上没有细菌理论之前，医生做手术不洗手是常态，产妇"产褥热"死亡率高达 20-30%。穿越者一句"碰伤口之前去洗个手"就能救下无数人。
