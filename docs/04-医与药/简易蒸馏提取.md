# 4.12 简易蒸馏提取

> 蒸馏能分离香味、水分和部分挥发物；它不是把植物变成安全药物的捷径。

## 原理

蒸馏提取基于**物质的沸点差异**：

1. 加热混合物→低沸点成分先气化
2. 蒸气通过冷凝管→遇冷液化
3. 收集到的液体就是目标物质

### 穿越者使用的三种蒸馏

| 类型 | 提取对象 | 操作温度 | 用途 |
|------|---------|---------|------|
| 酒精蒸馏 | 乙醇 | 78.5摄氏度 | 酒精风险、燃烧风险和历史制剂 |
| 水蒸气蒸馏 | 挥发油 | 100摄氏度 | 精油、纯露、香料 |
| 水蒸馏 | 水溶性成分 | 100摄氏度 | 浓缩思路，不等于安全药液 |

## 所需材料

| 材料 | 用途 | 替代品 |
|------|------|--------|
| 蒸馏锅 | 加热原料 | 陶锅/铁锅 |
| 冷凝管 | 冷却蒸气 | 斜放竹筒+冷水 |
| 接收器 | 收集馏液 | 碗/竹筒 |
| 密封泥 | 密封接口 | 黏土+面粉 |
| 冷却水 | 冷凝 | 流动冷水 |

## 操作步骤

### 简易蒸馏装置

1. **锅**：装原料液体
2. **锅盖**：倒扣的陶锅盖，或锥形竹编内衬布
3. **冷凝**：锅盖上方放一盆冷水（或盖湿布）
4. **收集**：锅盖内壁嵌一个竹筒/陶管引出冷凝液

**操作**：
1. 原料+水放入蒸馏锅
2. 大火烧开→转小火保持沸腾
3. 蒸气在冷凝面液化→沿斜面流入收集器
4. 前段（头液）含高浓度酒精/挥发油→单独收集

### 提取挥发油/精油

1. 采集芳香植物（薄荷、艾草、桂花、玫瑰）
2. 切碎放入蒸馏锅，加适量水
3. 蒸馏 1-2 小时
4. 馏出液分为两层：
   - 上层：挥发油（精油）——用滴管分离
   - 下层：纯露（花水）——可直接用于护肤

### 药液浓缩边界

1. 药材煎煮取液（药汤）
2. 将药汤放入蒸馏锅
3. 小火蒸发（不是蒸馏！是浓缩）
4. 蒸发至原体积的 1/3 至 1/5
5. 得浓缩膏（稠膏）

浓缩会同时放大有效成分、杂质、毒性和剂量误差。现实中不把未知草药浓缩膏入口；只可作为历史制剂、外用边界或小说工艺描写。

## 成功标准

- 装置全程不密闭，蒸气能顺利进入冷凝区，无鼓胀和喷液。
- 冷凝水保持低温，出口能稳定滴液，不大量漏气。
- 水蒸气蒸馏 1-2 小时后，纯露气味清晰，精油层少量浮在表面。
- 浓缩药液只做外用或进一步制剂参考，记录原液体积、浓缩比例和时间。

## 失败征兆

- 接口堵塞、锅盖跳动、管道鼓胀：压力积聚，立即停火降温。
- 酒精蒸气遇明火、闻到强烈酒气：火灾风险，先通风撤火。
- 馏液浑浊发臭、有焦糊味：原料烧焦或污染。
- 精油直接涂黏膜、入口或给儿童使用：越过安全边界。

## 常见问题

- **Q**: 没有玻璃管怎么做冷凝？
- **A**: 竹管 + 湿布包裹 + 冷水淋洒是最简单的替代。或者用金属勺做冷凝面。

- **Q**: 怎么增加蒸馏效率？
- **A**: ①原料切碎②延长冷凝路径③冷却水保持冷（随时换水）④密封接口用泥糊严实

- **Q**: 蒸馏出来的精油怎么保存？
- **A**: 深色小瓶密封，阴凉处可保存 1-2 年。精油有防腐作用，不易变质。

## 安全提醒

- 蒸馏过程中保持通气，防止高压爆炸
- 蒸馏酒注意防火（酒精蒸气易燃！）——远离明火
- 某些植物精油有刺激性，勿直接接触黏膜
- 精油浓度高，不内服，不接触眼口鼻、黏膜、婴幼儿皮肤和大面积破损皮肤

## 穿越小说梗

蒸馏是穿越者"化学科技树"的第一个分支。同一个装置可以解释酒精、精油、纯露和浓缩思路，但越靠近入口药物，风险越高。更可信的写法是主角先解决通气、防火、冷凝和标识，再把蒸馏产物用于香料、清洁和低风险外用，而不是直接宣布"三次蒸馏成药"。
