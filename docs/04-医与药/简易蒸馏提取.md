# 4.12 简易蒸馏提取

> 一次蒸馏获得的是酒精，二次蒸馏获得的是香水，三次蒸馏获得的是——药。

## 原理

蒸馏提取基于**物质的沸点差异**：

1. 加热混合物→低沸点成分先气化
2. 蒸气通过冷凝管→遇冷液化
3. 收集到的液体就是目标物质

### 穿越者使用的三种蒸馏

| 类型 | 提取对象 | 操作温度 | 用途 |
|------|---------|---------|------|
| 酒精蒸馏 | 乙醇 | 78.5°C | 消毒、药酒、烈酒 |
| 水蒸气蒸馏 | 挥发油 | 100°C | 精油、纯露 |
| 水蒸馏 | 水溶性成分 | 100°C | 浓缩药液 |

## 所需材料

| 材料 | 用途 | 替代品 |
|------|------|--------|
| 蒸馏锅 | 加热原料 | 陶锅/铁锅 |
| 冷凝管 | 冷却蒸气 | 斜放竹筒+冷水 |
| 接收器 | 收集馏液 | 碗/竹筒 |
| 密封泥 | 密封接口 | 黏土+面粉 |
| 冷却水 | 冷凝 | 流动冷水 |

## 操作步骤

### 简易蒸馏装置

1. **锅**：装原料液体
2. **锅盖**：倒扣的陶锅盖，或锥形竹编内衬布
3. **冷凝**：锅盖上方放一盆冷水（或盖湿布）
4. **收集**：锅盖内壁嵌一个竹筒/陶管引出冷凝液

**操作**：
1. 原料+水放入蒸馏锅
2. 大火烧开→转小火保持沸腾
3. 蒸气在冷凝面液化→沿斜面流入收集器
4. 前段（头液）含高浓度酒精/挥发油→单独收集

### 提取挥发油/精油

1. 采集芳香植物（薄荷、艾草、桂花、玫瑰）
2. 切碎放入蒸馏锅，加适量水
3. 蒸馏 1-2 小时
4. 馏出液分为两层：
   - 上层：挥发油（精油）——用滴管分离
   - 下层：纯露（花水）——可直接用于护肤

### 中药浓缩液

1. 药材煎煮取液（药汤）
2. 将药汤放入蒸馏锅
3. 小火蒸发（不是蒸馏！是浓缩）
4. 蒸发至原体积的 1/3 至 1/5
5. 得浓缩膏（稠膏）

## 常见问题

- **Q**: 没有玻璃管怎么做冷凝？
- **A**: 竹管 + 湿布包裹 + 冷水淋洒是最简单的替代。或者用金属勺做冷凝面。

- **Q**: 怎么增加蒸馏效率？
- **A**: ①原料切碎②延长冷凝路径③冷却水保持冷（随时换水）④密封接口用泥糊严实

- **Q**: 蒸馏出来的精油怎么保存？
- **A**: 深色小瓶密封，阴凉处可保存 1-2 年。精油有防腐作用，不易变质。

## 安全提醒

- 蒸馏过程中保持通气，防止高压爆炸
- 蒸馏酒注意防火（酒精蒸气易燃！）——远离明火
- 某些植物精油有刺激性，勿直接接触黏膜
- 精油浓度高，内服需谨慎（大部分精油不能内服）

## 穿越小说梗

蒸馏是穿越者"化学科技树"的第一个分支。同一个装置可以做：①酒精（消毒+酿酒）②精油（化妆品+香料）③纯露（护肤+芳香）④浓缩药液。一器多用。蒸馏瓶（烧瓶）是穿越者最想带的现代工具之一——但在没有玻璃的条件下，陶制蒸馏器也可以用。
