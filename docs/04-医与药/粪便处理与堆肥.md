# 4.13 粪便处理与堆肥

> 处理不好，粪便就是瘟疫的源头；处理得当，粪便就是土地的养料。

## 原理

粪便的危害主要来自**病原体**：

1. 人类/动物粪便中含有大量细菌、病毒和寄生虫卵
2. 如果污染水源或食物，引发霍乱、伤寒、痢疾等
3. 正确处理的粪便——经发酵后高温杀灭病原体→安全肥料

### 堆肥的生物学

| 阶段 | 温度 | 主要微生物 | 持续时间 |
|------|------|-----------|---------|
| 中温期 | 20-40摄氏度 | 嗜温菌 | 3-7 天 |
| 高温期 | 40-65摄氏度 | 嗜热菌 | 1-3 周 | 
| 降温期 | 下降 | 重新定植 | 2-4 周 |
| 成熟期 | 常温 | 放线菌、真菌 | 1-2 月 |

**高温期（50摄氏度以上 持续数天）可杀灭大部分病原体和杂草种子**。

## 所需材料

| 材料 | 用途 |
|------|------|
| 铲子 | 挖坑 |
| 草木灰/石灰 | 覆盖、除臭 |
| 干草/秸秆 | 堆肥碳源 |
| 粪便 | 堆肥原料 |
| 覆盖木板/草席 | 遮盖堆肥 |

## 操作步骤

### 一、厕所的建造（穿越第一要务）

**干厕（最简单的方案）**：
1. 挖一个深 50-80厘米的坑
2. 坑边放一桶草木灰或干土
3. 每次如厕后覆盖一层灰/土（除臭防蝇）
4. 坑满后填土，在旁边挖新坑

**发酵厕所（更好的方案）**：
1. 挖两个坑交替使用
2. 每次如厕后加一层草木灰+干草
3. 一坑满后用土密封，等待 6 个月以上自然发酵
4. 发酵后的产物是安全肥料

### 二、堆肥制作

**好堆肥 = 棕色料（碳） + 绿色料（氮） + 水 + 空气**

| 棕色料（碳源） | 绿色料（氮源） |
|---------------|--------------|
| 干草/秸秆 | 新鲜粪便 |
| 木屑/锯末 | 厨余/菜叶 |
| 枯叶 | 青草 |
| 纸/布（天然） | 豆渣/糟粕 |

**堆肥步骤**：
1. 选一个排水良好的位置
2. 先铺 20厘米棕色料（底部）
3. 交替铺绿色料和棕色料（每层 10-15厘米）
4. 最后盖一层干草或土（防虫防味）
5. 保持湿润（像拧干的湿海绵）
6. 每 1-2 周翻堆一次（补充氧气）
7. 2-4 个月后→成熟的黑色堆肥

### 三、粪便发酵的安全期

| 处理方式 | 安全等待期 | 说明 |
|---------|-----------|------|
| 密封堆放 | 6-12 个月 | 自然发酵 |
| 高温堆肥 | 2-3 个月 | 翻堆+保持高温 |
| 直接施用 | **严禁！** | 病原体污染作物 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 厕所选址 | 距离水源至少 30米，且不在水源上坡或上游 |
| 覆盖 | 每次如厕后立即覆盖干土、草木灰或锯末，减少苍蝇和臭味 |
| 堆温 | 高温堆肥核心保持 55-65摄氏度至少 3 天，翻堆后再次升温 |
| 湿度 | 手握像拧干湿海绵，能成团但不滴水 |
| 熟化 | 最终呈黑褐色土壤味，看不出生粪和厨余形态 |

## 失败征兆

- 粪坑渗水流向井、溪、池塘，是传染病高风险。
- 堆肥臭如腐肉或粪坑，通常是太湿、缺氧或碳源不足。
- 苍蝇大量繁殖，说明覆盖不够或堆体暴露。
- 温度长期低于 40摄氏度，不要把人畜粪肥用于食用作物。
- 未熟粪肥直接施在叶菜、瓜果上，容易造成粪口传播。

## 常见问题

- **Q**: 堆肥为什么发臭？
- **A**: 缺氧腐烂（厌氧）才会臭。好堆肥不缺氧气，闻起来是土壤味。翻堆不足或太湿是主要原因。

- **Q**: 堆肥里能加什么不能加什么？
- **A**: 可以加：素食厨余、粪便、草叶、木灰。不能加：肉类、骨头、油脂（引虫+难分解）。

## 安全提醒

- 处理粪便后必须彻底洗手
- 堆肥温度不足 40摄氏度说明厌氧或太湿→翻堆
- 厕所距离任何水源 30 米以上——这是底线
- 堆肥成熟前不要施用到食用作物上

## 穿越小说梗

穿越小说几乎不会写厕所和粪便处理，但这是穿越者**最没面子却最重要**的工程。古代城市瘟疫流行的一个重要原因就是粪便管理不善。如果穿越者能建立一套基本的旱厕系统和粪便堆肥方案，不仅能减少自己生病的概率，还能产出优质肥料→粮食增产→人口增长→国力增强。种田流的真实起点不是"开荒种地"，而是**挖厕所**。
