# 附录

> 查资料、看推荐、学安全——附录是一些难以归类但穿越者必须知道的东西。

## 使用顺序

1. 动手前先查安全：酸碱、高温、有毒物、火药相关内容必须先看安全条目。
2. 算量前先查单位：古代斤、尺、升差异很大，不换算就会错配方。
3. 写剧情前查真实性：判断原料、工具、时间和制度阻力。
4. 出门采集前查花期果期：季节错了，再懂技术也找不到材料。
