# C.3 在线资源推荐

> 穿越前先收藏这些网址，离线保存就是最强的知识金手指。

## 古籍与文献

| 资源名称 | 网址 | 内容 | 穿越价值 |
|---------|------|------|---------|
| 中国哲学书电子化计划 | ctext.org | 最大古籍在线数据库，全文检索 | 离线打包带走，古代相当于随身携带一座图书馆 |
| 搜韵 | sou-yun.cn | 诗词格律、韵书检索 | 穿越后写诗必备，避免押错韵 |
| **书格** | shuge.org | 大量古籍善本高清扫描版，自由开放的数字图书馆 | 穿越前下载高清古籍PDF，随身携带珍本藏书阁 |
| **国学大师** | guoxuedashi.com | 收录2万种古典图书、24亿字，含《四库全书》全文检索 | 一站式检索所有古籍，穿越后查资料最快 |
| **识典古籍** | shidianguji.com | 北京大学与抖音公益共建，免费古籍阅读平台 | 界面友好，适合快速查阅经典文本 |

## 如何离线保存

1. 上 GitHub 搜索上述项目，点击 Code → Download ZIP
2. 存到手机/平板/电子书阅读器中
3. 准备一块太阳能充电板保证设备续航
