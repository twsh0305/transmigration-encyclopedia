# C.3 在线资源推荐

> 穿越前先收藏这些网址，离线保存就是最强的知识金手指。

## 筛选标准

| 标准 | 合格 | 失败征兆 |
|------|------|----------|
| 来源 | 官方、大学、开源项目、可追溯文献 | 只有营销号摘要 |
| 格式 | PDF、HTML、TXT、图片可离线打开 | 只能联网检索 |
| 检索 | 有目录、索引或全文搜索 | 存了很多但找不到 |
| 版权 | 公开授权或合理个人备份 | 来源不明、随时失效 |

## 如何离线保存

1. 上 GitHub 搜索上述项目，点击 Code → Download ZIP
2. 存到手机/平板/电子书阅读器中
3. 准备一块太阳能充电板保证设备续航

## 备份比例与复查时间

| 项目 | 建议尺度 | 成功标准 |
|------|----------|----------|
| 主题覆盖 | 每个主题至少 3 份材料：入门、详细、图解 | 断网后仍能查到原理、边界和低风险步骤 |
| 存储备份 | 至少 2 个介质，分开放置 | 坏掉一个设备仍能读取 |
| 抽查 | 随机打开 10 个文件 | PDF、图片、文本都能离线打开 |
| 复查 | 每 3 个月复查链接和本地备份 | 失效来源能及时替换 |

## 成功标准：离线验收

1. 断网后随机打开 10 个文件，确认能读。
2. 搜索“盐、火、米、伤口、木炭、陶器”等高频词，确认能检索。
3. 每个主题至少保留 1 份入门材料、1 份详细材料、1 份图解材料。
4. 重要 PDF 另存一份纯文本或截图，避免阅读器不兼容。
5. 每 3 个月复查一次链接和本地备份，失效来源要补 1 个替代来源。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 文件名全是乱码 | 下载时编码或命名混乱 | 重新按主题改名 |
| 断网打不开 | 只保存了网页入口 | 保存 PDF/HTML 完整页面 |
| 找不到关键知识 | 无索引 | 建目录和关键词清单 |
| 链接失效 | 只依赖在线网址 | 本地备份，多来源保存 |

## 安全边界

医疗、化学、火药、冶炼类资料不能只看民间教程。至少交叉核对 2 个可靠来源；涉及人体、爆燃、腐蚀和高温时，优先写成风险识别、停手条件和求助路径。
