# B.4 有毒物质识别与处理

> 毒物处理的目标不是“找偏方解毒”，而是停止暴露、维持呼吸、保留证据、尽快求援。

## 常见高风险毒物

| 名称 | 典型风险 | 早期征兆 | 危险边界 |
|------|----------|----------|----------|
| 乌头、草乌、川乌 | 心律失常、麻痹 | 口舌发麻、恶心、出汗 | 不生食，不自制内服药 |
| 曼陀罗、洋金花 | 谵妄、呼吸抑制 | 口干、瞳孔大、幻觉 | 不作麻药试验 |
| 钩吻、断肠草 | 呼吸抑制 | 眩晕、吞咽困难 | 不采“不确定藤本”入药 |
| 毒鹅膏等毒蘑菇 | 肝肾衰竭 | 6-12 小时后呕吐腹泻 | 不凭银针、蒜变色判断 |
| 铅、汞、砷化合物 | 慢性或急性中毒 | 腹痛、神经症状、口腔异常 | 不入口、不入妆、不入药 |

## 识别原则

1. 同一地点有人畜同时不适，先怀疑水源、烟气、霉粮或毒物。
2. 发病越快，越要关注腐蚀性、神经毒性和窒息性风险。
3. “吃完一夜后才吐泻”不能说明安全，部分毒蘑菇有长潜伏期。
4. 民间试毒法不可靠，银针、变色、鸡犬先尝都不能证明无毒。

## 中毒处理步骤

1. 停止摄入或接触，把人移离毒源。
2. 保持呼吸道通畅。昏迷者侧卧，防呕吐物堵塞。
3. 皮肤污染用清水和皂液冲洗；眼睛污染流水冲 15 分钟。
4. 保留剩余食物、植物、药渣、容器和呕吐物样本。
5. 记录时间线：吃了什么、多少、何时发作、出现哪些症状。
6. 尽快找医者或有经验者处理，不把时间浪费在仪式化偏方上。

## 不做清单

- 不催吐，除非明确由专业人员要求。腐蚀性、石油类、昏迷者催吐会更危险。
- 不强灌水、酒、醋、油或草药。
- 不在家自行服用“活性炭粉”当通用解毒药；严重中毒需要监测气道和生命体征。
- 不把毒物样本丢掉，样本常比口述更有用。

## 成功标准

- 毒源被隔离，其他人没有继续暴露。
- 患者呼吸、意识、脉搏被持续观察。
- 样本和时间线完整，能交给救治者判断。

## 失败征兆

- 昏迷、抽搐、呼吸慢或喘不过气。
- 反复呕吐、血便、剧烈腹痛。
- 口唇发紫、心跳极快或极慢。
- 眼睛化学灼伤后仍疼痛、视物模糊。

## 安全边界

- 不把本页当作解毒药方；它只帮助停止暴露、保存证据和争取救治时间。
- 不用民间试毒法证明安全，不让儿童、孕妇、老人和病人参与试吃或试药。
- 不明植物、蘑菇、矿物、粉末和药渣应封存标记，不再入口、入妆或投喂牲畜。
