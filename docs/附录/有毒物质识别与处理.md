# B.4 有毒物质识别与处理

> 古人不知道毒物是什么——但你穿越后应该知道。

## 常见有毒植物

| 名称 | 毒素 | 症状 | 致死量 |
|------|------|------|--------|
| 乌头（川乌/草乌） | 乌头碱 | 口舌发麻→恶心→心律失常→呼吸麻痹 | 3~5g生乌头 |
| 曼陀罗（洋金花） | 莨菪碱 | 口干→幻觉→瞳孔散大→呼吸衰竭 | 果实数颗 |
| 断肠草（钩吻） | 钩吻碱 | 眩晕→吞咽困难→呼吸抑制 | 叶片数片 |
| 毒蘑菇（鹅膏菌） | 鹅膏毒素 | 潜伏6~12小时→呕吐腹泻→肝肾衰竭 | 一朵即可致命 |

## 中毒处理步骤

1. **停止摄入**——立刻不再食用/饮用可疑物
2. **大量饮水稀释**——清醒时可适量饮水
3. **催吐**——仅限清醒且非腐蚀性毒物时
   - 手指或筷子轻探咽喉
   - **不可**对昏迷者催吐（防窒息）
4. **保留样本**——剩下的毒物留作辨识，帮助救治
5. **最重要的解毒药是时间**——大部分中毒只要撑过毒性发作期就有生还可能

## 自制通用解毒方

活性炭烧制 → 碾成细粉 → 中毒后立即服用（约一汤匙混水）可吸附部分毒素。

> 注意：活性炭不能吸附所有毒素，应急使用后仍需专业救治。
