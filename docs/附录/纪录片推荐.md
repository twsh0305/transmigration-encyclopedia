# C.2 纪录片推荐

> 穿越前刷完，穿越后你就是先知。

## 观看方法

纪录片适合建立直觉，不适合替代手册。看每一集时只记三类信息：材料、工具、工序。能暂停画面画出流程图，才算真正吸收。

## 穿越前刷片计划

时间紧就看《原始技术》+《中国通史》，相当于古代三年学徒加十年经验的知识量。

## 成功标准

- 每部纪录片至少整理 5 条可执行观察，例如炉型、工具形状、材料处理、失败画面。
- 能把“看点”转写成低风险步骤、风险边界或待核实问题，而不是只记剧情。
- 能指出纪录片省略了什么：比例、温度、时间、尺寸或安全防护。

## 失败征兆

- 看完只会说“古人很厉害”，但说不出怎么做。
- 把影视复原当成考古定论，没有分清戏剧化画面和实操证据。
- 只看战争和权谋，不看农业、卫生、物流、工具维护。

## 安全边界

纪录片中的高温、刀具、野外、火器、毒物镜头往往有专业团队保护。现实复现必须回到本项目对应安全条目，不按视频画面直接模仿。
