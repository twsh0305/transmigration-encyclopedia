# B.3 高温作业安全

> 900°C的铁水和100°C的沸水没有本质区别——都能要你的命。

## 操作规范

1. **操作区保持干燥**——水遇高温金属会瞬间汽化膨胀，导致金属飞溅
2. **防护装备**：厚手套、护目镜（必要时加面罩）、皮围裙
3. **选址要求**：窑炉建在远离易燃物（茅草屋、木料堆）处
4. **防火准备**：配防火沙（不要用水灭油火/金属火）、干粉灭火剂

## 一氧化碳中毒预防

窑炉燃烧不充分会产生一氧化碳（无色无味），中毒症状：

- 轻度：头晕、恶心、乏力
- 中度：意识模糊、呼吸急促
- 重度：昏迷、死亡

**中毒后立即**：移到通风处，解开衣领，保持呼吸道通畅。

## 温度目测技巧

穿越者用颜色判断温度，不用全靠经验：

| 颜色 | 温度 | 用途 |
|------|------|------|
| 暗红色 | 约600°C | 低温锻造 |
| 樱桃红 | 约800°C | 铁器锻造 |
| 亮红色 | 约1000°C | 钢的淬火温度 |
| 橙色 | 约1200°C | 熔化生铁/玻璃 |
| 黄色/白色 | 1400°C+ | 炼钢/熔化铸铁 |

> 注意：温度低时不可贸然淬火，否则金属会开裂。
