# B.3 高温作业安全

900摄氏度以上的铁水和100摄氏度的沸水没有本质区别——都能要你的命。

## 操作前检查

| 检查项 | 合格标准 | 失败征兆 |
|--------|----------|----------|
| 地面 | 干燥、平整、不积水 | 有水洼、湿泥、易滑 |
| 燃料 | 干柴、木炭分区堆放 | 柴草贴近炉口 |
| 模具/坩埚 | 完全烘干预热 | 潮湿、冒汽、裂纹 |
| 人员 | 至少两人，一人操作一人看护 | 围观、酒后、疲劳 |
| 退路 | 炉前和侧面可快速撤离 | 工具杂物堵路 |

## 防护装备

- 护眼：护目镜或面罩，防火星和热渣。
- 手部：干燥厚皮手套；湿手套会导热烫伤。
- 身体：棉麻或皮围裙，不穿易熔化的化纤衣。
- 足部：厚底鞋，裤脚盖住鞋口。

## 一氧化碳中毒预防

窑炉燃烧不充分会产生一氧化碳（无色无味），中毒症状：

- 轻度：头晕、恶心、乏力
- 中度：意识模糊、呼吸急促
- 重度：昏迷、死亡

**中毒后立即**：移到通风处，解开衣领，保持呼吸道通畅。

## 高温事故处理

| 情况 | 处理 |
|------|------|
| 衣物着火 | 停、倒、滚，或用厚布压灭 |
| 皮肤烫伤 | 流动清水冷却 20 分钟，不涂油、不挑水泡 |
| 熔融物溅出 | 人先撤离，待冷却后清理，不用水泼 |
| 油火/金属火 | 用沙土覆盖隔绝空气，不用水冲 |

## 温度目测技巧

穿越者用颜色判断温度，不用全靠经验：

| 颜色 | 温度 | 用途 |
|------|------|------|
| 暗红色 | 约600摄氏度 | 低温锻造 |
| 樱桃红 | 约800摄氏度 | 铁器锻造 |
| 亮红色 | 约1000摄氏度 | 钢的淬火温度 |
| 橙色 | 约1200摄氏度 | 熔化生铁/玻璃 |
| 黄色/白色 | 1400摄氏度以上 | 炼钢/熔化铸铁 |

> 注意：温度低时不可贸然淬火，否则金属会开裂。

## 成功标准

- 操作 30 分钟内无人头晕、恶心、眼刺痛。
- 坩埚、模具、夹钳没有裂纹和松动。
- 火星落点都在清理过的非燃区内。
- 作业结束后余炭被集中覆盖，1 小时后复查无暗火。

## 安全边界

- 潮湿金属、潮湿石块、潮湿模具不得直接接触高温炉膛或熔融金属。
- 室内、低矮棚屋、茅草屋旁不做冶炼和熔炼。
- 只要出现连续头痛、胸闷、意识迟钝，立即停炉撤离。
