# B.2 强碱强酸操作安全

> 碱液和酸液不会区分你是主角还是炮灰。

## 基本守则

1. **操作前**：戴厚手套（橡胶或皮制）、护目镜、围裙
2. **容器**：必须显著标注内容物和浓度，严禁用食具装
3. **通风**：操作区必须通风，酸雾和碱尘吸入同样危险
4. **顺序**：稀释浓硫酸时，**一定要酸入水**（缓缓倒入水中搅拌），不可水入酸！

## 应急处理

| 情况 | 处理方法 |
|------|---------|
| 碱液溅到皮肤 | 大量清水冲15分钟以上，**不可用酸中和**（中和放热加重灼伤） |
| 酸液溅到皮肤 | 大量清水冲15分钟以上 |
| 溅入眼睛 | 撑开眼睑持续冲洗至少15分钟 |
| 误饮碱液 | **严禁催吐**，大量饮水稀释后就医 |
| 误饮酸液 | **严禁催吐**，大量饮水稀释后就医 |

## 安全储存

- 碱液用陶瓷或玻璃容器，不可用铝制（碱腐蚀铝）
- 酸液用玻璃容器，不可用金属
- 酸碱分开存放，远离可燃物
