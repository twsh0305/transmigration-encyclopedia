# A.1 金属熔点速查表

> 熔点只是第一道门槛。真正难的是炉温稳定、坩埚耐火、杂质控制和人员安全。

## 常用金属熔点

| 金属 | 熔点约值 | 古代条件 | 风险提醒 |
|------|----------|----------|----------|
| 锡 | 232摄氏度 | 炭火可达 | 熔液烫伤 |
| 铅 | 327摄氏度 | 炭火可达 | 铅烟和铅尘有毒 |
| 银 | 962摄氏度 | 强炭火/小炉 | 需耐火坩埚 |
| 金 | 1064摄氏度 | 强炭火/小炉 | 成本高，损耗难承受 |
| 铜 | 1085摄氏度 | 鼓风炉 | 氧化、夹渣明显 |
| 青铜 | 约 900-1000摄氏度 | 鼓风炉较可行 | 锡比例影响脆性 |
| 铁 | 1538摄氏度 | 传统多先炼海绵铁 | 重点是还原和锻打，不是直接化铁水 |

## 现场温度粗判

| 现象 | 大致温区 | 可做事项 |
|------|----------|----------|
| 锡铅熔化 | 230-330摄氏度 | 低温合金、小件配重 |
| 坩埚暗红 | 600-700摄氏度 | 退火、烘坩埚 |
| 橙红到黄 | 900-1100摄氏度 | 铜、青铜尝试 |
| 黄白刺眼 | 1200摄氏度以上 | 高风险冶炼，需鼓风和防护 |

肉眼看火色受阳光、炉膛、燃料影响很大，只能粗判，不能当精密温度计。

## 成功标准

- 金属完全熔成连续液面，表面浮渣可分离。
- 浇入预热模具后成形，无大量气孔和夹渣。
- 冷却后敲击声清，不一敲就碎。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 半熔半固 | 炉温不足或坩埚太大 | 减小批量，加强鼓风 |
| 浇注炸裂 | 模具潮湿 | 模具预热烘干 |
| 铸件脆裂 | 合金比例或杂质问题 | 小样试配，减少杂质 |
| 操作者头痛恶心 | 烟尘或一氧化碳 | 立即离炉通风 |

## 安全边界

- 铅、砷、汞、镉等有毒金属不做室内熔炼。
- 潮湿矿石、潮湿坩埚、潮湿模具不得接触熔融金属。
- 高温作业必须有旁人看护、水源用于冷却人员、沙土用于处理火情。

## 穿越要点

核心难题不是背出铁的 1538摄氏度，而是知道古代炼铁常走“固态还原出海绵铁，再锻打排渣”的路线。
