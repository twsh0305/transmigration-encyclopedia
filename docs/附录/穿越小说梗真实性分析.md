# D.1 穿越小说梗真实性分析

> 小说里三章搞定的事情，现实中可能需要三年。

## 常见穿越梗可行性评分

| 穿越梗 | 可行性 | 难点 | 判定理由 |
|--------|--------|------|---------|
| 造玻璃暴富 | ⭐⭐ | 极高 | 需1200°C以上高温，纯度要求高，古代燃料难达标 |
| 制火药称霸 | ⭐⭐⭐ | 中高 | 原料可得（硝石/硫磺/木炭），难点在纯度和安全储存 |
| 提纯精盐 | ⭐⭐⭐⭐ | 低 | 技术最简单（溶解→过滤→结晶），难点在官营盐制 |
| 青霉素救命 | ⭐ | 极高 | 无无菌条件，粗提注射可能致死，二次感染风险 |
| 造纸/印刷 | ⭐⭐⭐ | 中 | 技术上可行，难点在工艺熟练度和大规模生产 |
| 肥皂赚钱 | ⭐⭐⭐⭐ | 低 | 原理简单，原料易得，最难的是运输和销售 |
| 治瘟疫扬名 | ⭐⭐ | 高 | 需先让古人接受你的理论，隔离和消毒理念超前 |
| 蒸馏酒秒杀古人 | ⭐⭐⭐⭐ | 低 | 蒸馏器做出来就能碾压，难点在酒曲和发酵 |

> 1星=极难（月级准备），5星=穿越当天就能上手（日级准备）
