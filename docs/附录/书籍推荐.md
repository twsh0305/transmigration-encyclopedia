# C.1 书籍推荐

> 穿越前先把这些书装进脑子——知识才是最强的金手指。

## 使用方法

1. 先用 15 分钟看目录，标出“水、火、食物、医药、工具、制度”六类内容。
2. 每本书只摘三类信息：低风险可验证步骤、可换算的尺寸比例、必须避开的风险。
3. 古籍给方向，现代资料给校验；两者冲突时，以可重复实证和现代安全边界为准。
4. 每条笔记都写明出处、页码或章节，避免事后只记得“好像书里说过”。

## 成功标准

- 一本书读完后，至少能产出 10 条可判断笔记：低风险内容写用途、材料、步骤和限制；高风险内容写历史背景、禁区和停手条件。
- 关键工艺能找到两个来源互相印证，尤其是医药、冶金、发酵和建筑。
- 看到古代单位、药名、矿物名时，能查到现代对应或明确标注“不确定”。
- 复盘时能说清“适合小说设定”“适合低风险实操”“现实禁止尝试”三类边界。

## 失败征兆

- 只抄名词和典故，没有比例、时间、温度、尺寸或判断标准。
- 把古籍药方、火器、炼丹内容直接当现实教程。
- 引用只有书名没有章节，无法回查原文。
- 同一技术只看一个来源，就写成确定结论。

## 安全边界

- 医药、毒物、火药、武器、强酸强碱和高温冶炼内容只做历史研究，不能照古籍直接实操。
- 古籍中的“无毒”“久服”“神效”不等于现代安全；涉及入口、吸入和皮肤接触必须另查现代资料。
- 在线链接可能版本不同，引用前核对篇名和段落，不把误 OCR 当原文。
