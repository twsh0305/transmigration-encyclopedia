# 8.1 经商致富基础

> 穿越后不想打打杀杀？那就经商吧。低买高卖、货殖流通，从走街串巷到富可敌国。

## 核心原则

- **人弃我取，人取我与**——白圭的经商之道，低价囤积、高价卖出
- **货真价实，童叟无欺**——古代商誉是最重要的资产
- **信息差就是利润**——了解两地物价差异，倒卖获利
- **钱要流转**——死钱变活钱，存货变现才是王道

## 起步方式

| 方式 | 本钱 | 利润 | 风险 |
|------|------|------|------|
| 走贩（挑货郎） | 极低 | 低 | 低 |
| 集市摊位 | 低 | 中 | 低 |
| 店铺经营 | 中 | 中高 | 中 |
| 长途贩运 | 高 | 高 | 高 |
| 钱庄票号 | 极高 | 极高 | 极高 |

## 古代常见行业

| 行业 | 利润 | 门槛 | 说明 |
|------|------|------|------|
| 粮食 | 薄利但稳定 | 中 | 民以食为天，旱涝保收的基础行业 |
| 布匹 | 中 | 中低 | 家家户户都需要的刚需，周期短流转快 |
| 盐铁 | 极高 | 极高 | 官府专营，有牌照才有资格，但利润最丰厚 |
| 茶叶 | 高 | 中 | 茶马古道、海外贸易的重要商品 |
| 瓷器 | 高 | 中高 | 技术壁垒高，但海外需求极大 |
| 药材 | 中高 | 中 | 看天吃饭，识货才赚钱 |
| 马匹 | 高 | 高 | 古代的战略物资，养马成本极高 |
| 典当 | 极高 | 中高 | 暴利行业，但需要大量本金和人脉 |
| 酒楼 | 中高 | 中 | 信息集散地，适合收集情报 |
| 客栈 | 中 | 中 | 稳定现金流，适合起步积累 |

> 穿越者建议：起步选布匹或药材（门槛低、市场大），积累资本后再进军盐铁或典当。

## 起步步骤

1. 先做 7 天市价记录：同一货物早市、午市、晚市各记一次。
2. 选 1 种低损耗货物试卖，不同时做 5 种生意。
3. 本钱分三份：进货 60%，周转现金 30%，应急 10%。
4. 每日收摊后盘点：现银、欠账、剩货、损耗分开记。
5. 连续 3 轮交易都盈利，再扩大进货量。

## 资金比例与盘账时间

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 起步资金 | 进货 60%，周转现金 30%，应急 10% | 缺货和意外都能应对 | 全部压成库存 |
| 市价观察 | 连续 7 天，早市、午市、晚市各记 1 次 | 能看出真实波动 | 只听同行口头报价 |
| 试卖 | 先做 1 种低损耗货物，连续 3 轮盈利再加量 | 周转快，账能对上 | 同时铺太多品类 |
| 盘账 | 每日收摊后立刻盘点 | 现银、欠账、剩货、损耗分清 | 卖得热闹但不知道赚没赚 |

## 成功标准

- 毛利率能覆盖运输、摊费、损耗、人情费后仍有净利。
- 账本能对上现银和库存，误差不超过当日营业额的 2%。
- 同一客户愿意复购，说明不是一次性信息差。
- 最坏情况下能停业清货，不欠高利债。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 卖得热闹却没钱 | 没算损耗和赊账 | 停止赊账，重新盘点 |
| 货压仓 | 需求判断错 | 降价回本，换快周转货 |
| 被同行排挤 | 破坏行规或价格战 | 找中人调解，避开垄断 |
| 官差频繁找事 | 手续、税费、靠山不足 | 暂停扩张，补齐合规 |

## 安全边界

- 盐、铁、矿、酒、茶等可能有官营或重税，不明制度前不碰大额。
- 不借高利贷扩张，月息超过净利率就是慢性破产。
- 不把现代稀罕物一次性大量抛出，容易引来抢夺和官府盘问。

## 穿越小说梗

主角往往通过"现代知识碾压"发现商机——比如用玻璃镜代替铜镜、用肥皂抢占市场。但要注意：太超前可能被当妖孽，太落后赚不到第一桶金。
