# 8.4 权谋与驭人之术

> 权力不是地位，而是让别人做你想让他们做的事。

## 古代权谋经典

这些书是穿越者的"政治课本"，建议穿越前至少熟读其中两三本：

- **《孙子兵法》**——不战而屈人之兵，谋略的至高境界
  - 核心：知己知彼、避实击虚、以逸待劳
- **《韩非子》**——法、术、势，帝王术的集大成者
  - 核心：法术势三结合，不可偏废其一
- **《长短经》（反经）**——赵蕤著，专讲权变与博弈
  - 核心：没有永恒的敌人，只有永恒的利益
- **《资治通鉴》**——历代兴衰中的权谋案例大全
  - 核心：读史使人明智，每一段兴衰都是活教材
- **《三十六计》**——具体战术层面的计谋手册
  - 核心：胜战计、敌战计、攻战计……每计都是实战场景

## 驭人核心：赏与罚

管理下属的精髓就四个字——恩威并施。

- **赏贵信**——赏赐要及时、守信，才能激励人
  - 错误示范："等打完仗再赏" → 将士离心
  - 正确操作：当场兑现，哪怕赏少也比空头许诺强
- **罚贵必**——处罚要坚决、公平，才能震慑人
  - 错误示范：重罚小兵、轻饶将领 → 军心不服
  - 正确操作：王子犯法与庶民同罪，公平才有威信
- **恩威并施**——单用恩则骄，单用威则叛
  - 先威后恩：先立规矩再给甜头，下属觉得"这是恩赐"
  - 先恩后威：先给好处再立规矩，下属觉得"这是针对我"

## 穿越小说梗

穿越者最大的优势是"历史预知"——知道谁会是最终赢家，提前站队。但蝴蝶效应会让历史走偏，不能完全依赖。
