# 8.3 古代金融：钱庄、票号、汇兑

> 银子怎么存的？异地怎么转账？古代金融体系是穿越者积累财富不可或缺的工具。

## 钱庄

古代的钱庄类似于今天的银行+外汇兑换点，是穿越者管理财富的第一站。

- **起源**：明朝中期，主要经营银钱兑换业务
- **核心业务**：
  - 存款：存入白银收取保管费（或者倒给你利息，视行情而定）
  - 贷款：抵押贷款，月息2%~5%不等
  - 兑换：铜钱换白银、白银换黄金，赚取兑换差价
- **盈利模式**：存贷利差 + 兑换手续费，年化收益一般在20%~50%
- **风险**：钱庄倒闭跑路的不少，大额存款最好分散存放

## 票号

票号是古代的"跨行转账系统"，以山西票号最为著名，巅峰时期遍布全国。

- **起源**：清朝道光年间，山西商人首创
- **核心业务**：异地汇兑——在A地存钱，拿汇票到B地取钱
- **密押系统**：汇票上有暗语编码（如"谨防假票冒取"六个字对应一二三四五六），防伪极其成熟
- **分号网络**：遍布全国主要商埠，类似于现在的银行分行体系
- **穿越者用法**：大额资金转移首选票号，随身携带大量白银既不安全也不现实

## 典当行

古代的高利贷+二手交易市场，急用钱时的最后选择。

- **业务模式**：以物抵押，按物价值的30%~60%放款
- **利率**：月息2%~5%，年化利率可达30%~60%
- **赎期**：一般1~3年，过期不赎视为"死当"，物品归典当行所有
- **穿越者小技巧**：
  - 拿现代不值钱但古代稀罕的东西去当（如玻璃制品）
  - 死当物品可以低价从典当行回购转卖

## 古代汇率与货币

| 时期 | 主流货币 | 换算 |
|------|---------|------|
| 唐宋 | 铜钱为主 | 1两银 ≈ 1000文 |
| 明清 | 银两为主 | 1两银 ≈ 1000文铜钱 |
| 各朝 | 黄金 | 1两金 ≈ 10两银（浮动） |

## 穿越金融攻略

1. **先搞清楚当地货币体系**——铜钱、白银、纸币并行情况
2. **利用信息差套利**——两地银价差异
3. **和钱庄打好关系**——大额交易需要钱庄背书
4. **藏一部分实物**——战乱时纸币可能变废纸
