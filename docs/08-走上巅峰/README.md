# 第八卷·走上巅峰

> 活着只是开始，要在古代混出名堂，需要钱、权、人脉和谋略。本卷不讲道德高低，只讲规则和实操。

## 使用顺序

1. 先算现金流：没有稳定收入，不谈权谋和金融。
2. 再建信誉：账目、契约、交付记录，比一次暴利更重要。
3. 再找保护伞：商路、店铺、工坊都离不开地方势力和官府规则。
4. 最后扩张：跨城贸易、票号汇兑、官场晋升都属于高风险进阶。

## 时间尺度与退出线

| 决策 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 日常经营 | 每日记流水，每 7 天盘库存 | 银钱、货物、欠账能互相对上 | 只记收入，不记损耗和赊欠 |
| 扩张开店 | 连续 3 个月有净盈余再扩 | 新店不抽空老店现金 | 借钱扩张、账期越拖越长 |
| 结交势力 | 先观察 1 到 3 个月再深绑定 | 对方可兑现小承诺 | 一见面就索要大礼或担保 |
| 官场上位 | 每一步都有退路和替罪风险评估 | 失败后仍能保住家人和账本 | 把身家押在单个靠山上 |

## 成败总标准

| 阶段 | 成功标准 | 失败征兆 |
|------|----------|----------|
| 起步 | 每月收入覆盖口粮、租金、损耗 | 只看毛利，不算坏账 |
| 扩张 | 有账本、契约、库存盘点 | 货卖得多但银子不见 |
| 社交 | 有稳定供货人、客户、担保人 | 只靠主角个人魅力 |
| 上位 | 风险可退出，不把全家押上 | 一次失败就破产或入狱 |

## 安全边界

第八卷讨论的是制度和社会运行，不鼓励诈骗、勒索、贿赂、暴力扩张。小说可以写灰色地带，实操建议必须以保命、守约、可退出为底线。
