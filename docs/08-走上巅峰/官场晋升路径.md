# 8.2 官场晋升路径

> 学而优则仕。穿越后想登上权力之巅，先搞清楚古代官场的游戏规则。

## 三大晋升通道

### 科举入仕（最正统）
- **考试路径**：童试（秀才）→ 乡试（举人）→ 会试（贡士）→ 殿试（进士）
- **考试频率**：乡试三年一次，会试次年春
- **优势**：名正言顺，上升空间大，皇帝视为天子门生
- **劣势**：竞争惨烈，耗时长，很多人考到白头还是个秀才
- **穿越者优势**：现代知识储备（历史、经义、策论）降维打击
- **穿越者劣势**：字体丑、没有师承、不懂考场潜规则

### 军功晋升（乱世首选）
- **途径**：斩首、破阵、攻城累积军功
- **计功方式**：首级计数→升爵位→授实职
- **优势**：升迁快，乱世中尤为突出，容易建立嫡系部队
- **劣势**：风险极高，随时可能阵亡，和平时期军功难积
- **穿越者优势**：现代军事理论、火药知识、后勤管理理念

### 举荐/恩荫（捷径）
- **途径**：高官举荐、父祖余荫入仕
- **优势**：无需科举，起点高，直接做官
- **劣势**：上限有限，受制于人，容易被排挤
- **适合**：有靠山的穿越者，或者通过展现特殊才华（医术、工程、天文）被人举荐

## 古代官职体系（三省六部制）

| 部门 | 职能 | 现代类比 |
|------|------|---------|
| 吏部 | 官员任免考核 | 组织部 |
| 户部 | 财税户籍 | 财政部+统计局 |
| 礼部 | 典礼外交科举 | 外交部+教育部 |
| 兵部 | 军事 | 国防部 |
| 刑部 | 司法 | 司法部 |
| 工部 | 工程水利 | 住建部+水利部 |

## 权力生存法则

1. **站队是门艺术**——不站队难升迁，站错队掉脑袋
2. **功高震主是大忌**——学会自污、示弱、分功
3. **信息即权力**——建立自己的情报网络
4. **清官比贪官更难当**——水至清则无鱼

## 晋升准备清单

| 路径 | 先做什么 | 成功标准 |
|------|----------|----------|
| 科举 | 练字、背经义、找本地师承 | 文章能被当地塾师认可 |
| 军功 | 先管后勤、纪律、训练 | 队伍能按令行动，不乱抢 |
| 举荐 | 做出可验证专长 | 有见证人和成果样品 |
| 技术入仕 | 解决水利、疫病、税粮等痛点 | 成果能被官府量化 |

## 时间尺度与资格门槛

| 路径 | 时间尺度 | 可验证成果 | 失败征兆 |
|------|----------|------------|----------|
| 科举 | 以 3 年一轮乡试规划 | 字体、经义、策论被塾师认可 | 只背答案，不会按当朝文体写 |
| 军功 | 先做 1 到 3 个月后勤与纪律 | 队伍少逃亡、少哄抢 | 只谈战术，不管粮草 |
| 举荐 | 先积累 3 件可展示成果 | 有保人、见证人、样品 | 一开口就求官求印 |
| 技术入仕 | 先做小范围试点 | 水利、疫病、税粮有账面变化 | 成果不能复验，全靠吹嘘 |

## 失败征兆

- 只会背现代知识，不会用当朝话术写奏疏和策论。
- 功劳全揽在自己身上，挡了上级和同僚的路。
- 过早暴露“预知”或超时代技术，引发猜忌。
- 没有户籍、保人、师承，连入场资格都拿不到。

## 安全边界

官场路线最怕“越级、越权、越制”。没有正式身份时不插手刑狱、军务、税粮；有功也要让合法上级能分到功劳。

## 穿越小说梗

"三年科举、十年宰相"是常见的爽文路线，现实中的官场晋升往往需要二十年以上的经营。
