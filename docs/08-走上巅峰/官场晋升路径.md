# 8.2 官场晋升路径

> 学而优则仕。穿越后想登上权力之巅，先搞清楚古代官场的游戏规则。

## 三大晋升通道

### 科举入仕（最正统）
- **考试路径**：童试（秀才）→ 乡试（举人）→ 会试（贡士）→ 殿试（进士）
- **考试频率**：乡试三年一次，会试次年春
- **优势**：名正言顺，上升空间大，皇帝视为天子门生
- **劣势**：竞争惨烈，耗时长，很多人考到白头还是个秀才
- **穿越者优势**：现代知识储备（历史、经义、策论）降维打击
- **穿越者劣势**：字体丑、没有师承、不懂考场潜规则

### 军功晋升（乱世首选）
- **途径**：斩首、破阵、攻城累积军功
- **计功方式**：首级计数→升爵位→授实职
- **优势**：升迁快，乱世中尤为突出，容易建立嫡系部队
- **劣势**：风险极高，随时可能阵亡，和平时期军功难积
- **穿越者优势**：现代军事理论、火药知识、后勤管理理念

### 举荐/恩荫（捷径）
- **途径**：高官举荐、父祖余荫入仕
- **优势**：无需科举，起点高，直接做官
- **劣势**：上限有限，受制于人，容易被排挤
- **适合**：有靠山的穿越者，或者通过展现特殊才华（医术、工程、天文）被人举荐

## 古代官职体系（三省六部制）

| 部门 | 职能 | 现代类比 |
|------|------|---------|
| 吏部 | 官员任免考核 | 组织部 |
| 户部 | 财税户籍 | 财政部+统计局 |
| 礼部 | 典礼外交科举 | 外交部+教育部 |
| 兵部 | 军事 | 国防部 |
| 刑部 | 司法 | 司法部 |
| 工部 | 工程水利 | 住建部+水利部 |

## 权力生存法则

1. **站队是门艺术**——不站队难升迁，站错队掉脑袋
2. **功高震主是大忌**——学会自污、示弱、分功
3. **信息即权力**——建立自己的情报网络
4. **清官比贪官更难当**——水至清则无鱼

## 穿越小说梗

"三年科举、十年宰相"是常见的爽文路线，现实中的官场晋升往往需要二十年以上的经营。
