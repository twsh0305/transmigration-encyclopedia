# 6.23 鼓的制作

> 鼓声要响，靠的是鼓腔刚、鼓皮匀、张力稳。

## 材料与尺寸

| 部件 | 推荐 | 合格标准 |
|------|------|----------|
| 鼓腔 | 木筒、掏空圆木、陶筒 | 壁厚均匀，无贯穿裂 |
| 鼓皮 | 牛皮、羊皮、鹿皮 | 厚薄接近，无腐臭 |
| 固定 | 木钉、皮绳、铁箍 | 能均匀拉紧 |
| 小试尺寸 | 直径 20-30厘米，高 20-40厘米 | 新手容易控制 |

## 操作步骤

1. 鼓腔内外刮平，边缘磨圆，避免割破鼓皮。
2. 生皮去肉去脂，清水浸软。脱毛可用稀石灰水，但之后必须反复漂洗到不滑手。
3. 湿皮铺在鼓口，四周预留 5-8厘米边。
4. 先按十字方向固定四点，再补中间点，逐步拉紧。
5. 阴干 2-5 天。干燥中每天轻敲检查，发现松边及时补拉。

## 成功标准

- 鼓面干后平整，中心无明显皱褶。
- 轻敲中心声音集中，敲边缘音色差异不大。
- 连续敲击 5 分钟，鼓皮不滑脱、不裂。

## 调音与维护

| 场景 | 做法 | 成功标准 |
|------|------|----------|
| 阴雨变闷 | 放通风阴凉处慢干 | 声音恢复集中 |
| 边缘松 | 轻微润湿后补拉固定点 | 边缘音色接近中心 |
| 长途传令 | 同规格鼓统一尺寸和鼓点 | 远处能分辨节奏含义 |
| 保存 | 离地悬挂或垫高 | 不受潮、不虫蛀 |

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 声音闷 | 鼓皮太松或太厚 | 重新润湿后拉紧，或换薄皮 |
| 边缘破裂 | 鼓口锋利或拉力不均 | 磨圆鼓口，十字均匀固定 |
| 有腐臭 | 生皮处理不净 | 弃用，重新去脂漂洗 |
| 音高乱 | 厚薄不均或张力不匀 | 换皮或局部调整固定点 |

## 安全边界

- 石灰水会灼伤皮肤，处理皮料时戴手套，溅到皮肤立刻冲洗。
- 腐败动物皮可能带病原，不处理发臭、蛆虫、黑斑严重的皮。
- 鼓皮干燥不要暴晒急烘，开裂后很难补救。

## 穿越小说梗

穿越者用不同尺寸鼓腔编排节奏可以写，但更可信的细节是先建立固定鼓点含义，并用同规格鼓保证声音能被远处听懂。
