# 6.14 灯芯材料选择

> 灯芯是灯具的心脏——好灯芯让燃料充分燃烧，省油又亮堂。

## 可选材料对比

| 材料 | 来源 | 毛细效果 | 燃烧稳定性 | 采集难度 |
|------|------|---------|-----------|---------|
| 灯芯草髓心 | 湿地采集灯芯草茎秆剥取 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 低 |
| 棉线 | 棉花纺线搓捻 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 中（需有棉） |
| 麻纤维 | 麻皮剥取梳理 | ⭐⭐⭐ | ⭐⭐⭐ | 低 |
| 芦苇绒 | 芦苇花絮收集 | ⭐⭐⭐ | ⭐⭐ | 低 |
| 丝线 | 蚕丝捻合 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 高（蚕丝珍贵） |

## 操作步骤

### 灯芯草灯芯（推荐）
1. 夏秋采集灯芯草茎秆，越老越粗壮
2. 用小刀沿茎秆纵向划开，剥去外皮
3. 取出白色海绵状的髓心
4. 晒干后裁成合适长度（一般8~12cm，露出盏沿3mm最佳）

### 棉线灯芯
1. 取3~5根棉线松散地编在一起（不要拧太紧，以防毛细效果变差）
2. 一端浸入融化的蜡或油脂中晾干，使灯芯挺直不易烧断

## 灯芯长度与亮度

灯芯露出越长约亮但也越冒烟。最佳比例：露出油面约3mm，此时燃烧最充分、亮度最大、黑烟最少。
