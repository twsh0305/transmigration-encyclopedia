# 6.7 天然香水提取

> 蒸馏出来的是水，但留在水里的是花的灵魂——古代花露水制作指南。

## 所需材料

| 材料 | 用途 |
|------|------|
| 新鲜花瓣（玫瑰/茉莉/桂花等） | 芳香来源 |
| 清水 | 蒸馏介质 |
| 陶罐+竹管冷凝通道 | 蒸馏装置 |
| 高度蒸馏酒（可选） | 浸泡提取法 |

## 操作步骤

### 蒸馏法（提取纯露+精油）
1. 搭建简易蒸馏器：陶罐装水+花瓣，罐口接竹管作为冷凝通道
2. 竹管外包裹湿布降温，下方放收集容器
3. 加热陶罐，水蒸气带出花瓣中的芳香物质
4. 蒸气经竹管冷却凝结成液体
5. 收集液静置后分层：上层为**精油**（极少量）、下层为**纯露**（芳香花水）
   - 注意：从古法蒸馏中提取纯精油效率极低，1千克玫瑰花瓣只能得到约0.5毫升精油

### 酒精浸泡法（简易香水）
1. 花瓣浸泡在高度蒸馏酒（50°以上）中
2. 花瓣:酒≈1:3，密封
3. 放置 1 到 3 个月，每周摇晃一次
4. 过滤取液，即得花香水
5. 可滤出花瓣后换新花瓣再泡一次，浓度更高

## 比例与时间尺度

| 方法 | 建议尺度 | 成功标准 |
|------|----------|----------|
| 蒸馏法 | 花瓣装到陶罐三分之一到二分之一，不压实 | 出液连续，香气清楚，无焦糊味 |
| 冷凝 | 竹管外湿布持续保持凉湿 | 出口滴液，不大量冒白汽 |
| 酒精浸泡 | 花瓣:酒约 1:3 | 花瓣完全浸没，不浮出发霉 |
| 熟化 | 密封 1 到 3 个月，每周摇晃 1 次 | 过滤后无腐败味、无明显花渣 |
| 贴肤测试 | 手腕内侧 24 小时 | 不红肿、不刺痒 |

## 不同花材出香效率

| 花材 | 香气强度 | 持久度 | 最佳方法 |
|------|---------|--------|---------|
| 玫瑰 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 蒸馏法 |
| 茉莉 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 脂吸法 |
| 桂花 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 酒精浸泡法 |
| 薰衣草 | ⭐⭐⭐ | ⭐⭐ | 蒸馏法 |

## 成功标准

- 蒸馏液清澈或微浑，有明显花香，无焦糊味。
- 冷凝管出口持续滴液，不大量跑蒸汽。
- 静置后若有油层，油层极薄且气味纯净。
- 酒精浸泡液过滤后无花渣腐败味。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 焦味 | 干烧或火太猛 | 加足水，小火蒸馏 |
| 没香味 | 花材少或冷凝差 | 增加新鲜花材，强化冷却 |
| 液体浑浊发臭 | 花材腐败 | 丢弃，换新鲜花 |
| 酒味刺鼻 | 酒精浓、熟化短 | 密封阴放 1-2 周再评估 |

## 安全边界

- 酒精浸泡液远离明火，过滤和灌装不在炉灶旁进行。
- 不使用夹竹桃、曼陀罗、乌头等有毒植物提香。
- 纯露不是无菌药水，不能滴眼、注射或处理深伤口。
