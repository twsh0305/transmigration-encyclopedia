# 6.7 天然香水提取

> 蒸馏出来的是水，但留在水里的是花的灵魂——古代花露水制作指南。

## 所需材料

| 材料 | 用途 |
|------|------|
| 新鲜花瓣（玫瑰/茉莉/桂花等） | 芳香来源 |
| 清水 | 蒸馏介质 |
| 陶罐+竹管冷凝通道 | 蒸馏装置 |
| 高度蒸馏酒（可选） | 浸泡提取法 |

## 操作步骤

### 蒸馏法（提取纯露+精油）
1. 搭建简易蒸馏器：陶罐装水+花瓣，罐口接竹管作为冷凝通道
2. 竹管外包裹湿布降温，下方放收集容器
3. 加热陶罐，水蒸气带出花瓣中的芳香物质
4. 蒸气经竹管冷却凝结成液体
5. 收集液静置后分层：上层为**精油**（极少量）、下层为**纯露**（芳香花水）
   - 注意：从古法蒸馏中提取纯精油效率极低，1kg玫瑰花瓣只能得到约0.5ml精油

### 酒精浸泡法（简易香水）
1. 花瓣浸泡在高度蒸馏酒（50°以上）中
2. 花瓣:酒≈1:3，密封
3. 放置1~3个月，每周摇晃一次
4. 过滤取液，即得花香水
5. 可滤出花瓣后换新花瓣再泡一次，浓度更高

## 不同花材出香效率

| 花材 | 香气强度 | 持久度 | 最佳方法 |
|------|---------|--------|---------|
| 玫瑰 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 蒸馏法 |
| 茉莉 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 脂吸法 |
| 桂花 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 酒精浸泡法 |
| 薰衣草 | ⭐⭐⭐ | ⭐⭐ | 蒸馏法 |
