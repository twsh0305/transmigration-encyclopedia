# 6.8 常见古代游戏复原

> 古人下班后玩什么？投壶、蹴鞠、马球——了解它们，穿越后社交不尴尬。

## 投壶

| 项目 | 说明 |
|------|------|
| 道具 | 铜壶口直径约15cm，箭矢长约20~40cm |
| 距离 | 投掷线距壶口2~5米（随水平调整） |
| 规则 | 宾主依次投掷，中多者胜，不中罚酒 |
| 技巧 | 箭身竖直下落更易入壶，末端加羽毛可增加稳定性 |

**制作方法**：陶罐或竹筒可替代铜壶，筷子或细竹枝替代箭矢。

## 蹴鞠

- **球体制作**：皮革缝制球皮，内填毛发或干草，直径约15~20cm
- **单人玩法**：颠球，计数为胜
- **多人玩法**：两队对抗，踢进球门得分（类似现代足球）
- **穿越者优势**：现代足球训练方法+充气气球技术，降维打击

## 打马球

- **道具**：长柄曲棍（约1.2m），木球（约8cm直径）
- **规则**：两队骑手用曲棍击球入门，进球多者胜
- **门槛**：需要马匹和场地，不是平民游戏
- **穿越者注意**：马球是唐代贵族社交的重要场合，打得好=社交通行证

## 其他流行游戏

- **弹棋**——用手指弹棋子入坑，类似桌游
- **樗蒲**——掷骰子的赌博游戏，规则类似飞行棋
- **六博**——先秦流行的棋类游戏，现已失传
