# 6.8 常见古代游戏复原

> 古人下班后玩什么？投壶、蹴鞠、马球——了解它们，穿越后社交不尴尬。

## 投壶

| 项目 | 说明 |
|------|------|
| 道具 | 铜壶口直径约 15 厘米，箭矢长约 20-40 厘米 |
| 距离 | 投掷线距壶口 2-5 米（随水平调整） |
| 规则 | 宾主依次投掷，中多者胜，不中罚酒 |
| 技巧 | 箭身竖直下落更易入壶，末端加羽毛可增加稳定性 |

**制作方法**：陶罐或竹筒可替代铜壶，筷子或细竹枝替代箭矢。

**成功标准**：新手距离 2 米，10 投能中 2-3 次；熟练者提高到 3-5 米。壶不晃、箭不反弹伤人。

## 蹴鞠

- **球体制作**：皮革缝制球皮，内填毛发或干草，直径约 15-20 厘米
- **单人玩法**：颠球，计数为胜
- **多人玩法**：两队对抗，踢进球门得分（类似现代足球）
- **穿越者优势**：现代足球训练方法+充气气球技术，降维打击

**成功标准**：球体圆整，滚动不明显偏向一侧；连续踢 20 分钟不裂线、不散填料。

## 打马球

- **道具**：长柄曲棍（约 1.2 米），木球直径约 8 厘米
- **规则**：两队骑手用曲棍击球入门，进球多者胜
- **门槛**：需要马匹和场地，不是平民游戏
- **穿越者注意**：马球是唐代贵族社交的重要场合，打得好=社交通行证

**安全边界**：不会骑马的人不直接上马球场。先用步行木球练击球，再练慢马控向。

## 其他流行游戏

- **弹棋**——用手指弹棋子入坑，类似桌游
- **樗蒲**——掷骰子的赌博游戏，规则类似飞行棋
- **六博**——先秦流行的棋类游戏，现已失传

## 失败征兆

- 道具太硬太重，玩一次就砸伤人或砸坏家具。
- 规则没有先约定，最后变成争输赢而不是社交。
- 赌博彩头过大，引发债务和冲突。

## 复原原则

先复原“社交功能”，再复原“考据细节”。宴席上用安全材料玩起来，比拿着危险硬木器具争论尺寸更有用。
