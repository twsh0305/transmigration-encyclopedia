# 6.9 律管定音原理

> 一根竹管决定了中国音乐的基调——黄钟大吕不是成语，是物理。

## 原理

三分损益法是中国古代音律的核心计算方式：

- **三分损一**：取标准管长度L，截去三分之一（得2/3L），发出高五度音
- **三分益一**：增加三分之一（得4/3L），发出低四度音
- 反复损益十二次，得到十二个半音（十二律）

> 这和现代十二平均律的差别在于：三分损益法不是等分十二个半音，所以不同调式的音程略有差异。

## 操作步骤

1. 选内径均匀的竹管（内径约1cm，壁厚均匀）
2. 截取基音管（黄钟管）长度约9寸（约30cm），一端用竹节封底
3. 从管口横吹发出基础音（类似吹瓶口）
4. 以黄钟为基准，按三分损益依次计算各律管长度：
   - 黄钟→林钟：×2/3
   - 林钟→太簇：×4/3
   - 以此往复十二次
5. 各律管按计算结果截取，逐一校音
