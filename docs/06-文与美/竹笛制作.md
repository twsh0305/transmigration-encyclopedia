# 6.16 竹笛制作

> 笛子不是“竹管打洞”这么简单。孔位、孔径、内壁和调音决定它能不能成乐器。

## 选材与预处理

| 项目 | 合格标准 |
|------|----------|
| 竹龄 | 3-4 年，竹壁坚实 |
| 竹径 | 内径尽量均匀，新手选 16-22毫米 |
| 竹节 | 一端保留竹节作堵头，管内通畅 |
| 干燥 | 阴干 2-4 周，敲击声音清脆 |

弯曲处可小火烘热后缓慢校直。烘烤时竹面温热即可，不烤焦。

## 关键尺寸

- 管内径：新手优先 16-22毫米，太细声音尖且难吹，太粗耗气。
- 吹孔：距堵头内侧 30-50毫米，先开到约 8毫米，再按发声修边。
- 音孔：先小后大，初孔约 5-6毫米，最终多在 6-9毫米之间。
- 孔边：内外毛刺都磨掉，边缘利落但不割嘴。

## 开孔步骤

1. 先定吹孔：距堵头内侧约 30-50毫米，开椭圆孔，长约 8-10毫米。
2. 吹孔下方 60-80毫米处开膜孔；没有笛膜时可先不做。
3. 音孔先开小，不一次开到最终大小。常见音孔直径约 6-9毫米。
4. 从靠近吹孔的一端逐孔调音。孔开大，音会升高；孔无法“缩小”，所以下刀要保守。
5. 每开一孔，用蜡临时封住后续孔，逐音校对。

## 成功标准与调音标准

- 同一口风重复吹 3 次，音高差不大。
- 相邻音阶听起来递进，不出现明显“某一孔特别低”。
- 低音容易发声，高音加急气也不破音。

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 吹不响 | 吹孔边缘钝或角度错 | 修薄吹孔内侧边缘 |
| 音偏低 | 孔小或位置偏远 | 少量扩孔 |
| 音偏高 | 孔过大或位置偏近 | 用蜡临时缩孔，下一支修正 |
| 竹管开裂 | 干燥不足或火烤过猛 | 阴干后再做，裂管弃用 |

## 安全边界

- 开孔用烧红铁钎时要固定竹管，手离孔位至少 10厘米。
- 不吸入焦竹烟；室内烘烤要通风。
- 裂纹贯穿音孔或吹孔的笛管不要继续修，音准和强度都不可靠。

## 穿越小说梗

穿越者可以用“先小孔、逐孔校音、孔大升高”的原则稳定出品，比只凭手感一次打孔可靠得多。
