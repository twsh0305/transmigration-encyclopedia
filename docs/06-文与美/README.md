# 第六卷·文与美

> 生存之上，才有生活。琴棋书画、胭脂水粉、灯火蜡烛——穿越了也要活得体面。

## 使用顺序

1. 先做照明：油灯、灯芯、灯具和蜡烛能延长夜间劳动时间。
2. 再做文房：毛笔、墨、砚、印章解决记录、契约和身份凭证。
3. 接着做妆护：口脂、香脂、发油、米粉只在小量试用合格后上脸。
4. 最后做乐器与游戏：先按尺寸复原，再按音准或规则稳定性调整。

## 成功标准

- 照明：同一盏灯连续燃烧 30 分钟，火焰稳定、不黑烟、不溢油。
- 文房：能连续写满一页纸，笔锋不散、墨色不糊、印文可辨。
- 妆护：手腕内侧试用 24 小时无红肿刺痛，再小面积上脸。
- 乐器：同一人重复演奏 3 次，音高和节奏差异可被听者接受。
- 游戏：两个新手按规则玩完一局，没有反复争议关键判定。

## 失败征兆

- 只好看但不能重复制作：缺少比例、尺寸或时间记录。
- 灯具冒黑烟、油面过热、蜡烛偏烧：材料和结构还没达标。
- 粉、脂、香出现霉味、酸败味或分层渗水：不要继续上脸。
- 乐器越修越跑音：应回到小样试制，不在成品上猛改。

## 安全边界

- 任何接触皮肤、眼口鼻的制品都先做小样和皮试，不用铅、汞、砷等有毒矿物。
- 明火照明旁 30厘米内不放布帘、纸张、干草和酒精类液体。
- 打磨石、骨、竹、墨时避开粉尘，至少保持通风并遮住口鼻。
- 游戏和表演不以摔打、射击、纵火为噱头；娱乐不能压过安全。
