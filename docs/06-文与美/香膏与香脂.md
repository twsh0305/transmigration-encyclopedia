# 6.21 香膏与香脂

> 穿越到古代，体味问题比温饱问题更紧迫——香膏就是古代除臭+香水二合一。

## 原理

香膏的核心是「脂吸法」：用纯净无味的动物脂肪吸收花瓣中的芳香物质，脂肪会慢慢吸收并保留花香。原理类似现代香水中的「浸膏」工艺。

## 操作步骤

### 脂吸法（最传统）
1. 炼制基底脂：猪脂或羊脂切碎，小火炼化后用纱布过滤去杂质
2. 瓷盘上涂一层精炼脂（厚约3~5mm）
3. 铺上新鲜花瓣（玫瑰、茉莉、桂花等），花瓣不能重叠
4. 每天更换一次新鲜花瓣，连续7~10天
   - 每天换花时，先用竹片刮一下吸收花香的油脂表面促进混合
5. 最后加少量蜂蜡（约油:蜡=5:1）调节稠度
6. 隔水加热使蜂蜡熔化，搅匀后灌入小瓷盒密封保存

### 直接浸泡法（简易版）
1. 花瓣浸泡在芝麻油或山茶油中
2. 密封放置7~15天，每日摇晃
3. 过滤取油即可（质地比脂吸法稀，更适合夏天）

## 香膏VS现代香水

| 对比 | 香膏 | 现代香水 |
|------|------|---------|
| 持久度 | 约4~6小时 | 约6~8小时 |
| 扩香力 | 贴身距离 | 数米内可闻 |
| 制作难度 | 低（材料易得） | 极高（需酒精+蒸馏） |
| 古代价值 | 上等香膏价比黄金 | 不存在 |

## 穿越小说梗

女主靠自制香膏在上流社交场合香压群芳，靠的就是现代人对香氛层次的理解。
