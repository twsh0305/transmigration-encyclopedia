# 6.2 发油与发蜡

> 古代男子的束发，少不了一抹油光——发油就是古代的定型喷雾。

## 所需材料

| 材料 | 用途 | 获取方式 |
|------|------|---------|
| 山茶油/芝麻油 | 基础油 | 榨油或购买 |
| 新鲜桂花/玫瑰花 | 增香 | 采集 |
| 蜂蜡 | 增加稠度，定型 | 蜂巢提取 |

## 操作步骤

### 桂花发油
1. 新鲜桂花阴干去掉水分（不必全干，去表面水分即可）
2. 放入山茶油或芝麻油中，花:油≈1:5
3. 密封放在日晒处，每天摇晃一次
4. 浸泡7~15天后过滤取油
5. 油呈淡金色，有桂花香气

### 发蜡
1. 按油:蜡≈3:1的比例准备发油和蜂蜡
2. 蜂蜡切碎，放入发油中
3. 隔水加热（不可直火），搅拌至蜂蜡完全熔化
4. 稍微降温后倒入容器，冷却凝固成膏状
5. 取少量（约黄豆大小）涂抹于干发上

## 其他配方

| 类型 | 基底 | 香味来源 | 效果 |
|------|------|---------|------|
| 桂花油 | 山茶油 | 桂花 | 柔顺有光泽 |
| 玫瑰蜡 | 蓖麻油+蜂蜡 | 玫瑰花瓣 | 定型强 |
| 松香发蜡 | 猪油+松香 | 无 | 极强定型（戏班常用） |

## 穿越小说梗

穿越者知道不同油脂的护发效果差异——山茶油养发、蓖麻油生发，可在古代美发界开创商机。
