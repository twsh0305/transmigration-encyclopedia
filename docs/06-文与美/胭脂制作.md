# 6.17 胭脂制作

> 一抹胭脂红，千年女儿心。

## 原理

胭脂核心来自红蓝花（红花）中的红花素。红花素不溶于水但可溶于碱性溶液。先用酸性水去除黄色素，再用草木灰碱液溶出红色素，最后加酸中和沉淀得到纯净红色染料调入油脂基底。

## 所需材料

| 材料 | 说明 |
|------|------|
| 红蓝花（红花） | 主要色素来源 |
| 草木灰碱液 | 碱性溶液提取红花素 |
| 醋或发酵淘米水 | 去除黄色素 |
| 动物脂肪（猪脂/羊脂） | 基底 |
| 米粉或滑石粉 | 调节质地 |

## 操作步骤

1. 红花洗净阴干捣碎，清水浸泡一夜
2. 放入发酵淘米水中揉搓，换水3~5次去除黄色素
3. 加入草木灰碱液搅拌揉压，静置1~2小时溶出红色素，过滤得深红色滤液
4. 滤液中逐滴加醋搅拌，出现红色絮状沉淀后静置2~4小时
5. 倾去上层清液，收集深红色膏状沉淀
6. 动物脂肪熔化过滤，温热时加入红花素搅匀，灌入瓷瓶冷却凝固

## 穿越小说梗

女主靠改良红花提取工艺做出纯正胭脂发家致富，经典穿越桥段。
