# 2.23 蜂蜜采集与处理

> 蜜蜂不生产糖，它们只是大自然的搬运工。

## 原理

蜜蜂采集花蜜（蔗糖溶液）→ 混入唾液（含转化酶）→ 带回蜂巢 → 蒸发水分至 18% 以下 → 成熟蜂蜜。

蜂蜜为什么能永久保存不坏？
- **低含水量**（< 18%）：细菌无法生长
- **高酸性**（pH 3.5-5.5）：抑制大多数微生物
- **过氧化氢**：蜂蜜中天然含有微量葡萄糖氧化酶，遇水缓慢释放过氧化氢（杀菌成分）

## 操作步骤

### 寻找野生蜂巢

1. **水源侦察**：蜜蜂觅食半径约 3 公里，水源附近常见
2. **路线追踪**：找到采蜜的蜜蜂，观察飞行方向（回巢路线）
3. **诱饵法**：放糖水盘，等待蜜蜂"叫来"更多同伴
4. **位置特征**：树洞、岩缝、老墙缝隙

### 取蜜操作

1. **时间选择**：最好在白天温暖时（蜜蜂活跃，大部分在外采蜜）
2. **烟熏**：用喷烟器或干草闷烟在蜂巢口熏几下（烟使蜜蜂吸入大量蜂蜜，行动变缓）
3. **开巢**：小心打开蜂巢或蜂箱
4. **取脾**：取出蜜脾（封盖的巢脾才是成熟蜜）
5. **割盖**：用刀割去封盖蜡层
6. **摇蜜**：放入摇蜜机，离心甩出蜂蜜
7. **过滤**：通过滤网去除蜡屑和杂质
8. **装瓶**：密封保存

**留足蜂粮**：取蜜时至少留 1/3 给蜜蜂过冬，否则整群会饿死。

### 蜂蜜处理

1. **过滤**：新鲜蜂蜜中混有蜡屑、花粉粒、死蜂等杂质，需过滤
2. **静置消泡**：刚取出的蜂蜜含大量气泡，静置 1-2 天
3. **结晶处理**：蜂蜜放久了会结晶（葡萄糖析出），隔水加热至 40摄氏度即可恢复液态
4. **保存**：密封、避光、15-20摄氏度最佳。**不要冷藏**（加速结晶）

### 蜂蜡处理

取蜜后剩下的巢脾是宝贵的蜂蜡：

1. 将巢脾放入锅中加水煮沸
2. 蜡熔化后浮在水面
3. 冷却后取出蜡块
4. 反复熔化-冷却几次提纯
5. 纯蜂蜡可用于制作蜡烛、润唇膏、木器保养

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 取蜜时间 | 白天温暖、无雨、大多数外勤蜂离巢时操作 |
| 蜂群保留 | 至少留 1/3 蜜脾给蜂群，不毁掉全部幼虫区 |
| 成熟蜜 | 优先取封盖蜜脾，流出的蜂蜜浓稠、不像水一样滴落 |
| 过滤 | 过滤后无死蜂、粗蜡屑和明显杂质，静置 1-2 天消泡 |
| 保存 | 密封避光，15-20摄氏度阴凉处保存，瓶口不进水 |

## 失败征兆

- 蜂蜜有明显酒味、酸味或大量气泡，说明含水高或已发酵。
- 取蜜时蜂群大规模攻击，说明烟熏不足、天气不合适或动作太粗暴。
- 被蜇后出现喘不过气、喉咙紧、全身荨麻疹，是严重过敏信号。
- 蜜中混入大量幼虫、泥沙、死蜂，不能直接作高品质食用蜜。
- 为取蜜烧毁蜂巢，会失去长期蜜源，也可能引发山火。

## 常见问题

- **Q**: 没有摇蜜机能取蜜吗？
- **A**: 可以。将蜜脾切碎，用纱布包裹，挤压出蜜。效率低但原理一样。

- **Q**: 蜂蜜发酵了怎么办？
- **A**: 含水量过高（> 20%）的蜂蜜会发酵产生酒精味。隔水加热至 60摄氏度保持 30 分钟可杀死酵母，然后密封保存。

- **Q**: 被蜜蜂蜇了怎么办？
- **A**: 刮去毒刺（不要用镊子夹，会挤入更多毒液），肥皂水清洗，冷敷。对蜂毒过敏者可能休克，需立即就医。

## 安全提醒

- 蜂蜇可能引发严重过敏反应，第一次接触先少量试探
- 取蜜时穿浅色衣服（蜜蜂对深色衣物有攻击性）
- 不要靠近蜂巢喷洒杀虫剂——蜜蜂对化学喷雾极为敏感
- 蜂蜜虽好，一岁以下婴儿禁食（可能含肉毒杆菌孢子）

## 穿越小说梗

"主角从树洞里掏出一大块野蜂蜜"——理论上可行，实操上你大概率会被蜇到怀疑人生。野生蜂群极难对付，除非有烟熏设备。更靠谱的策略是发现蜂巢后用烟熏法慢慢操作，或者干脆自己养一箱蜜蜂。对了，蜂蜜在古代不仅是甜味剂，更是**轻伤特效药**——涂在伤口上杀菌促愈合，这个知识点才值钱。
