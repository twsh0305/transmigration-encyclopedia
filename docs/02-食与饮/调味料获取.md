# 2.24 调味料获取

> 没有调味料的食物只是燃料，有了调味料才是美食。

## 原理

五种基本味觉：甜、咸、酸、苦、鲜。调味料的作用是激活味觉受体，提升食物的适口性，同时部分调味料有抑菌保鲜功能。

## 各类调味料获取

### 盐（咸味）

已详述，见第一卷"粗盐提纯"等条目。

### 醋（酸味）

**原理**：酒在醋酸菌作用下氧化为醋酸

**制作**：
1. 将淡酒（米酒、果酒）敞口放置
2. 醋酸菌自然接种（来自空气）
3. 表面形成一层白色薄膜（醋母）
4. 放置 2-4 周，酒变酸即为醋
5. 更好的方法：将酒与醋母（已有醋的渣）混合
6. 温度 25-30摄氏度最宜

### 酱油（鲜味+咸味）

已列入大纲 2.4 条目，此处简述：
- 大豆蒸煮 → 拌面粉 → 制曲 → 加盐水发酵 → 榨取酱汁 = 酱油

### 香料

| 香料 | 获取方式 | 替代品 |
|------|---------|--------|
| 花椒 | 花椒树果 | 野生花椒 |
| 姜 | 种植或野外采集 | 沙姜 |
| 葱 | 种植 | 野葱/薤白 |
| 蒜 | 种植 | 野蒜 |
| 辣椒 | 明代后传入中国 | 茱萸（古代替代品） |
| 八角 | 八角树果实 | — |
| 桂皮 | 桂树皮 | 肉桂 |
| 茴香 | 种植 | — |

**⚠️ 穿越时间线注意**：辣椒明代才传入中国，穿越到唐代想吃辣只能用茱萸。

### 糖（甜味）

见 2.6 条目。

### 味精/鲜味

穿越后无法制作味精（需要发酵法制谷氨酸钠），但可以通过以下方式获得鲜味：
- 骨汤熬煮（含谷氨酸）
- 海带/昆布煮汤（含谷氨酸，味精的原始来源）
- 蘑菇（含鸟苷酸，与谷氨酸协同增鲜）
- 酱油/豆酱（发酵产生的氨基酸）

## 成功标准

- 咸味：盐能稳定获取、干燥保存，不夹苦卤味和泥沙。
- 酸味：醋液有清爽酸香，表面醋母为白色薄膜，无黑绿霉。
- 甜味：糖浆或饴糖能拉丝，不焦苦、不发霉。
- 鲜味：骨汤、海带、蘑菇、酱类至少有一种可稳定供应。
- 香料：每一种野生香料都能说出叶、花、果或气味特征，不混采。

## 失败征兆

- 醋有腐臭、霉味、黑绿霉或虫卵，不能食用。
- 野生香料只凭气味相似就下锅，容易混入毒草。
- 糖浆熬到焦黑发苦，说明火候过头。
- 用不洁容器长期存酱、醋、糖浆，表面反复长霉。

## 常见问题

- **Q**: 穿越到古代能吃到什么味道？
- **A**: 咸（盐）、酸（醋）、甜（饴糖/蜂蜜）、鲜（酱/骨汤）。花椒提供麻味。辣椒明代后才传入中国——宋代以前的中国菜是不辣的。

## 安全提醒

- 自制醋需要卫生条件，杂菌污染可能产生有害物质
- 野生香料需确认种类，部分植物外观相似但有毒

## 穿越小说梗

穿越到古代想吃麻辣火锅？没有辣椒就没有辣味。明代以前的中国菜基本不辣——四川人也不是天生吃辣的，辣椒是美洲作物，大航海时代后才传入。
