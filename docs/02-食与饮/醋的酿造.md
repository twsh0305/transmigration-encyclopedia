# 2.29 醋的酿造

> 酒放坏了就是醋——但人类花了几千年才学会控制这个过程。

## 原理

醋的酿造本质是**醋酸菌（Acetobacter）**将酒精氧化为醋酸：

```
C₂H₅OH + O₂ →(醋酸菌)→ CH₃COOH + H₂O
乙醇    氧气          醋酸       水
```

关键条件：
- **酒精**：任何含酒精的液体都可制醋（米酒、果酒、啤酒等）
- **氧气**：醋酸菌是需氧菌，必须有充足的氧气
- **温度**：25-35摄氏度最适宜
- **醋酸菌**：空气中自然存在，无需额外添加

## 所需材料

| 材料 | 用途 | 说明 |
|------|------|------|
| 含酒精液体 | 原料 | 米酒、果酒、啤酒、甚至剩酒 |
| 敞口容器 | 发酵 | 陶瓷/玻璃，宽口最好 |
| 纱布 | 防虫 | 盖在容器口 |
| 种醋（可选） | 加速启动 | 市售未灭菌的醋 |

## 操作步骤

### 基础法（自然发酵）

1. 将含酒精液体倒入敞口容器，装至 2/3 满
2. 用纱布盖口（透气防虫）
3. 放置于 25-35摄氏度的温暖处
4. 静置 2-4 周
5. 表面逐渐形成一层菌膜（醋衣/醋母）
6. 定期闻一下——酒味渐弱、酸味渐浓即可

### 接种法（更快更稳）

1. 在酒液中加入 10-20% 的种醋（含醋酸菌）
2. 同样敞口、纱布覆盖
3. 1-2 周即成
4. 成品醋留一部分做下次的种醋

### 判断标准

- **时间到**：酒味几乎消失、有明显的酸味
- **检验**：取一小勺尝——酸得够劲就可以过滤装瓶了
- **老熟**：过滤后的醋密封放置 1-3 个月，酸味更柔和

## 成功标准

- 发酵 1-2 周后酒味减弱、酸香增强，气味清爽不腐臭。
- 表面醋母为白色或米色薄膜，无黑绿霉、粉红霉和毛霉。
- 过滤后煮沸或加热杀菌，再密封避光保存。
- 留种醋时只留气味正常、无虫、无霉的一批。

## 失败征兆

- 腐臭、霉味、臭鸡蛋味或黏滑拉丝：杂菌污染。
- 表面有醋蝇幼虫、虫卵或大量小虫：防虫失败，小批量家庭制作应废弃。
- 酸味很弱但酒味消失：可能被其他微生物消耗，不能当合格醋。
- 容器沾油或长期暴露灰尘，风味和安全都不稳定。

## 不同酒源的风味

| 酒源 | 醋的风味 | 备注 |
|------|---------|------|
| 米酒 | 米醋，清淡微甜 | 最常见的家用醋 |
| 糯米酒 | 甜米醋 | 酸中带甜 |
| 果酒 | 果醋 | 苹果醋、葡萄酒醋等 |
| 啤酒 | 麦芽醋 | 欧洲传统调味醋 |
| 高粱酒 | 陈醋风格 | 酸味浓烈 |

## 常见问题

- **Q**: 为什么我的酒变酸了但没有菌膜？
- **A**: 菌膜不是必需的——醋酸菌也可以在没有明显菌膜的情况下工作。只要闻起来是醋味而不是腐臭味，就成功了。

- **Q**: 醋里面有小白虫怎么办？
- **A**: 那通常是醋蝇幼虫，说明防虫失败。家庭小批量制作不建议继续入口；应整批废弃，清洗消毒容器，换更密的纱布或水封。

- **Q**: 醋能保存多久？
- **A**: 密封避光保存数年不坏。醋酸本身就是天然防腐剂。

## 安全提醒

- 醋酸腐蚀牙齿和食道黏膜，纯醋建议稀释后饮用
- 高浓度醋酸对皮肤有刺激，操作时注意
- 过滤后的醋渣可作肥料

## 穿越小说梗

穿越后"开个醋坊发家致富"——技术上不难，关键是你得有稳定的酒源。醋的本质就是酒被醋酸菌"吃"了。先学会酿酒，醋自然就有了。而且古代醋坊最核心的竞争力不是配方，而是**种醋**——传了几代的老醋母比新酿的香得多。
