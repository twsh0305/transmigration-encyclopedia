# 2.6 常见可食野生植物辨识

> 辨识植物是穿越者最重要的技能——吃对了活命，吃错了要命。

## 原理

全球约有 30 万种植物，其中可食用的约 8 万种，但有毒的也有数千种。辨识必须依靠**逐种确认**，不存在万能识别法则。

**⚠️ 不存在"所有红色果实都有毒"之类的通用法则。** 很多红色浆果可食（草莓、覆盆子），有些看起来无害的植物却剧毒（毒芹）。

## 按气候带分类的可食植物

### 温带（中国大部分地区）

| 植物 | 可食部分 | 季节 | 识别要点 |
|------|---------|------|---------|
| 蒲公英 | 叶、花、根 | 全年 | 黄花→白绒球，叶基生莲座状，折断有白浆 |
| 荠菜 | 叶、花 | 春 | 基生叶莲座状，白色小花，三角扁平角果 |
| 马齿苋 | 茎叶 | 夏 | 茎红色匍匐，叶肉质对生，味酸 |
| 蕨菜 | 嫩叶（需处理） | 春 | 卷曲如拳头状嫩芽，**必须煮熟去毒** |
| 野蒜/薤白 | 全株 | 春 | 蒜味，细长叶，白色球状花 |
| 车前草 | 叶 | 全年 | 叶基生宽卵形，平行脉，穗状花序 |
| 野苋菜 | 叶、嫩茎 | 夏秋 | 叶互生，茎绿色或红色，穗状花序 |
| 榆树 | 嫩叶、榆钱 | 春 | 榆钱（翅果）可生食，嫩叶煮食 |
| 桑树 | 果、叶 | 夏 | 桑葚紫黑可食，叶可煮食或养蚕 |
| 野生草莓 | 果 | 夏 | 小型红色聚合果，白色花五瓣 |
| 栗子 | 果 | 秋 | 坚果包在刺壳中，烤熟食用 |
| 核桃 | 果 | 秋 | 青皮硬壳坚果，砸开取仁 |
| 松子 | 种子 | 秋冬 | 松果中取出，可生食 |
| 莲 | 根、子、叶 | 夏秋 | 莲藕、莲子可食，荷叶可包食 |

### 热带

| 植物 | 可食部分 | 识别要点 |
|------|---------|---------|
| 香蕉 | 果 | 大型草本，穗状花序 |
| 椰子 | 果汁、果肉 | 高大棕榈，大果 |
| 木瓜 | 果 | 高大草本，黄色果 |
| 野生山药 | 块根 | 藤本，心形叶，**必须煮熟** |
| 竹笋 | 嫩茎 | 禾本科，**多数需焯水去苦** |

## 通用采集原则

1. **确认再吃**：不确定的绝对不吃
2. **先少量试**：确认可食后先吃少量，等 8 小时看有无不良反应
3. **年轻植物优先**：嫩叶嫩茎通常比老叶好吃且毒素少
4. **远离污染源采集**：路边、农田边的植物可能喷过农药

## 常见问题

- **Q**: 有没有万能试毒法？
- **A**: **没有**。所谓"银器变黑""动物吃了我也能吃"都是不可靠的。唯一可靠的方法是逐种学习辨识。

## 安全提醒

- 本条目仅为入门参考，**不可替代专业植物辨识**
- 每种植物在食用前应确认至少 2 个独立来源
- 野外求生以安全为第一原则，宁饿勿赌

## 穿越小说梗

主角"看了一下就认出了几十种草药和野菜"——现实中，植物辨识需要长期学习。建议现代人穿越前先把自己所在地区的常见可食/有毒植物背下来，这比背元素周期表有用得多。
