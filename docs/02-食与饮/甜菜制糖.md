# 2.14 甜菜制糖

> 没有甘蔗的地方，甜菜就是唯一的糖源。

## 原理

甜菜（Beta vulgaris）的根茎中含有约 15-20% 的蔗糖，与甘蔗中的糖完全一样（都是蔗糖 C₁₂H₂₂O₁₁）。区别在于：
- **甘蔗**：茎秆含糖，压榨取汁
- **甜菜**：根部含糖，用热水浸出（扩散法）

甜菜制糖的温度控制比甘蔗更严格——甜菜汁中杂质更多，处理不当糖分会分解。

## 所需材料

| 材料/工具 | 用途 |
|-----------|------|
| 甜菜根 | 原料 |
| 水 | 浸出糖分 |
| 大锅 | 煮糖 |
| 擦丝器 | 将甜菜擦成丝 |
| 滤布/纱布 | 过滤 |
| 醋或石灰 | 澄清去杂 |

## 操作步骤

### 第一步：备料

1. 甜菜洗净，去掉须根
2. 切成细丝或擦成丝（越细越好，增大接触面积）
3. 越细出糖率越高，但别碎成泥（难过滤）

### 第二步：浸出（扩散法）

1. 将甜菜丝放入锅中，加水没过
2. 加热至 **70-80°C**（不要沸腾）
3. 保温浸泡 30-45 分钟
4. 倒出糖水过滤
5. 甜菜丝加新水再浸泡一次
6. 合并两次糖水

**不要煮沸**：高温会使甜菜中的果胶大量溶出，糖水变黏稠，后续结晶困难。

### 第三步：澄清

甜菜汁比甘蔗汁更"脏"——含有更多蛋白质和色素：

1. 糖水加热至 80°C
2. 慢慢加入石灰水或少量醋（调 pH 至中性）
3. 蛋白质絮凝沉淀，浮在表面
4. 撇去浮沫
5. 用细布过滤，得到淡黄色清液

如果过滤后有颜色，可以加少量蛋清搅拌后加热——蛋清会吸附色素，形成沉淀后过滤。

### 第四步：浓缩结晶

1. 澄清后的糖水大火煮沸
2. 不断搅拌防止糊底
3. 水分蒸发，糖液变浓
4. 浓缩至糖液温度达 110-115°C
5. 离火继续搅拌，蔗糖结晶析出
6. 结晶后得红糖色粗糖

### 第五步：提纯

1. 粗糖加少量水加热溶解
2. 加入蛋清或骨炭脱色
3. 过滤后再次浓缩结晶
4. 重复 2-3 次可得接近白色的糖

## 甜菜糖 vs 甘蔗糖

| 对比 | 甜菜糖 | 甘蔗糖 |
|------|--------|--------|
| 原料分布 | 温带/寒带 | 热带/亚热带 |
| 提取方式 | 水浸出 | 压榨 |
| 风味 | 中性，适合精制 | 有独特甘蔗香气 |
| 结晶难度 | 较难（杂质多） | 较易 |
| 出糖率 | 10-15%（干重） | 10-15%（干重） |

## 常见问题

- **Q**: 甜菜糖水太黏稠，结晶不出来？
- **A**: 可能是煮沸时间太长，果胶溶出太多。下次控制温度在 80°C 以下浸出，不要煮到沸腾。

- **Q**: 甜菜糖有怪味？
- **A**: 甜菜本身含有土腥味物质（土味素，geosmin）。原因：①浸出温度过高 ②澄清不充分。加蛋清澄清和骨炭脱色可以有效去味。

- **Q**: 甜菜能种在穿越世界？
- **A**: 甜菜适合温带气候，不耐热。如果你穿越到北方地区（黄河流域以北），种甜菜制糖比种甘蔗靠谱得多。

## 安全提醒

- 煮糖温度高（> 100°C），防止烫伤
- 甜菜丝榨糖后不要扔——含大量纤维和营养，可做饲料
- 糖浆结晶困难时不要强行食用——半结晶的糖浆在密封容器中可能发酵

## 穿越小说梗

如果你穿越到了北方或高纬度地区（欧洲、中亚、中国北方），甘蔗是种不活的。这时候甜菜就是唯一的本地糖源。拿破仑战争时期，英国人封锁了甘蔗航线，欧洲人把甜菜制糖技术推到了极致——这个历史真实事件说明：穿越后要不要种甘蔗，先看你落在哪个纬度。
