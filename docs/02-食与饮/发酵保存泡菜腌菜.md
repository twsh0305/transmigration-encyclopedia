# 2.1 发酵保存（泡菜、腌菜）

> 发酵是微生物替你打工——好菌占满了地盘，坏菌就挤不进来。

## 原理

乳酸菌在无氧环境下将蔬菜中的糖分转化为乳酸，使环境 pH 降至 3.5-4.0。在这个酸度下，大多数腐败菌和致病菌无法生存，食物因此得以保存。

发酵 ≠ 腐败。发酵是可控的微生物活动，腐败是不可控的。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 蔬菜 | 新鲜、无虫害 | 白菜、萝卜、黄瓜等 |
| 盐 | 纯净 | — |
| 水 | 煮沸后冷却 | — |
| 容器 | 陶瓷缸、玻璃罐 | 木桶（不能含油脂） |
| 重石 | 压住蔬菜 | 装水的密封袋 |

## 操作步骤

### 基础泡菜

1. **选菜**：选新鲜紧实的蔬菜（白菜、萝卜最佳）
2. **洗菜**：洗净后晾干表面水分
3. **配制盐水**：每升水加盐约 30-50克（3-5%浓度），煮沸后冷却
4. **装缸**：
   - 将蔬菜紧密码入缸中
   - 倒入冷却的盐水，完全淹没蔬菜
   - **关键**：蔬菜必须完全没入水面以下——露出水面的部分会腐烂
5. **压石**：在蔬菜上压一块干净的石头，确保蔬菜不浮出水面
6. **密封**：盖上盖子或用布封口
7. **发酵**：
   - 室温 20-25摄氏度：3-5 天即可
   - 室温 15-20摄氏度：5-7 天
   - 室温 10-15摄氏度：7-14 天
8. **判断成功**：
   - 水面有少量白色泡沫 = 正常（酵母活动）
   - 菜色微黄、有酸味 = 成功
   - 发臭、发黏、长有色霉 = 失败，丢弃

### 防止失败的要点

1. **全程无油**：容器和手不能有油脂（油会阻隔乳酸菌）
2. **完全浸没**：蔬菜不接触空气就不会烂
3. **盐量适当**：太少会腐烂，太多会太咸且抑制乳酸菌
4. **温度适宜**：太高会腐烂，太低发酵太慢

## 成功标准

- 盐水浓度保持 3-5%，蔬菜全程低于液面 2厘米以上。
- 20-25摄氏度下 24-48 小时出现细小气泡，3-5 天有清爽酸味。
- 菜体酸脆、颜色略变黄，盐水可浑但不黏、不臭。
- 取食工具干净无油，取完后剩余蔬菜仍被盐水淹没。

## 失败征兆

- 黑色、绿色、粉红色毛霉出现：整缸丢弃。
- 盐水拉丝、黏滑、腐臭或有粪臭味：腐败，不可食用。
- 蔬菜浮出液面变软发黑：浸没失败，污染风险高。
- 食用后口唇发紫、头晕、恶心：按中毒处理，保留样品。

## 常见问题

- **Q**: 泡菜表面长了白膜怎么办？
- **A**: 如果是白色薄膜（酵母膜），捞掉即可，下面的菜还能吃。如果是绿色/黑色/粉红色霉斑，整缸丢弃。

- **Q**: 能保存多久？
- **A**: 持续在盐水中可保存数月。越酸保存越久。转入低温环境可延长保存。

## 安全提醒

- 发酵失败的泡菜可能含亚硝酸盐，**不要食用发臭或长有色霉的泡菜**
- 亚硝酸盐中毒症状：口唇发紫、头晕、恶心
- 自制发酵食品有一定风险，注意卫生

## 穿越小说梗

穿越后想吃一口酸脆的泡菜？没问题——白菜+盐水+石头+等待，简单到不需要任何现代设备。这是人类最早的食品保存技术之一，已有数千年历史。
