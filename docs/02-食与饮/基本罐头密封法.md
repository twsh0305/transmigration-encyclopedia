# 2.5 基本罐头密封法（玻璃瓶 + 蜡封）

> 把食物装进罐子里封死，你就锁住了时间。

## 原理

罐头保存的核心：**加热杀灭微生物 + 密封隔绝再污染**。

1. 加热使食物中的微生物和酶失活
2. 密封防止空气中的微生物再次进入
3. 内部形成微真空，好氧菌无法生存

现代罐头用金属罐和高压灭菌，古代条件下可以用玻璃瓶 + 水浴灭菌 + 蜡封实现基础版本。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 玻璃瓶+盖 | 有密封胶圈的广口瓶 | 陶罐+木塞 |
| 蜡 | 蜂蜡、石蜡 | 松香 |
| 食物 | 新鲜食材 | — |
| 大锅 | 水浴灭菌用 | — |

## 操作步骤

### 1. 瓶子消毒

1. 玻璃瓶和盖子在沸水中煮 10 分钟
2. 取出倒扣在干净布上晾干
3. **全程不要用手接触瓶口内侧**

### 2. 装瓶

1. 食物预先烹煮至半熟或全熟
2. 趁热将食物装入瓶中
3. 留约 2cm 的顶部空间（食物膨胀预留）
4. 倒入煮食物的汤汁或糖水/盐水
5. 用干净的筷子沿瓶壁滑动排气泡
6. 擦净瓶口

### 3. 密封

**方法A：旋盖瓶**
1. 拧紧瓶盖
2. 放入大锅中水浴
3. 加热至水沸腾，保持 20-30 分钟
4. 取出冷却——冷却时瓶盖会凹陷（真空）

**方法B：软木塞 + 蜡封**
1. 用软木塞塞紧瓶口
2. 将蜂蜡加热融化
3. 将瓶口倒扣浸入蜡液中
4. 取出后蜡层凝固形成密封
5. 再水浴灭菌 30 分钟

### 4. 检验

1. 冷却后检查瓶盖是否凹陷（真空标志）
2. 按压瓶盖——不弹起 = 密封好
3. 倾斜瓶子——没有气泡从瓶口溢出 = 密封好

## 不同食物的水浴时间

| 食物 | 酸度 | 水浴时间 | 备注 |
|------|------|---------|------|
| 酸性水果（番茄、梅、杏） | pH<4.5 | 15-20分钟 | 容易保存 |
| 糖水水果 | 酸+糖 | 20-30分钟 | |
| 低酸蔬菜 | pH>4.5 | **至少30分钟** | 风险较高 |
| 肉类 | pH>4.5 | 不推荐水浴 | 需高压灭菌 |

**⚠️ 警告**：低酸食物（肉类、大部分蔬菜）用水浴法不够安全，可能残留肉毒杆菌孢子。现代罐头用 121°C 高压灭菌。古代条件下，肉类的长期保存建议用盐腌/烟熏代替罐头。

## 常见问题

- **Q**: 罐头鼓盖了能吃吗？
- **A**: **绝对不能**。鼓盖说明内部产气（微生物活动），食物已变质，可能含致命的肉毒杆菌毒素。

- **Q**: 能保存多久？
- **A**: 酸性水果罐头 6-12 个月。开盖后必须尽快食用。

## 安全提醒

- 肉毒杆菌是罐头食品最大的威胁——其毒素 1 微克即可致死
- 低酸食物**不要**用简易水浴法做罐头
- 罐头鼓盖、漏气、异味 = 丢弃
- 吃之前煮沸 10 分钟可破坏肉毒毒素（但不可靠，不要依赖此法）

## 穿越小说梗

穿越后想做罐头？酸性水果罐头难度不大，但肉类罐头在没有高压锅的情况下几乎不可能安全制作。肉毒杆菌是看不见闻不到的杀手——穿越者最大的饮食风险不是饿死，而是食物中毒。
