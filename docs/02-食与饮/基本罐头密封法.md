# 2.5 基本罐头密封法（玻璃瓶 + 蜡封）

> 把食物装进罐子里封死，你就锁住了时间。

## 原理

罐头保存的核心：**加热杀灭微生物 + 密封隔绝再污染**。

1. 加热使食物中的微生物和酶失活
2. 密封防止空气中的微生物再次进入
3. 内部形成微真空，好氧菌无法生存

现代罐头用金属罐和高压灭菌，古代条件下可以用玻璃瓶 + 水浴灭菌 + 蜡封实现基础版本。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 玻璃瓶+盖 | 有密封胶圈的广口瓶 | 陶罐+木塞 |
| 蜡 | 蜂蜡、石蜡 | 松香 |
| 食物 | 新鲜食材 | — |
| 大锅 | 水浴灭菌用 | — |

## 操作步骤

### 1. 瓶子消毒

1. 玻璃瓶和盖子在沸水中煮 10 分钟
2. 取出倒扣在干净布上晾干
3. **全程不要用手接触瓶口内侧**

### 2. 装瓶

1. 食物预先烹煮至半熟或全熟
2. 趁热将食物装入瓶中
3. 留约 2厘米的顶部空间（食物膨胀预留）
4. 倒入煮食物的汤汁或糖水/盐水
5. 用干净的筷子沿瓶壁滑动排气泡
6. 擦净瓶口

### 3. 密封

**方法A：旋盖瓶**
1. 拧紧瓶盖
2. 放入大锅中水浴
3. 加热至水沸腾，保持 20-30 分钟
4. 取出冷却——冷却时瓶盖会凹陷（真空）

**方法B：软木塞 + 蜡封**
1. 用软木塞塞紧瓶口
2. 将蜂蜡加热融化
3. 将瓶口倒扣浸入蜡液中
4. 取出后蜡层凝固形成密封
5. 再水浴灭菌 30 分钟

### 4. 检验

1. 冷却后检查瓶盖是否凹陷（真空标志）
2. 按压瓶盖——不弹起 = 密封好
3. 倾斜瓶子——没有气泡从瓶口溢出 = 密封好

## 不同食物的处理边界

| 食物 | 酸度 | 简易水浴 | 备注 |
|------|------|----------|------|
| 酸性水果、果酱、醋渍菜 | pH≤4.6 | 可按可靠配方水浴 | 仍需足量糖、酸和时间 |
| 番茄制品 | 边界酸度 | 需额外酸化才稳妥 | 不同品种酸度差异大 |
| 低酸蔬菜 | pH>4.6 | 不推荐 | 需要压力罐头达到 121摄氏度左右 |
| 肉类、鱼类、禽类 | pH>4.6 | 不推荐 | 古代条件优先盐腌、烟熏、风干 |

**⚠️ 警告**：低酸食物（肉类、大部分蔬菜）用水浴法不够安全，可能残留肉毒杆菌孢子。现代罐头用 121摄氏度高压灭菌。古代条件下，肉类的长期保存建议用盐腌/烟熏代替罐头。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 食材 | 只用新鲜、无霉烂食材；低酸肉菜不做简易水浴罐头 |
| 酸度 | 水浴法只用于 pH≤4.6 的高酸食品，并按可靠配方加酸 |
| 消毒 | 瓶和盖沸水煮 10 分钟，装瓶时不摸瓶口内侧 |
| 密封 | 冷却后瓶盖内凹，按压不弹，瓶口无渗漏 |
| 储存 | 阴凉避光，贴日期；开盖后尽快吃完并冷藏或重新加热 |

## 失败征兆

- 鼓盖、漏液、喷气、异味、泡沫、霉点，整瓶丢弃。
- 肉类、豆类、低酸蔬菜用蜡封或水浴常温存放，是肉毒风险。
- 只把热食倒进瓶里就封口，没有瓶内加热灭菌。
- 自改配方减少糖、醋、盐或加低酸配料，会改变安全边界。
- 婴儿、孕妇、老人和免疫低下者不要吃来源不明自制罐头。

## 常见问题

- **Q**: 罐头鼓盖了能吃吗？
- **A**: **绝对不能**。鼓盖说明内部产气（微生物活动），食物已变质，可能含致命的肉毒杆菌毒素。

- **Q**: 能保存多久？
- **A**: 酸性水果罐头 6-12 个月。开盖后必须尽快食用。

## 安全提醒

- 肉毒杆菌是罐头食品最大的威胁——其毒素 1 微克即可致死
- 低酸食物**不要**用简易水浴法做罐头
- 罐头鼓盖、漏气、异味 = 丢弃
- 不要用“吃前煮一煮”弥补错误罐藏；安全边界应在制作时保证

## 穿越小说梗

穿越后想做罐头？酸性水果罐头难度不大，但肉类罐头在没有高压锅的情况下几乎不可能安全制作。肉毒杆菌是看不见闻不到的杀手——穿越者最大的饮食风险不是饿死，而是食物中毒。
