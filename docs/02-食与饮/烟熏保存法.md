# 2.12 烟熏保存法

> 烟熏不只是风味，更是最古老的食品保存技术之一。

## 原理

烟熏保存食物依靠三重作用：
1. **脱水**：烟熏伴随加热，降低食物水分含量
2. **抑菌**：木烟中的酚类、醛类、有机酸具有抑菌作用
3. **隔氧**：烟熏在食物表面形成一层保护膜，减少氧化

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 肉/鱼 | 新鲜 | — |
| 木柴 | 硬木最佳，避免松木 | 果木（苹果木、樱桃木） |
| 熏房/熏箱 | 可密封、可排烟 | 简易棚+竹架 |
| 盐 | 先腌制 | — |

## 操作步骤

### 前处理：腌制

烟熏前**必须先腌制**，盐分是抑菌的基础：

1. 肉/鱼切条，抹盐腌制 12-24 小时（盐量约肉重的 5-8%）
2. 腌好后清水冲洗表面盐粒
3. 晾干表面水分

### 冷熏法（长期保存）

温度 20-30°C，熏制时间长，保存效果好：

1. 搭建简易熏房（小棚屋，顶部留小排烟口）
2. 在棚内搭竹架/木架，将肉条悬挂
3. 在棚外或棚底点燃木柴
4. **控制火力**：只要烟不要明火——用湿木屑或半干柴产生浓烟
5. 每天熏 6-8 小时，持续 3-7 天
6. 熏好后肉色深红/褐色，表面干燥
7. 悬挂在通风干燥处保存

### 热熏法（快速食用）

温度 50-80°C，熏制时间短，适合即食：

1. 在简易灶上方搭架悬挂食材
2. 下方点火产生热烟
3. 熏制 2-4 小时
4. 熏好的食物可以直接食用
5. **保存时间较短**（1-2 周），不如冷熏持久

### 木材选择

| 木材 | 烟熏风味 | 评价 |
|------|---------|------|
| 栎木/橡木 | 浓郁 | ★★★★★ 最佳 |
| 山核桃木 | 甜香 | ★★★★★ |
| 苹果木 | 果香 | ★★★★ |
| 樱桃木 | 果香微甜 | ★★★★ |
| 桦木 | 淡雅 | ★★★ |
| 松木 | **苦涩含树脂** | ❌ 禁用 |

## 常见问题

- **Q**: 烟熏食品能保存多久？
- **A**: 冷熏 + 干燥保存的肉可达 6-12 个月。热熏的只能保存 1-2 周。

- **Q**: 可以只熏不腌吗？
- **A**: 不建议。烟熏的抑菌作用有限，不腌制的话内部容易腐败。

## 安全提醒

- 熏房注意防火
- 烟熏食品含亚硝胺（致癌物），不宜长期大量食用
- 保存期间定期检查有无发霉或虫蛀

## 穿越小说梗

"主角把肉往火堆上一挂就不管了"——烟熏需要控制烟量和温度，明火烤出来的是烤肉不是熏肉。正确的做法是浓烟低温，连续数天。这是耐心活，不是一把火的事。
