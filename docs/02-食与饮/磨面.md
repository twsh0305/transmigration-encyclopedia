# 2.17 磨面（石磨原理与操作）

> 石磨是人类最早的"食品加工机械"——原理两千年不变。

## 原理

石磨的工作核心是**剪切力 + 碾压**：
- 上下两扇磨盘接触面刻有磨齿（沟槽）
- 上扇旋转，下扇固定
- 谷物从中心的磨眼进入，被磨齿剪切和碾压成粉
- 面粉从磨盘边缘的缝隙中落下

磨齿的走向（螺旋/放射）和深浅决定了研磨的粗细。

## 所需材料

| 材料 | 用途 | 最佳选择 |
|------|------|---------|
| 石材 | 磨盘 | 花岗岩、砂岩（硬度高、不掉砂） |
| 木料 | 支架、推杆 | 硬木 |
| 铁（可选） | 磨心轴 | 铁棒，减少摩擦 |

### 选石关键

- **花岗岩**：最佳。硬度高、耐磨、不易掉砂
- **砂岩**：可用。表面粗糙有助于磨碎，但易掉砂
- **石灰岩**：不推荐。掉粉严重，石灰粉混入面粉有害健康

## 操作步骤

### 手工推磨（人推或驴拉）

1. 谷物从磨眼倒入（每次少量，不要塞满）
2. 推动上扇旋转（顺时针）
3. 面粉从两扇之间落下，收集在磨盘下的容器中
4. 第一遍磨出的粉 + 粗粒（麸皮较粗）
5. 用细筛筛分——细粉为精面，粗粒再磨第二遍
6. 通常要重复 3-4 遍，越往后出粉越粗

### 磨盘转速

- 人推：10-15 转/分钟
- 驴/马推：15-20 转/分钟
- 水磨：20-30 转/分钟

**太快**：研磨不充分、发热过高破坏面筋
**太慢**：效率低

### 判断面粉粗细

| 筛号 | 用途 |
|------|------|
| 粗筛 | 全麦面、窝头 |
| 中筛 | 馒头、面条 |
| 细筛 | 包子、点心 |

### 水磨（水力驱动）

1. 建造水轮（卧式或立式）
2. 水轮轴连接石磨上扇
3. 水流带动水轮→磨盘旋转
4. 一个人照看即可，24 小时可磨数百斤

**水磨比人力磨节省 90% 的劳力**——是古代最重要的食品加工技术革新之一。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 开磨 | 新磨盘先用粗粮或废粮磨 2-3 遍，弃掉最初带砂的粉 |
| 进料 | 每次少量连续加料，磨眼不堵、不需要用手塞 |
| 温度 | 刚出的面粉不烫手，温度不超过 40摄氏度 |
| 粗细 | 筛分后细粉无明显砂感，粗粒回磨 2-4 遍 |
| 磨盘 | 转动声音均匀，无跳动、卡死或石屑掉落 |

## 失败征兆

- 面粉有沙味或牙碜，说明石材掉砂或开磨不足。
- 面粉发苦、发热、有焦味，说明转速太快或一次加料太多。
- 磨盘异响、跳动，可能轴心偏移，应停磨校正。
- 手伸进磨眼推料，是夹伤高风险动作，必须改用木棒。
- 磨后不清理，残粉 1-2 天内可能受潮发霉。

## 常见问题

- **Q**: 面粉里有沙子/石头粉？
- **A**: 新磨盘的前几次使用会掉砂，先用粗粮磨几次"开磨"后再磨细粮。如果石材选择不当（如石灰岩），会持续掉粉。

- **Q**: 磨出的面发苦？
- **A**: 研磨温度过高烧坏了面粉。减少一次加料量、放慢转速。理想的面粉温度不超过 40摄氏度。

- **Q**: 没有大石头怎么做磨盘？
- **A**: 可以用小块石板拼接，或者用硬木刻齿（木磨盘，但耐用性差很多）。

## 安全提醒

- 磨盘很重（数百斤），搬动时注意安全
- 磨眼进料时不要用手直接塞——用木棒
- 磨盘使用后需清洁，残留在磨齿间的面粉会发霉

## 穿越小说梗

"主角做了一个水磨，生产效率翻了几十倍"——这是真的。水磨在古代绝对是黑科技级别的技术革新。有了水磨，一个人一天能干的活相当于几十人推磨。而且不只是磨面，水碓舂米、水碓打纸浆——**水力机械**是穿越者最值得投入的技术，没有之一。
