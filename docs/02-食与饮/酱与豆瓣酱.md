# 2.26 酱与豆瓣酱

> 酱是古代中国人的"味精"——没有它，饭菜寡淡得让人怀疑人生。

## 原理

酱的酿造原理与酱油类似，都是利用**米曲霉**分解大豆蛋白质产生氨基酸（鲜味），区别在于：
- **酱油**：发酵后取液体
- **酱**：发酵后吃固体（整料）
- **豆瓣酱**：加辣椒/蚕豆再发酵

制酱的核心是**曲**——有曲则有鲜，无曲则无味。

## 所需材料

| 材料 | 大豆酱 | 豆瓣酱 |
|------|--------|--------|
| 主料 | 大豆 | 蚕豆/黄豆 |
| 辅料 | 面粉 | 辣椒、面粉 |
| 盐 | 15-20% | 10-15% |
| 曲种 | 米曲霉 | 米曲霉 |
| 水 | 适量 | 适量 |

## 操作步骤

### 大豆酱（传统黄酱/大酱）

1. **泡豆蒸豆**：大豆浸泡 8-12 小时，蒸至软烂
2. **制曲**：豆子摊凉至 35°C，拌入面粉（豆:面=10:3），接种曲种，30-35°C 培养 36-48 小时
3. **入缸**：成曲掰碎，加入 20% 盐水，搅拌成糊状
4. **日晒**：封纱布，暴晒发酵
5. **打耙**：每天搅拌 1-2 次（"打耙"），持续 2-3 个月
6. **成熟**：酱色红褐、酱香浓郁即可

### 豆瓣酱（四川风味）

1. **处理蚕豆**：去壳、浸泡、蒸熟
2. **制曲**：接种曲种，培养 2-3 天
3. **晒干**：成曲晒 1-2 天
4. **入坛**：加 15% 盐水、碎辣椒（鲜红辣椒剁碎，比例豆:椒≈1:1）
5. **发酵**：封坛、日晒 3-6 个月
6. **翻晒**：每周搅拌一次
7. **成品**：红油渗出、酱香带辣

### 快速版（豆豉）

如果不想等那么久：

1. 大豆浸泡蒸熟
2. 30°C 制曲 3-5 天
3. 加盐 15%、姜丝、陈皮
4. 密封发酵 2-4 周
5. 得豆豉——这是酱的"半成品"，但已经很好吃了

## 常见问题

- **Q**: 酱没有鲜味？
- **A**: 鲜味来自蛋白质分解为氨基酸。可能是发酵时间不够、盐浓度过高抑制了酶活性、或者曲种质量不好。

- **Q**: 酱生虫了？
- **A**: 纱布盖严。酱缸里的蛆一般是酱蝇产的卵——不影响食用（传统做法甚至认为"有虫的酱才香"），但现代人估计接受不了。过滤掉继续用。

- **Q**: 酱可以存多久？
- **A**: 高盐酱（>15%）密封避光可存 1-2 年。每次取酱用干净干爽的勺子。

## 安全提醒

- 发酵时出现黑色、绿色霉斑且散发恶臭 = 污染，全部丢弃
- 盐量不足时可能产生肉毒杆菌（尤其无氧环境）
- 容器务必清洁

## 穿越小说梗

穿越后"开个酱园"在古代绝对是一门好生意。古人吃饭离不开酱，但没有现代的曲种和卫生知识，酱园的核心竞争力就是**传了几十年的酱缸**——里面积累了无数代菌种，新开的酱园根本比不了。所以如果你带了冷冻干燥的米曲霉孢子穿越，才是真正的降维打击。
