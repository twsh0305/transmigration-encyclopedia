# 2.34 风干与晒干

> 太阳和风是免费的干燥机。

## 原理

微生物繁殖需要水分。将食物水分含量降至 15% 以下（肉）或 20% 以下（蔬果），绝大多数微生物无法生存，酶活性也大幅降低，食物可长期保存。

## 操作步骤

### 风干肉（肉干/肉脯）

1. 将瘦肉切成长条（顺着肌肉纤维方向，厚约 0.5cm）
2. 用盐和调料腌制 4-6 小时
3. 将肉条挂在通风阴凉处
4. 避免阳光直射（阳光会使脂肪氧化变味）
5. 风干 3-7 天（取决于温度和湿度）
6. 判断干燥程度：肉条弯曲不断裂但无弹性 = 干透了
7. 保存：用油纸包裹或放入密封容器

**最佳条件**：温度 10-20°C，湿度 < 60%，持续微风

### 晒干蔬菜

1. 蔬菜洗净，切成薄片或细条
2. 在沸水中焯烫 1-2 分钟（杀酶，防止变色变味）
3. 捞出沥干
4. 摊开放在竹席或干净石板上
5. 放在阳光下暴晒 2-4 天
6. 每天翻动 1-2 次
7. 完全干燥后收储（体积约为鲜品的 1/10）

### 晒干水果

1. 水果洗净去核
2. 切成薄片
3. 可浸入淡盐水或蜂蜜水（防止氧化变色）
4. 放在阳光下晒 3-5 天
5. 完全干燥后储存

### 风干鱼

1. 鱼剖开去内脏，沿脊骨展开成片
2. 用盐腌制 12 小时
3. 清水冲洗去表面盐分
4. 挂在通风处风干 3-5 天
5. 避免蝇虫（可用纱布罩住）

## 干燥后的保存

| 食物 | 保存方式 | 保存时间 |
|------|---------|---------|
| 肉干 | 密封、阴凉干燥 | 6-12个月 |
| 干菜 | 密封、防潮 | 6-12个月 |
| 干果 | 密封 | 6-12个月 |
| 干鱼 | 悬挂通风 | 3-6个月 |

**最大的敌人是潮气和虫蛀。**

## 常见问题

- **Q**: 阴雨天怎么办？
- **A**: 在火堆旁用热气烘干（但不要用烟熏——除非你想要烟熏味）。温度控制在 50-60°C。

- **Q**: 肉干发霉了还能吃吗？
- **A**: 表面少量白霉可以擦掉后重新烘干。绿霉/黑霉 = 丢弃。

## 安全提醒

- 肉类必须先用盐腌制再风干
- 风干环境要防蝇虫
- 雨天潮湿天不适合风干
- 干燥不彻底的肉会从内部腐烂
