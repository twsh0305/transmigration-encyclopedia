# 2.3 土灶搭建

> 灶台是厨房的基础，厨房是家的基础。

## 原理

土灶通过合理的烟道设计实现：
1. **聚火**：灶膛集中热量
2. **导烟**：烟道引导热气绕锅底，提高热效率
3. **拔风**：烟囱形成负压抽风，燃烧更充分

好的土灶比明火烹饪省柴 30-50%。

## 操作步骤

### 最简易土灶（三石灶）

1. 找三块大小相近的石头
2. 围成 U 形或三角形
3. 锅架在石头上
4. 从开口处添柴
5. **5 分钟建成，适合临时使用**

### 单锅土灶（标准版）

1. **挖灶膛**：在地面上挖一个直径 30cm、深 20cm 的圆坑
2. **砌灶壁**：
   - 用土坯砖或石块沿坑壁砌起
   - 灶膛直径约 25cm
3. **留灶门**：在正对操作者的方向留一个宽 15cm、高 12cm 的添柴口
4. **放置锅口**：灶顶留一个与锅底匹配的圆口
5. **砌烟囱**：
   - 在灶膛后方用土坯砌一个烟囱
   - 烟囱高 1-1.5 米
   - 内截面 12×12cm
6. **抹泥**：整个灶体内外用泥浆抹平，填补缝隙
7. **试火**：点小火检查烟道是否通畅

### 回风灶（省柴版）

在标准灶基础上增加回风道：

1. 烟气从灶膛出来后不是直接上烟囱
2. 而是绕锅底一圈再从烟囱排出
3. 热量更多传递给锅，效率更高
4. 省柴约 30-40%

## 常见问题

- **Q**: 灶冒烟不抽风怎么办？
- **A**: ①烟囱太矮，加高 ②灶门太大，缩小 ③烟道有堵塞 ④刚点火时烟囱冷，用引火物在烟囱底部加热一下

- **Q**: 土灶能用多久？
- **A**: 日常使用下土坯灶约 1-3 年。外部抹一层石灰可延长寿命。

## 安全提醒

- 灶旁不要堆放易燃物
- 烟囱穿过屋顶处必须用泥/瓦隔开
- 定期清理烟道积灰
