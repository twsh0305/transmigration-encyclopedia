# 2.9 果酒酿造（葡萄酒、梅酒）

> 水果自带糖分和酵母——酿酒是最自然的保存方式。

## 原理

水果表面天然附着酵母菌，果肉中的糖分可直接被酵母发酵为酒精。果酒是所有酒类中**最不需要额外设备**的——你只需要容器和时间。

## 所需材料

| 材料 | 要求 | 替代方案 |
|------|------|---------|
| 水果 | 成熟、新鲜 | 葡萄、梅子、枣、柿子 |
| 糖 | 辅助发酵 | 蜂蜜 |
| 容器 | 陶瓷罐/玻璃瓶 | — |

## 操作步骤

### 葡萄酒

1. **采摘**：选成熟度高的葡萄
2. **除梗破碎**：
   - 去掉葡萄梗（梗含苦涩的单宁）
   - 将葡萄粒捏破（不用去籽和皮——皮提供颜色和单宁）
3. **装缸**：
   - 破碎的葡萄连同皮籽一起装入缸中
   - 装至 3/4 满（发酵会膨胀起泡）
4. **初发酵**（5-7 天）：
   - 缸口盖纱布（防虫，允许排气）
   - 每天用干净的木棍将浮起的皮渣压入液面下 2 次
   - 温度 20-28°C
   - 3 天后开始冒泡发酵
5. **压榨**：
   - 发酵 5-7 天后，皮渣颜色变浅、液面不再冒泡
   - 用纱布过滤，分离酒液和皮渣
   - 轻压皮渣出酒（不要榨太狠——籽中的苦涩味会被榨出）
6. **后发酵**（2-4 周）：
   - 将酒液装入容器，装满
   - 加盖密封（可加气闸或水封）
   - 温度 15-20°C
   - 酵母继续将残糖发酵完
7. **换桶**：
   - 发酵结束后，酒液底部有沉淀物（酒泥）
   - 用虹吸管将上层清酒液抽到新容器
   - 不要搅起底部沉淀
8. **陈酿**：密封存放 3-6 个月

### 梅酒

1. 青梅洗净晾干，用牙签去掉果蒂
2. 一层梅一层糖码入罐中
3. 倒入烧酒（酒精度 35%+）至完全覆盖梅子
4. 密封存放 3 个月以上
5. **注意**：梅酒是用酒浸泡而非发酵，需要已有蒸馏酒

## 常见问题

- **Q**: 不加糖能酿吗？
- **A**: 葡萄糖度足够可以不加。但其他水果（如梅子）酸度高糖度低，需要加糖才能出酒。

- **Q**: 酒面上长白膜怎么办？
- **A**: 轻微的酵母膜可以捞掉。面积大、有异味 = 醋酸菌污染 = 正在变醋。

## 安全提醒

- 果酒有甲醇风险（水果中果胶在果胶酶作用下产生甲醇）
- 核果类（梅、杏、樱桃）的果核含氰苷——酿酒前务必去核
- 甲醇中毒症状：剧烈头痛、视力模糊——立即就医

## 穿越小说梗

穿越后"用野葡萄酿了一坛酒"——这确实是最简单的酿酒方式。前提是有葡萄或类似水果。不过没有现代酵母菌株，野生酵母发酵的不确定性很大，成功率可能只有 50-70%。
