# 2.21 蒸煮烤的基础工具

> 工欲善其事，必先利其器。穿越者的厨房，从这几件工具开始。

## 原理

烹饪的四种基本传热方式：
- **传导**：直接接触热源（煎、烙）
- **对流**：液体/气体传热（煮、蒸）
- **辐射**：红外线直接加热（烤）
- **压力**：密闭加压提高沸点（高压锅——穿越后极难实现）

### 无锅烹饪法

| 方法 | 原理 | 适用食材 |
|------|------|---------|
| 竹筒煮 | 竹筒装水+食材，火上加热 | 任何可煮的食物 |
| 香蕉叶包 | 食材包在叶中，埋入炭火 | 鱼、肉、根茎 |
| 泥裹烤 | 食材裹泥，放入火堆 | 鸡、鱼、根茎 |
| 石板煎 | 扁平热石上煎 | 肉、面饼 |
| 地坑烤 | 地坑中铺热石，放食材覆土 | 整只动物、根茎 |

## 基础尺寸与使用标准

| 工具 | 建议尺寸 | 成功标准 |
|------|----------|----------|
| 陶罐 | 口径 15-25厘米，壁厚 0.6-1厘米 | 小火煮 30 分钟不裂、不漏 |
| 竹筒 | 直径 6-10厘米，单节完整 | 水位高于食材 2厘米，煮沸不渗漏 |
| 石板 | 厚 3-5厘米，表面平整 | 滴水能迅速成珠滚动，食物不粘满砂粒 |
| 蒸笼 | 屉高 8-12厘米，底孔均匀 | 蒸汽能从四周冒出，食物受热一致 |

## 成功标准

- 煮：水能持续沸腾 1 分钟以上，容器不裂、不漏、不倒。
- 蒸：上汽后 5 分钟，笼内各处都有明显热蒸汽，不只中间热。
- 烤：食物外层上色但不焦黑，切开中心熟透、无血水或硬心。
- 工具：连续使用 3 次后不变形、不脱屑、不散架。

## 失败征兆

- 竹筒水位下降过快、有焦味：快烧干，立即离火补水或弃用。
- 陶罐出现细响、裂纹、渗水：受热过急或坯体有暗裂。
- 石板掉砂、炸裂、冒白烟：石材不合适或含水，不能继续用。
- 蒸笼只有底层熟、上层夹生：孔太少、盖不严或火力不足。

## 常见问题

- **Q**: 竹筒煮食物，竹子不会烧着吗？
- **A**: 竹筒内部有水的情况下，水的沸点限制温度在 100摄氏度，竹子不会超过这个温度所以不会燃烧。但水烧干后竹筒就会着火。

## 安全提醒

- 刀具使用注意安全
- 热锅热油防烫伤
- 竹筒煮时注意水位

## 穿越小说梗

穿越后"拿出铁锅开始炒菜"——你确定穿越到了有铁锅的时代？铁锅在宋代才开始普及，之前大部分人用陶罐煮食。更原始的方式：竹筒煮、石头煎、泥裹烤。
