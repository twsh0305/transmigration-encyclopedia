# 第二卷·食与饮

> 吃饱了才有力气种田、打仗、搞发明。从采集到酿造，从狩猎到烘焙，一应俱全。

## 使用顺序

1. 先解决安全饮食：净水、熟食、生熟分离和餐具清洁。
2. 再采集与捕猎：只吃能逐种确认的食材，不靠颜色、气味或动物试吃判断。
3. 接着做主食加工：脱壳、磨粉、蒸煮和烘烤先稳定，发酵再跟上。
4. 最后做保存：盐渍、干燥、烟熏、糖渍、发酵和窖藏都要小批量试验。

## 成功标准

- 净：处理食物前后洗手 20 秒，生肉、熟食、净水和污水不交叉。
- 熟：禽肉最厚处达到约 74摄氏度；肉馅约 71摄氏度；整块肉和鱼约 63摄氏度。
- 稳：同一菜式能按记录复做，水量、盐量、火力和时间误差可控。
- 存：保存食品无霉斑、黏滑、胀气、腐臭和异常苦味。

## 失败征兆

- 食材来源不明，却靠“煮熟就没事”安慰自己。
- 生肉刀板直接切熟食，或洗手水与饮用水混用。
- 发酵容器鼓胀、黑绿霉、腐臭、黏液增多仍继续食用。
- 保存食物外干内湿，切开中心仍软黏或有酸败味。

## 安全边界

- 未确认的野菜、蘑菇、浆果、贝类和腐败肉类不入口。
- 罐藏、腌制和发酵失败时不靠重煮挽救；毒素和厌氧风险不能凭味道排除。
- 酒精蒸馏、烟熏和炭火烹饪必须通风，避免火灾和一氧化碳中毒。
