# 2.28 酿酒基础原理

> 一切发酵都是微生物在替你干活——你只需要给它创造合适的环境。

## 原理

酿酒的核心是**酵母菌（Saccharomyces cerevisiae）**将糖分转化为乙醇和二氧化碳：

```
C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂↑
葡萄糖    乙醇      二氧化碳
```

**关键条件**：
- **糖源**：水果自带糖分；谷物需先糖化（淀粉→糖）
- **厌氧环境**：有氧条件下酵母优先进行有氧呼吸（产CO₂不产酒），缺氧才走酒精发酵
- **温度**：20-30摄氏度最佳
- **酵母**：可自然接种（空气中的野生酵母）或用酒曲/老面引种

## 谷物酿酒 vs 水果酿酒

| 对比 | 谷物酒 | 水果酒 |
|------|--------|--------|
| 原料 | 大米、小麦、高粱 | 葡萄、梅子、枣 |
| 糖化 | 需要糖化步骤（淀粉→糖） | 不需要（水果自带糖分） |
| 酒曲 | 必须使用 | 不必须 |
| 难度 | 较高 | 较低 |
| 代表 | 米酒、黄酒、白酒 | 葡萄酒、梅酒 |

## 酒曲的作用

谷物酿酒必须经过**糖化**步骤（将淀粉转化为糖），再由酵母将糖发酵为酒精。酒曲中同时含有：

1. **霉菌**（根霉、曲霉）：分泌淀粉酶，将淀粉分解为糖
2. **酵母**：将糖发酵为酒精

酒曲是中国酿酒的独特发明——**糖化和发酵同时进行**（双边发酵），而西方酿酒糖化和发酵是分开的（啤酒先麦芽糖化再加酵母）。

## 基础酿酒流程

```
原料 → 处理 → 糖化（谷物必须）→ 发酵 → 过滤 → 成品
```

1. **原料处理**：谷物蒸煮/水果破碎
2. **糖化**：加酒曲，保温 30摄氏度左右 1-3 天
3. **发酵**：密封容器，20-30摄氏度，7-30 天
4. **过滤**：去除酒糟，取酒液
5. **陈酿**（可选）：存放数月改善口感

## 酒精度规律

- 酵母在酒精度约 15-18% 时自行死亡
- 因此**自然发酵酒的酒精度上限约 15-18%**
- 超过这个度数需要蒸馏（见蒸馏酒条目）

## 成功标准

- 发酵开始后 24-72 小时出现气泡、酒香和轻微升温。
- 温度大多保持 20-30摄氏度，过冷发酵慢，过热杂菌多。
- 发酵后期气泡减少、液体变清、沉淀集中在底部。
- 每批记录原料重量、糖化时间、发酵温度、发酵天数和异常气味。

## 失败征兆

- 3 天无气泡、无香气：酵母不足、温度不对或原料被烫死。
- 表面黑绿霉、腐臭、臭鸡蛋味、拉丝：污染，不饮用。
- 容器完全密闭且鼓胀：二氧化碳无法排出，有爆裂风险。
- 饮后剧烈头痛、视力模糊、恶心：停止饮用，警惕甲醇或污染。

## 常见问题

- **Q**: 发酵不启动怎么办？
- **A**: 可能原因：①温度太低（15摄氏度以下）②酵母不足（加酒曲或水果皮上的野生酵母）③容器未清洁（杂菌占优势）

- **Q**: 酒变酸了怎么回事？
- **A**: 醋酸菌将酒精氧化为醋酸。原因：密封不好，空气中的醋酸菌进入。变酸的酒 = 正在变成醋。

## 安全提醒

- 自酿酒有甲醇风险（尤其果酒中果胶含量高的原料）
- 甲醇中毒可致失明甚至死亡
- 首次自酿量少试喝，如有头痛剧烈应警惕甲醇
- 发霉的酒曲不可使用

## 穿越小说梗

穿越后"随手酿了一坛美酒"——酿酒需要：原料、酒曲、合适温度、7-30 天等待、密封容器。每一步都不能省。最快的米酒也要 3-5 天才能喝，不是插根稻草就能出酒的。
