# 2.2 啤酒酿造

> 啤酒的酿造是化学工程的起源——从大麦到金色液体的精密过程。

## 原理

啤酒酿造流程：大麦发芽（产生酶）→ 烘干麦芽 → 糖化（淀粉→糖）→ 煮沸加花（杀菌+苦味）→ 冷却加酵母 → 发酵 → 成品

啤酒与米酒的核心区别：啤酒**先糖化再发酵**（分步法），米酒**糖化发酵同时进行**（双边法）。

## 所需材料

| 材料 | 用途 | 替代方案 |
|------|------|---------|
| 大麦 | 原料 | 小麦 |
| 啤酒花 | 苦味+防腐 | 其他苦味草药 |
| 酵母 | 发酵 | — |
| 水 | — | — |

## 操作步骤

### 第一步：制麦芽

1. 将大麦浸泡在水中 8-12 小时
2. 捞出摊在湿润的布上，保持湿润通风
3. 每天翻动 1-2 次
4. 3-5 天后麦粒长出约等于粒长 1-1.5 倍的芽根
5. **停止发芽**：将绿麦芽在 50-80摄氏度烘干（低温烘干保持酶活性）
6. 干燥后去根，即为麦芽

### 第二步：糖化

1. 将麦芽碾碎（不必太细）
2. 放入大锅中，加 65-68摄氏度热水
3. 保温 60-90 分钟（淀粉酶将淀粉转化为糖）
4. 碘测试：取几滴糖化液加碘——不变蓝 = 淀粉已完全糖化
5. 过滤：将甜味麦汁与麦糟分离

### 第三步：煮沸加花

1. 将麦汁煮沸 60-90 分钟
2. 煮沸开始时加入 1/3 啤酒花
3. 煮沸中期加入 1/3
4. 煮沸结束前 15 分钟加入最后 1/3
5. 啤酒花的作用：苦味平衡甜味、天然防腐、促沉淀

### 第四步：冷却

1. 将煮沸的麦汁尽快冷却至 20-25摄氏度
2. **快速冷却很重要**——减少杂菌感染风险
3. 沉淀物留在锅底

### 第五步：发酵

1. 冷却后的麦汁转入发酵容器
2. 加入酵母
3. 20-25摄氏度发酵 7-14 天
4. 密封加气闸（让 CO₂ 排出，不让空气进入）
5. 发酵结束后装瓶

### 第六步：装瓶

1. 在酒液中加少量糖（瓶中二次发酵产生碳酸）
2. 装瓶密封
3. 室温放置 1-2 周碳酸化
4. 冷藏后饮用

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 麦芽 | 3-5 天芽根约粒长 1-1.5 倍，50-80摄氏度烘干后仍有麦香 |
| 糖化 | 65-68摄氏度保温 60-90 分钟，碘测试不变蓝 |
| 煮沸 | 麦汁滚沸 60-90 分钟，泡沫和蛋白絮凝物能分离 |
| 发酵 | 20-25摄氏度 7-14 天，气泡由旺盛转弱，酒液变清 |
| 装瓶 | 二次发酵 1-2 周有适度气泡，开瓶无喷涌和酸臭 |

## 失败征兆

- 麦芽发霉或酸臭，整批弃用。
- 糖化温度超过 75摄氏度太久，酶失活，麦汁不甜。
- 冷却太慢、容器不洁，酒液酸败或长膜。
- 装瓶糖太多，瓶压过高可能爆瓶。
- 成品有醋味、腐臭、霉味，不要饮用。

## 啤酒花替代

没有啤酒花时可用：
- 艾草（苦味）
- 当归叶
- 蒲公英根
- 迷迭香

这些提供苦味和部分防腐效果，但风味不同。

## 常见问题

- **Q**: 没有大麦能酿啤酒吗？
- **A**: 任何含淀粉的谷物都可以，但大麦酶系最完善。小麦啤酒（白啤）也很常见。

- **Q**: 酿啤酒需要多长时间？
- **A**: 从制麦芽到成品约 3-5 周。

## 安全提醒

- 发酵容器注意卫生
- 瓶中二次发酵时不要加太多糖——过压可能爆瓶
- 酒精度一般 3-8%

## 穿越小说梗

穿越到古代酿啤酒？中国没有啤酒花原生种，用艾草/迷迭香替代可以做出类似物。不过中国古人更习惯米酒——谷物酿酒走的是完全不同的路线。如果你穿越到了欧洲中世纪，啤酒才是王道。
