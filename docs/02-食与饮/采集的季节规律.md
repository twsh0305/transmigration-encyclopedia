# 2.30 采集的季节规律

> 什么季节有什么吃的，这是大自然的日历。

## 原理

植物的生长和结果严格受季节（温度、日照、降水）控制。掌握季节规律，你才能知道什么时候采集什么，提前储备过冬食物。

## 春季（3-5月）

### 可采集
| 类别 | 具体食物 |
|------|---------|
| 野菜 | 荠菜、蒲公英嫩叶、蕨菜、野蒜、榆钱、香椿 |
| 树叶 | 桑叶、榆叶（嫩叶可食） |
| 鱼 | 春汛期鱼活跃，易捕 |
| 虫 | 蚕蛹、越冬后出蛰的蝗虫 |

### 关键任务
- 采集嫩芽野菜
- 播种作物
- 准备渔具捕鱼

## 夏季（6-8月）

### 可采集
| 类别 | 具体食物 |
|------|---------|
| 浆果 | 野草莓、覆盆子、桑葚、蓝莓 |
| 瓜果 | 野黄瓜、各种瓜类 |
| 坚果（早熟） | 部分榛子 |
| 菌类 | 雨后大量蘑菇（**需严格辨识**） |
| 鱼 | 最佳捕鱼季节 |
| 昆虫 | 蝗虫、蟋蟀最多 |

### 关键任务
- 大量采集浆果（部分制干/制酱保存）
- 捕鱼
- 准备食物保存设备

## 秋季（9-11月）

### 可采集
| 类别 | 具体食物 |
|------|---------|
| 坚果 | 栗子、核桃、榛子、松子、橡子 |
| 果实 | 野柿、野山楂、野葡萄、枣 |
| 根茎 | 莲藕、山药、葛根 |
| 粮食 | 稻、麦、粟（如有种植） |

### 关键任务
- **最重要的采集季节**——为冬季储备食物
- 采集坚果和根茎（最耐储存）
- 风干、烟熏、腌制保存食物
- 收割作物

## 冬季（12-2月）

### 可采集
| 类别 | 具体食物 |
|------|---------|
| 根茎 | 冬季部分根茎仍可挖出（需记住位置） |
| 坚果（残余） | 地面落叶下的坚果 |
| 猎物 | 动物觅食活跃（设陷阱） |
| 鱼 | 冰面凿洞钓鱼 |
| 树皮 | 松树内皮（应急食物） |

### 关键任务
- **消耗储备**
- 狩猎（动物在雪地上易追踪）
- 冰钓
- 减少活动节省能量

## 全年可采

| 食物 | 说明 |
|------|------|
| 蒲公英 | 全年可采，冬季也有（地下部分） |
| 车前草 | 全年可采 |
| 松针 | 富含维C，可煮茶（全年） |
| 野蒜 | 全年可采（鳞茎） |
| 鱼 | 全年可捕 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 季节计划 | 每季列出 5-10 种可确认食物，标注采集窗口和保存方法 |
| 储备量 | 秋季按“预计消耗量 × 1.5”准备，留出霉变、鼠耗和歉收余量 |
| 新食材 | 每次只新增 1 种，先少量试吃并观察 2 小时 |
| 保存 | 采后 24 小时内完成晒干、腌制、烟熏或入窖 |
| 盘点 | 每周检查一次储粮，发现霉味、虫蛀、潮湿立即分拣处理 |

## 失败征兆

- 只采当天吃的量，没有干制或腌制计划。
- 雨后见蘑菇就采，无法 100% 确认种类。
- 到冬季才开始准备陷阱、地窖和干粮。
- 储粮堆底发热、结块或有霉味，说明含水过高或通风不足。
- 采集点被反复掏空，来年产量明显下降，应轮换区域。

## 常见问题

- **Q**: 如何判断当前季节？
- **A**: 观察树木（落叶=秋，新芽=春）、日照长度（夏长冬短）、温度变化。如果穿越到了南半球，季节是反的。

## 安全提醒

- 秋季采集是生死攸关的——冬季没有新鲜食物来源
- 不要在秋天偷懒，冬天没有后悔药
- 储备量按 "计划量 × 1.5" 计算

## 穿越小说梗

穿越后"秋天到了开始准备过冬"——已经晚了。真正的过冬准备从夏末就要开始，采集、腌制、风干、挖地窖，每一步都需要时间。古人说"秋收冬藏"，藏的工作从秋天第一天就开始了。
