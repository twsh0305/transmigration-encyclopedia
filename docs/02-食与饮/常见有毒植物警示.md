# 2.7 常见有毒植物警示

> 记住毒的比记住能吃的更紧急——吃了毒草，没有医院救你。

## 原理

植物毒素是植物的防御机制，常见类型：
- **生物碱**：影响神经系统（乌头碱、毒芹碱）
- **氰苷**：释放氢氰酸（苦杏仁、木薯）
- **草酸钙**：刺激黏膜（天南星科）
- **毒蛋白**：破坏细胞（蓖麻毒素）

## 极度危险——碰都不要碰

| 植物 | 毒性 | 致死量 | 识别要点 |
|------|------|--------|---------|
| 毒芹（水毒芹） | ★★★★★ | 几克可致死 | 沼泽地，空心茎有紫色斑点，伞形白花 |
| 乌头 | ★★★★★ | 几毫克乌头碱 | 蓝紫色盔形花，块根 |
| 断肠草（钩吻） | ★★★★★ | 极少量 | 黄色漏斗花，全株有毒 |
| 蓖麻 | ★★★★ | 凾粒种子 | 大叶掌状裂，刺球果 |
| 曼陀罗 | ★★★★ | 种子几粒 | 白/紫色喇叭花，刺球果 |
| 夹竹桃 | ★★★★ | 几片叶 | 常绿灌木，粉红/白花，窄长叶 |

## 常见易混淆——最危险的情况

| 有毒植物 | 容易误认为 | 区别 |
|---------|-----------|------|
| 毒芹 | 野芹菜/水芹 | 毒芹茎有紫色斑点，有鼠尿味 |
| 断肠草 | 金银花 | 断肠草黄花无绒毛，金银花白转黄有绒毛 |
| 曼陀罗 | 秋葵 | 曼陀罗果有刺，秋葵果有棱无刺 |
| 马桑 | 桑葚 | 马桑果红色小而圆，桑葚紫黑长圆形 |
| 苍耳 | — | 幼苗剧毒，像很多可食野菜的幼苗 |
| 白屈菜 | 野菜 | 黄色汁液，味苦 |

## 毒素类型与症状

| 毒素 | 代表植物 | 症状 | 急救 |
|------|---------|------|------|
| 神经毒（生物碱） | 乌头、毒芹 | 麻木、抽搐、呼吸麻痹 | 催吐，立即送医 |
| 氰苷 | 苦杏仁、木薯 | 呼吸困难、苦杏仁味 | 催吐 |
| 草酸钙 | 天南星、芋 | 口腔灼痛、肿胀 | 牛奶或蛋清漱口 |
| 毒蛋白 | 蓖麻 | 呕吐、腹泻、器官衰竭 | 无特效解毒，送医 |

## 通用安全原则

1. **不认识的绝不吃**
2. **不生吃陌生植物**（很多毒素可被热分解）
3. **闻起来苦或刺鼻的不吃**
4. **白色汁液的不吃**（除确认安全的如蒲公英）
5. **鸟类吃的未必人能吃**（鸟类对很多毒素免疫）
6. **煮熟可减毒但不能去所有毒**

## 常见问题

- **Q**: 煮熟能去毒吗？
- **A**: 部分可以（氰苷、部分草酸钙），但生物碱和毒蛋白通常不能靠加热去除。乌头碱高温也很难完全分解。

- **Q**: 动物吃了没事，人也能吃？
- **A**: **错**。山羊能吃毒芹不死，人吃了就完了。不同物种对毒素的耐受性差异极大。

## 安全提醒

- 本条目仅为警示，不构成完整毒植清单
- 疑似中毒：立即催吐（大量饮水后刺激咽部），保留植物样本
- 在没有医院的时代，中毒 = 极大概率死亡
