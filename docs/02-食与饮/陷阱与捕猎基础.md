# 2.32 陷阱与捕猎基础

> 在没有超市的时代，蛋白质不会自己走到你碗里来。

## 原理

陷阱利用动物的行为习惯（沿固定路线行走、钻洞觅食等），通过机械装置在无人值守时捕获猎物。

陷阱的三要素：
1. **诱饵**：吸引动物
2. **触发机构**：检测动物到位
3. **捕获机构**：限制动物行动

## 所需材料

| 材料 | 用途 | 替代方案 |
|------|------|---------|
| 绳索 | 绑扎、套索 | 藤条、树皮纤维 |
| 木棍 | 框架、触发 | 竹子 |
| 石头 | 配重 | — |
| 诱饵 | 吸引猎物 | 任何食物残渣 |

## 操作步骤

### 陷阱一：弹簧套索（最实用）

捕捉中小型动物（兔子、松鼠等）：

1. 找一棵有弹性的小树，将树顶弯下
2. 在树顶绑一根绳，绳另一端打一个活结套索
3. 套索直径约 10-15cm（兔子大小）
4. 在动物必经的小路上固定套索，底部离地约 10cm
5. 制作触发器：
   - 在地上插一根带凹槽的竖桩
   - 将弯曲的树顶用一根横棍卡在凹槽中
   - 横棍上放诱饵或连接踏板
6. 动物经过套索踩到触发器→横棍脱出→弹力树弹回→套索收紧

### 陷阱二：落坑陷阱

1. 在动物必经之路挖一个坑（深 50-80cm，宽 30-40cm）
2. 坑底插削尖的木桩（如需致命）
3. 用细枝和树叶覆盖坑口（伪装）
4. 在伪装层上撒少量泥土和落叶使其与周围地面一致
5. 可在坑旁放诱饵引诱

### 陷阱三：翻板陷阱

1. 在深坑上搭一块平衡板（中间有支点）
2. 一端放诱饵，另一端配重
3. 动物踏上放诱饵的一端→平衡板翻转→动物落坑

### 陷阱四：鱼笼（捕鱼用）

1. 用竹篾/藤条编一个锥形笼子
2. 入口做成内翻的漏斗形（鱼能进不能出）
3. 笼内放诱饵（骨头、虫子）
4. 放入溪流中固定
5. 每天检查取鱼

## 陷阱布置原则

1. **设在动物通道上**：找兽径（被踩出的小路）、水源旁、食物源附近
2. **不要留人气**：设置时避免留下过多气味（戴手套、不抽烟）
3. **定期检查**：至少每天检查一次，猎物被困太久可能挣脱或被其他捕食者吃掉
4. **多设几个**：不要只靠一个陷阱，设 5-10 个增加成功概率

## 常见问题

- **Q**: 陷阱合法吗？
- **A**: 现代法律下多数地区禁用陷阱猎杀野生动物。本条目仅限极端求生场景参考。穿越到古代……法律取决于你穿越到的朝代。

- **Q**: 什么诱饵最好？
- **A**: 肉食动物用肉，杂食动物用水果/谷物。本地常见食物通常是最好的诱饵。

## 安全提醒

- 自己设的陷阱自己要记得位置，**不要误伤自己**
- 陷阱附近做好标记（只有自己认识的标记）
- 搬运猎物时注意是否还活着——受伤的动物非常危险
- 处理猎物时注意寄生虫和疾病

## 穿越小说梗

主角"设了几个陷阱，第二天就抓到了兔子"——现实中新手陷阱成功率可能只有 1-5%。需要反复调整位置和触发灵敏度。专业猎人靠的是经验和对动物行为的理解，不是运气。
