# 2.4 地窖冷藏原理

> 大地的恒温层是天然的冰箱——冬暖夏凉，四季如春。

## 原理

地下约 2-3 米深处的温度几乎全年恒定，等于当地年平均气温（中国大部分地区约 10-18°C），不受地表日夜温差和季节变化影响。

| 深度 | 温度波动 | 典型温度 |
|------|---------|---------|
| 地表 | ±30°C/年 | — |
| 1米 | ±5°C/年 | 接近年均温 |
| 2米 | ±2°C/年 | ≈年均温 |
| 3米+ | 几乎恒定 | ≈年均温 |

这个温度虽然不是现代冰箱的 4°C，但比地表的 35°C 夏天低得多，足以大幅延长食物保存时间。

## 操作步骤

### 挖掘地窖

1. **选址**：
   - 地势高、排水好（地下水位以下不能挖）
   - 土质坚实（黏土最佳，砂土易塌）
   - 靠近居住地但不太近（防鼠虫、防火）

2. **挖掘**：
   - 竖井式：挖直径 0.8-1 米的竖井，深 2.5-3 米
   - 在底部向侧面扩展储物空间
   - 侧壁如果土质松软，用砖石或木板加固

3. **通风**：
   - 用竹管或中空木棍从窖底通到地面
   - 通风管高出地面 0.5 米，顶部加防雨帽
   - 冷空气从窖口下沉，暖空气从通风管上升——自然对流

4. **防水**：
   - 窖底铺一层碎石（排水）
   - 窖壁涂石灰浆（防潮）
   - 窖口周围挖排水沟

5. **窖口**：
   - 用厚木板或石板做盖
   - 盖上再铺草和土保温
   - 加锁或标记防止他人误入

### 储存方法

| 食物 | 储存方式 | 保存时间 |
|------|---------|---------|
| 根茎类（土豆、红薯） | 筐装，离地放置 | 3-6个月 |
| 白菜 | 竖直排列，根部朝下 | 2-3个月 |
| 苹果/梨 | 用纸/草逐个包裹 | 2-4个月 |
| 腌制品 | 密封缸中 | 6-12个月 |
| 粮食 | 袋装/缸装 | 6-12个月 |

### 储存注意事项

1. 不同食物分开放——苹果释放乙烯会加速其他果蔬成熟腐烂
2. 定期检查——发现腐烂立即取出
3. 冬季寒冷地区注意防冻——温度低于 0°C 根茎类会冻伤

## 常见问题

- **Q**: 地窖会不会缺氧？
- **A**: 通风不好的深窖可能积聚 CO₂（比空气重，沉在窖底）。下窖前先点一根蜡烛——蜡烛灭了 = 缺氧 = 不能下。

- **Q**: 雨季窖内渗水怎么办？
- **A**: 在窖底挖一个集水坑，定期舀水。如果渗水严重，需要在外围挖排水沟。

## 安全提醒

- 下窖前先通风 5 分钟
- 蜡烛测试氧气——蜡烛熄灭绝不下窖
- 儿童不要单独接近窖口
- 窖口盖板必须牢固

## 穿越小说梗

穿越后没有冰箱——挖个地窖就是你的冰箱。这不是穿越者的发明，是人类用了几千年的方法。中国北方农村几乎家家有地窖，冬天存白菜土豆能吃一整个冬天。
