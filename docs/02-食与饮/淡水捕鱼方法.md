# 2.11 淡水捕鱼方法

> 有水的地方就有鱼，有鱼的地方就有活路。

## 原理

鱼类需要氧气、食物和庇护。了解鱼的行为模式（觅食时间、栖息位置）比工具本身更重要。

鱼的活动规律：
- **清晨和黄昏**：最活跃的觅食时间
- **白天**：大型鱼躲在水草/深水区
- **阴天**：比晴天更活跃（光线弱，有安全感）
- **下雨前**：气压低，鱼频繁浮头

## 方法一：鱼笼/鱼篓（最省力）

1. 用竹篾/藤条编一个锥形笼子（长 50-80厘米）
2. 入口做成内翻漏斗形——鱼能游进，游不出
3. 笼内放诱饵（虫子、骨头、米饭）
4. 放入溪流或池塘中固定
5. 过夜后第二天早上检查
6. **优点**：设好就不用管，被动捕鱼

## 方法二：手抓/摸鱼（最原始）

浅水区有效：

1. 找一块有石缝或水草的浅水区
2. 双手慢慢伸入水中，手掌张开贴底
3. 感觉到鱼后，双手合拢抓鱼腹部
4. **技巧**：从鱼尾方向接近，手指扣住鱼鳃
5. 夜间更好——用手电/火把照水面，鱼会被光吸引

## 方法三：刺鱼（需工具）

1. 削一根尖头长木棍（或绑上骨刺/铁钉）
2. 站在高处或浅水中观察
3. 看到鱼后迅速刺下
4. **注意**：水中光线折射，鱼的实际位置比看到的更深——需要向下修正

## 方法四：拦鱼堰（群体捕鱼）

1. 在溪流窄处用石头和泥筑一道弧形矮坝
2. 坝中留一个缺口
3. 在缺口处放鱼笼或网
4. 鱼沿水流游入鱼笼
5. 适合小溪，需要一定工程量

## 方法五：毒鱼（历史方法，不推荐）

某些植物汁液可使鱼暂时昏迷或死亡，但这种做法会污染水体、误伤其他水生生物，也可能让人吃到有毒残留。它在现代许多地区违法，穿越语境中也只适合作为“历史上有人这么做过”的风险知识。

**判断边界**：凡是会让整段溪流死鱼、死虾、死蛙的做法，都不是可持续捕鱼，而是破坏水源。优先使用鱼笼、鱼堰、钩钓和网捕。

## 处理渔获

1. 用手指从鱼鳃插入，折断鳃与脊椎连接处（快速杀鱼）
2. 刮鳞（从尾部向头部刮）
3. 剖腹去除内脏（保留鱼鳔、鱼肝可食）
4. 鱼肠鱼鳃丢弃（含大量寄生虫）
5. 必须煮熟——淡水鱼寄生虫风险极高

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 鱼笼 | 长 50-80厘米，入口内翻，固定后过夜不被水流冲走 |
| 放置 | 选水草边、石缝旁、缓流回水处，第二天能看到鱼虾进入或诱饵被啃 |
| 渔获处理 | 鱼鳃、内脏和苦胆去除干净，鱼肉煮到全白且容易剥离鱼骨 |
| 水域保护 | 捕鱼后水体无大片死鱼、无异味，不破坏上游饮水点 |
| 人身安全 | 下水穿鞋或裹脚，水深超过膝盖时不用手摸深洞 |

## 失败征兆

- 鱼笼入口太大或没有内翻，诱饵被吃光但没有鱼。
- 笼子未固定，夜里被水流冲走。
- 赤脚下水后划伤、皮肤瘙痒或发热，应停止涉水并处理伤口。
- 鱼肉半透明、带血水或贴骨发红，说明没有煮熟。
- 使用毒鱼后出现大片死鱼、死虾或水面异味，应停止取水和食用渔获。

## 常见问题

- **Q**: 没有鱼钩怎么钓鱼？
- **A**: 可以用荆棘/骨刺做成简易鱼钩。或者直接用鱼笼/鱼堰，不需要鱼钩。

- **Q**: 淡水鱼能生吃吗？
- **A**: **绝对不能**。淡水鱼普遍携带肝吸虫、异尖线虫等寄生虫。必须彻底煮熟。

## 安全提醒

- 淡水鱼必须煮熟
- 注意水域是否有血吸虫（中国南方水网地区）
- 捕鱼时注意水蛇
- 不要赤脚在不明水域行走

## 穿越小说梗

"主角跳进河里摸了几条鱼"——现实中摸鱼成功率很低，鱼比你的手快得多。鱼笼/鱼堰才是靠谱的方式：花半天搭个鱼堰，之后每天都有鱼吃，这才叫可持续发展。
