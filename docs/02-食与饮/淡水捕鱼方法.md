# 2.11 淡水捕鱼方法

> 有水的地方就有鱼，有鱼的地方就有活路。

## 原理

鱼类需要氧气、食物和庇护。了解鱼的行为模式（觅食时间、栖息位置）比工具本身更重要。

鱼的活动规律：
- **清晨和黄昏**：最活跃的觅食时间
- **白天**：大型鱼躲在水草/深水区
- **阴天**：比晴天更活跃（光线弱，有安全感）
- **下雨前**：气压低，鱼频繁浮头

## 方法一：鱼笼/鱼篓（最省力）

1. 用竹篾/藤条编一个锥形笼子（长 50-80cm）
2. 入口做成内翻漏斗形——鱼能游进，游不出
3. 笼内放诱饵（虫子、骨头、米饭）
4. 放入溪流或池塘中固定
5. 过夜后第二天早上检查
6. **优点**：设好就不用管，被动捕鱼

## 方法二：手抓/摸鱼（最原始）

浅水区有效：

1. 找一块有石缝或水草的浅水区
2. 双手慢慢伸入水中，手掌张开贴底
3. 感觉到鱼后，双手合拢抓鱼腹部
4. **技巧**：从鱼尾方向接近，手指扣住鱼鳃
5. 夜间更好——用手电/火把照水面，鱼会被光吸引

## 方法三：刺鱼（需工具）

1. 削一根尖头长木棍（或绑上骨刺/铁钉）
2. 站在高处或浅水中观察
3. 看到鱼后迅速刺下
4. **注意**：水中光线折射，鱼的实际位置比看到的更深——需要向下修正

## 方法四：拦鱼堰（群体捕鱼）

1. 在溪流窄处用石头和泥筑一道弧形矮坝
2. 坝中留一个缺口
3. 在缺口处放鱼笼或网
4. 鱼沿水流游入鱼笼
5. 适合小溪，需要一定工程量

## 方法五：毒鱼（应急）

某些植物汁液可使鱼暂时昏迷（非致命剂量下鱼可恢复）：

1. 将**鱼藤**（Derris）根捣碎，投入水中
2. 鱼藤酮使鱼鳃麻痹，鱼浮头
3. 捞取浮头鱼
4. **注意**：剂量过大会杀死所有水生生物，且某些植物毒素对人体有害

**此方法对水体生态破坏极大，仅在极端求生时使用。**

## 处理渔获

1. 用手指从鱼鳃插入，折断鳃与脊椎连接处（快速杀鱼）
2. 刮鳞（从尾部向头部刮）
3. 剖腹去除内脏（保留鱼鳔、鱼肝可食）
4. 鱼肠鱼鳃丢弃（含大量寄生虫）
5. 必须煮熟——淡水鱼寄生虫风险极高

## 常见问题

- **Q**: 没有鱼钩怎么钓鱼？
- **A**: 可以用荆棘/骨刺做成简易鱼钩。或者直接用鱼笼/鱼堰，不需要鱼钩。

- **Q**: 淡水鱼能生吃吗？
- **A**: **绝对不能**。淡水鱼普遍携带肝吸虫、异尖线虫等寄生虫。必须彻底煮熟。

## 安全提醒

- 淡水鱼必须煮熟
- 注意水域是否有血吸虫（中国南方水网地区）
- 捕鱼时注意水蛇
- 不要赤脚在不明水域行走

## 穿越小说梗

"主角跳进河里摸了几条鱼"——现实中摸鱼成功率很低，鱼比你的手快得多。鱼笼/鱼堰才是靠谱的方式：花半天搭个鱼堰，之后每天都有鱼吃，这才叫可持续发展。
