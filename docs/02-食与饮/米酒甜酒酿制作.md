# 2.18 米酒/甜酒酿制作

> 米酒是中国最容易自酿的酒——只要有大米和酒曲。

## 原理

糯米中的淀粉在酒曲中根霉菌分泌的淀粉酶作用下糖化为葡萄糖（甜味来源），部分葡萄糖再被酵母发酵为乙醇（酒味来源）。甜酒酿是糖化为主、发酵程度低的产物，米酒则是继续发酵后的酒液。

## 所需材料

| 材料 | 用量 | 替代方案 |
|------|------|---------|
| 糯米 | 1kg | 大米也可但口感差 |
| 酒曲 | 5-8g（约1小包） | 中药铺/网上购买 |
| 水 | 适量 | — |
| 容器 | 陶罐/玻璃罐 | — |

### 酒曲是什么

酒曲是含有霉菌（根霉、曲霉）和酵母的固体培养物。中国各地有不同类型的酒曲：
- **小曲**：米粉制成的小丸子，适合米酒
- **大曲**：麦麸制成的大块，适合白酒
- **红曲**：含红曲霉，产红色素，做红曲酒

穿越后没有商品酒曲怎么办？→ 需要自制。采集野生酵母+霉菌培养，难度极高，建议找到现有酒曲后再酿酒。

## 操作步骤

### 甜酒酿（醪糟）

1. **泡米**：糯米淘洗干净，浸泡 4-6 小时（至米粒可轻易碾碎）
2. **蒸饭**：
   - 将泡好的米沥干，放入蒸笼/纱布中
   - 大火蒸 30-40 分钟至米粒熟透但不烂
   - **标准**：米粒完整、透亮、无硬心
3. **冷却**：
   - 将蒸好的米饭摊开晾凉
   - 至 30-35°C（手摸温热不烫）
4. **拌曲**：
   - 酒曲碾成粉末
   - 边撒酒曲边翻拌米饭，使曲粉均匀
   - 可喷少量凉开水帮助曲粉附着（米饭不能太湿）
5. **装缸**：
   - 将拌好的米饭装入容器
   - 在中间挖一个坑（直达底部，便于观察出酒）
   - 盖上盖子（不要密封太紧，发酵需要少量排气）
6. **保温发酵**：
   - 维持 28-32°C
   - 冬天可用棉被包裹保温
   - **24-36 小时后**中间的坑出现清亮甜液 = 成功
7. **出酿**：48-72 小时即可食用。越久越甜转酒味越浓。

### 米酒（甜酒→老酒）

1. 按甜酒酿法做出酒酿
2. 发酵 3 天后，加入约等量凉开水
3. 继续密封发酵 7-15 天
4. 过滤取酒液 = 米酒
5. 酒精度约 8-15%

## 常见问题

- **Q**: 米饭发黑/长绿毛怎么办？
- **A**: 杂菌污染。整缸丢弃。原因是：容器不干净、米饭太湿、温度过高。

- **Q**: 为什么不甜？
- **A**: 发酵时间太长，糖分都被转化为酒精了。想吃甜的，48 小时左右取出。

- **Q**: 出酒少怎么办？
- **A**: 米饭可能太干。下次蒸饭时适当多喷些水。

## 安全提醒

- 所有器具必须干净无油
- 发酵温度不超过 35°C
- 如有异味（臭味、霉味）立即丢弃
