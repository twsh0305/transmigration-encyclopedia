# 2.10 淀粉提取（红薯淀粉、土豆淀粉）

> 淀粉是厨房的隐身主角——勾芡、挂糊、做粉条都离不开它。

## 原理

淀粉提取的本质是**水洗沉淀法**：

1. 植物细胞中的淀粉颗粒不溶于冷水
2. 将原料磨碎，淀粉颗粒释放到水中
3. 通过过滤去除纤维残渣
4. 淀粉在静置后沉淀到底部
5. 分离水分、干燥即得纯淀粉

常见淀粉来源的淀粉含量：
- 红薯：15-25%
- 土豆：15-20%
- 木薯：25-35%
- 玉米：60-70%（干玉米）
- 小麦：65-75%（面粉本来就是淀粉+面筋）

## 所需材料

| 材料 | 用途 | 最佳选择 |
|------|------|---------|
| 淀粉原料 | 主料 | 红薯、土豆、木薯 |
| 水 | 冲洗 | 大量清水 |
| 磨碎工具 | 破碎细胞 | 磨碎机、石磨、擦丝器 |
| 纱布/滤袋 | 过滤 | 细密布料 |
| 容器 | 沉淀 | 大盆/缸 |

## 操作步骤

### 红薯淀粉

1. **清洗**：红薯洗净，去泥去根
2. **磨碎**：磨成细浆（越细出粉率越高）
3. **加水搅拌**：1 份薯浆加 2-3 份水
4. **过滤**：用细纱布过滤，滤液接在盆中
5. **沉淀**：静置 4-8 小时，淀粉沉底
6. **去水**：倒掉上层黄褐色液体（含可溶性糖和杂质）
7. **水洗**：再加清水搅拌，再次沉淀（重复 2-3 次）
8. **干燥**：取出底层白色淀粉，晾晒或烘干
9. **粉碎**：干淀粉块碾碎成粉

### 土豆淀粉

步骤与红薯淀粉基本相同，但有两点区别：
- 土豆磨碎后要**立即加水**防止氧化变色
- 沉淀后上层液体中含有土豆蛋白（可作饲料）

### 判断淀粉纯度

| 状态 | 纯净淀粉 | 含杂质 |
|------|---------|-------|
| 颜色 | 纯白或微黄 | 灰暗 |
| 沉淀分层 | 明显白层 | 模糊 |
| 水洗次数 | 2-3 次 | 越多越纯 |

## 淀粉的用途

| 用途 | 原理 | 操作 |
|------|------|------|
| 勾芡 | 淀粉糊化增稠 | 冷水调开，倒入热汤中搅拌 |
| 挂糊 | 包裹食物油炸 | 淀粉+水+蛋清调糊 |
| 做粉条 | 淀粉糊化+成型 | 调糊→漏入沸水→冷却晾干 |
| 做凉粉 | 淀粉胶凝化 | 浓淀粉糊煮熟→冷却凝固 |
| 做糕点 | 降低面筋含量 | 掺入面粉 |

### 手工粉条做法

1. 红薯淀粉加少量温水调成糊
2. 冲入开水搅拌至半透明（打芡）
3. 再加入干淀粉揉成团
4. 装入漏勺（底部有孔），拍打使粉条落入沸水
5. 粉条浮起即熟，捞出过冷水
6. 挂起晾干

## 常见问题

- **Q**: 沉淀后淀粉不白？
- **A**: 水洗次数不够。杂质中的色素和糖分导致发黄。多洗几次就好了。

- **Q**: 出粉率低？
- **A**: 磨得不够细（细胞没充分破碎）、过滤不充分（淀粉随残渣流失）。

- **Q**: 淀粉生虫了？
- **A**: 没有干透。淀粉必须彻底晒干才能密封保存。含水量 < 14% 才不会发霉生虫。

## 安全提醒

- 制粉过程中产生的废水富含有机物，不要直接排入水源
- 干燥过程中防尘防虫
- 淀粉在空气中会形成粉尘，注意远离明火（粉尘爆炸）

## 穿越小说梗

穿越后用土豆/红薯制淀粉，然后做粉条——这是很靠谱的致富路径。粉条耐储存、好运输，在古代就是硬通货。而且粉条的制作技术门槛不高，只要有红薯/土豆、水、石磨和漏勺，就能批量生产。
