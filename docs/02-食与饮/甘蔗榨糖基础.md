# 2.13 甘蔗榨糖基础

> 从甘蔗到白糖，是物理和化学的双重魔法。

## 原理

甘蔗茎秆中含有约 12-18% 的蔗糖，榨糖流程：
1. **压榨**：将甘蔗压碎，挤出含糖汁液
2. **澄清**：去除杂质（蛋白质、色素、纤维碎屑）
3. **浓缩**：煮沸蒸发水分，蔗糖结晶析出
4. **分离**：用离心或重力分离糖晶体和糖蜜

蔗糖（C₁₂H₂₂O₁₁）是自然界中最常见的二糖，甜度纯正稳定。

## 所需材料

| 材料/工具 | 用途 | 替代方案 |
|-----------|------|---------|
| 甘蔗 | 原料 | 甜菜（另一种制糖原料） |
| 石碾/压榨机 | 榨汁 | 两段硬木棍挤压 |
| 大锅 | 煮糖 | 铁锅 |
| 纱布/滤布 | 过滤 | 细密布料 |
| 草木灰/石灰 | 澄清去杂 | 蛋清（传统方法） |
| 竹席 | 冷却结晶 | 平板 |
| 离心机（可选） | 分蜜 | 重力滴漏 |

## 操作步骤

### 第一步：压榨取汁

1. 甘蔗去皮，切段
2. 用石碾或木榨反复碾压，挤出汁液
3. 镇压榨汁越彻底，出糖率越高
4. 收集的汁液立即过滤，去除粗渣

**古代方法**：用两段硬木棒作杠杆挤压，效率低。畜力驱动的石碾是古代主流方案。

### 第二步：澄清去杂

甘蔗汁呈深绿色/褐色（含叶绿素、蛋白质、多酚）：

1. 甘蔗汁加热至 60-70°C
2. 加入石灰水或草木灰水（调 pH 至 7 左右）
3. 杂质凝结成絮状浮起
4. 撇去浮沫，过滤得到清澈的糖水

**原理**：杂质在加热和碱性条件下凝聚沉淀。

### 第三步：浓缩结晶

1. 澄清后的糖水倒入大锅
2. 大火煮沸浓缩
3. 水分蒸发，糖液逐渐变浓
4. 温度达到 **110-115°C** 时，糖液过饱和
5. 加入少量晶种（上次剩下的糖粉）或继续搅拌
6. 蔗糖开始结晶出来

**三代糖法**：
- **一锅**：煮到起泡浓稠，得红糖/黑糖
- **二锅**：煮得更纯，得黄糖/黄砂糖
- **三锅**：反复结晶提纯，得白糖

### 第四步：分蜜（分离糖蜜）

1. 将糖浆倒入底部有孔的容器或离心机
2. 糖蜜（深色黏稠液体）从孔中流出
3. 留在容器的浅色固体 = 粗糖
4. 粗糖加水溶解→过滤→再结晶→可得精白糖

**红糖 vs 白糖**：
- **红糖**：未分离糖蜜，含矿物质和风味物质
- **白糖**：分离糖蜜并经脱色处理，纯蔗糖

## 关于糖蜜

糖蜜是制糖的副产品，黏稠深色液体：
- 含约 50% 的糖分（无法结晶的蔗糖+还原糖）
- 用途：发酵酿酒（朗姆酒）、酱油增色、饲料
- 不要扔掉——在穿越世界，糖蜜也是宝

## 常见问题

- **Q**: 没有甘蔗怎么制糖？
- **A**: 用甜菜（含糖 15-20%），制法类似。北方地区无甘蔗，甜菜是替代品。也可以做饴糖（见"饴糖"篇）。

- **Q**: 糖水煮不稠？
- **A**: 水分太多了。继续煮，直到温度达到 110°C 以上，水分含量降到 10% 以下。

- **Q**: 为什么我的糖发苦？
- **A**: 煮过头了——蔗糖在高温下会焦化（> 160°C）。控制火候，糖液变黄褐色就马上离火。

## 安全提醒

- 煮糖温度高，防止烫伤
- 热糖浆黏性强，沾到皮肤先冲冷水再慢慢剥离
- 澄清用的石灰水不要加太多（pH > 8 会产生苦涩味）

## 穿越小说梗

"穿越古代靠制糖富可敌国"——这是历史上真实发生过的故事。唐宋时期，蔗糖技术从印度传入中国，后来成了重要出口商品。如果你穿越到蔗糖技术尚未普及的时代，一套完整的制糖工艺足以让你富甲一方。但注意：制糖需要大量燃料（煮一锅糖要烧很多柴），而且榨甘蔗需要畜力石碾——先解决能源问题。
