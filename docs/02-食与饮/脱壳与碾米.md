# 2.20 脱壳与碾米

> 一粒谷到一粒米，中间隔着一层壳和一套手艺。

## 原理

稻谷的结构由外到内：
- **颖壳（谷壳）**：最外层，坚硬，不可食
- **果皮+种皮（麸层）**：含纤维、维生素 B 族
- **胚乳**：我们吃的大米（淀粉为主）
- **胚芽**：营养最丰富的部分

脱壳 = 去颖壳，碾米 = 去麸层。前者只去壳（得到糙米），后者进一步去麸皮（得到精白米）。

## 所需工具

| 工具 | 用途 | 达到的效果 |
|------|------|-----------|
| 杵臼 | 舂捣脱壳 | 糙米（最原始） |
| 水碓 | 水力舂米 | 省力，可批量 |
| 土砻 | 旋转磨搓脱壳 | 效率最高 |
| 风车/簸箕 | 分离谷壳和米 | 去壳后必用 |
| 筛子 | 分离碎米和整米 | 分级 |

## 操作步骤

### 杵臼舂米（最原始，够一人吃）

1. 将稻谷放入石臼中，装入约 1/3 满
2. 用杵（硬木或石制）反复舂捣
3. 舂约 50-100 下，期间翻动使受力均匀
4. 倒出混合物（谷壳碎 + 糙米 + 未脱壳谷粒）
5. 用簸箕扬去谷壳碎（或风车吹去）
6. 拣出未脱壳的谷粒，放回臼中继续舂
7. 如需精米，重复舂捣进一步去麸层

**效率**：一人一小时约出 1-2 斤米（糙米）
**体力**：非常累，两手起茧是常态

### 水碓舂米（有水力资源时）

1. 筑水坝引水，建水轮
2. 水轮带动一组碓杆（通常 4-8 个）
3. 碓杆轮流抬起-落下，自动舂捣臼中的稻谷
4. 省去了人力，效率提升数倍

**条件**：需要稳定的水流。没有河流的平原地区无法使用。

### 土砻脱壳（效率最高）

1. 竹篾编成砻身，内壁填黄泥和木屑，呈螺旋纹
2. 上下两扇砻盘，上扇有进料孔
3. 转动上扇（人力或畜力），稻谷从中间进入
4. 螺旋纹的摩擦力将谷壳搓掉
5. 糙米和谷壳从边缘落下

**效率**：一人一天可碾 50-100 斤
**优势**：碎米率低，米粒完整

## 糙米 vs 精白米

| 对比 | 糙米 | 精白米 |
|------|------|--------|
| 营养 | 完整，含 B 族维生素、纤维 | 营养流失严重（只剩淀粉） |
| 保存 | 麸层含油脂，易生虫，不耐存 | 保存时间长 |
| 口感 | 较硬，需更长时间浸泡和蒸煮 | 软糯 |
| 穿越版 | 前期推荐糙米（营养全面） | 有富余劳力后再精加工 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 原粮 | 稻谷晒到手咬发硬、无霉味，含水率约 14% 以下 |
| 舂米 | 每次装臼 1/3 满，50-100 下后多数谷壳裂开 |
| 扬壳 | 谷壳被风带走，米粒落回簸箕，未脱壳谷粒能分出回舂 |
| 糙米 | 去壳率达到 80% 以上，碎米少，米粒仍带浅褐麸层 |
| 精米 | 去麸后米色变白，但不要追求全白到损失过多米粒 |

## 失败征兆

- 稻谷太湿，舂后变成碎米和糊粉，应先再晒半天到 1 天。
- 猛砸导致整米率低，说明力度过大或臼内装得太满。
- 扬壳时粉尘呛鼻、眼痒，应换上风口并遮口鼻。
- 米有霉味、黑点或结块，不能靠碾白“修好”。
- 水碓传动部位有松动、夹手点外露，应停机修整。

## 常见问题

- **Q**: 没有臼和杵怎么办？
- **A**: 任何硬石挖个坑就是臼，任何硬木棒就是杵。石器时代的工具要求——一块硬石、一根粗木棍即可。

- **Q**: 如何提高出米率？
- **A**: 谷粒先晒干（含水率 14% 以下），太湿的谷子脱壳容易碎米。舂捣时力度均匀，不要猛砸。

## 安全提醒

- 使用杵臼时注意手指安全
- 水碓的传动部件容易夹伤
- 扬壳时戴口罩，谷壳粉尘刺激呼吸道

## 穿越小说梗

穿越小说里"主角拿出一袋白米震惊乡里"——在脱壳靠杵臼的年代，精白米就是奢侈品。普通人吃的是糙米、甚至带壳煮的"谷饭"。一袋脱壳干净的白米，在生产力低下的古代，就是硬通货。
