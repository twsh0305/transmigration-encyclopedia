# 2.8 昆虫食用指南

> 当你饿到一定程度，虫子就是蛋白质棒。

## 原理

昆虫是地球上种类最多、数量最大的动物类群，许多种类富含优质蛋白（蛋白质含量 40-75% 干重）、脂肪和微量元素。

全球约 2000 种昆虫被人类食用，中国云南、广东等地至今有食虫传统。

## 不可食/危险昆虫

| 昆虫 | 原因 |
|------|------|
| 有鲜艳警告色的 | 通常有毒（警戒色） |
| 有刺毛/毒毛的 | 皮肤刺激或过敏 |
| 分泌白色/彩色液体的 | 化学防御 |
| 气味刺鼻的 | 化学防御 |
| 蜘蛛/蜈蚣/蝎子 | 有毒（部分可食但需专业处理） |
| 苍蝇/蟑螂 | 病原体携带者 |

## 通用安全原则

1. **必须煮熟**——加热可灭杀寄生虫和部分毒素
2. **去掉翅膀、腿和硬壳**——难消化且可能划伤消化道
3. **去掉头部**——某些昆虫头部含毒素腺体
4. **避免采集在粪便/腐肉附近的昆虫**——病原体风险
5. **第一次少量尝试**——可能有过敏反应

## 烹饪方法

### 炒/煎（最推荐）

1. 清洗昆虫
2. 去翅去腿
3. 热锅少油
4. 炒至酥脆（5-10 分钟）
5. 加盐调味

### 烤

1. 串在细枝上
2. 在火堆旁烤至酥脆
3. 适合蝗虫、蟋蟀等

### 煮汤

1. 洗净后加水煮沸 15 分钟
2. 捞出虫体吃肉喝汤
3. 适合幼虫类

## 常见问题

- **Q**: 生吃可以吗？
- **A**: **强烈不建议**。寄生虫、细菌、毒素都可能存在。至少烤或煮 5 分钟以上。

- **Q**: 所有蚂蚁都能吃吗？
- **A**: 大多数可以，但火蚁、子弹蚁等有剧毒刺。普通黑蚁和红蚁通常安全，酸味来自蚁酸，无毒。

- **Q**: 昆虫能当主食吗？
- **A**: 短期可以，但单一昆虫来源营养不全面。应搭配植物性食物。

## 安全提醒

- 首次食用任何昆虫，先少量试吃并等待 2 小时
- 对甲壳类过敏的人可能对昆虫也过敏
- 昆虫是食物的补充，不是唯一来源
