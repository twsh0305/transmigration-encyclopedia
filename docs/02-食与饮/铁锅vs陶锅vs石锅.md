# 2.31 铁锅 vs 陶锅 vs 石锅

> 锅决定了你能怎么做饭，也决定了你的饮食天花板。

## 三种锅对比

| 特性 | 铁锅 | 陶锅 | 石锅 |
|------|------|------|------|
| 传热性 | ★★★★★ | ★★ | ★★★ |
| 耐用性 | ★★★★ | ★★ | ★★★ |
| 制作难度 | ★★★★★ | ★★★ | ★★★ |
| 重量 | 中 | 重 | 重 |
| 可炒菜 | ✅ | ❌（易裂） | ⚠️ |
| 可煮汤 | ✅ | ✅ | ✅ |
| 需要铁器时代 | 是 | 否 | 否 |

## 铁锅

**优点**：传热快、可炒菜、耐用
**缺点**：需要冶铁技术、易锈、需要养锅
**穿越时机**：宋代（960-1279）以后铁锅才在中国民间普及

### 铁锅养锅法

新铁锅必须"开锅"：

1. 清水洗净
2. 小火烘干
3. 用肥猪肉/食用油涂抹内壁
4. 小火加热 10 分钟使油脂渗入铁的微孔
5. 冷却后擦去多余油脂
6. 重复 2-3 次
7. 每次用完清洗后涂薄油防锈

## 陶锅

**优点**：可用泥土制作、保温性好、煮汤炖菜一流
**缺点**：不耐温差（热锅加冷水会裂）、不能干烧、重且易碎

### 陶锅使用规则

1. 加水后再上火（不要空烧）
2. 不要热锅加冷水
3. 文火慢煮，不要大火猛烧
4. 煮完自然冷却，不要放在冷石板上

### 石锅

**优点**：可找天然石料、保温极好
**缺点**：极重、传热慢、选择有限

### 石锅选择

- 选择无裂纹的整块石头
- 花岗岩最佳——致密、耐热、无辐射
- 用凿子和锤子挖出凹槽
- 首次使用前在火中慢慢加热至稳定（释放内部应力）

## 常见问题

- **Q**: 没有锅怎么办？
- **A**: 竹筒、香蕉叶、泥裹——大自然提供很多替代方案。参见"蒸煮烤的基础工具"条目。

## 安全提醒

- 铁锅生锈后必须打磨除锈再使用
- 陶锅温差大时可能炸裂——注意防烫
- 石锅可能有微量重金属溶出，首次使用煮几次倒掉
