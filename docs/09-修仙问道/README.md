# 第九卷·修仙問道

> 既然穿越了，谁说不能修仙？功法、丹药、异兽、秘境——梳理修仙小说的通用设定，为你的穿越之旅加个外挂。

## 使用边界

本卷偏向世界观、小说设定和民俗想象，不把“丹药”“符箓”“尸患”等内容当作现实医疗、化学或防身教程。凡涉及进食、吸入、燃烧、放血、驱赶野兽和接触尸体的情节，都以现实安全优先。

## 使用顺序

1. 先看功法体系，确定境界、资源、寿命和战力的等级尺。
2. 再看丹药与天材地宝，只作为小说资源设定，不自行吞服未知矿物或草药。
3. 遇到异兽、僵尸、丧尸等桥段，先按现实动物、传染病和尸体卫生处理。
4. 写符箓法术时先定代价、限制和失败后果，避免万能外挂。

## 成功标准

- 设定能自洽：同一境界的速度、力量、寿命、资源消耗不前后矛盾。
- 情节能落地：主角每次突破都有材料、时间、风险和失败代价。
- 读者能判断：看到异兽或丹药条目时，知道“小说可写”和“现实不可做”的边界。
- 复用方便：每个设定能转化成角色选择、冲突或限制，而不只是名词表。

## 失败征兆

- 功法没有代价，丹药没有副作用，异兽没有生态位置：设定会变空。
- 把现实有毒矿物、未知草药、腐败尸体写成可随便试验：风险判断失真。
- 所有问题都靠更高境界碾压：读者无法判断策略成败。

## 安全边界

- 不按小说丹方服用矿物、动物组织、未知草药或自制丸散。
- 不接触、解剖、搬运腐败尸体；现实尸体和动物尸骸按传染源处理。
- 不在室内焚烧符纸、硫磺、香料或未知粉末，避免烟气中毒和火灾。
- 修行设定可以夸张，现实行动只采用医学、消防和公共卫生原则。
