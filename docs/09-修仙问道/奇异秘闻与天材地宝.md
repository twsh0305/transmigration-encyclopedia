# 9.4 奇异秘闻与天材地宝

> 修仙世界里，"奇遇"往往意味着天材地宝的出现。认识它们，才能不错过机缘。

## 使用方法

1. 先确定功能：突破、疗伤、炼器、交易、剧情诱饵，不要一件宝物包办全部。
2. 再确定代价：守护兽、禁制、采摘窗口、保存时间、服用副作用。
3. 最后确定限制：境界不够不能用、属性不合会反噬、来历不明会招祸。

## 成功标准

- 每件天材地宝都有外观、产地、成熟时间和争夺理由。
- 读者能判断主角该采、该逃、该交易还是该上交宗门。
- 资源稀有度与剧情后果匹配：越珍贵，越要有守护、代价或追杀。

## 失败征兆

- 宝物只写名字，不写生态、采摘限制和保存方式。
- “千年”“万年”随口乱用，导致世界资源泛滥。
- 把现实蘑菇、矿石、动物器官写成可随便入口的灵药。

## 安全边界

- 现实中不采食未知灵芝、蘑菇、草根、矿石粉和动物组织。
- 不为“寻宝”进入矿洞、火山口、冰裂缝、深山崖壁等危险地形。
- 小说可写奇遇，现实以植物辨识、野外安全和毒物处理原则为准。

## 天材地宝分类

### 灵药类

灵药的年份决定药效——百年起步、千年为宝、万年可遇不可求。

- **千年灵芝**——固本培元，起死回生
  - 辨识：紫芝为上品，赤芝次之，不可与毒芝混淆
  - 小说用法：直接吞服效力有限，入丹方才发挥最大效果；现实蘑菇不得凭外观采食
- **朱果**——百年一熟，增修为
  - 辨识：通体赤红，异香扑鼻，周围必有守护兽
  - 用法：成熟后三日内采摘，过时灵气消散
- **玄冰草**——极寒之地生长，可炼寒属性丹药
  - 辨识：叶片呈冰晶状，手触即冻伤
  - 采摘：需玉器或寒气护体，不可直接手摘
- **火灵芝**——火山口附近，炼体圣品
  - 辨识：通体赤红带焰纹，生长处寸草不生
  - 用法：配合寒属性药材中和，否则暴烈能量伤经脉

### 灵矿类

灵矿是炼器的基础材料，品级决定法器的上限。

- **玄铁**——比凡铁坚硬数倍，是入门级法器的主要材料
  - 来源：深埋地底千米以上，开采难度大
  - 熔炼：需高温地火，普通炭火无法熔化
- **星辰石**——蕴含星辰之力，可炼法器
  - 来源：陨石坠落处，或极高山巅受星辉千年照耀形成
  - 用法：适合炼制星空属性的法器，对推演天机有帮助
- **灵石**——修炼通用货币，分下中上极品四等
  - 下品：常见，日常修炼消耗
  - 中品：稀有，宗门月俸级别
  - 上品：罕见，可换高阶丹药
  - 极品：世间难寻，大型交易才会出现

### 异宝类

异宝多出自上古遗迹或大能传承，功能各异，价值无法用灵石衡量。

- **储物戒**——空间法宝，穿越者标配
  - 容量：小的数尺见方，大的可装一座山
  - 注意：低阶储物戒不能存放活物
- **传讯玉简**——远距离通讯
  - 范围：数十里到数万里不等
  - 限制：需要神识注入，消耗精神力
- **符箓**——封印法术的符纸，即用即弃
  - 品级：从火球符到禁咒符，价格天差地别
  - 用法：注入灵力即可触发，适合战斗时瞬发

## 上古秘闻

这些是修仙世界的历史大事件，知道它们能帮你避开不少坑。

- **封神之战**——阐教截教之争，改写天地规则
  - 起因：商周更替背后的神仙博弈
  - 后果：众神归位，天地灵气开始衰退
  - 穿越启示：封神榜上有名者身死道消，不上榜的才是自由身
- **巫妖大战**——上古巫族与妖族的灭世之战
  - 起因：争夺天地气运
  - 后果：巫族近乎灭绝、妖族退守北俱芦洲
  - 穿越启示：巫妖遗迹中宝物极多，但禁制也最凶险
- **龙凤初劫**——龙族凤族麒麟三族争霸起源
  - 起因：三族争夺霸主地位
  - 后果：三族元气大伤，人族趁势崛起
  - 穿越启示：龙族遗留的炼体秘法至今仍是最强传承之一
- **道祖鸿钧**——合道于天，传道三千
  - 地位：万道之祖，三千大道的源头
  - 传说：紫霄宫讲道，有缘者得闻
  - 穿越启示：鸿钧道祖的传承碎片散落各处，是最大的"奇遇"来源

## 穿越小说梗

"捡漏"是穿越者最爱——别人不认识的宝物，主角靠现代知识或前世记忆认出。但天材地宝通常有守护兽或有主之物，直接伸手往往要付出代价。
