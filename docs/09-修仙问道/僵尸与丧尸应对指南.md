# 9.6 僵尸与丧尸应对指南

> 穿越后遇到会蹦的、会咬的、死了还在动的……别慌，大概率是僵尸或丧尸。本篇教你分清敌我、活到最后。

## 现实边界

本篇是恐怖、玄幻和末世题材设定，不是现实暴力或尸体处置教程。现实中遇到咬伤、狂犬病风险、尸体、群体攻击或精神危机，应按急救、报警、传染病防护和撤离处理。

## 成功标准

- 小说中先定义传播方式、行动能力、弱点和感染后果，再写应对策略。
- 主角优先撤离、隔离、阻断传播和保护同伴，而不是逞强清场。
- 每次接触尸体、血液、咬伤和封闭空间都有后果和防护代价。

## 失败征兆

- 被咬、抓伤、血液污染后仍隐瞒伤情。
- 队伍为抢物资进入密闭楼房、地下室、尸体堆和火场。
- 把民俗道具写成万能护盾，现实感染、创伤和烟气风险被忽略。

## 安全边界

- 现实咬伤先冲洗、止血、就医并评估狂犬病和破伤风风险。
- 不搬运、焚烧、解剖或围观尸体；尸体处置应由专业人员执行。
- 遇到暴力威胁时优先撤离、报警、躲避和保护生命，不主动冲突。

## 僵尸 vs 丧尸——先分清楚

很多人把僵尸和丧尸混为一谈，但它们是两种完全不同的存在，应对方式也截然不同。

| 对比项 | **僵尸（中国僵尸）** | **丧尸（西方丧尸）** |
|-------|-------------------|-------------------|
| **来源** | 尸体不腐，积聚阴气所化 | 病毒/寄生虫/辐射/巫毒术 |
| **移动方式** | 跳跃前进，双臂前伸 | 蹒跚行走，后期可奔跑 |
| **意识** | 有本能残识，会追人气 | 基本无意识，只有进食本能 |
| **弱点** | 额头贴符、糯米、桃木剑 | 摧毁大脑 |
| **怕什么** | 阳光、火焰、法器等**阳气**之物 | 火焰、爆头、断脊椎 |
| **是否感染** | 被咬不会变僵尸（但会阴气入体） | 被咬**100%感染**变丧尸 |
| **是否有智慧** | 高阶僵尸有灵智 | 基本无智慧 |
| **经典形象** | 林正英电影中的清朝僵尸 | 《行尸走肉》中的活死人 |

## 第一篇：应对僵尸

### 僵尸的成因

传统说法：人死之后，咽气时若有怨气、未了心愿，或葬于**养尸地**（风水极阴之处），尸体不腐，吸收月华阴气，日久化僵。

僵尸的进化路线：
```
白僵（浑身白毛，行动慢）→ 黑僵（黑毛，力大）→ 跳僵（经典僵尸）→ 飞僵（会飞，有灵智）→ 旱魃（可引发旱灾）→ 犼（近乎神级）
```

### 僵尸的特征

- **闭气**：僵尸靠嗅觉追踪活人气息。屏住呼吸，它就找不到你
- **关节僵硬**：不能弯腰、不能曲臂——躺平装死或蹲在低矮处可能有效
- **怕糯米**：糯米撒地能让僵尸脚底打滑、行动受阻
- **怕墨线**：墨斗线（墨汁+鸡血）弹在僵尸身上如烙铁
- **怕桃木**：桃木剑、桃木钉是克制僵尸的主要法器
- **怕符箓**：额头贴上镇尸符可定住僵尸（前提是符画对了）

### 应对方法

#### 防御工事

| 方法 | 操作 | 效果 |
|------|------|------|
| **糯米防线** | 门口/窗户撒一层糯米 | 僵尸不敢踏过（民间说法） |
| **墨斗网** | 墨线拉成网格状封门窗 | 僵尸碰触会被弹回 |
| **八卦镜** | 挂于大门上方 | 反射煞气，防尸气入侵 |
| **五谷** | 门口撒黄豆、黑豆、大米等 | 僵尸踩到会滑倒（据说） |
| **桃木桩** | 房屋四角钉桃木桩 | 形成结界 |

#### 主动对抗

| 武器/手段 | 使用方法 | 效果 |
|----------|---------|------|
| **桃木剑** | 刺心口或额头 | 主武器，配合符箓效果翻倍 |
| **镇尸符** | 贴僵尸额头 | 可定住低中级僵尸 |
| **墨斗** | 弹墨线于僵尸身上 | 灼伤效果，限制行动 |
| **糯米** | 撒向僵尸或塞入口中 | 僵尸接触糯米如被火烧 |
| **火攻** | 火把/火油 | 最终手段，僵尸惧火 |
| **阳光** | 拖至日光下 | 低级僵尸遇阳光自燃 |

#### 逃生技巧

1. **屏住呼吸**：僵尸靠嗅觉追踪，发现你后马上憋气，它会失去目标
2. **不要直线跑**：僵尸跳跃速度很快，但转弯不灵活，利用障碍物绕圈
3. **往高处跑**：僵尸关节僵硬，不会爬楼梯（飞僵除外）
4. **过水**：僵尸怕水（属阴，遇水阴气更重还是怕水说法不一，但涉水确实能阻碍行动）
5. **躲米缸**：大米/糯米对僵尸有克制作用，躲进米缸是民间传说中的保命法

## 第二篇：应对丧尸

### 丧尸的成因（穿越者视角）

穿越世界中的丧尸通常有以下来源：

| 成因 | 典型场景 | 危害等级 |
|------|---------|---------|
| **病毒泄露** | 实验室事故、生化武器 | ⭐⭐⭐⭐⭐ 全球性 |
| **巫毒术失控** | 非洲/南美巫毒仪式 | ⭐⭐ 局部性 |
| **寄生虫** | 真菌/线虫感染脑部 | ⭐⭐⭐ 可控 |
| **辐射变异** | 核辐射导致死体复活 | ⭐⭐⭐⭐ 区域性 |
| **古代诅咒** | 某位邪术师的亡灵军团 | ⭐⭐⭐⭐ 可破解 |

### 丧尸的特征

- **只有进食本能**：看到活的就想咬，没有其他目的
- **没有痛觉**：打身体其他部位没用，必须摧毁大脑
- **会腐烂**：时间越长，丧尸越弱（腐烂到最后只剩骨架）
- **听觉敏感**：丧尸对声音反应强烈，嘈杂声会吸引大量丧尸
- **视觉退化**：多数丧尸视力很差，主要靠听觉和嗅觉
- **不会累**：丧尸能持续追你直到你体力耗尽

### 应对方法

#### 生存三原则

> **跑 > 躲 > 打**

1. **跑**：遭遇丧尸，第一选择永远是跑。不要恋战。
2. **躲**：跑不掉就找坚固掩体。丧尸不会开门、不会开锁（基础设定）。
3. **打**：迫不得已才战斗，且必须一击毙命。

#### 武器选择

| 武器类型 | 优缺点 | 推荐度 |
|---------|-------|-------|
| **长柄钝器**（铁锹/镐柄） | 安全距离大，不易被咬 | ⭐⭐⭐⭐⭐ 首选 |
| **刀**（砍刀/长刀） | 一刀断头效率高，但需要技术和体力 | ⭐⭐⭐⭐ |
| **弓箭/弩** | 远程无声，但弹药有限 | ⭐⭐⭐ |
| **火器**（枪/铳） | 远程击杀，但声音会引来更多丧尸 | ⭐⭐ 谨慎使用 |
| **短刃**（匕首） | 近身肉搏用——不到万不得已不要用 | ⭐ 拼命用 |

**不推荐**：斧头（容易卡住）、剑（需要技巧，不如刀实用）、徒手（绝对不要）

#### 防御工事

| 方案 | 材料 | 效果 |
|------|------|------|
| **围墙** | 砖石+木板+铁丝 | ⭐⭐⭐⭐⭐ 终极防御 |
| **壕沟** | 挖深沟，插尖桩 | ⭐⭐⭐⭐ 阻止大量丧尸 |
| **栅栏** | 粗木桩围成一圈 | ⭐⭐⭐ 临时防御 |
| **门窗加固** | 木板钉死门窗 | ⭐⭐⭐ 基础防护 |
| **楼梯拆除** | 拆掉一楼的楼梯 | ⭐⭐⭐⭐ 二层以上安全 |

#### 丧尸爆发生存策略

**第一阶段：爆发初期（黄金24小时）**

1. **确认情况**：是局部事件还是全面爆发
2. **收集物资**：食物、水、药品、武器、火种
3. **找避难所**：坚固+易守难攻+有水源（警察局、军营、医院是首选，但人多风险也大）
4. **通知同伴**：集结可信的人

**第二阶段：爆发中期（第1-4周）**

1. **建立水源**：收集雨水、找井水、附近河流
2. **食物计划**：收集+保存+种植（食用罐头、干货优先）
3. **建立警戒**：24小时轮流值班
4. **噪音管理**：严格控制声音，做饭用暗火

**第三阶段：长期生存（1个月以上）**

1. **自给自足**：开辟菜地、养殖家禽
2. **收集情报**：是否有其他幸存者、是否有救援
3. **建立社区**：人多力量大，但人多了也容易出问题

#### 医疗急救

| 情况 | 处理 |
|------|------|
| **被咬** | 如果是病毒型丧尸，**100%会感染**。截肢可能争取几分钟，但基本无解。 |
| **被咬（非病毒型）** | 按普通伤口处理：清洗+消毒+包扎 |
| **伤口污染** | 用丧尸的血/组织污染伤口，按被咬同样处理 |
| **精神创伤** | 同伴变成丧尸是最常见的精神打击——相互支持，不要独处 |

## 常见问题

- **Q**: 僵尸和丧尸哪个更危险？
- **A**: 看情况。僵尸个体战力更强（铜皮铁骨），但不会**扩散感染**；丧尸个体很弱，但**数量多+会感染**，总体威胁更大。一个城市爆发丧尸，几周内全城沦陷；僵尸一般单个出现，不会形成"尸潮"。

- **Q**: 穿越到古代遇到丧尸怎么办？
- **A**: 好消息：古代人口密度低，丧尸传播慢。坏消息：古代没有现代武器、没有抗生素。穿越者策略：①利用城池围墙防御②火攻最有效③用道士/和尚的身份获得话语权，推动隔离和清理

- **Q**: 有没有什么"万金油"对付这两者？
- **A**: **火**——僵尸怕火、丧尸也怕火。火攻是两者通用的克制手段。其次是**爆头/斩首**，对两者都有效（僵尸砍头+火烧以绝后患）。

## 穿越小说梗

"穿越者丧尸末世求生"和"穿越者茅山道士捉僵尸"是两类完全不同的题材，但有一个共同点：

**知识就是战斗力**。

在丧尸世界，穿越者知道丧尸的弱点、知道怎么建立避难所、知道不要被声音吸引——这些知识本身就是最大的金手指，比拳头有用得多。

在僵尸世界，穿越者如果恰好看过林正英的电影，知道"屏住呼吸""糯米克制僵尸""桃木剑驱邪"这些知识点，那简直就是半个茅山传人。

**最关键的忠告**：无论遇到哪种，**不要被咬是第一原则**。保护好自己和同伴，比消灭所有敌人更重要。
