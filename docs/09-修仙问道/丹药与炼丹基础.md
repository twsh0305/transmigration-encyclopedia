# 9.2 丹药与炼丹基础

> 炼丹不是瞎煮锅。火候、丹方、药材，差一分则丹毁人伤。

## 现实边界

本篇是修仙小说设定资料，不是现实制药、炼丹或草药内服教程。现实中的矿物药、未知草药、动物组织和高温密闭加热都可能中毒、爆裂或违法。

## 成功标准

- 小说中每种丹药都有适用境界、资源成本、服用限制和失败代价。
- 炼丹过程能写出温度、火候、投料顺序和等待时间，但不提供现实可复刻的危险配方。
- 成丹率不是固定外挂：炉、火、药材、神识和环境至少影响其中两项。
- 丹药不是万能药，突破、疗伤、延寿和洗灵根各有边界。

## 失败征兆

- 主角随手混药就成丹，没有报废、反噬、毒性和资源消耗。
- 把朱砂、雄黄、水银、铅粉等现实有毒矿物写成可安全吞服。
- 丹药效果无限叠加，导致境界体系失去张力。

## 安全边界

- 不服用自制丹丸、矿物粉、未知草药酒和来历不明的“秘方”。
- 不密闭加热未知粉末、油脂、酒精和矿物，不吸入炼制烟气。
- 现实药物和草药处理应回到医学资料，不用修仙设定指导身体实践。

## 炼丹基础条件

炼丹不是有了丹炉扔药材进去就行，缺一不可：

### 丹炉

| 类型 | 级别 | 特点 |
|------|------|------|
| 铁炉 | 入门 | 炼制基础丹药尚可，导热不均 |
| 铜炉 | 中阶 | 导热均匀，适合中阶丹药 |
| 紫砂炉 | 高阶 | 能锁住药性不流失 |
| 灵石炉 | 顶阶 | 自带聚灵阵，大幅提升成丹率 |

### 火焰

| 火焰类型 | 温度特性 | 适用 |
|---------|---------|------|
| 凡火 | 温度不稳，难控 | 仅能处理基础药材 |
| 地火 | 稳定持久 | 中高阶丹药必备 |
| 异火/三昧真火 | 可熔炼天材地宝 | 极品丹药，极难获取 |

### 丹方

- 药材配比、投放顺序、凝丹时机，错一步全炉报废
- 注意：丹方中的"文火三刻"指的是具体时间，不能凭感觉
- 丹方品级：凡方 → 灵方（一至九品）→ 天方 → 仙方 → 神方

### 神识

- 感知炉内变化，精神力越强控火越精
- 神识不足时最好找低阶丹药练手，别浪费好药材

## 常见丹药分类

丹药按等级分为：灵丹（一至九品）→ 天丹 → 仙丹 → 神丹
品质从低到高：下品 → 中品 → 上品 → 极品

### 各境界常用丹药

| 丹药 | 对应境界 | 功效 | 稀有度 |
|------|---------|------|--------|
| 辟谷丹 | 练气-筑基 | 短时间不必进食 | 常见 |
| 黄龙丹/金髓丸 | 练气期 | 增进修为 | 常见 |
| **筑基丹** | 练气→筑基 | **突破筑基瓶颈，极其珍贵** | 稀有 |
| 合气丹 | 筑基期 | 筑基修士修炼灵药 | 较稀有 |
| 培元丹 | 筑基-金丹 | 固本培元，稳固修为 | 一般 |
| **结金丹** | 筑基→金丹 | **辅助结丹** | 极稀有 |
| 昊元丹 | 金丹期 | 突破瓶颈、增进修为 | 稀有 |
| 定灵丹 | 金丹-元婴 | 定心安魂，减轻心魔 | 较稀有 |
| **结婴丹** | 金丹→元婴 | **辅助凝结元婴** | 极稀有 |
| **化神丹** | 元婴→化神 | **辅助突破化神** | 传说级 |
| 延寿丹 | 任何境界 | 增加寿元 | 极稀有 |
| **补天丹** | 金丹以上 | **洗炼灵根，提升纯度** | 传说级 |

## 炼丹步骤

1. 预热丹炉
2. 药材提纯（去除杂质）
3. 按顺序投入药液
4. 文火武火交替
5. 凝丹成形
6. 开炉取丹

## 穿越小说梗

穿越者常利用"现代化学知识"改良炼丹——比如精确温控、提纯工艺。但要记住：炼丹的本质是灵气反应，不是化学反应。
