# 5.21 简易炼铁炉

> 从石头到铁水，需要的只是一堆炭、一块矿石和一些巧思。

## 原理

炼铁的本质是还原反应：用一氧化碳（CO）将氧化铁中的氧夺走。

```
Fe₂O₃ + 3CO → 2Fe + 3CO₂↑
```

**条件**：
1. 高温（1100摄氏度以上）——让矿石熔化
2. 还原气氛（缺氧）——防止铁再被氧化
3. 熔剂（石灰石）——去除杂质（脉石），形成炉渣

## 所需材料

| 材料 | 用途 |
|------|------|
| 铁矿石 | 原料（粉碎至核桃大小） |
| 木炭 | 燃料+还原剂 |
| 黏土/耐火土 | 建炉 |
| 石灰石（CaCO₃） | 熔剂——去杂质 |
| 风箱 | 鼓风 |
| 陶管/铁管 | 鼓风管（风口） |

## 操作步骤

### 一、块炼炉（最古老的炼铁炉）

**适用**：穿越初期炼铁。温度较低（1100-1200摄氏度），铁不熔化，生成海绵铁。

**建造**：
1. 用黏土+沙砌一个圆筒形炉（高 60-80厘米，内径 20-30厘米）
2. 底部留一个炉门（出渣取铁）
3. 一侧开风口（离底部 5-10厘米，插入陶管接风箱）
4. 炉壁厚 5-8厘米
5. 自然晾干 3-5 天

**操作**：
1. 底部铺一层木炭
2. 一层木炭一层矿石交替装入（比例约 5:1）
3. 加少量石灰石（矿石量的 10-15%）
4. 点燃木炭，开始鼓风
5. 连续鼓风 4-8 小时，不断加炭和矿石
6. 结束时打开炉门，取出海绵铁块
7. 海绵铁反复加热锻打→排除杂质→铁锭

### 二、竖炉（更高温度）

1. 砌高 1-1.5米的炉子（内径 30-40厘米）
2. 底部较大，向上逐渐收窄
3. 炉底设出铁口和出渣口
4. 鼓风更强劲

## 温度控制

| 炉温 | 现象 | 产物 |
|------|------|------|
| 800-1000摄氏度 | 矿石变红但不熔化 | 无法冶铁 |
| 1100-1200摄氏度 | 海绵铁形成（不熔） | 块炼铁 |
| 1200-1300摄氏度 | 铁开始部分熔化 | 生铁（含碳高） |
| 1300-1500摄氏度 | 铁水完全熔出 | 生铁水 |
| 1500摄氏度以上 | 钢的熔化 | 炼钢 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 炉体 | 高 60-80厘米、内径 20-30厘米、炉壁厚 5-8厘米，晾干 3-5 天 |
| 原料 | 矿石碎到核桃大小，木炭干燥，石灰石为矿石量 10-15% |
| 鼓风 | 连续 4-8 小时，炉内保持亮橙到黄白火色，火焰不频繁倒灌 |
| 产物 | 取出海绵铁块，红热锻打能挤出炉渣并逐渐结成铁坯 |
| 记录 | 每炉记录矿石、木炭、石灰石、鼓风时间和产物重量 |

## 失败征兆

- 炉料只是烧红，没有海绵铁，说明温度或还原气氛不足。
- 炉渣黏死炉底，可能熔剂不足或温度分布不均。
- 铁块一锻就碎成渣，说明还原不足或夹渣太多。
- 烟气倒灌、操作者头晕恶心，是一氧化碳风险。
- 炉壁出现贯穿裂缝或风口烧塌，应停炉冷却后修补。

## 常见问题

- **Q**: 炉子烧不到炼铁温度怎么办？
- **A**: ①鼓风不够强（升级风箱）②木炭质量不好③炉体散热。改进：用双风箱交替鼓风、使用优质硬木炭。

- **Q**: 海绵铁取出来之后怎么处理？
- **A**: 海绵铁是铁+炉渣的混合物。加热到红热状态，反复锻打（"挤"出炉渣）。锻打次数越多，铁越纯。

- **Q**: 第一次炼铁成功概率？
- **A**: 穿越后 **前 3-5 次** 大概率失败。这是正常的——古人也是反复试验。每次记录失败原因：风量？炭质？矿石粒度？

## 安全提醒

- 炼铁炉温度极高（1100摄氏度以上），保持安全距离
- 炉渣和铁水可能喷溅，穿戴皮围裙和护目镜
- 一氧化碳中毒风险——炼铁炉必须在室外通风处
- 建炉地点远离易燃物

## 穿越小说梗

"主角三天就炼出了铁"——现实中穿越后从建炉到出第一块铁需要**数周**。第一次大概率失败，第二三次可能只出一块小铁疙瘩。穿越者的知识优势：①知道要加石灰石做熔剂②知道鼓风是关键③知道海绵铁需要反复锻打提纯。如果能造出**水力鼓风**（水排），产能可以提升一个量级。
