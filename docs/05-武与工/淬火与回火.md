# 5.13 淬火与回火

> 同样一块钢，淬与不淬是天壤之别——淬得好是宝刀，淬不好是一块废铁。

## 原理

淬火的核心是**相变**：

1. 钢加热到 750-900°C（奥氏体化）→ 铁晶格结构变为面心立方（奥氏体）
2. 快速冷却 → 碳来不及扩散出去 → 形成**马氏体**（体心四方、极硬）
3. 未淬火 → 缓慢冷却 → 形成珠光体/铁素体（较软）

### 钢的热处理三步骤

| 步骤 | 目的 | 操作 |
|------|------|------|
| 退火 | 软化工件 | 加热→缓冷 |
| 淬火 | 提高硬度 | 加热→快冷 |
| 回火 | 降低脆性 | 再加热→中低温 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 炭炉 | 加热 |
| 水槽（淬火液） | 冷却 |
| 铁钳 | 夹持 |
| 铁锤+铁砧 | 校正 |
| 不同淬火液（可选） | 不同冷却速度 |

### 淬火介质的冷却速度

| 介质 | 冷却速度 | 硬度 | 风险 |
|------|---------|------|------|
| 盐水 | 极快 | 极高 | 易开裂 |
| 清水 | 快 | 高 | 可能开裂 |
| 动物油 | 中 | 中高 | 安全 |
| 植物油 | 中 | 中 | 安全 |
| 熔铅/盐浴 | 可控 | 可调 | 专业 |

## 操作步骤

### 一、淬火（使钢变硬）

1. **加热**：刀剑/工具加热到**樱桃红色**（约 780-850°C）
   - 用磁铁测试：钢在 768°C 以上失去磁性→合适的淬火温度
2. **均热**：保持温度几分钟（每 mm 厚度约 1 分钟）
3. **淬入**：快速、垂直地浸入淬火液中
   - 刀剑：刃口先入，垂直，上下小幅度移动（防止蒸汽膜）
   - 整体淬入：动作果断，不要犹豫
4. **冷却**：完全冷却（约 10-30 秒）
5. **取出**：钢变得极硬（锉刀打滑）

### 二、回火（消除脆性）

淬火后的钢硬如玻璃——一摔就断。必须回火：

1. **低温回火（200-300°C）**：
   - 刀剑类：回火 1-2 小时
   - 颜色变化：浅黄（200°C）→ 黄（230°C）→ 紫（270°C）→ 蓝（300°C）
   - 刀剑一般回火到**紫色或蓝色**（对应 270-300°C）

2. **回火方法**：
   - **炉中回火**：在炉中加热至所需温度，保温 1-2 小时，缓冷
   - **火焰回火**：用炭火在刀背处加热，观察刃口颜色变化（传统方法）

### 三、局部淬火（刀剑刃硬背韧）

1. 在刀背涂抹一层黏土泥浆（隔热）
2. 加热至淬火温度
3. 淬入水中：刃口无泥→直接接触水→极硬
4. 刀背有泥隔热→冷却慢→韧而不硬
5. **效果**：刃口坚硬可削铁、刀背柔韧不断裂

## 常见问题

- **Q**: 淬火后钢裂了怎么办？
- **A**: ①加热温度太高（过烧）②冷却太快（用油代替水）③钢本身品质差（含硫高、杂质多）。避免：降低加热温度、用油淬。

- **Q**: 没有温度计怎么判断温度？
- **A**: 用钢材加热的颜色判断：
  - 暗红色 ≈ 650°C
  - 樱桃红 ≈ 750°C
  - 亮樱桃红 ≈ 800-850°C（淬火温度）
  - 亮红带白 ≈ 900°C（过高）

- **Q**: 淬火后钢太硬但一敲就断？
- **A**: 回火不够。回火温度升高，硬度略降但韧性大增。

## 安全提醒

- 红热钢淬入水中会产生蒸汽爆炸（蒸汽瞬间膨胀）——脸部保持距离
- 淬火液槽边不要放置易燃物
- 淬火后的钢非常脆，轻拿轻放
- 淬火油使用多次后会变稀（闪点降低），注意防火

## 穿越小说梗

"主角淬出了绝世好剑"——淬火是技术活中的技术活。穿越者的优势：①知道"马氏体"是什么（古人只知其然）②知道回火的科学原理③知道局部淬火（粘土法）让刃硬背韧。但最重要的是：**每块钢的含碳量不同，淬火温度也不同**。一个详细的穿越桥段：主角第一次淬火把剑淬断了→意识到温度不对→用磁铁测试→成功了。
