# 5.1 刀剑基础锻造

> 一把好刀，不是打出来的，是折叠出来的。

## 原理

刀剑锻造核心是控制含碳量和晶粒结构。纯铁太软、生铁太脆——刀剑需要含碳0.4%~1.2%的钢。折叠锻打将不同含碳量铁料分层复合，排除杂质，形成均匀细密的晶体结构。

## 所需材料

| 材料 | 说明 |
|------|------|
| 铁料/钢料 | 高碳钢（刃口）+低碳钢（刀身） |
| 木炭 | 加热燃料 |
| 水/油 | 淬火介质 |
| 砂石/磨石 | 研磨开刃 |
| 硬木 | 刀柄材料 |

## 操作步骤

1. 确定用途（厨刀/猎刀/战刀），画出1:1纸样
2. 铁料加热到 900 到 1000 度，锻打出刀形；夹钢法将高碳钢条嵌入刀身锻接
3. 加热至 760 到 800 度浸入淬火介质，再加热至 200 到 300 度回火降低脆性
4. 粗磨（80 目）到中磨（400 目）到精磨（1000 目），开刃角度 15 到 30 度
5. 刀茎插入木柄用胶加铆钉固定

## 常见问题

Q: 没有钢料怎么做刀？A: 铁加热后放入炭粉保温做渗碳处理。

Q: 水淬和油淬区别？A: 水淬硬度更高但易开裂；油淬韧性更好。

## 工艺验收

| 项目 | 成功标准 | 失败征兆 |
|------|----------|----------|
| 外形 | 刀身直，刃线连续 | 扭曲、局部过薄 |
| 热处理 | 刃口能咬住软铁或木料 | 一磨就卷刃 |
| 韧性 | 轻度侧压能回弹 | 崩口、裂纹 |
| 装柄 | 摇晃不松 | 铆钉松、柄裂 |

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 淬后开裂 | 温度过高、冷却太猛 | 降低加热，改油淬 |
| 刃口卷 | 含碳低或回火过度 | 重新热处理或换钢 |
| 刃口崩 | 太硬太脆 | 提高回火温度或延长回火 |
| 刀身弯 | 加热不匀或锻打偏 | 回火后校直 |

## 安全提醒

锻打时铁屑飞溅——必须戴护目镜。淬火时水/油可能沸腾溅射。

## 安全边界

- 本条优先用于工具刀、厨刀、农具刃具的工艺理解，不用于违法武器制造。
- 淬火油远离明火，油面起火用盖子或沙土处理，不泼水。
- 新刀只做材料和工具验收，不做对人或动物的伤害测试。

## 穿越小说梗

穿越者的真正优势不是手巧，而是知道夹钢法和淬火回火温度判断（用颜色判断温度）。
