# 5.1 刀剑基础锻造

> 一把好刀，不是打出来的，是折叠出来的。

## 原理

刀剑锻造核心是控制含碳量和晶粒结构。纯铁太软、生铁太脆——刀剑需要含碳0.4%~1.2%的钢。折叠锻打将不同含碳量铁料分层复合，排除杂质，形成均匀细密的晶体结构。

## 所需材料

| 材料 | 说明 |
|------|------|
| 铁料/钢料 | 高碳钢（刃口）+低碳钢（刀身） |
| 木炭 | 加热燃料 |
| 水/油 | 淬火介质 |
| 砂石/磨石 | 研磨开刃 |
| 硬木 | 刀柄材料 |

## 操作步骤

1. 确定用途（厨刀/猎刀/战刀），画出1:1纸样
2. 铁料加热到900~1000度，锻打出刀形；夹钢法将高碳钢条嵌入刀身锻接
3. 加热至760~800度浸入淬火介质，再加热至200~300度回火降低脆性
4. 粗磨（80目）到中磨（400目）到精磨（1000目），开刃角度15~30度
5. 刀茎插入木柄用胶加铆钉固定

## 常见问题

Q: 没有钢料怎么做刀？A: 铁加热后放入炭粉保温做渗碳处理。

Q: 水淬和油淬区别？A: 水淬硬度更高但易开裂；油淬韧性更好。

## 安全提醒

锻打时铁屑飞溅——必须戴护目镜。淬火时水/油可能沸腾溅射。

## 穿越小说梗

穿越者的真正优势不是手巧，而是知道夹钢法和淬火回火温度判断（用颜色判断温度）。
