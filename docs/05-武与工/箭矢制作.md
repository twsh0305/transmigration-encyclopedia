# 5.24 箭矢制作

> 好弓配好箭——十支箭里能飞直线的不过三支。

## 原理

箭矢飞行稳定性取决于三点：重心位置（偏前于箭杆中点）、尾羽的空气动力稳定力矩、箭杆直线度。

## 所需材料

| 材料 | 说明 |
|------|------|
| 箭杆（桦木/白杨/竹/芦苇） | 桦木最直，竹最耐弯，芦苇最轻 |
| 箭头（铁/骨/燧石） | 铁最佳，骨和燧石为替代 |
| 箭羽（鹰羽/鹅羽/鸡羽） | 鹰羽最硬挺，鹅羽最常见 |
| 动物胶/筋线 | 粘合与捆扎 |

## 操作步骤

1. 选直径8~10mm直枝，刮去树皮，火烤校正直度
2. 铁条加热打薄成柳叶形（长5~8cm），尾部留銎管插入箭杆
3. 每支箭用3片羽毛，均匀黏在箭杆后端120度分布，旋向一致使箭旋转稳定
4. 重心在箭杆前1/3到2/5处为理想
5. 箭杆后端削出V形弦槽，贴骨片防磨损

## 常见问题

Q: 没有鹰羽用什么？A: 鹅羽、鸡羽，甚至木片或皮革片都可。

Q: 竹箭杆需处理吗？A: 小火烘烤去水分，竹节处加热压直。

## 穿越小说梗

新手做10支箭能有3~4支合格就不错了。穿越者的优势是知道标准化——每批箭矢规格保持一致。
