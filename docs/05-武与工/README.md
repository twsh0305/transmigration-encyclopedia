# 第五卷·武与工

> 冷兵器时代的材料、机械与组织能力。从打一把刀到造一台投石机，从百炼钢到火工风险边界。

## 使用顺序

1. 先读安全规范和高温作业，确认场地、人员、通风、灭火和退出路线。
2. 再学材料与工具：矿石、木炭、炉、砧、钳、量具先稳定，成品才有重复性。
3. 工程机械先做小比例模型，确认受力、磨损和失效方式，再考虑实物。
4. 武器、火药和火器条目只作历史、安全和小说可信度参考，不作为现实攻击或试放教程。
