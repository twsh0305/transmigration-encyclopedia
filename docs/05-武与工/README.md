# 第五卷·武与工

> 冷兵器时代的材料、机械与组织能力。从打一把刀到造一台投石机，从百炼钢到火工风险边界。

## 使用顺序

1. 先读安全规范和高温作业，确认场地、人员、通风、灭火和退出路线。
2. 再学材料与工具：矿石、木炭、炉、砧、钳、量具先稳定，成品才有重复性。
3. 工程机械先做小比例模型，确认受力、磨损和失效方式，再考虑实物。
4. 武器、火药和火器条目只作历史、安全和小说可信度参考，不作为现实攻击或试放教程。

## 成功标准

- 冶金锻造：能稳定控制颜色、火候、淬火介质和回火时间，并记录每炉结果。
- 木工机械：空载运行 30 分钟无裂纹、异响、偏摆和连接松动。
- 防具盾具：先做钝器、低能量、无人穿戴测试，确认结构不散裂。
- 危险工艺：能说清隔离距离、停工条件、事故信号和撤离路线。

## 失败征兆

- 先追求威力、速度和产量，却没有夹具、护具、灭火和旁站观察。
- 结构靠“差不多”拼接，受力点没有余量，第一次加载就变形。
- 粉末、燃料、热金属、儿童和围观者出现在同一工作区。
- 试验记录只写成功品，不记录裂纹、炸裂、过热和报废原因。

## 安全边界

- 不在居住区、粮仓、柴草堆、牲畜棚和人群附近做高温、粉尘、撞击或爆燃相关实验。
- 不制造、改装、试射现实武器或火器；涉及法律风险的内容以当地法规为准。
- 不用手接近拉满的弓弩、悬吊重物、旋转齿轮、热金属和未知粉末。
- 一旦出现冒烟、异味、异常发热、裂纹扩展、绳索断股，立即停工撤人。
