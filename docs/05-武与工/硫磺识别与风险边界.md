# 5.20 硫磺识别与风险边界

> 硫磺是矿物、药材和火工史里常见的黄色物质；识别它首先是为了避免中毒、火灾和误用。

## 原理

硫磺易燃，燃烧会产生刺激性二氧化硫；天然硫磺常见于火山、温泉、硫磺矿区，也可能和硫化氢、二氧化硫、酸性水、塌陷、烫伤等风险同场出现。历史上它被用于药材、杀虫和火工行业，本条只用于矿物识别、风险管理和小说可信度判断。

### 历史来源与风险

| 来源 | 识别线索 | 主要风险 | 安全边界 |
|------|----------|----------|----------|
| 火山地区黄色沉积物 | 鲜黄、结晶或粉末状 | 二氧化硫、硫化氢、塌陷、烫伤 | 只观察和标记，闻到刺鼻味或头晕立刻撤离 |
| 温泉沉积物 | 温泉边缘黄色泉华 | 高温水、酸性水、缺氧或有毒气体 | 不进入低洼、密闭或无通风区域 |
| 硫铁矿矿石 | 黄铜色矿物、金属光泽 | 误认、粉尘、焙烧有毒气体 | 只作矿物识别，不焙烧、不密闭加热 |
| 药材铺石硫黄 | 有标签的药材或矿物 | 误服、误混、儿童接触 | 只作标识和封存，不自行内服 |
| 历史提纯概念 | 古籍提到升华或冷凝 | 刺激性烟气、火工前体风险 | 不提供装置、温度、收集和提纯流程 |

## 识别与隔离用品

| 材料 | 用途 |
|------|------|
| 纸包或小陶罐 | 临时封存疑似样品，外侧写明来源和风险 |
| 标签和炭笔 | 写清地点、日期、外观、气味风险和“不得加热” |
| 手套或夹具 | 避免粉尘和皮肤长期接触 |
| 清水 | 接触后洗手、冲洗皮肤刺激处 |
| 隔离盒 | 与食物、药材、火源、氧化剂和儿童用品分开放置 |

## 历史获取逻辑（不作为操作步骤）

### 一、野外出现环境

**常见地点**：
1. 火山地区——火山口、熔岩附近常有黄色硫磺结晶
2. 温泉区——含硫温泉的泉华沉积物
3. 硫磺矿——黄色或绿色矿脉

**风险判断**：火山、温泉和硫磺矿区可能出现黄色沉积物，但这些地点也可能有二氧化硫、硫化氢、塌陷、烫伤和酸性水风险。闻到刺鼻气味、头晕、咳嗽或眼刺痛，应立刻撤到上风处。

### 二、历史提纯概念

升华法是历史上的提纯思路，但加热硫磺可能产生刺激性烟气，且提纯后更容易进入火工用途。本项目只保留“可通过升华提高纯度”的历史概念，不提供温度、装置和收集流程。

### 三、硫铁矿风险

硫铁矿焙烧会产生有毒、刺激性含硫气体，还涉及高温密闭容器风险。本项目不提供焙烧条件、收集装置和产量估算。

### 四、外观风险判断（不做燃烧测试）

| 指标 | 低风险记录方式 | 失败征兆 |
|------|--------|--------|
| 颜色 | 记录鲜黄、灰黄、棕色等外观 | 仅凭颜色就当药用或火工材料 |
| 气味 | 只记录“有/无刺鼻味”，不贴近深闻 | 低头猛闻、头晕仍停留 |
| 来源 | 写明火山、温泉、矿区、药材铺等 | 黄色粉末无标签混放 |

## 时间与距离尺度

| 场景 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 野外观察 | 站在上风侧，先观察再靠近 | 无刺鼻气体、无头晕咳嗽 | 眼刺痛仍继续采 |
| 样品标记 | 当场写明地点、日期、外观、气味风险 | 不与食物和药材混装 | 黄色粉末无标签 |
| 临时存放 | 远离火源、硝石、木炭、油脂和儿童 | 单独密封、清楚标识 | 与火工材料同柜 |
| 接触后处理 | 操作后洗手，皮肤不适即停止接触 | 无持续刺痛、皮疹 | 粉尘沾衣后带回卧室 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 用途边界 | 只用于历史、矿物识别、杀虫/药材风险知识，不与硝石木炭混合试验 |
| 采集 | 优先识别火山、温泉黄色沉积物，避开密闭含硫气体区域 |
| 提纯 | 只保留历史概念，不进行升华、焙烧、冷凝或收集 |
| 存放 | 干燥密封、远离火源、氧化剂和儿童 |
| 标识 | 容器写清“硫磺、易燃、燃烧有毒烟气” |

## 失败征兆

- 在室内加热硫磺，闻到刺激性气体仍继续操作。
- 把硫磺和硝石、木炭放在同一研钵或同一储物袋。
- 用燃烧火焰来判断纯度，增加吸入二氧化硫和失火风险。
- 采集温泉或火山沉积物时头晕、咳嗽、眼刺痛，应立即撤离上风处。
- 黄色矿物身份不明就作药用或火工用途。

## 常见问题

- **Q**: 硫磺和硝石一起研磨会爆炸吗？
- **A**: 可能发生剧烈燃烧或爆炸风险。不要混合研磨；本书不提供火工混合操作。

- **Q**: 找不到天然硫磺怎么办？
- **A**: 小说或历史设定里可写贸易购买、矿区运输或药材铺流通；现实中不要为火工用途寻找替代配方。

- **Q**: 硫磺有毒吗？
- **A**: 硫磺本身低毒，但燃烧产生的SO₂有毒。硫磺蒸气对人体也有刺激。

## 安全提醒

- 硫磺遇火易燃，远离明火
- 硫磺燃烧产生有毒SO₂气体，操作时通风
- 不进行硫磺升华、焙烧、冷凝或收集；硫磺蒸气和燃烧烟气都有刺激性和毒性风险
- 硫磺粉与皮肤长期接触可能引起皮炎

## 穿越小说梗

"主角发现了一个硫磺矿，大发其财"——在中国，天然硫磺主要分布在西南（云南、四川、台湾）和西北（甘肃）。如果你穿越到中原地区，硫磺比硝石更难获取。更可信的写法是：主角知道硫磺矿区的气体风险、火工隔离要求和贸易价值，能把硫磺当危险物资管理，而不是当普通黄色石粉随便混放。
