# 5.14 灌钢法

> 生铁做刀太脆，熟铁做刀太软——那把它们混在一起呢？这就是灌钢法的天才思路。

## 原理

灌钢法利用**生铁（高碳）** 和**熟铁（低碳）** 相互扩散：

1. 生铁熔点较低（约 1150°C），先熔化
2. 熔化的生铁"灌"入熟铁块的孔隙中
3. 碳从高碳的生铁向低碳的熟铁扩散
4. 最终整块铁达到均匀的中等含碳量→钢

### 炒钢 vs 灌钢

| 对比 | 炒钢法 | 灌钢法 |
|------|--------|--------|
| 原理 | 生铁→脱碳 | 生铁+熟铁→混合 |
| 原料 | 生铁一种 | 生铁+熟铁两种 |
| 产品均匀性 | 一般 | 好 |
| 适合 | 农具、工具 | 刀剑刃口 |
| 操作难度 | 中 | 较高 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 生铁（碎块） | 提供碳 |
| 熟铁（条/板） | 吸碳基体 |
| 黏土 | 封口 |
| 木炭 | 加热 |

| 铁钳 | 夹持 |

## 操作步骤

### 灌钢法（沈括《梦溪笔谈》记载）

**方法一：包裹灌钢法**

1. **准备**：
   - 熟铁条弯成 U 形
   - 生铁碎块放入 U 形槽中
   - 将 U 形口夹紧，用黏土封口

2. **加热**：
   - 放入炭火中加热至 >1200°C
   - 生铁熔化，渗入熟铁

3. **锻打**：
   - 取出热铁，迅速锻打
   - 锻打使生熟铁混合均匀
   - 折叠再锻打（反复 3-5 次）

4. **成钢**：
   - 锻打至铁块颜色均匀
   - 断口呈均匀的灰色→钢

**方法二：生铁淋口法（更简单）**

1. 熟铁刀坯加热至红热
2. 将熔化的生铁水滴在刀坯刃口处
3. 生铁水渗入刃口表层
4. 趁热锻打，使碳层深入
5. 淬火

**优点**：只在刃口处渗碳，刀背保持韧性（"刚柔相济"）。

## 常见问题

- **Q**: 生铁和熟铁的比例多少合适？
- **A**: 一般是 1:3 到 1:5（生铁:熟铁）。生铁太多→钢太硬太脆；生铁太少→钢不够硬。具体比例需要根据原料质量试验。

- **Q**: 没有熟铁怎么办？
- **A**: 把生铁反复加热锻打多次，碳会逐渐减少，变成"近似熟铁"。或者直接买/收集废铁。

- **Q**: 怎样判断灌钢成功？
- **A**: ①断口颜色均匀（灰色有光泽）②锤击韧性好（不崩不裂）③磨出刃口后有足够的硬度。简易测试：用锉刀锉刃口——打滑（太硬？）还是吃进（太软？）。

## 安全提醒

- 灌钢温度高，溅出的铁水可能引燃周围物品
- 锻打时铁屑飞溅，注意防护
- 生铁水遇水会爆炸飞溅，防止水滴进入操作区

## 穿越小说梗

"主角用灌钢法做出了绝世宝剑"——灌钢法确实是中国古代领先世界的冶金技术，穿越者知道这个原理可以复现。但真正让灌钢法厉害的，是**折叠锻打**（将生熟铁反复折叠、锻打、再折叠），每一次折叠都使碳分布更均匀，一把好刀需要折叠 10-20 次。注意：灌钢法做出的剑在宋代已经很常见，穿越者不能说"这是我发明的"——沈括一千年前就写了。
