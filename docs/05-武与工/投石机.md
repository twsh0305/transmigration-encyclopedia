# 5.6 投石机

> 投石机是中世纪的重炮——物理定律是唯一的火药。

## 原理

投石机利用杠杆原理将配重物的重力势能转化为投射物的动能。配重式（重力，1~10吨配重，投50~500kg，射程200~400m）远比牵引式（人力，30~100人拉绳）强大。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（橡木/榆木） | 主梁（6~12m）、支架 |
| 麻绳/铁链 | 牵引绳 |
| 铁/铜 | 轴承、铰链 |
| 石块 | 配重 |
| 皮革/帆布 | 弹射袋 |

## 操作步骤

1. 力比重臂=4:1~6:1，配重一般为投射物的100倍
2. 建造A字形支架（高4~6m），顶部装铁轴承座
3. 主梁长8~12m，短臂端装配重箱，长臂端装弹射袋
4. 用绞盘拉下长臂，卡扣固定，释放时抽出卡扣

## 常见问题

Q: 投石机打一炮需要多久？A: 配重式每5~10分钟一发，牵引式每2~5分钟一发。

Q: 能打移动目标？A: 不能，投石机是区域面杀伤/攻城武器。

## 安全提醒

释放时主梁回弹力量极大——操作人员必须站在安全距离外。绳索断裂有回弹伤人风险。

## 穿越小说梗

投石机最大缺点是射速太慢（十分钟一发）。穿越者的真正优势是知道配重计算——投100kg石弹需要多少配重、力臂多长——背后是简单的物理公式。
