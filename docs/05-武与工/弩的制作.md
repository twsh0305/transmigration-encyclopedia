# 5.5 弩的制作

> 弩让农民也能杀死骑士——技术的平等力量。

## 原理

弩的本质是**弓的机械化**——将弓横装在弩臂上，用弩机（扳机机构）扣住弓弦，射手扣动扳机释放。弩的核心优势：人力拉开弓弦后可机械锁定，射手可以从容瞄准。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（柘木/桑木/榆木） | 弩臂材料，需高抗弯强度 |
| 弓片（木/竹/钢） | 弩弓，比弓稍短但更厚 |
| 牛筋/麻绳 | 弓弦 |
| 铁/青铜 | 弩机（扳机系统） |

## 操作步骤

1. 选硬木截成长80~120cm，前端开出弓片槽，后端开出弩机槽
2. 弩弓安装于弩臂前端槽中
3. 铸铁或青铜锻造弩机三件：牙（挂弦）、望山（瞄准）、悬刀（扳机）
4. 上弦方式：手拉（<80磅）、蹶张（80~200磅）、绞盘（200~1000磅）
5. 望山上刻刻度对应不同距离的瞄准点

## 常见问题

Q: 弩和弓哪个更强？A: 弩精度高易用好，弓射速快（8~12发/分 vs 1~2发/分）。

Q: 弩的射程？A: 40磅弩约50m有效射程，150磅弩约150m。

## 安全提醒

上弦后绝不将弩对着人或动物。长期上弦不放箭会导致弩臂材料疲劳。

## 穿越小说梗

弩在古代从未完全取代弓——射速太慢是致命弱点。穿越者最佳策略：步兵用弩、骑兵用弓。
