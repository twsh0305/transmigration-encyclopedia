# 5.7 折叠锻打与百炼钢

> 一块铁反复折叠锻打三十次——它就变成了钢。这就是"百炼"的含义。

## 原理

折叠锻打的本质：

1. **除杂质**：每次加热锻打，炉渣被挤出
2. **均匀碳含量**：折叠使高碳和低碳层混合
3. **细化晶粒**：反复加热-锻打-冷却使铁晶粒细化，韧性提高
4. **形成花纹**：折叠产生明暗交错的层状结构（大马士革钢的效果）

### "百炼"的次数

| 折叠次数 | 层数 | 效果 |
|---------|------|------|
| 1 次 | 2 层 | 开始混合 |
| 5 次 | 32 层 | 初步均匀 |
| 10 次 | 1024 层 | 很均匀 |
| 15 次 | 32768 层 | 非常均匀 + 花纹 |
| 20 次 | 100万+层 | 极致均匀 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 钢/铁块 | 锻打原料 |
| 炭火炉 | 加热 |
| 大铁锤 | 锻打 |
| 铁砧 | 锻打平台 |
| 铁钳 | 夹持 |
| 水槽 | 冷却 |
| 铁刷 | 清除氧化皮 |

## 操作步骤

### 一、基础折叠锻打

1. **加热**：铁块在炭火中加热至橙黄色（约 1000-1100摄氏度）
2. **锻打**：在铁砧上锻打，将铁块打成扁条形
3. **折叠**：
   - 用凿子在扁条中间切一半（不切断）
   - 将铁条对折
   - 注意：折痕处不要有缝隙
4. **焊接**：
   - 加热至接近熔化温度（亮黄色，≈1200摄氏度）
   - 撒少量石英砂/硼砂（助焊剂，帮助铁层黏合）
   - 锻打使折痕融合
5. **重复**：以上步骤重复 10-20 次

### 二、百炼钢（传统工艺）

古人说"三十炼""五十炼""百炼"，其实指的就是折叠锻打的次数：

1. 每次折叠=一次"炼"
2. 三十炼：折叠 30 次 → 层数 2³⁰ ≈ 10 亿层
3. 实际 10-15 次折叠已经足够，更多次边际效益递减

### 三、大马士革花纹效果

1. 用两块不同的钢（高碳和低碳）焊在一起
2. 折叠锻打多次
3. 磨光后用稀酸（醋+少量铁锈水）浸泡
4. 高碳层颜色深、低碳层颜色浅 → 明暗花纹

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 火候 | 锻打温度约 1000-1100摄氏度，焊合温度接近 1200摄氏度但不过烧 |
| 折叠 | 10-15 次后层纹细密，肉眼不见大夹层 |
| 助焊 | 石英砂、碎玻璃或硼砂只薄撒一层，锻后氧化皮能排出 |
| 试样 | 小条淬火回火后能削木、刃口不卷不崩 |
| 体力 | 每锻打一小时休息，保持夹持稳定，不疲劳硬撑 |

## 失败征兆

- 折叠处总有黑缝，说明氧化皮夹入或焊接温度不足。
- 铁料表面冒火花、边缘像烧酥饼，说明过烧不可救。
- 层数盲目追到 20 次以上，材料越打越薄、脱碳越多。
- 刃口淬火后满是裂纹，说明水温、钢材含碳或加热不均有问题。
- 锤击时红热铁块滑出铁钳，应立即停工调整夹具。

## 常见问题

- **Q**: 锻打时铁块裂开了怎么办？
- **A**: ①温度太高（过烧）②温度太低（没烧透）③铁料含硫高（热脆）。重新加热到合适温度再打。

- **Q**: 没有硼砂做助焊剂怎么办？
- **A**: ①石英砂（河沙）②碎玻璃③草木灰（效果差一些）。助焊剂的作用是防止铁层之间氧化。

- **Q**: 古代说的"百炼钢"真的炼一百次吗？
- **A**: 大部分只有 10-30 次。"百"在古文中常是虚指，表示"很多次"。

## 安全提醒

- 锻打时铁屑飞溅，护目镜和皮围裙必备
- 铁钳夹紧，防止红热铁块飞出
- 水淬时产生蒸汽，注意蒸气烫伤
- 锻打一小时要休息——体力消耗极大

## 穿越小说梗

"主角用百炼钢打了一把削铁如泥的宝剑"——折叠锻打确实可以提高钢材质量，但效果被现代小说神化了。事实上，好的折叠锻打钢比普通钢好 20-30%，而不是"十倍"。穿越者的知识优势：①知道折叠次数-层数的数学关系②知道助焊剂的重要性（防止层间氧化）③知道"百炼"不是一百次。最重要的是：**锻打工是技术活，不是有知识就能打好的**——主角得真练。
