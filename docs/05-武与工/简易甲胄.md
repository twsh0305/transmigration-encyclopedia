# 5.22 简易甲胄

> 一件皮甲能让你的生存概率翻三倍。

## 原理

甲胄的本质是分散冲击力、阻挡切割、使刀刃偏离要害，而不是完全挡住攻击。

对比：皮甲（3~5kg，制作简单）、札甲（8~15kg，数百铁片+皮绳）、锁子甲（10~15kg，数万铁环，制作周期数月）。

## 所需材料

| 材料 | 说明 |
|------|------|
| 牛皮/猪皮/鹿皮 | 皮甲主体 |
| 铁片/钢片 | 札甲/鳞甲 |
| 皮绳/麻绳 | 穿连甲片 |
| 动物胶 | 皮革硬化 |
| 麻布衬甲 | 穿在甲内缓冲 |

## 操作步骤

1. 鞣制后的牛皮浸泡热水+蜂蜡溶液硬化，晾干后按胸/背/肩/裙剪裁
2. 边缘打孔，皮绳绑接，趁湿塑形
3. 铁片裁成标准尺寸（5~8cm乘2~4cm），四角打孔，皮绳穿连如瓦片叠压
4. 内穿厚麻衣缓冲防磨，肩和腰共同分担重量

## 常见问题

Q: 皮甲能挡住箭吗？A: 5mm硬皮甲30m外可挡30磅弓，但60磅弓20m内会射穿。

Q: 没有铁皮怎么办？A: 竹片替代做竹篾甲，多层叠压。

## 穿越小说梗

大多数古代士兵穿皮甲。穿越者用标准化尺寸加流水线作业可大幅降低成本。知道甲胄配重比（肩腰分担比例）能让士兵在战场上更持久。
