# 5.19 硝石识别与提取

> 没有硝石，黑火药就是一捧烟灰。硝石是火药工业的"瓶颈原料"。

## 原理

硝石的主要成分是**硝酸钾**（KNO₃），自然界中以**土硝**形式存在。

形成原理：有机物（粪便、腐殖质）在微生物作用下分解出氨 → 硝化细菌将氨氧化为硝酸 → 硝酸与土壤中的钾离子结合 → 形成硝酸钾结晶。

### 穿越者获取硝石的途径

| 途径 | 难度 | 产量 | 纯度 | 说明 |
|------|------|------|------|------|
| 老墙土/厕所土提取 | ⭐⭐ | 中 | 低 | 传统方法 |
| 洞穴土提取 | ⭐⭐ | 中 | 低 | 燕子洞等 |
| 重结晶提纯 | ⭐⭐⭐ | — | 高 | 提高纯度的关键 |
| 硝石矿开采 | ⭐⭐⭐ | 高 | 中 | 仅部分地区有矿 |

| 材料 | 用途 |

| 草木灰 | 提供钾离子 |
| 大锅 | 蒸发浓缩 |

## 操作步骤

### 一、含硝土壤采集

**最佳地点**：
1. **老墙土**——几十年以上的土墙、猪圈墙底部——白色结晶状物
2. **厕所/畜圈土**——有机质丰富、经年累月的土地
3. **洞穴土**——蝙蝠洞、燕子洞的堆积物

**采集**：刮取地表白色霜状物或地下 10-30cm 的表层土

### 二、土法提取

1. **捣碎**：将含硝土壤打碎（越细越好）
2. **浸泡**：土 + 水（1:3），搅拌浸泡 4-8 小时
3. **静置**：沉淀 1-2 小时
4. **取上清液**：用布过滤
5. **加草木灰**：每升滤液加一把草木灰（补充钾，促进硝石形成）
6. **蒸发浓缩**：大火煮沸，蒸发至原体积的 1/5
7. **冷却结晶**：将浓缩液倒入盆中，静置过夜
8. **收集结晶**：盆底白色针状晶体——粗硝石

### 三、重结晶提纯（提高纯度）

1. 粗硝石 + 少量热水溶解（刚好溶解即可）
2. 加少量草木灰搅匀，静置沉淀（去钙镁杂质）
3. 取上清液再次蒸发浓缩
4. 冷却后得白色针状结晶——纯度可达 90%+
5. 重复 2-3 次可得 95%+ 的纯硝石

### 四、纯度测试

| 测试 | 方法 | 合格指标 |
|------|------|---------|
| 火焰测试 | 取少量撒在炭火上 | 剧烈燃烧，紫色火焰（钾的焰色） |
| 溶解测试 | 100ml 水溶解 | 20°C 时约 30g/100ml 水 |
| 火药测试 | 按配方制火药点燃 | 快速燃尽，无残渣 |

## 常见问题

- **Q**: 100斤土能提多少硝？
- **A**: 好的老墙土每 100 斤可提 2-5 斤粗硝。质量差的厕所土可能只有 1 斤。

- **Q**: 硝石和食盐怎么区分？
- **A**: ①硝石在舌头上是凉丝丝的（溶解吸热）②硝石遇炭火剧烈燃烧（食盐不燃）③硝石溶解度随温度变化极大（食盐变化小）。

## 安全提醒

- 硝石是强氧化剂，与易燃物（硫磺、木炭）混合后极度易燃
- 硝石粉末不要接触有机物（木屑、布料）——有自燃风险
- 硝石不要和硫酸混合（生成硝酸——强腐蚀性）

## 穿越小说梗

"主角要做火药→需要硝石→想到了厕所老墙"——这是正确的思路。但小说常忽略：①提取硝石需要大量的土、大量的水和大量的燃料②重结晶是提纯的关键步骤（不提纯的火药质量很差）③最理想的硝石来源不是厕所，而是**天然硝石矿**（四川、甘肃、新疆等地）。如果你穿越到四川，恭喜你，中国最大的硝石产区就在脚下。
