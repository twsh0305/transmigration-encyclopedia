# 5.3 火药安全操作规范

> 火药不是玩具。一个错误操作足以炸掉半间屋子。

## 核心原则

**火药安全第一定律**：所有火药操作（研磨、混合、干燥、储存），都要假设下一秒可能爆炸。

## 操作禁区

### 绝对禁止的操作

| 操作 | 原因 |
|------|------|
| 金属工具研磨火药 | 撞击产生火花 |
| 在有明火处操作 | 火花引燃 |
| 大量混合（>1kg） | 爆炸威力过大 |
| 暴晒火药 | 热量积累可能自燃 |
| 摔打/撞击火药容器 | 摩擦引爆 |
| 吸烟/打火 | 明火 |
| 铁制容器储存 | 跌落产生火花 |

### 操作场所要求

| 要求 | 说明 |
|------|------|
| 远离居住区 | 至少 20m 外操作 |
| 通风良好 | 避免粉尘积聚 |
| 地面洒水 | 减少静电 |
| 无易燃物 | 干草、秸秆等清除 |
| 备灭火材料 | 沙土、水桶 |
| 单人操作 | 减少失误面 |

## 操作步骤

### 一、操作前准备

1. **着装**：棉布衣服（不要化纤——静电！），头发束起
2. **工具检查**：所有工具为木/竹/铜制，无铁制
3. **地面洒水**：湿润防静电
4. **环境确认**：附近无明火、无其他人
5. **灭火准备**：沙土桶、水桶放在手边

### 二、研磨安全

1. **每种材料单独研磨**——混合研磨极度危险
2. 每次研磨量 <50g
3. 研磨动作**轻柔、缓慢**
4. 硝石单独研磨时注意——不要有木炭、硫磺残留

### 三、混合安全

1. 用**木桶滚动法**混合（将粉倒入木桶，密封后滚动）
2. 严禁用研钵混合
3. 混合量每次 <200g
4. 混合前确认材料没有受潮结块

### 四、储存安全

1. **少量分散**——分装多个小容器，不要全部放在一起
2. 容器：密封陶罐/竹筒（**严禁金属容器**）
3. 标注：标记"火药"字样和日期
4. 位置：阴凉干燥、远离住所、远离火源
5. 防潮：储存处放生石灰吸潮

### 五、报废处理

过期/受潮结块的火药**不要倒进火里**：

1. 少量（<50g）：摊开在空地上，用**长引信**远距离点燃
2. 大量：用大量水溶解（硝石溶于水后不再可燃），然后倒掉
3. 严禁：倒入火堆、下水道（爆炸）或随地丢弃

## 应急处理

| 事故 | 处理 |
|------|------|
| 火药起火 | 立即用沙土覆盖（**不要用水！**水冲散后更危险） |
| 皮肤烧伤 | 冷水冲洗 20 分钟 |
| 火药溅入眼睛 | 大量清水冲洗 |
| 衣物着火 | 躺地上翻滚灭火 |

## 常见问题

- **Q**: 火药受潮了还能用吗？
- **A**: 可以。在阴凉处晾干（不要暴晒），干燥后敲碎结块即可使用。但威力会有所下降。

- **Q**: 家里有小孩怎么做火药？
- **A**: **不要在有小孩的环境中操作。** 火药制作和储存都必须远离儿童。

## 安全提醒（重点）

- 这可能是你穿越后**最危险的操作**
- 新手操作时，量控制在 20-30g 以内
- 第一次做火药，必须有心理准备：它真的会爆炸
- 安全不是妥协的——一次失误可能意味着失去手指、眼睛或生命

## 穿越小说梗

大部分穿越小说对火药安全写得太轻描淡写了。"主角随手就配了几斤火药"——现实中，熟练的火药匠人也会偶尔出事故。穿越者的知识优势不是"不怕炸"，而是**知道要防静电、知道不能用铁器、知道要少量分装**。正是这些看似琐碎的安全知识，让穿越者比古代的工匠更安全。记住：火药把你炸死只需要 0.01 秒。
