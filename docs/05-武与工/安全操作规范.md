# 5.3 火药安全操作规范

> 火药不是玩具。一个错误操作足以炸掉半间屋子。

## 核心原则

**火工安全第一定律**：所有疑似火工材料、未知粉末和旧弹药，都要假设下一秒可能爆燃或释放有毒烟气。

## 操作禁区

### 绝对禁止的操作

| 操作 | 原因 |
|------|------|
| 金属工具研磨火药 | 撞击产生火花 |
| 在有明火处操作 | 火花引燃 |
| 任何非专业混合、研磨、筛分 | 摩擦、静电、撞击都可能引燃 |
| 暴晒火药 | 热量积累可能自燃 |
| 摔打/撞击火药容器 | 摩擦引爆 |
| 吸烟/打火 | 明火 |
| 自行选择容器长期储存 | 容器材料、密封、静电和法规都可能出问题 |

### 操作场所要求

| 要求 | 说明 |
|------|------|
| 远离居住区 | 危险品区至少远离住所、粮仓、牲畜棚和儿童活动区 20 米 |
| 通风良好 | 避免粉尘积聚 |
| 湿式清扫 | 减少粉尘飞扬，不处理疑似火工材料本身 |
| 无易燃物 | 干草、秸秆等清除 |
| 备灭火材料 | 沙土、水桶 |
| 单人操作 | 减少失误面 |

## 安全边界

本页保留危险品识别、隔离、标识和事故处置原则，不提供火药制造、混合、造粒、测试、储存优化或销毁的可复现流程。现实中应遵守当地爆炸物法规；穿越叙事中也应把火药工坊写成高风险专业场所，而不是厨房手工活。

### 临时隔离边界

1. 不把疑似火工材料带回住宅、厨房、粮仓、牲畜棚或儿童活动区。
2. 不更换容器、不打开试闻、不敲碎、不晾晒、不加热。
3. 从外侧做清楚标识：名称不明、来源、发现日期、负责人和危险性。
4. 清走无关人员，设置禁入边界，等待专业人员或有经验工匠处理。
5. 不把生石灰、水、油、木屑、硫磺、硝石或金属工具放到同一区域。

### 报废处理

过期、受潮、结块或来源不明的火药不要试图“点一下看看”。能交给专业机构就交给专业机构；穿越叙事中应安排专门空旷场、远程处置、封锁警戒和有经验工匠负责。

严禁：倒入火堆、下水道、炉膛、垃圾坑，或在住宅附近随地丢弃。

## 时间与隔离尺度

| 场景 | 可执行尺度 | 成功标准 | 失败征兆 |
|------|------------|----------|----------|
| 发现未知粉末 | 10 秒内停止靠近、喊停、清人 | 现场无人围观 | 还在试闻、试点、试磨 |
| 标签记录 | 当场写明名称、日期、来源、负责人 | 任何人一看知道危险性 | 无标签、混装、用食具盛放 |
| 受潮结块 | 立即隔离，不敲、不晒、不烤 | 等专业人员或有经验工匠处置 | 试图恢复性能 |
| 烧伤冲洗 | 冷水连续冲洗 20 分钟 | 先救人后抢物料 | 中途停止冲洗去找药膏 |

## 应急处理

| 事故 | 处理 |
|------|------|
| 火药起火 | 优先撤离并隔离，能安全覆盖时才用沙土，不抢救物料 |
| 皮肤烧伤 | 冷水冲洗 20 分钟 |
| 火药溅入眼睛 | 大量清水冲洗至少 15 分钟 |
| 衣物着火 | 躺地上翻滚灭火 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 场地 | 隔离区远离居住区、粮仓、牲畜棚和儿童活动区至少 20米，现实从当地法规和专业指导 |
| 分区 | 原料、成品、废品、工具、火源分区，不交叉堆放 |
| 标识 | 所有危险容器写明名称、日期、危险性和负责人 |
| 纪律 | 不吸烟、不围观、不疲劳操作，不把火药带回住处 |
| 事故准备 | 备沙土、清水、撤离路线和烧伤冲洗点 |

## 失败征兆

- 为赶进度提高批量、减少距离或让无关人员帮忙。
- 容器无标识，分不清硝石、硫磺、火药、废料。
- 操作间有干草、木屑、油布、明火或儿童。
- 受潮结块后用敲、砸、晒、烤来恢复。
- 事故后先抢材料而不是撤人、冷却烧伤和封锁现场。

## 常见问题

- **Q**: 火药受潮了还能用吗？
- **A**: 不建议尝试恢复。受潮、结块、来源不明或标签不清的危险粉末应隔离标记，交给专业人员或有经验工匠处理。

- **Q**: 家里有小孩时发现疑似火工材料怎么办？
- **A**: 立即清人、隔离、标识，不让儿童和围观者接触；不要把材料带回住处，也不要自行打开、试闻、敲碎或点火。

## 安全提醒（重点）

- 这可能是你穿越后**最危险的操作**
- 新手不做火药制造、混合、造粒、测试和销毁操作
- 第一次接触未知火工材料，默认它真的会爆炸
- 安全不是妥协的——一次失误可能意味着失去手指、眼睛或生命

## 穿越小说梗

大部分穿越小说对火药安全写得太轻描淡写了。"主角随手就配了几斤火药"——现实中，熟练的火药匠人也会偶尔出事故。穿越者的知识优势不是"不怕炸"，而是**知道要防静电、知道不能用铁器、知道要少量分装**。正是这些看似琐碎的安全知识，让穿越者比古代的工匠更安全。记住：火药把你炸死只需要 0.01 秒。
