# 5.18 盾牌制作

> 最好的盔甲，是你拿得起的那面盾。

## 原理

盾牌是移动掩体——通过倾斜角度使攻击偏斜。圆形盾不论从哪个方向来的攻击都能以一定角度偏斜。

盾牌类型：圆盾（直径30~80cm，1~4kg，单手格斗）、筝形盾（高120~150cm，3~6kg，全身防护）、塔盾（高150~180cm，6~12kg，攻城/盾阵）。

## 所需材料

| 材料 | 说明 |
|------|------|
| 板材（椴木/松木/梧桐） | 椴木最轻且不易开裂 |
| 牛皮 | 蒙皮加固 |
| 铁/青铜 | 包边、盾心 |
| 铁钉/动物胶 | 固定与层压 |

## 操作步骤

1. 木板交错拼接，厚度圆盾1~1.5cm、筝形盾1.5~2cm
2. 盾面从中心到边缘由厚渐薄，向外微微鼓起（使箭矢滑开）
3. 湿牛皮紧绷在盾面上从背面钉固，干后收缩紧绷
4. 铁/铜皮沿盾牌边缘包覆防裂
5. 盾心后方装横木把手，重心落在盾心附近

## 常见问题

Q: 没有木板怎么拼大盾？A: 藤条编扎成盾面外蒙多层牛皮，轻便有弹性。

## 安全提醒

1.5cm松木加单层牛皮可阻挡30磅弓20m外的箭矢。蒙皮湿操作时刀具锋利谨防割伤。

## 穿越小说梗

好盾牌在不同方向上有不同强度——中心最强，边缘最薄。这个差异就是举盾和用盾的区别。
