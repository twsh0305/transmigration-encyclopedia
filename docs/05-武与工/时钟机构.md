# 5.9 时钟机构

> 精准计时，是现代科学的第一推动力。

## 原理

时钟三要素：能量源（重力/弹簧/水流）、振荡器（摆锤/擒纵/滴水）、计数器（齿轮系）。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（黄杨/枣木） | 齿轮、框架 |
| 铁/铜 | 擒纵轮、轴、摆锤 |
| 绳/链条 | 重锤悬挂 |
| 水容器（陶/铜） | 水钟主体 |
| 细沙/磨细蛋壳粉 | 沙漏用 |

## 操作步骤

1. 日晷：竖垂直指针，以针根为中心画圆12等分，正北对准12点
2. 水钟（浮子型）：上容器滴水到下容器，浮子上升带动指针。加溢流管保持恒压
3. 沙漏：两等大瓶口对口密封，连接处开1~2mm孔，装入干燥细沙
4. 摆钟：摆长T=2pi乘根号(L/g)，标准秒摆长0.99m。木齿轮加铁擒纵轮加重锤驱动
5. 机械钟：重锤驱动到减速齿轮系到擒纵机构（卡住释放卡住）到摆锤控节奏到指针显示

## 常见问题

Q: 水钟越走越慢？A: 泄水型通病，加溢流管保持恒压解决。

Q: 没有金属能做摆钟？A: 木擒纵轮精度差磨损快，至少需要一个铁制擒纵轮。

## 穿越小说梗

精准计时是科学实验的基础——航海经度、天文观测、物理实验都需要精确时钟。穿越者用木制擒纵轮加水的简易方案，每天差10~15分钟，在古代已是革命性进步。
