# 5.4 弓的制作

> 弓是人类第一件"储能武器"——拉开时蓄力，释放时杀敌。

## 原理

弓的本质是**弹性势能转化器**。拉弓时弓臂弯曲变形储存弹性势能，撒手时弓臂回弹将能量传递给箭矢。弓的效率取决于弓材的弹性回复率——好的弓材能将70%~90%储存能量传递给箭矢。

**单体弓 vs 复合弓**：单体弓用单一种木材弯曲而成，制作简单但弓身长（1.5~2m）；复合弓用木胎+牛筋+牛角+动物胶层压，弓身短、储能密度高，但制作周期长（数月）、怕潮湿。

## 所需材料

| 材料 | 说明 |
|------|------|
| 弓材（紫杉/榆木/白蜡/竹） | 紫杉弹性与抗压均衡最佳 |
| 弓弦（牛筋/麻绳/丝线） | 牛筋弹性最佳，麻绳便宜但寿命短 |
| 动物胶（鱼鳔胶/皮胶） | 复合弓层间粘合用 |
| 刮刀/玻璃片 | 修整弓臂形状 |
| 油脂（动物油/桐油） | 防潮保护 |

## 操作步骤

1. 选树龄5~10年、纹理笔直的树木，冬季砍伐，阴干6~12个月
2. 弓坯两端削细（弓梢），中间保持粗壮（弓把）
3. 弓臂从弓把到弓梢均匀变细，用细绳拉线法检查对称性
4. 上弦后弦到弓把距离约一拳高（15~20cm）
5. 先拉半弓检查，不对称处用刮刀微调

## 常见问题

Q: 弓做出来拉不开怎么办？A: 拉力设计过大。新手从20~30磅开始。

Q: 弓臂开裂了能修吗？A: 小裂纹用动物胶+筋线修补，大裂纹报废。

## 安全提醒

炸弓是最大风险——首次拉弓前仔细检查弓臂有无裂纹。拉弓时不要对着人。

## 穿越小说梗

一把合格的弓从选材到完成至少需半年。穿越者真正的优势是知道用烘干法把干燥时间从一年压缩到一周。
