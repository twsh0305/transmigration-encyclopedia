# 5.8 攻城与守城器械

> 攻城和守城，是古代工程学的最高竞技场。

## 原理

攻城战本质是攻守双方时间和资源的消耗竞赛。攻方需突破城墙，守方需尽可能延长守城时间以待援军。

## 攻城器械

**攻城槌**：粗木槌（长8~15m，直径40~60cm）悬挂在移动棚架内，槌头包铁，10~30人摆动撞击城门。

**攻城塔**：多层木塔（高10~20m，超过城墙），底部装轮，顶部设放下吊桥，外蒙湿牛皮防火。

**云梯**：顶端装铁钩钩住城墙，梯长超过城墙高度至少2m。

**盾车**：木制框架（宽2~3m，高2~2.5m），前端斜面板加铁皮，底部装轮。

## 守城器械

礌石滚木从城墙推下。金汁：粪便加油脂煮沸浇下，高温加细菌感染。弩炮：重型箭矢射穿攻城器械。护城河（宽大于5m，深大于3m）、马面（城墙凸出侧射）、瓮城（两道城门之间的夹道）。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（橡木/榆木/松木） | 器械主体 |
| 铁/铜 | 包边、固定件、槌头 |
| 麻绳/皮绳 | 捆扎 |
| 湿牛皮/泥土 | 防火涂层 |

## 常见问题

Q: 攻城塔会被火烧毁吗？A: 会。必须覆盖湿牛皮，塔上备水桶和沙土灭火。

Q: 没有石材做礌石？A: 用砖块或石灰混凝土浇筑。

## 穿越小说梗

攻城战胜负往往不取决于器械，而取决于后勤——城里有水有粮能守一年，城外大军围城三月自己先饿死了。
