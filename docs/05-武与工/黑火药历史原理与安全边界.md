# 5.27 黑火药历史原理与安全边界

> 古代火药改变了战争史；今天阅读它，重点应是理解风险、制度和历史影响，而不是复现配方。

## 现实安全边界

本篇只用于历史和小说可信度分析，不作为现实制备、混合、测试、储存或使用火药的教程。现实中制造、改性、运输、储存和点燃火药可能违法，并可造成死亡、失明、烧伤和火灾。

## 成功标准

- 读者能理解黑火药依赖氧化剂、燃料和硫类材料的危险组合，而不是把它当普通粉末。
- 小说描写能体现供应链、纯度、受潮、事故、保密和法律风险。
- 现实阅读后能做到“不混合、不摩擦、不点燃、不试验”，只进行风险识别和远离。

## 失败征兆

- 想把比例、工具、测试方法转化为现实试制计划。
- 未知粉末被混放、研磨、筛分、加热或靠近明火。
- 现场出现硫味、焦味、烟、静电、粉尘飞扬或容器发热。

## 原理

黑火药燃烧的化学反应（简化）：

```
2KNO₃ + S + 3C → K₂S + N₂↑ + 3CO₂↑ + 大量热
```

关键是**瞬间产生大量气体**（N₂ + CO₂），体积在极短时间内膨胀数千倍，产生爆炸推力。

## 历史配方为什么难稳定

古籍中的口诀往往是经验表达，真正影响结果的是原料纯度、含水量、颗粒均匀度、混合一致性、储存环境和操作纪律。任何一个环节失控，都可能从“燃烧慢”变成“突然爆燃”。

## 风险识别表

| 风险点 | 典型征兆 | 处理 |
|--------|----------|------|
| 受潮 | 结块、变色、气味异常 | 不揉碎，不烘烤，隔离标记 |
| 摩擦撞击 | 粉尘飞扬、工具刮擦、容器摇晃 | 停止搬动，撤离无关人员 |
| 火源 | 油灯、炉火、烟斗、火星 | 熄灭火源或远离现场 |
| 混装 | 无标签、不同粉末同容器 | 不打开试闻，不混合，等待专业处理 |

## 常见问题

- **Q**: 为什么不保留具体配方和测试步骤？
- **A**: 因为它们会把历史知识变成现实爆燃风险。本项目的火药内容只保留科学原理、历史背景、风险识别和小说可信度判断。

- **Q**: 小说里怎样写才可信？
- **A**: 写原料难得、纯度不稳、工坊隔离、工匠训练、事故记录、运输防潮和保密制度，比写“背出口诀马上成功”更可信。

- **Q**: 一硝二磺三木炭这个比例对吗？
- **A**: 它是古代流传很广的经验口诀，但现实中不应据此尝试制备。写作时可把它当作“穿越者知道的历史线索”，真正难点放在安全制度和供应链。

## 安全提醒

- **绝对不能**用金属工具研磨或混合火药
- **绝对不能**在有明火、火星的环境操作
- 不自行混合、测试、点燃或储存危险粉末
- 发现疑似火药或旧弹药，撤离、隔离、交专业人员处理

## 穿越小说梗

"一硝二磺三木炭"在现代穿越小说里被说烂了。但真正关键不是主角背出口诀，而是他能不能建立隔离工坊、原料台账、防潮运输、事故问责和训练制度。更可信的写法是：穿越者先因为一次小事故意识到火药不是玩具，再把它纳入军工和安全体系，而不是在卧室里随手搓一包。
