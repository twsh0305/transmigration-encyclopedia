# 5.12 水车与风车

> 水车是人类第一台不需要人力驱动的机器。

## 原理

水车和风车都是自然能量转化器——将水流或风的动能转化为机械能。水的密度是空气的800倍，同样体积水能是风能的800倍。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（橡木/榆木） | 主结构 |
| 竹材 | 水轮叶片（轻便耐水） |
| 铁/铜 | 转轴、轴承 |
| 石材 | 磨盘 |
| 桐油/沥青 | 防水处理 |

## 操作步骤

1. 上击式水车（效率60~85%，需水槽引水从上冲击）、下击式（效率20~40%，直接投入水中）
2. 选择水流大于0.5m/s的河段，两岸立支架，制作直径2~4m水轮（轮轴+轮幅+叶片）
3. 立轴风车：中心柱高5~8m，柱顶装转盘，转盘上装6~8片帆布
4. 水碓联动：水车转轴上凸轮拨动碓杆，碓头升起落下舂米

## 常见问题

Q: 流速慢的小河能用吗？A: 用拦截坝使水流集中（堰坝法），或做大直径水轮。

Q: 木制水轮能用多久？A: 保养得当可用3~5年，水中部分每2年更换。

## 穿越小说梗

水车的安装比建造更难——几百公斤木轮准确架到轴承上是工程难题。穿越者用滑轮组加杠杆组合吊装，这就是技术代差。
