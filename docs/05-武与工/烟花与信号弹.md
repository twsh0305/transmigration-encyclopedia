# 5.17 烟花与信号弹

> 火药不只能杀人，还能放烟花——而在古代，一场精心设计的烟花秀就是顶级的"实力展示"。

## 原理

烟花利用火药的**不同配比 + 不同金属离子 + 不同封装结构**产生效果：

1. **升空**：底部发射药将烟花弹推上高空
2. **爆炸**：中心火药炸开弹体
3. **着色**：添加金属盐产生不同颜色
4. **声响**：特定配比产生爆音或啸声

### 颜色来源

| 颜色 | 添加物 | 来源 |
|------|--------|------|
| 红色 | 硝酸锶/碳酸锶 | 含锶矿物 |
| 绿色 | 硝酸钡/氯化钡 | 含钡矿物 |
| 蓝色 | 氯化铜/碳酸铜 | 铜盐（最难） |
| 黄色 | 硝酸钠/冰晶石 | 食盐/钠盐 |
| 紫色 | 锶+铜混合 | 两种混合 |
| 白色 | 镁粉/铝粉 | 银白色耀眼 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 黑火药 | 发射+爆炸 |
| 各种金属盐 | 着色 |
| 纸/竹筒 | 烟花壳体 |
| 引信 | 延时点燃 |
| 黏土/封口泥 | 密封 |
| 木棍（竹签） | 平衡尾翼 |

## 操作步骤

### 一、信号弹（最实用）

**彩色信号弹**：

1. 取一段竹筒（口径 2-3cm，长 10-15cm）
2. 底部封泥（约 2cm 厚）
3. 装入发射火药（约 15g 黑火药）
4. 上面放着色药（10-20g 黑火药+金属盐）
5. 顶部封纸/泥
6. 装引信（从底部小孔穿入）
7. 固定尾杆（竹片/木棍，保持飞行稳定）

**发射**：点燃引信 → 底部火药爆发推升 → 高空爆炸呈色

### 二、简易烟花（娱乐用）

**地面小喷泉**：
1. 竹筒内装入黑火药+铁粉（喷出火星效果）
2. 压实（不能太紧——会炸筒）
3. 顶部开口
4. 点燃——喷出金色火花

**升空烟花**：
1. 用纸卷成圆筒（多层纸）
2. 一端加黏土底
3. 装入火药+着色剂
4. 加引信
5. 装尾杆
6. 用发射管（竹筒/铁管）定向发射

### 三、烟花安全发射

1. **固定**：烟花筒插在土中或沙桶中
2. **距离**：观众距离 >30m
3. **风向**：在上风处点火
4. **哑弹处理**：等待 5 分钟再靠近（不要立即查看）

## 常见问题

- **Q**: 古代为什么要做烟花？
- **A**: ①展示实力（火药技术=军工技术）②吸引皇帝/高官注意③收入来源（宋朝已有烟花产业）④军事信号（信号弹）

- **Q**: 古代没有金属盐怎么做彩色烟花？
- **A**: ①铜绿（碳酸铜）→绿色②铁粉→金色火花③骨粉→银色④天然矿物可提供部分颜色。但古代彩色烟花远不如现代丰富。

## 安全提醒

- 烟花制作与火药操作等同，遵守所有火药安全规范
- 发射管必须结实（否则炸管伤人）
- 信号弹不要对着人发射
- 烟花储存同火药储存要求

## 穿越小说梗

"主角用一场烟花秀征服了所有人"——这确实可行。古代对烟花有天然的好奇和喜爱。穿越者如果能制造出彩色信号弹（红/绿/黄），用途不止于娱乐：①军事信号②节日表演③与外界远距离通信。如果是"种田流"开局，烟花还可以作为商品卖给富裕阶层——烟花在古代是奢侈品。
