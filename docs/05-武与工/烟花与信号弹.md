# 5.17 烟花与信号弹

> 烟花和信号弹展示的是火工组织能力，也展示了爆燃、抛射、有毒烟气和哑弹风险。

## 原理

烟花利用燃烧、抛射、显色和声光效应制造视觉信号。对本项目而言，重点不是复原结构，而是理解它为什么危险：封装会积聚压力，金属盐可能有毒，哑弹和残骸可能二次爆燃。

### 颜色来源

| 颜色 | 历史线索 | 风险 |
|------|--------|------|
| 红、绿、蓝等彩光 | 不同金属盐会改变焰色 | 钡、铜、锶等来源不明时可能有毒 |
| 白光和火花 | 金属粉末或高温颗粒可产生强光 | 粉尘、灼伤和眼损伤风险高 |
| 声响 | 封装压力突然释放 | 哑弹、炸裂和碎片风险高 |

## 历史结构概览（不提供制作步骤）

烟花与信号弹属于火工品，风险来自高温、爆燃、抛射和有毒烟气。历史上常见结构包括壳体、引信、发射药、效果药和稳定结构；这些信息适合写小说、理解古代军工和识别危险品，不适合作为手工制作教程。

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 内容边界 | 只讨论历史原理、颜色来源和通信用途，不给出装药量和制作流程 |
| 观演距离 | 合法成品烟花燃放时，人员保持足够距离，至少 30米外观察 |
| 场地 | 空旷、无草垛、无房屋屋檐、无易燃堆料，备水和沙土 |
| 哑弹 | 失败未响时等待至少 15 分钟，不低头查看、不二次点火 |
| 储存 | 远离火源、潮湿、撞击和儿童，少量、分开、清楚标识 |

## 失败征兆

- 为追求效果提高装药量、压得更紧或改壳体，风险会急剧上升。
- 发射管开裂、倾斜或固定不稳，可能横飞伤人。
- 彩色金属盐来源不明，可能含铅、钡等有毒物。
- 烟雾刺鼻、人员咳嗽流泪，应撤到上风处。
- 哑弹马上靠近查看，是最常见的伤人场景之一。

## 常见问题

- **Q**: 古代为什么要做烟花？
- **A**: ①展示实力（火药技术=军工技术）②吸引皇帝/高官注意③收入来源（宋朝已有烟花产业）④军事信号（信号弹）

- **Q**: 古代没有现代化学品怎么写彩色烟花？
- **A**: 可以写成颜色有限、烟大、失败多、材料昂贵且有毒性风险。不要把矿物粉、金属粉和火药写成家庭可混配材料。

## 安全提醒

- 烟花制作与火药操作等同，现实中只使用合法成品并遵守当地规定
- 不靠近、不捡拾、不拆解哑弹或残骸
- 观演时站在上风侧和安全距离外，儿童不得接触火工品
- 烟花和疑似火工品远离火源、潮湿、撞击和儿童
- 本条目不提供自制烟花、信号弹或火工配方

## 穿越小说梗

"主角用一场烟花秀征服了所有人"可以写，但更可信的重点是组织、采购、封锁场地、观演距离、哑弹处置和有毒烟气，而不是手搓配方。烟花在古代也可作为军工实力和财富的展示，但它首先是高风险火工品。
