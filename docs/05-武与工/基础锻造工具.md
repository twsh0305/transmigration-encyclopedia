# 5.2 基础锻造工具

> 没有好铁砧和好锤子，再好的铁匠也打不出好东西。

## 原理

锻造是**加热 + 锤击**的机械加工：

1. 铁加热到塑性状态（红热以上）
2. 锤击使铁变形——沿受力方向流动
3. 控制变形方向和量→得到所需形状

### 必备工具体系

| 工具 | 用途 | 穿越初期的替代 |
|------|------|--------------|
| 铁砧 | 锻打底座 | 大块花岗岩/青石 |
| 锤子 | 施加力 | 石锤→铁锤 |
| 铁钳 | 夹持热铁 | 树枝夹（临时） |
| 炉子 | 加热 | 挖地灶+鼓风 |
| 水槽 | 冷却 | 木桶 |
| 锉刀 | 修整 | 硬石片 |
| 凿子 | 切割/开槽 | 在铁砧上凿 |
| 冲子 | 打孔 | 锥形铁棒 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 大块硬石/铸铁块 | 铁砧 |
| 铁棒（买或炼） | 锻造原料 |
| 木炭 | 加热燃料 |
| 皮风箱/木风箱 | 鼓风 |
| 水桶 | 淬火冷却 |

## 操作步骤

### 一、制作铁砧（穿越第一件铁器）

1. 找一块大花岗岩/玄武岩（至少 30×30×20厘米）
2. 顶部磨平（作为工作面）
3. 埋在土中一半（稳固，减少震动）
4. 石砧效率低但可用——初期打铁足够了

### 二、制作铁锤

1. **小锤**：1-2斤，单手用（精确锻打工）
2. **大锤**：5-8斤，双手用（粗加工）
3. 锤面微凸（不是平的！），这样打铁不会留下棱角

### 三、建造锻炉

1. 在地上挖一个 30×30厘米的浅坑
2. 一面砌矮墙（约 20厘米高）作挡风墙
3. 一侧底部装陶管（接风箱风口）
4. 炉膛铺耐火黏土

### 四、基础锻造技术

| 技术 | 操作 | 用途 |
|------|------|------|
| 拉伸 | 在铁砧上斜向锤击 | 把铁块打成条形 |
| 镦粗 | 垂直锤击缩短 | 增加截面 |
| 弯曲 | 在砧角上弯折 | 各种曲线形状 |
| 冲孔 | 用冲子边打边转 | 打孔 |
| 焊接 | 两铁同时加热到接近熔化，撒砂→锤接 | 接合两铁块 |
| 开刃 | 斜向锤击打薄刃口 | 刀剑刃口 |

## 尺寸、温度与动线尺度

| 项目 | 建议尺度 | 成功标准 | 失败征兆 |
|------|----------|----------|----------|
| 石砧 | 至少 30 厘米×30 厘米×20 厘米，埋入一半 | 锤击不晃、不翻 | 砧面碎裂飞片 |
| 锻炉 | 炉膛约 30 厘米×30 厘米，挡风墙约 20 厘米 | 火色稳定到橙红 | 黑烟大、风口堵灰 |
| 锤具 | 小锤约 1 到 2 斤，大锤约 5 到 8 斤 | 锤柄楔紧不松 | 锤头晃动、柄裂 |
| 工作区 | 热铁区、燃料区、水槽分开，通道无绊脚物 | 转身不碰热铁 | 冷铁热铁混放 |

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 石砧 | 至少 30厘米×30厘米×20厘米，埋入地面一半，锤击不晃 |
| 锤面 | 微凸、无裂纹、无卷边，锤柄楔紧不松 |
| 锻炉 | 炉膛约 30厘米×30厘米，风口不被灰堵，火色能稳定到橙红 |
| 夹持 | 热铁夹起后摇晃不脱落，铁钳或临时夹具不烧手 |
| 工作区 | 水槽、燃料、热铁区分开，行走路径无绊脚物 |

## 失败征兆

- 铁砧跳动或翻倒，锻打力会反弹并砸脚。
- 锤头松动、锤柄开裂，必须停用重装。
- 石砧表面碎裂飞片，说明石材不合适或锤击过猛。
- 热铁被随手放在地上且没有标记，极易误触烫伤。
- 炉火忽明忽暗、黑烟大，说明鼓风或燃料干燥度不足。

## 常见问题

- **Q**: 没有铁砧能不能打铁？
- **A**: 可以。在石头上打铁是古代常见做法。但石头会碎、会吸收热量、表面不平整，效率很低。**获取铁砧是冶炼后第二重要的事**。

- **Q**: 风箱怎么做？
- **A**: 最简单的：竹筒活塞风箱——大竹筒内装活塞，推拉鼓风。进阶：木制双作用风箱（推拉都出风），效率高一倍。

- **Q**: 铁钳没有怎么做？
- **A**: 先用两段树枝夹住，外面绑湿布隔热。虽然不好用，但可以坚持到打出铁钳。

## 安全提醒

- 锻打时铁屑乱飞，必须戴护目镜
- 铁砧要稳固，防止翻倒砸脚
- 热铁和冷铁看起来一样！在铁钳手柄上绑绳子/布条——触感区分
- 地面洒水防止火灾，但水槽远离脚边

## 穿越小说梗

"穿越者建了一个铁匠铺"——这是种田流的经典转折点。但很多时候小说跳过了一个关键事实：**铁砧和铁锤本身就是铁做的**。你的第一件铁器不是刀剑，而是**铁锤**（用来打下一件铁器）和**铁钳**（用来夹热铁）。换句话说：你需要的工具本身就需要工具来制造。这个"先有鸡还是先有蛋"的困境，靠一块大石头做临时代用铁砧来解决。
