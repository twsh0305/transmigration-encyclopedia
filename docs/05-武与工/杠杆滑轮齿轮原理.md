# 5.11 杠杆滑轮齿轮原理

> 给我一个支点，我不仅能撬动地球，还能建造一个帝国。

## 原理

杠杆三类：第一类（撬棍，支点中间）、第二类（独轮车，重物中间）、第三类（镊子，力点中间）。公式：力乘力臂=重乘重臂。

滑轮：定滑轮（改变方向不省力）、动滑轮（省一半力）、滑轮组（F=W/n，n为承重绳数）。

齿轮：传动比=驱动齿轮齿数/从动齿轮齿数。大于1增速减扭，小于1减速增扭。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（橡木/枣木） | 杠杆臂、齿轮 |
| 麻绳/皮绳 | 滑轮绳索 |
| 木棍/铁棍 | 转轴 |
| 动物油脂 | 润滑 |

## 操作步骤

1. 杠杆制作：选 1.5 到 3 米硬木棍，一端削平接触重物，支点尽量靠近重物
2. 滑轮制作：木轮直径 10 到 30 厘米，中心打孔装轴，圆周刻绳槽
3. 齿轮制作：木板锯成圆形，圆周均匀刻齿，两齿轮齿距必须一致
4. 绞盘加棘轮：滚筒加手柄加棘轮止回爪防止倒转

## 比例与尺寸尺度

| 机构 | 建议尺度 | 成功标准 |
|------|----------|----------|
| 杠杆 | 木棍长 1.5 到 3 米，支点靠近重物 | 人能缓慢撬动，木杆不弯裂 |
| 滑轮 | 木轮直径 10 到 30 厘米，绳槽圆滑 | 拉 20 次不磨绳、不卡轴 |
| 齿轮 | 两轮齿距一致，中心距稳定 | 转动 20 圈不跳齿 |
| 绞盘 | 手柄越长越省力但越慢 | 停手后止回可靠 |

## 计算与验收

| 机构 | 快速公式 | 成功标准 |
|------|----------|----------|
| 杠杆 | 力 x 力臂 = 重量 x 重臂 | 人能缓慢撬动，支点不滑 |
| 滑轮组 | 理想拉力约为重量 / 承重绳数 | 拉绳不磨边，重物平稳上升 |
| 齿轮 | 从动转速 = 主动转速 x 主动齿数 / 从动齿数 | 转动 20 圈不跳齿 |
| 绞盘 | 手柄越长越省力，速度越慢 | 停手后棘轮能止回 |

## 失败征兆与修正

| 征兆 | 原因 | 修正 |
|------|------|------|
| 杠杆折断 | 木料有节疤或力臂太长 | 换直纹硬木，缩短力臂 |
| 滑轮卡死 | 轴歪或槽太窄 | 修正轴孔，加宽绳槽 |
| 齿轮啃齿 | 齿距不同或中心距错 | 重新画等分线 |
| 绞盘倒转 | 无止回或止回爪弱 | 加棘轮和保险绳 |

## 安全边界

- 所有省力机构都会把能量储存在绳、杆、重物中，断裂时会回弹。
- 试吊和试撬时，人员站在侧面，不站在重物滚落方向。
- 绳索有断股、霉烂、焦痕时不用于承重。

## 常见问题

Q: 绳子在滑轮上打滑？A: 轮槽内贴一层皮革增加摩擦力。

Q: 木齿轮不耐磨？A: 齿面包铁皮延长寿命。

## 穿越小说梗

杠杆、滑轮、齿轮组合就是工业革命的基石。穿越者知道精确计算传动比——30人拉一根绳和5人操作滑轮组的区别就是知识的差距。
