# 5.10 木炭的选材与烧制

> 木炭是火药中最容易被忽视的变量——不同的炭，决定火药是"快"还是"慢"。

## 原理

黑火药中的木炭不是燃料，而是**还原剂**。木炭的多孔结构提供了巨大的反应表面积。

### 不同木材的效果

| 木材 | 炭特性 | 火药效果 | 评价 |
|------|-------|---------|------|
| 柳木 | 轻、多孔、灰分低 | 燃速最快 | ⭐⭐⭐⭐⭐ 首选 |
| 桐木 | 极轻、多孔 | 中速 | ⭐⭐⭐⭐ 好 |
| 杉木 | 轻、纹理直 | 中速 | ⭐⭐⭐ 可用 |
| 松木 | 含树脂、灰分高 | 较慢、烟大 | ⭐⭐ 差 |
| 竹 | 质硬、孔隙适中 | 好 | ⭐⭐⭐⭐ 好 |
| 麻秆 | 极轻多孔 | 快 | ⭐⭐⭐⭐ 好 |
| 硬木（橡/榉） | 密实 | 慢 | ⭐ 太慢 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 木材（首选柳木） | 原料 |
| 铁桶/陶罐 | 烧制容器 |
| 泥土/沙 | 密封 |
| 火源 | 加热 |
| 密封容器 | 储存（防潮） |

## 操作步骤

### 一、选材（关键）

1. 选**嫩枝**（手指粗细，1-2年生），不要老枝
2. 剥去树皮（树皮灰分高，降低火药质量）
3. 晒干或烘干至含水量 <10%

### 二、传统土法烧炭

1. **装桶**：将干燥木条填入铁桶/陶罐，压实
2. **密封**：桶盖留一个小孔，其余接口用泥密封
3. **加热**：桶外堆柴烧火
4. **排烟**：小孔中冒出白烟（水分）→ 黄烟（木质素分解）→ 蓝烟（接近完成）
5. **封孔**：蓝烟出现时，用泥堵住小孔
6. **冷却**：桶自然冷却（不要提前开封！——遇空气会燃烧）
7. **取出**：完全冷却后开桶，得黑色木炭

### 三、火药用炭的特殊处理

1. 烧好的木炭趁热密封保存（防潮）
2. 使用时研磨——越细越好
3. **质量判断**：
   - 轻（浮于水）→ 好
   - 一捏就碎 → 好
   - 敲击声音清脆 → 好
   - 敲击声闷 → 碳化不足

## 常见问题

- **Q**: 烤肉用的木炭能用吗？
- **A**: 不能用。烧烤炭块大、密度高、含杂质多，不适合做火药。

- **Q**: 烧炭的温度和时间？
- **A**: 300-500°C 是最佳温度。时间取决于木材量和容器大小，一般 2-6 小时。判断标准：蓝烟出现即可封孔。

- **Q**: 为什么柳木炭最好？
- **A**: 柳木结构疏松、多孔、灰分含量极低（<1%）。灰分会吸收硝石中的硝酸根，降低火药威力。

## 安全提醒

- 烧炭时注意通风（木炭燃烧产生一氧化碳）
- 刚烧好的炭遇空气可能自燃，必须完全冷却后再开封
- 木炭粉极易飞扬，吸入有害
- 木炭易燃，远离火源

## 穿越小说梗

"主角烧了一堆木炭做火药"——但大多数穿越小说忽略了一个关键细节：**柳木炭 > 其他木炭**。黑火药的最佳搭档是柳木炭（西方也是用柳木炭做最好火药），因为灰分最低、比表面积最大。穿越者知道这个细节，火药性能就比别人好。另外，古人烧炭全凭经验，穿越者可以用**颜色判断法**（蓝烟出现就是碳化完成）来确定烧制终点。
