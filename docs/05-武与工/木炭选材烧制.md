# 5.10 木炭的选材与烧制

> 木炭是冶炼、取暖、过滤和保存火种的重要材料。涉及火药用途时，只保留历史常识和安全边界。

## 原理

木炭是在缺氧条件下加热木材得到的多孔碳材料。它比木柴更耐烧、烟少、热值集中，也能用于冶炼、吸附异味和简易过滤。粉末木炭遇明火和粉尘环境有燃烧风险，不能和氧化剂、硫磺等危险材料混放。

### 不同木材的效果

| 木材 | 炭特性 | 适合用途 | 评价 |
|------|-------|----------|------|
| 柳木 | 轻、多孔、灰分低 | 火种、过滤、细炭 | 易碎，需防潮 |
| 桐木 | 极轻、多孔 | 火种、轻质炭 | 易燃，储存要分开 |
| 杉木 | 轻、纹理直 | 一般燃料 | 可用 |
| 松木 | 含树脂、烟较大 | 取暖、野外燃料 | 烟味重 |
| 竹 | 质硬、孔隙适中 | 炊事、过滤、除味 | 易得 |
| 麻秆 | 极轻多孔 | 应急火种 | 易飞灰 |
| 硬木（橡/榉） | 密实、耐烧 | 冶炼、长时间炊事 | 火力持久 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 木材 | 原料 |
| 铁桶/陶罐 | 烧制容器 |
| 泥土/沙 | 密封 |
| 火源 | 加热 |
| 密封容器 | 储存（防潮） |

## 操作步骤

### 一、选材（关键）

1. 选**嫩枝**（手指粗细，1-2年生），不要老枝
2. 剥去树皮（树皮灰分高，燃烧烟尘和杂质更多）
3. 晒干或烘干至含水量 <10%

### 二、传统土法烧炭

1. **装桶**：将干燥木条填入铁桶/陶罐，压实
2. **密封**：桶盖留一个小孔，其余接口用泥密封
3. **加热**：桶外堆柴烧火
4. **排烟**：小孔中冒出白烟（水分）→ 黄烟（木质素分解）→ 蓝烟（接近完成）
5. **封孔**：蓝烟出现时，用泥堵住小孔
6. **冷却**：桶自然冷却（不要提前开封！——遇空气会燃烧）
7. **取出**：完全冷却后开桶，得黑色木炭

### 三、质量判断

| 指标 | 合格 | 失败征兆 |
|------|------|----------|
| 外观 | 通体黑色，断面无褐色木心 | 中间发褐，碳化不足 |
| 手感 | 轻、干、敲击清脆 | 潮黏、发霉、闷声 |
| 燃烧 | 能稳定生火，烟不过大 | 冒白烟、焦油味重 |
| 存放 | 干燥密封，远离火源和氧化剂 | 粉尘飞扬、与危险粉末混放 |

## 成功标准

- 木材入罐前充分干燥，折断声脆，不冒湿气。
- 排烟从白烟转黄烟，再转蓝烟后封孔；全程不让空气大量进入。
- 冷却到手触容器不烫后再开封，木炭黑而轻、一捏易碎。
- 普通燃料炭能稳定生火；粉末炭远离火源、氧化剂、硫磺和儿童。

## 失败征兆

- 开封后木炭发红、冒烟：未冷却或进氧，可能自燃。
- 木炭中间褐色、敲击闷：碳化不足，含挥发物和水分高。
- 木炭变灰白粉末多：过烧，灰分高。
- 烧炭者头痛、恶心、犯困：一氧化碳风险，立即离开通风。

## 常见问题

- **Q**: 烤肉用的木炭能用吗？
- **A**: 可作为普通炊事燃料。不要把未知烧烤炭粉末用于药物、饮水或火工用途。

- **Q**: 烧炭的温度和时间？
- **A**: 300-500摄氏度是最佳温度。时间取决于木材量和容器大小，一般 2-6 小时。判断标准：蓝烟出现即可封孔。

- **Q**: 为什么柳木炭最好？
- **A**: 柳木炭轻、多孔、易点，适合火种和细炭。不同用途要选不同木炭，不存在“一种炭万能”。

## 安全提醒

- 烧炭时注意通风（木炭燃烧产生一氧化碳）
- 刚烧好的炭遇空气可能自燃，必须完全冷却后再开封
- 木炭粉极易飞扬，吸入有害
- 木炭易燃，远离火源

## 穿越小说梗

"主角烧炭"比看起来更有技术含量。可信细节不是“随便一堆木头变黑就行”，而是控制缺氧、观察排烟、完全冷却再开封，并把炭按冶炼、炊事、过滤、火种分开存放。粉末炭一律按易燃粉尘管理，不能和危险粉末混在一起。
