# 5.27 黑火药配方

> 一硝二磺三木炭——这七个字，改变了战争史。

## 原理

黑火药燃烧的化学反应（简化）：

```
2KNO₃ + S + 3C → K₂S + N₂↑ + 3CO₂↑ + 大量热
```

关键是**瞬间产生大量气体**（N₂ + CO₂），体积在极短时间内膨胀数千倍，产生爆炸推力。

### 配方比例（按重量）

| 用途 | 硝石（KNO₃） | 硫磺（S） | 木炭（C） | 说明 |
|------|-------------|----------|----------|------|
| 经典配比 | 75% | 10% | 15% | "一硝二磺三木炭"≈1:0.13:0.2 |
| 火箭/烟花 | 60-70% | 10-15% | 15-20% | 燃烧稍慢 |
| 爆破用 | 70-75% | 10-12% | 13-15% | 爆炸力强 |
| 引信 | 50-60% | 20-25% | 15-20% | 燃烧稳定 |
| 最差可用 | 60% | 10% | 30% | 效率低、烟大 |

## 所需材料

| 材料 | 要求 | 获取方式 |
|------|------|---------|
| 硝石（硝酸钾） | 纯度 >95% | 见"硝石识别提取" |
| 硫磺 | 黄色结晶粉 | 见"硫磺识别获取" |
| 木炭 | 轻质、多孔 | 见"木炭选材烧制" |
| 研钵（石/木） | 不产生火花 | **严禁金属** |
| 细筛（100目+） | 混合均匀 | 纱布/丝绢 |

## 操作步骤

1. **单独粉碎**：将硝石、硫磺、木炭分别研成细粉（每种材料**单独研磨**！）
2. **过筛**：分别过细筛，确保细度一致
3. **混合**：按 75:10:15 的比例称量
4. **混合方式**：
   - 方法一（安全）：将所有粉倒入木桶，密闭后滚动混合 15-30 分钟
   - 方法二（高效）：用木/竹勺轻轻翻拌（**不要研磨！不要冲击！**）
5. **粒度**：粉越细，燃烧越快，爆炸力越强
6. **测试**：取一小撮点燃——应该快速燃烧，发出"嘶"的一声，不留残渣
7. **密封保存**：黑火药易吸潮，密封于干燥容器中

## 质量测试

| 测试 | 方法 | 合格标准 |
|------|------|---------|
| 燃烧速度 | 取 1g 粉末堆成条状点燃 | 瞬间燃尽（<1秒） |

## 常见问题

- **Q**: 为什么我的火药燃烧很慢？
- **A**: ①硝石纯度不够②木炭太粗③材料受潮④比例不对。先检查硝石纯度。

- **Q**: 为什么火药不放火？
- **A**: ①配方中硝石不足50%不会爆炸②水分超标③木炭不是活性炭而是普通炭灰。

- **Q**: 一硝二磺三木炭这个比例对吗？
- **A**: 这是**体积比**（目测比例），换算成重量比大约是 75:10:15。按体积算大概是 1份硝石:0.1份硫磺:0.15份木炭。

## 安全提醒

- **绝对不能**用金属工具研磨或混合火药
- **绝对不能**在有明火、火星的环境操作
- 一次只混合少量（<50g）
- 操作区备沙土/水桶灭火
- 存储远离住所和火源

## 穿越小说梗

"一硝二磺三木炭"在现代穿越小说里被说烂了。但真正关键的不是配方比例（古人早就知道），而是**原料纯度**和**混合均匀度**。古代黑火药质量差主要是因为硝石提纯不够。穿越者如果会重结晶法提纯硝石，做出来的火药就会比同时代的好一个量级。注意：火药在穿越早期用处有限（没有火枪），但做信号弹和炸药用得着。
