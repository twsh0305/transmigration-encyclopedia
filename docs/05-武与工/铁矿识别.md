# 5.25 铁矿识别

> 没有铁，你就永远停留在石器时代。找到铁矿，才是工业化的第一步。

## 原理

铁在自然界中不以纯铁存在，而是以**氧化铁**为主的矿物形式。

### 常见铁矿石

| 矿石 | 分子式 | 含铁量 | 颜色 | 穿越后首选 |
|------|--------|--------|------|-----------|
| 赤铁矿 | Fe₂O₃ | 60-70% | 红褐色 | ⭐⭐⭐⭐⭐ 首选 |
| 磁铁矿 | Fe₃O₄ | 65-72% | 黑色 | ⭐⭐⭐⭐⭐ 磁性易识别 |
| 褐铁矿 | Fe₂O₃·nH₂O | 35-55% | 黄褐 | ⭐⭐⭐ 可用 |
| 菱铁矿 | FeCO₃ | 30-40% | 灰白/黄 | ⭐⭐ |
| 黄铁矿 | FeS₂ | 45% | 金黄 | ⭐ 不宜炼铁（含硫高） |

## 所需材料

| 材料 | 用途 |
|------|------|
| 铁矿石样品 | 待识别 |
| 磁铁（天然磁石） | 测试磁性 |
| 硬石/锤子 | 敲碎观察断口 |
| 陶片（白色） | 划痕测试 |
| 火源 | 加热测试 |

## 操作步骤

### 一、野外找矿标志

**地表线索**：
1. **红色/褐色土壤**——赤铁矿风化后的土壤颜色
2. **黑色重砂**——河床中的黑色磁铁矿砂
3. **露头矿脉**——山坡或河岸露出的矿层
4. **铁锈色泉水**——含铁高的泉水
5. **地名线索**——"铁山""铁坑""冶铁"之类的老地名

### 二、矿物的快速识别

| 测试 | 赤铁矿 | 磁铁矿 | 褐铁矿 | 黄铁矿 |
|------|--------|--------|--------|--------|
| 颜色 | 红褐/钢灰 | 黑色 | 黄褐 | 金黄 |
| 条痕（划陶片） | 樱红色 | 黑色 | 黄褐色 | 绿黑色 |
| 磁性 | 无 | 强 | 无 | 无 |
| 比重 | 较重 | 很重 | 中等 | 重 |
| 断口 | 贝壳状 | 不规则 | 土状 | 不规则 |

### 三、还原测试（最确定）

取少量矿石在木炭火中加热——如果矿石表面出现**金属光泽的小珠**→确实含铁。

### 四、铁矿质量判断

| 指标 | 好铁矿 | 差铁矿 |
|------|--------|--------|
| 含铁量 | >50% | <30% |
| 硫含量 | 低（无臭） | 高（有硫味） |
| 磷含量 | 低 | 高（铁变脆） |
| 脉石 | 少 | 多（需要更多熔剂） |
| 块度 | 大块 | 粉末状 |

## 常见问题

- **Q**: 黄铁矿和黄金怎么区分？
- **A**: 黄铁矿（愚人金）比黄金轻、脆、条痕为绿黑色（黄金条痕金黄色）。

- **Q**: 河床里的黑色沙是铁矿吗？
- **A**: 很可能是**磁铁矿砂**。可以用磁铁吸出——磁铁矿砂是古代炼铁的重要原料（"铁砂"）。

- **Q**: 什么样的铁矿最好炼？
- **A**: 高品位赤铁矿（Fe₂O₃ >60%）最容易还原。磁铁矿含铁最高但更难还原。

## 安全提醒

- 采矿时注意落石和塌方
- 黄铁矿在空气中自然氧化会放热，大量堆积可能自燃
- 含硫高的铁矿石炼出的铁"热脆"（加热后易裂）

## 穿越小说梗

"主角发现了铁矿，开始打造兵器"——这是种田流的经典走向。但真正的问题不是找到铁矿，而是①品位够不够②运输成本③有没有树木烧炭④有没有水源（水碓）。如果找到的是高硫铁矿（黄铁矿），炼出来的铁一烧就碎，做不了好兵器。穿越者最关键的知识：**先用磁铁测试**，快速识别高品位磁铁矿。
