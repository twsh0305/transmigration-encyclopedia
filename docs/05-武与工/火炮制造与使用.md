# 5.28 火炮制造与使用

> 火药改变了战争，而火炮是火药时代真正的王。

## 原理

火炮通过火药在封闭炮膛内瞬间燃烧产生大量气体，推动弹丸射出。核心参数：

- **膛压**：火药燃烧产生的高压气体（约 50-200 MPa）
- **初速**：弹丸离开炮口时的速度（300-800 m/s）
- **射程**：取决于装药量、仰角、弹丸重量

### 火炮基本结构

```
      炮口 ────── 炮管 ────── 火药室 ── 引信孔 ── 尾球
      │                                              │
      └────────── 炮架 (轮式/固定式) ─────────────────┘
```

| 部件 | 功能 | 材料要求 |
|------|------|---------|
| 炮管 | 承受膛压，赋予弹丸方向 | 铸铁/青铜，需无砂眼 |
| 火药室 | 装填发射药，位于炮管后部 | 管壁最厚处 |
| 引信孔 | 插入引信点燃火药 | 小孔，需防漏气 |
| 炮架 | 支撑炮管，调节角度 | 硬木+铁件 |
| 驻退装置 | 缓冲后坐力（现代火炮） | 弹簧/液压 |

## 火炮类型

### 按制造工艺

| 类型 | 制作方法 | 优缺点 | 代表 |
|------|---------|-------|------|
| **铸炮** | 泥模/铁模铸造 | 成本低，可量产，但管壁容易有气孔 | 红夷大炮 |
| **锻炮** | 铁条缠绕锻焊 | 强度高，不易炸膛，但制作周期长 | 欧洲早期锻铁炮 |
| **钻膛炮** | 铸造实心胚后钻孔 | 内膛光洁，精度高，但需要大型钻床 | 阿姆斯特朗炮 |

### 按用途

| 类型 | 口径 | 射程 | 用途 |
|------|------|------|------|
| 臼炮/迫击炮 | 大（200-300mm） | 近（100-500m） | 高抛弹道，轰击城墙内侧 |
| 加农炮 | 中（75-150mm） | 远（500-2000m） | 直射，攻击城墙/阵列 |
| 榴弹炮 | 中（100-200mm） | 中（300-1500m） | 曲射，杀伤人员 |
| 回旋炮/佛朗机炮 | 小（30-60mm） | 近（100-300m） | 子铳快速装填，船上/城墙上 |

## 所需材料与工具

| 材料 | 用途 | 获取方式 |
|------|------|---------|
| 青铜/铸铁 | 炮管主体 | 青铜熔点低（~950°C），易铸但成本高；铸铁强度大但需高温（~1200°C） |
| 硬木（橡木/榆木） | 炮架 | 木工加工 |
| 铁件 | 炮架连接件、轮轴 | 锻造 |
| 黑火药 | 发射药 | 见"黑火药配方" |
| 引信 | 点燃发射药 | 纸捻+慢燃火药 |
| 弹丸（铁/石） | 杀伤/破城 | 铸铁圆形弹丸效果最佳 |

## 操作步骤

### 一、铸炮工艺（泥模法）

1. **制模**：用粘土制作炮管外模，阴干，内腔用泥芯定型
2. **烘干**：模具彻底烘干（否则遇铁水会炸裂）
3. **熔炼**：铜（90%铜+10%锡）或铁在坩埚中熔化
4. **浇铸**：铁水沿浇口注入模具，一次完成
5. **冷却**：自然冷却 24-48 小时
6. **拆模**：打碎外模，取出炮管粗胚
7. **修整**：清理毛刺，打磨炮口
8. **试射**：先用减量火药试射，逐步增加

### 二、装填与射击

1. **清洁炮膛**：用炮刷清理上次发射残留
2. **装发射药**：定量黑火药装入药室（用铜勺/绸布袋）
3. **捣实**：用木槌轻捣火药，使其密实
4. **装入弹丸**：放入铁球/石球，用木槌推至紧贴火药
5. **装引信**：将引信插入引信孔
6. **瞄准**：调整炮口仰角和方向
7. **点火**：用火绳/烧铁棍点燃引信
8. **后坐处理**：炮车后退后推回原位

### 三、弹丸类型

| 弹丸类型 | 用途 | 制作 |
|---------|------|------|
| 实心铁球 | 砸穿城墙、杀伤密集阵型 | 铸铁球形，直径略小于口径 |
| 霰弹 | 近距离（<200m）杀伤人员 | 铁砂/碎石装布袋 |
| 链弹 | 破坏敌船桅杆/帆索 | 两枚半球+铁链连接 |
| 燃烧弹 | 焚烧建筑/船只 | 铁壳内装火药+沥青+油脂 |
| 开花弹 | 爆炸杀伤 | 铁壳内装火药+引信定时 |

## 安全操作规范

### 防炸膛

- **严格检查炮管**：每次装填前检查有无裂纹、砂眼
- **控制装药量**：不要为了追求射程而超量装药
- **适当壁厚**：炮管最薄处壁厚不小于口径的 0.5 倍
- **试射原则**：新炮先用 1/3 装药量试射，逐步增加到设计装药量

### 操作安全

- 点火者站在炮尾侧面（不是正后方！）
- 每次发射后炮刷蘸水清洁炮膛，清除余烬
- 装填时**绝对禁止**有明火靠近火药
- 引信长度要足够，给点火者留够撤离时间
- 炮管过热时停止射击，用水冷却或自然降温

### 日常维护

- 炮膛内部每次射击后清洁并涂抹油脂防锈
- 炮架定期检查，松动部件及时紧固
- 火药密封防潮储存
- 铜炮比铁炮耐锈，适合潮湿环境

## 常见问题

- **Q**: 铸铁炮容易炸膛怎么办？
- **A**: ①提高铸造质量（模具彻底烘干、铁水温度足够）②采用"铁心铜体"工艺（内膛用铸铁，外层包铜）③减少装药量，用射程换安全

- **Q**: 没有青铜也没有铸铁怎么造炮？
- **A**: 锻铁炮（熟铁条缠绕锻焊）是技术最简单的方式。需要：铁匠炉 + 锻锤 + 耐心。将熟铁条烧红后绕芯棒螺旋缠绕，锻焊成一体。

- **Q**: 古代城墙能抗住火炮吗？
- **A**: 传统夯土墙/砖墙无法抵抗持续炮击。明代城墙在火炮面前如纸糊。但花岗岩条石砌筑的城墙可以承受更多发。

- **Q**: 射不准怎么办？
- **A**: ①每次装药量一致 ②弹丸直径与口径间隙越小越准（间隙大会漏气，威力下降但准度也下降）③固定仰角后不要轻易调整 ④批次生产中记录每一门炮的弹道特性

## 穿越小说梗

"穿越者造出火炮横扫天下"——这是最经典的穿越桥段之一，但现实远没有小说那么容易。几个关键难点：

1. **冶金是瓶颈**：造出能承受膛压而不炸膛的炮管，需要成熟的铸造工艺。穿越初期大概率只能做粗制铁炮，威力不如想象的大。

2. **火药是消耗品**：一场攻城战可能消耗数百斤火药。穿越者需要建立硝石、硫磺、木炭的稳定供应链。

3. **人才比技术重要**：一个穿越者无法同时当铸炮匠、药剂师、炮手。真正的优势是"知道该做什么"然后指挥专业工匠去做。

4. **时机问题**：大规模造炮会引起周边势力的警觉。穿越者需要在技术优势和保密之间找到平衡点——先造几门精品小炮保命，站稳脚跟后再量产。
