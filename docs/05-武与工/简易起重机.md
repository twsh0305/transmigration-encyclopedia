# 5.23 简易起重机

> 一个人搬不动石头，但一台起重机可以——这就是工程的力量。

## 原理

通过杠杆加滑轮加绞盘加棘轮的组合，将人力小力放大为提升重物的大力。三角架（3~5m，100~500kg）、桅杆吊（5~15m，200~2000kg）、井式起重机（5~30m，50~500kg）。

## 所需材料

| 材料 | 说明 |
|------|------|
| 硬木（松木/杉木） | 柱、梁 |
| 麻绳/棕绳 | 起重绳 |
| 滑轮（木制或铁制） | 改变方向/省力 |
| 绞盘（木制滚筒） | 收绳 |
| 棘轮（木或铁） | 防逆转 |

## 操作步骤

1. 三角架：3根5~6m圆木一端捆扎，撑开成等边三角形，顶端挂滑轮
2. 桅杆吊：竖立8~12m主柱，四角拉索固定，前端装滑轮组
3. 绞盘：木滚筒（直径20~40cm）加手柄（力臂30~60cm）
4. 棘轮：滚筒端齿轮加止回爪防重物倒转

## 常见问题

Q: 没有铁轴怎么做绞盘？A: 硬木轴（枣木/柘木）加动物油脂润滑。

Q: 棘轮做不好重物一松手就落？A: 棘轮齿必须是斜齿，止回爪角度与齿面垂直。

## 安全提醒

起重物下方绝对禁止站人。每次使用前检查绳索磨损和地基下沉。

## 穿越小说梗

棘轮才是起重机的灵魂——没有棘轮人就得一直拉着绳子。棘轮让人可以停下来休息，这个改进在古代就是巨大的工程革命。
