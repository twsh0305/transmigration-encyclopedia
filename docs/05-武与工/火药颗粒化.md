# 5.15 火药颗粒化

> 粉末火药燃烧慢、冒烟大、受潮快——颗粒化之后，这些问题全部解决。

## 原理

火药粉末 → 压制成颗粒 → 颗粒之间的孔隙允许火焰快速传播，燃速提高数倍。

### 粉末 vs 颗粒对比

| 特性 | 粉末状 | 颗粒状 |
|------|--------|--------|
| 燃速 | 较慢 | 快 2-3 倍 |
| 密度 | 松堆 | 密实 |
| 防潮性 | 差（易吸潮） | 好 |
| 运输安全 | 粉尘易爆 | 安全 |
| 储存稳定性 | 差 | 好 |
| 威力 | 低 | 高 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 黑火药粉（调配好的） | 原料 |
| 水/酒精 | 润湿 |
| 明胶/阿拉伯胶（可选） | 粘合剂 |
| 粗布/竹筛 | 过筛造粒 |
| 平底容器 | 干燥 |
| 研钵（木/石） | 破碎大块 |

## 操作步骤

### 一、润湿混合

1. 火药粉放入木盆（**严禁金属容器！**）
2. 少量多次喷水/酒精（每100g火药加10-15ml水）
3. 用手轻轻揉搓（戴手套）——不要研磨！不要用力！
4. 直到粉末呈**潮湿的沙状**（捏成团、放下散）

### 二、造粒

**手工造粒法**：
1. 将湿润的火药粉在粗布上揉搓
2. 通过竹筛或粗布筛下
3. 筛下的颗粒大小均匀（通常 1-2mm）

**压块法（更好的方法）**：
1. 将湿润火药粉放入木模中压紧（用木槌轻敲压实）
2. 脱模后得扁平的"火药饼"
3. 晾至半干后，用木槌敲碎成小颗粒
4. 过筛分选（大颗粒→再敲碎）

### 三、干燥

1. 将颗粒平铺在干净的布/纸上
2. 阴凉通风处晾干（**严禁暴晒！严禁明火！**）
3. 约 24-48 小时完全干燥
4. 测试：取几粒在手中捏——不粘手、不沾指纹 → 干燥完成

### 四、抛光（可选）

1. 干燥后的火药颗粒放入布袋中
2. 布袋中加少量石墨粉（或细腻木炭粉）
3. 抖动布袋 5-10 分钟
4. 颗粒表面形成光滑层——进一步防潮

## 粒度选择

| 用途 | 最佳粒度 | 说明 |
|------|---------|------|
| 火枪/火炮 | 1-3mm | 大颗粒燃速适中 |
| 爆破/地雷 | 3-5mm | 更大颗粒 |
| 引信 | <1mm | 小颗粒燃速稳定 |
| 烟花 | 1-2mm | 适中 |

## 常见问题

- **Q**: 颗粒火药和粉末火药威力差多少？
- **A**: 颗粒化后燃速提高 2-3 倍，威力提高 50-100%。**火药颗粒化是火药史上一次重要飞跃**。

- **Q**: 没有明胶怎么做粘合剂？
- **A**: ①糯米糊（最天然）②土豆淀粉糊③只用纯水也可以，但颗粒不够牢固。

## 安全提醒

- **刚压好的湿火药饼仍然危险！** 不要在阳光下暴晒
- 干燥过程中远离任何火源
- 在操作台上铺皮革或布（防静电）
- 一次只造少量（<500g）
- 工具只能用木/竹/铜，不能用铁（防止火花）

## 穿越小说梗

"主角造出了威力更大的颗粒火药"——这确实是一个技术飞跃。历史上，欧洲在 15 世纪引入颗粒化技术后，火炮威力提升了数倍。穿越者知道"压饼→破碎→过筛"这个流程，做出的火药就是"升级版"。注意：颗粒化后的火药燃速更快，枪管/炮管需要更结实，否则会炸膛。
