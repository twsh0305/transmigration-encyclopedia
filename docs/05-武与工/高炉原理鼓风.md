# 5.26 高炉原理与鼓风设备

> 从块炼铁到生铁水——中间差的不是温度，而是鼓风。

## 原理

高炉炼铁的核心：**逆流热交换 + 化学还原**

1. 矿石 + 木炭 + 石灰石从顶部加入
2. 鼓风从底部吹入
3. 木炭在底部燃烧 → 产生 CO 和高温
4. 热气体上升，预热下降的矿石
5. CO 还原矿石中的铁氧化物
6. 铁熔化后滴落到底部聚集

### 高炉 vs 块炼炉

| 特性 | 块炼炉 | 高炉 |
|------|--------|------|
| 温度 | 1100-1200摄氏度 | 1300-1500摄氏度 |
| 产物 | 海绵铁（固态） | 生铁水（液态） |
| 产量 | 几公斤/次 | 几百公斤/次 |
| 连续性 | 批次 | 连续 |
| 建炉难度 | 低 | 高 |
| 燃料需求 | 中 | 极大 |

## 所需材料

| 材料 | 用途 |
|------|------|
| 耐火砖/耐火黏土 | 炉体 |
| 铁矿石 | 原料 |
| 木炭 | 燃料 |
| 石灰石 | 熔剂 |
| 强力鼓风设备 | 鼓风（关键） |
| 陶管/铁管 | 风口 |

## 操作步骤

### 一、鼓风设备

**皮风箱（最简单）**：
1. 用整张羊皮/牛皮缝成袋状
2. 一端装竹管（出风口）
3. 另一端装木把手
4. 拉→袋膨胀吸气；压→鼓风

**双作用木风箱（更好的选择）**：
1. 木制方箱体
2. 内装活塞
3. 进风口和出风口用皮瓣控制方向
4. 推拉都出风——效率是单作用风箱的 2 倍

**水排（水力鼓风，终极方案）**：
1. 水车带动曲柄连杆
2. 连杆拉动风箱
3. 连续自动鼓风——人力解放
4. 这是东汉杜诗发明的技术

### 二、高炉建造

**小型实验炉**（高 1-1.5米）：
1. 底部宽 40厘米，向上逐渐收窄至 25厘米
2. 炉壁用耐火黏土砌筑（厚 15-20厘米）
3. 底部留出铁口（平时用泥封住）
4. 距底部 20厘米开风口（接风箱）
5. 自然干燥一周以上

### 三、操作流程

1. 底部铺木炭，点燃
2. 先加木炭，再加矿石+石灰石
3. 每次加料比例：木炭:矿石:石灰石 ≈ 5:2:0.3
4. 持续鼓风
5. 每 4-6 小时出铁一次（捅开出铁口）
6. 出铁后用泥封住出铁口

## 成功标准

| 项目 | 合格表现 |
|------|----------|
| 炉体 | 高 1-1.5米小炉先试，炉壁厚 15-20厘米，干燥一周以上 |
| 配料 | 木炭:矿石:石灰石约 5:2:0.3，原料粒度基本均匀 |
| 鼓风 | 风量连续稳定，火焰不倒灌，炉底能维持 1300摄氏度以上 |
| 出铁 | 出铁口能流出液态生铁或高温铁渣，出铁后可重新封口 |
| 运行 | 记录每 4-6 小时加料、出渣、出铁和风量变化 |

## 失败征兆

- 温度只到块炼水平，出海绵铁不出铁水，说明鼓风或炉高不足。
- 炉底凝死，出铁口打不开，可能需要拆炉。
- 炉壁开裂、风口烧穿，应停炉撤人。
- 出铁区有水、湿泥或湿工具，可能发生蒸汽爆溅。
- 木炭供应不足还强行开炉，会半途停炉报废。

## 常见问题

- **Q**: 鼓风不足会怎样？
- **A**: 温度不够→铁不熔化→得到海绵铁而不是铁水。**鼓风是高炉的灵魂**。

- **Q**: 高炉停炉了怎么重新启动？
- **A**: 凝固的铁水会堵住炉底。严重时需要拆炉重建（所以高炉一旦启动就连续数周不停炉）。

- **Q**: 高炉炼出来的生铁有什么用？
- **A**: 生铁很脆，不能直接打刀剑。只能铸造成型（锅、犁）。要变成钢需要炒钢或灌钢处理。

## 安全提醒

- 高炉温度 1300摄氏度以上，铁水喷溅极度危险
- 出铁时蒸汽爆炸风险，人员保持距离
- 高炉 CO 浓度高，必须露天操作
- 燃料消耗巨大——一座高炉一天烧掉几吨木炭

## 穿越小说梗

"主角建了高炉，炼出了铁水"——高炉是古代钢铁工业的皇冠。建高炉意味着：①你已有足够的铁矿储备②你已有规模化木炭供应③你有足够的人力。穿越小说的误区：把高炉描述得太容易建了。事实上，建高炉需要耐火材料（耐火黏土/耐火砖）+ 连续数天的燃料 + 强力鼓风。穿越者应该从块炼炉开始，等积累了经验再建高炉。
